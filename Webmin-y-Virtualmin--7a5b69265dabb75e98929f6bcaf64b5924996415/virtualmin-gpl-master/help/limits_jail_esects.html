Extra sections for Jailkit to enable
By default, only essential files are copied from Jailkit config sections into new or updated jails. To include additional files, list the desired sections from /etc/jailkit/jk_init.ini here.

The sections in /etc/jailkit/jk_init.ini file are named as [git] or [apacheutils]. For example, to include git and apacheutils sections in this domain chroot environment, put git apacheutils (separated by space) in this field.
