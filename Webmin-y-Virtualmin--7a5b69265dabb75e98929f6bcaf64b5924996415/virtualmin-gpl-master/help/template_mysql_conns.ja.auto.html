ドメイン所有者の最大接続数
このフィールドは、仮想サーバー所有者のアカウントが MySQL に同時に接続できる数を制御します。これには、Wordpress や Drupal など、Virtualmin によってインストールされた Web アプリケーションからの接続が含まれます。デフォルトでは無制限の接続数が許可されますが、各接続は MySQL サーバー プロセス内でメモリを消費するため、ビジー状態のシステムでは合計接続数を制限する必要がある場合があります。

ここで設定された制限は、新しい仮想サーバーが作成される場合、または既存のサーバーに対して MySQL ログインが有効になっている場合にのみ適用されます。
