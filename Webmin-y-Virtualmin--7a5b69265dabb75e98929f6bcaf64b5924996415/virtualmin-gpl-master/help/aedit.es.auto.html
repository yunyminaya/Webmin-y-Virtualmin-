Editar alias manualmente
Esta página le permite crear, editar o eliminar múltiples alias de correo electrónico a la vez editándolos en formato de texto. No se pueden editar todos los alias: aquellos con destinos complejos o autorespondedores no son adecuados para el formato de texto utilizado en esta página, por lo que no se mostrarán.

En el cuadro de texto, debe enumerar un alias por línea, y cada línea debe tener el siguiente formato:

name:[comment]:destination1[:destination2:...]

Los significados de los campos son:

nombre
El nombre del buzón del alias, con como ventas . Para crear un alias que coincida con todo el correo al dominio, ingrese * aquí.
comentario
Este campo opcional se puede usar para ingresar un comentario descriptivo para el alias, como Ventas corporativas .
destino
Se pueden ingresar uno o más destinos, generalmente en forma de direcciones de correo electrónico para reenviar el correo, como joe@yourdomain.com o simplemente joe . También se admiten los siguientes destinos especiales:
  • bounce
    Crea un alias que devuelve el correo electrónico al remitente.
  • local username
    Entrega correo electrónico directamente al buzón del usuario especificado.
