Cliente de servicios de Cloudmin
Esta página le permite configurar Virtualmin para conectarse a un servidor de servicios de Cloudmin, de modo que las bases de datos MySQL, las zonas DNS y otros servicios puedan descargarse en sistemas remotos. Idealmente, debería usarse antes de que se hayan creado servidores virtuales, ya que la combinación de bases de datos y zonas remotas y servidas localmente es ineficiente.

Los campos que deben completarse son:

Maestro de servicios de Cloudmin
Este nombre de host de una máquina Cloudmin con el complemento de servicios de Cloudmin instalado. Esto puede ser administrado por usted, su empresa de alojamiento o proveedor de VPS.

Número de puerto
Esto casi siempre debe establecerse en 10000, que es el puerto predeterminado de Webmin / Cloudmin.

Nombre de usuario para el servidor
Este campo debe contener el nombre de usuario de una cuenta de propietario del sistema en el maestro de servicios de Cloudmin que tenga permiso para crear bases de datos, zonas DNS u otros servicios.
Características para alojar
Esta sección le permite seleccionar qué características de Virtualmin se descargarán a los sistemas remotos. Solo se pueden seleccionar aquellos que su cuenta en el sistema maestro puede aprovisionar.
