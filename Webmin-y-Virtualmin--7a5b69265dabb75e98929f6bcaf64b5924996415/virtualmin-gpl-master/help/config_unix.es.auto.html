¿Creación de usuario de administración habilitada?
Esta opción determina si la creación de un usuario de Unix para un nuevo servidor virtual de nivel superior siempre se realiza (y la opción está oculta), o si es opcional al crear un servidor. Por lo general, es una buena idea establecer esta opción en Siempre , ya que se necesita un usuario de Unix a menos que se cree un dominio solo de DNS o solo de proxy web.
