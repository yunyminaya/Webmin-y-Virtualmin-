Standaard virtueel server IP adres
Virtuele servers waarbij de SSL of FTP virtuele host opties niet zijn aangezet zullen allemaal achter een enkel IP adres zitten. Deze optie specificeert dit adres. Indien u dit op standaard laat staan zal Virtualmin het primaire adres gebruiken van de interface die gespecificeerd is door de *Netwerk interface voor virtuele adressen* hierboven.
