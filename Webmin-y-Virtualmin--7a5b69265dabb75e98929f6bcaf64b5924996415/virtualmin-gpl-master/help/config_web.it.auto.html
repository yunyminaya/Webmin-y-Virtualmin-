Installazione e configurazione di Apache abilitate?
Questa opzione abilita o disabilita la creazione di voci Apache VirtualNameHost e imposta tutte le altre funzionalità dell'host virtuale nel file di configurazione di Apache. Se si utilizza un server Web alternativo, è possibile disabilitare questa opzione. Potrebbe essere possibile utilizzare l'interfaccia del modulo plug-in per creare supporto per il server preferito (anche se al momento non sono supportati altri server Web) oppure è possibile scegliere di chiamare uno script esterno insieme alle variabili tratte dalle informazioni fornite a Virtualmin all'indirizzo tempo di creazione del dominio per configurare il tuo server web preferito.
