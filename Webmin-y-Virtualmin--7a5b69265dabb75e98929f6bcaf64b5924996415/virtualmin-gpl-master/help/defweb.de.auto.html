Standardwebsite für IP-Adresse?
Wenn dieses Feld auf " Ja" gesetzt ist , wird die Website dieses virtuellen Servers von Apache bereitgestellt, wenn eine Anforderung empfangen wird, die keinem anderen virtuellen Server auf dem System entspricht. Dies ist normalerweise der Fall, wenn ein Benutzer eine URL mit einer darin enthaltenen IP-Adresse oder einen Hostnamen verwendet, der in Ihr System aufgelöst wird, aber keiner Virtualmin-Domäne entspricht.
