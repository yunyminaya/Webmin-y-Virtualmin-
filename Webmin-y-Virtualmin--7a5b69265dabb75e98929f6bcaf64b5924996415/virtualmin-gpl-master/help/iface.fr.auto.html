Interface réseau
Si vous souhaitez configurer un hôte virtuel basé sur IP (requis pour les sites Web SSL ou l'hébergement FTP virtuel), cette option doit être définie sur Interface virtuelle avec IP et une adresse non utilisée doit être entrée. Virtualmin activera cette IP sur votre système lors de la création du serveur et l'associera au serveur.

Si l'option Partagé est sélectionnée, l'adresse d'interface réseau principale du système sera utilisée à la place pour le site Web et le domaine DNS de ce serveur.
