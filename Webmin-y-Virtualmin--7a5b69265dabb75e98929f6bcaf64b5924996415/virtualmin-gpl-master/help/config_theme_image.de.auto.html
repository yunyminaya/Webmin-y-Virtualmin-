URL für das Firmenlogo
Wenn diese Option auf die URL zu einem Bild festgelegt ist, wird sie in der oberen linken Ecke der Virtualmin-Benutzeroberfläche angezeigt, wenn das gerahmte Thema verwendet wird. Sie können das Logo für Ihr Hosting-Unternehmen festlegen. Wenn Sie darauf klicken, wird die URL geöffnet, die in der Option Link-Ziel für Firmenlogo angegeben ist .

Obwohl die genaue zulässige Größe des Bildes vom Browser abhängt, der für den Zugriff auf Virtualmin verwendet wird, sollte das Bild höchstens 220 Pixel breit sein. Die Höhe ist nicht begrenzt, aber 120 Pixel scheinen eine vernünftige Größe zu sein.
