Modèles de fichiers à remplacer uniquement
Si les variables de substitution dans le contenu? est définie sur Oui , Virtualmin remplacera les variables dans tous les fichiers à l'exception des images lors de leur copie à partir du répertoire squelette. Cependant, cela peut casser des scripts PHP ou perl qui utilisent le même format pour les variables que le langage de modèle de Virtualmin.

Pour éviter ce problème, vous pouvez entrer une liste de modèles de fichiers dans ce champ, comme *.conf . Seuls les fichiers correspondant à ces modèles et non les modèles des modèles de fichiers à ne pas remplacer ne verront pas leur contenu modifié.
