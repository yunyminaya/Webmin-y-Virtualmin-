Набор символов для ответного сообщения
Если ваше сообщение автоответа не содержит символов US-ASCII, вам следует ввести правильный набор символов, чтобы убедиться, что оно правильно отформатировано в почтовом клиенте получателя. По умолчанию используется набор символов, который в данный момент используется вашим браузером.
