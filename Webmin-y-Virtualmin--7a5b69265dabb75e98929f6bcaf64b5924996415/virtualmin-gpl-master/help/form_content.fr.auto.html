Créer une page Web initiale ?
Pour créer une page Web initiale pour le nouveau serveur virtuel, vous pouvez choisir de ne créer aucune page Web en sélectionnant « Non », ou opter pour une page par défaut du site Web Virtualmin en choisissant l'option « Oui, avec le modèle de page par défaut du site Web ».

Alternativement, sélectionner « Oui, avec le contenu ci-dessous » vous permet de saisir du code HTML ou du texte personnalisé à utiliser comme contenu de la page Web.
