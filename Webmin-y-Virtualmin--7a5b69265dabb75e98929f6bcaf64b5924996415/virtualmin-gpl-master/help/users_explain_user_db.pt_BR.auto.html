Criar usuário de banco de dados
Cria um novo usuário de banco de dados capaz de gerenciar apenas bancos de dados, aproveitando as cotas do proprietário do servidor virtual para todos os bancos de dados gerenciados. Este usuário opera exclusivamente no ambiente de banco de dados, sem criar uma conta Unix separada. Além disso, esse usuário pode ser perfeitamente integrado mesclando-se com uma conta Unix criada usando o mesmo nome de usuário.
