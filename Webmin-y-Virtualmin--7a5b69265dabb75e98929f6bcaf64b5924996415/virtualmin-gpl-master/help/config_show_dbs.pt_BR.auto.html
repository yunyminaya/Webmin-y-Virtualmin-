Mostrar bancos de dados na lista de usuários?
Quando definido como Sim , o número de bancos de dados aos quais cada usuário tem acesso será exibido em uma coluna separada na página Editar usuários . Este recurso fornece uma visão geral rápida da acessibilidade do banco de dados para cada usuário, complementando as informações já disponíveis na coluna “Acesso de Login”. Optar por Não ocultará "Bancos de dados?" coluna da lista de usuários.
