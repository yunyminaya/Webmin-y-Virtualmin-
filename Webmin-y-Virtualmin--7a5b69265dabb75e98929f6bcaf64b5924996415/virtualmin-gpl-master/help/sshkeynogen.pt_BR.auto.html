Chave pública SSH
Esta opção permite que o acesso SSH seja concedido ao usuário recém-criado usando autenticação de chave pública. As opções são:

Nenhuma chave padrão
Não configure uma chave SSH. Somente serão permitidos logins com senha.

Use a chave pública abaixo
Logins SSH serão permitidos usando a chave privada que corresponde à chave pública inserida na caixa de texto abaixo.
