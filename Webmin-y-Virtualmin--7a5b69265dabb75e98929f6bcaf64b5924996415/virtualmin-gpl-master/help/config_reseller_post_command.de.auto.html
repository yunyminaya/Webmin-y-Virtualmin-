Befehl zum Ausführen, nachdem Änderungen an einem Reseller vorgenommen wurden
Diese Option kann verwendet werden, um ein Skript oder einen Befehl anzugeben, der ausgeführt werden soll, nachdem ein Reseller erstellt, geändert oder gelöscht wurde. Mithilfe der Umgebungsvariablen RESELLER_NAME und RESELLER_ACTION kann ermittelt werden, welcher Reseller geändert wird und was damit geschieht.
