Cuota para usuario de administración
El propietario de un dominio es como una cuenta de usuario normal, ya que puede recibir correo electrónico, y también se pueden aplicar cuotas de uso de espacio en disco. Los archivos dentro del sitio web del dominio normalmente serán propiedad del usuario, por lo que puede ser útil especificar el tamaño del sitio web utilizando esta cuota, al tiempo que permite que la cuota total para todo el servidor limite el total utilizado para el correo y los datos del sitio web que pertenecen a todos los usuarios dentro del dominio.
