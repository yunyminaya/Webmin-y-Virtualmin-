Website komt overeen met alle sub-domeinen?
Wanneer deze optie is aangezet, zullen aanvragen voor iedere URL onder het domein (zoals http://abc.voorbeeld.com) de virtuele server website terug sturen. Dit kan handig zijn indien u een applicatie host die verschillende soorten inhoud kan versturen gebaseerd op de aangevraagde domein naam, zoals Wordpress MU.

Als de virtuele server DNS domein word gehost op dit systeem zal Virtualmin ook de wildcard * DNS record toevoegen wanneer Ja is geselecteerd.
