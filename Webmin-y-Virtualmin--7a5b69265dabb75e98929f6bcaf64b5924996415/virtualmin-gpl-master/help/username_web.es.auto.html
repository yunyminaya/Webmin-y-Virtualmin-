Nombre de usuario
Este campo establece la parte del nombre de usuario del inicio de sesión del usuario, que debe ser único dentro de la lista de usuarios del servidor web. El inicio de sesión del usuario para acceder al servidor web se derivará del nombre de usuario ingresado aquí, combinado con el nombre de dominio. Por ejemplo, si ingresa el nombre de usuario web-user1 y el dominio es example.com , el inicio de sesión resultante será web-user1@example.com .
