Automatisches BCCing ausgehender E-Mails zulassen?
Wenn diese Option auf Ja gesetzt ist , können Sie jeden virtuellen Server so konfigurieren, dass von seinen E-Mail-Benutzern gesendete E-Mails an eine separate Adresse gesendet werden. Dies wird jedoch nur funktionieren , wenn der Postfix Mail - Server verwendet wird und wenn die sender_bcc_maps Richtlinie in seiner gesetzten main.cf - Datei.

Wenn Postfix auch die hat recipient_bcc_maps Richtlinie Satz, werden Sie auch die Möglichkeit haben , eine Adresse zu setzen für BCCing eingehende Nachrichten.
