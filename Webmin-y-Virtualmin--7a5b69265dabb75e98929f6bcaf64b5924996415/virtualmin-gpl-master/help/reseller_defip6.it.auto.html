Indirizzo IPv4 per server basati sul nome
Questa opzione può essere utilizzata per fare in modo che i server virtuali creati per questo rivenditore utilizzino un indirizzo IPv6 diverso da quello predefinito del sistema, che in genere è condiviso da tutti i siti Web virtuali basati sul nome. Ciò può essere utile per i server di "branding" creati da rivenditori diversi, facendo in modo che l'indirizzo IPv4 di ciascun rivenditore si risolva in un nome host diverso.

L'IP inserito deve essere già attivo sul sistema, cosa che può essere fatta nel modulo Network Configuration di Webmin. L'IP non verrà utilizzato per server virtuali con i propri IP privati, poiché ognuno ottiene sempre un indirizzo distinto.
