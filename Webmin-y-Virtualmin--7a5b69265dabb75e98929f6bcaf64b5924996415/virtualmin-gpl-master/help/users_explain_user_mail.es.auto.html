Crear usuario de correo
Crea una nueva cuenta de usuario Unix, asignada al mismo grupo que el propietario del servidor virtual con funcionalidad de correo electrónico, lo que permite la gestión específica del envío y recepción de correos electrónicos. Este usuario está configurado inicialmente con cuotas predeterminadas y todas las funciones adicionales deshabilitadas, que se pueden habilitar más adelante si es necesario.
