Sous-répertoire FTP anonyme à créer
Si les téléchargements FTP anonymes sont autorisés, vous pouvez spécifier le sous-répertoire où les fichiers à télécharger vivront. Si vous modifiez les directives de l'option précédente, vous devez inclure une section anonyme et elle doit correspondre à l'emplacement que vous fournissez ici. Notez que l'accès anonyme via un nom de domaine spécifique nécessite la configuration d'un hôte FTP virtuel basé sur IP.
