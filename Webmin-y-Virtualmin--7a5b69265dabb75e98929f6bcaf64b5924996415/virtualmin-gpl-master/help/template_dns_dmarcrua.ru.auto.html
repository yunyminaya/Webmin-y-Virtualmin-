Отчетный URI для сводных отчетов
По умолчанию DNS-запись DMARC для новых доменов будет использовать mailto:postmaster@domain.com в качестве адреса для получателей спама, чтобы сообщать сводные сводки. Однако вы можете ввести новый URI здесь, чтобы настроить адрес электронной почты, например, чтобы использовать общий адрес для всех доменов.
