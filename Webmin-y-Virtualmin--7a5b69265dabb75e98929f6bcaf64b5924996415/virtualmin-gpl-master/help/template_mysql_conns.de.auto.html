Maximale Verbindungen für Domänenbesitzer
Dieses Feld steuert die Anzahl gleichzeitiger Verbindungen zu MySQL, die das Konto eines virtuellen Serverbesitzers herstellen kann. Dazu gehören Verbindungen von Webanwendungen, die von Virtualmin installiert wurden, wie z. B. Wordpress oder Drupal. Standardmäßig ist eine unbegrenzte Anzahl von Verbindungen zulässig. Da jedoch jede Verbindung Speicher innerhalb des MySQL-Serverprozesses verbraucht, möchten Sie die Gesamtzahl der Verbindungen auf einem ausgelasteten System möglicherweise begrenzen.

Das hier festgelegte Limit wird nur angewendet, wenn ein neuer virtueller Server erstellt wird oder wenn ein MySQL-Login für einen vorhandenen Server aktiviert wird.
