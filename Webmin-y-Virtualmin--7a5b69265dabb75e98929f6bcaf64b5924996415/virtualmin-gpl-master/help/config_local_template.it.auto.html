File modello per e-mail a nuovi utenti locali
Gli utenti locali sono utenti che non appartengono a un dominio Virtualmin, ad esempio, se dispongono solo di posta e / o servizio Web all'interno del dominio nativo del sistema. È possibile inviare l'e-mail predefinita per il nuovo utente oppure è possibile specificare un file modello diverso. Le variabili standard del modello Virtualmin sono disponibili all'interno del modello.
