Пароль MySQL для новых доменов
Эта опция контролирует исходный пароль MySQL для новых виртуальных серверов. Варианты:
Синхронизация с администрацией входа
Пароль для MySQL всегда остается таким же, как логин администратора виртуального сервера.

То же, что и начальный логин администратора
Пароль изначально совпадает с логином администратора, но не изменяется при изменении пароля администратора.

Генерировать случайным образом
Когда виртуальный сервер создан, случайно сгенерированный пароль MySQL выбирается и не изменяется.
