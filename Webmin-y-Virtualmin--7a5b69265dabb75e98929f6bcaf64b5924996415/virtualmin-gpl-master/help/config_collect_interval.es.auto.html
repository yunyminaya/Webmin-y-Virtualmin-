Intervalo entre ejecuciones de trabajos de recopilación de estado
Virtualmin tiene un proceso en segundo plano llamado collectinfo.pl que se ejecuta de forma predeterminada cada 5 minutos desde Cron, para recopilar información utilizada para la página de Información del sistema y gráficos históricos. Si esto es demasiado frecuente para su sistema, puede aumentar la brecha entre ejecuciones hasta 30 minutos utilizando este campo.

Alternativamente, seleccione Nunca ejecutar para desactivar este proceso por completo. Si rara vez inicia sesión en la interfaz web de Virtualmin como root y nunca mira los gráficos de carga de CPU y otras estadísticas, esta opción reducirá la carga de CPU en su sistema. Sin embargo, hará que la página de información del sistema tarde más en cargarse.
