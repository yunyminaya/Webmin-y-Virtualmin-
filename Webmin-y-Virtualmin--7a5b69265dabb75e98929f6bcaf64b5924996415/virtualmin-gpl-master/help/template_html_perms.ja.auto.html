Web サイトのディレクトリに対するアクセス許可
このオプションでは、前のオプションで指定した Web サイト ディレクトリ内のファイルとディレクトリに対する初期アクセス許可を指定します。 suEXEC が有効になっており、Apache ユーザーが仮想サーバー ユーザー グループに追加されている場合、これらのアクセス許可を強化して、他のユーザーが Web サイト ディレクトリの内容を参照できないようにすることができます。シェル アクセス権を持つすべてのユーザーが信頼できるユーザーでない限り、この強化された権限セットはあらゆる展開に推奨されます。推奨される権限は750で、これはrwxr-x---と同等です。つまり、所有者は読み取り、書き込み、実行が可能で、グループは読み取りと実行が可能ですが、その他のユーザーにはディレクトリ内のファイルの読み取り、書き込み、実行の権限がありません。
