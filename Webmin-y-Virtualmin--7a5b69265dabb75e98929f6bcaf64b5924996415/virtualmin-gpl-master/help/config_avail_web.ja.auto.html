Apache Web サーバー (仮想ホスト用)
ドメイン所有者が自分のドメインの Apache 設定を編集できるようにする場合は、これを有効にします。このレベルの制御は、経験の浅いユーザーにとっては手に負えないかもしれませんが、経験豊富なユーザーにとっては大きな柔軟性を提供します。また、デフォルトのアクセス制御設定を変更することで、Apache の設定ファイルの仮想ドメイン セクションに直接アクセスできるようにすることもできます。悪意のあるアクションを防ぐための保護は施されていますが、完璧ではない可能性があります。デフォルトでは、付与された権限は、共有ホスティング環境での使用に一般的に安全です。
