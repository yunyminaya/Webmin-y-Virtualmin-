Zusätzliche Parameter zu den Befehlen ssh und scp
Wenn Virtualmin eine Remote-Sicherung oder Wiederherstellung mit SSH durchführt, werden die Befehle ssh und scp zum Übertragen von Dateien verwendet. In diesem Feld können Sie zusätzliche Parameter für diese Befehle angeben, z. B. das Flag -i gefolgt vom Pfad zu einer alternativen Identitätsdatei für den privaten SSH-Schlüssel.
