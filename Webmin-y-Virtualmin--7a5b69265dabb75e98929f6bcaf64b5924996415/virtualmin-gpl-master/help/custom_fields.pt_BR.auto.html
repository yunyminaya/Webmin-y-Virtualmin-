Os campos personalizados
Os campos personalizados permitem que o administrador inclua dados adicionais sobre contas de domínio. Os campos serão exibidos ao editar o domínio. Eles podem ser usados para praticamente qualquer tipo de dados. Por exemplo, pode ser útil ter informações de contato adicionais anexadas à conta do domínio ou um campo de anotações usado para explicar por que um domínio está desativado ou possui algum recurso ou limite adicional além de outros domínios criados com o mesmo modelo. Se você estiver desenvolvendo ferramentas para interagir com o Virtualmin, esses campos podem ser úteis para outros fins.

Para cada campo, você deve inserir o seguinte:

Nome do campo
Um código curto para esse campo, usado para se referir ao campo em modelos e scripts de pós-criação. Cada campo deve ter um nome de campo exclusivo.

Rótulo
O texto que será exibido ao usuário ao inserir o valor para este campo. Se o rótulo contiver um; , qualquer coisa depois dela será usada como uma dica de ferramenta que aparece ao passar o mouse sobre o campo na página Editar servidor virtual.

Tipo
Isso determina que tipos de valores podem ser inseridos para este campo. A maioria é auto-explicativa, exceto a opção Menu . Quando isso é selecionado, você deve inserir o caminho para um arquivo no sistema Virtualmin na caixa de texto adjacente. Cada linha do arquivo deve conter um código e o texto exibido para uma entrada no menu desse campo personalizado, separados por espaços.

Na lista de servidores?
Se isso estiver definido como Sim , uma coluna para este campo personalizado será mostrada na página Listar Servidores Virtuais .
