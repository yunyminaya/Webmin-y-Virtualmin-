Nombre de usuario
Este campo establece la parte del nombre de usuario del inicio de sesión del usuario, que debe ser única dentro de la base de datos. El inicio de sesión del usuario para acceder a la base de datos se derivará del nombre de usuario ingresado aquí, combinado con el nombre de dominio. Por ejemplo, si ingresa el nombre de usuario db-user1 y el dominio es example.com , el inicio de sesión resultante será db-user1@example.com .
