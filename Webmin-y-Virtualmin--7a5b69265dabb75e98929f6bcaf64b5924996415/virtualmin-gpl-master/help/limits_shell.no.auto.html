Tillatt innloggingstype
Denne innstillingen bestemmer hvordan administratoren for dette domenet får lov til å logge på, ved å endre Unix-skallet. Standard tilgjengelige alternativer er:

Bare e-post
Både FTP- og SSH-pålogginger er blokkert, men POP3 og IMAP vil fungere.
E-post og FTP
Brukeren kan koble til via FTP, men ikke via SSH eller telnet.
E-post, FTP og SSH
Innlogging via FTP, SSH og telnet er tillatt.
Disse kan imidlertid tilpasses av hovedadministratoren på Custom Shells -siden.

Fordi denne funksjonen fungerer ved å endre den virtuelle serveren Unix-brukerens skall, antar den at FTP-serveren din bruker filen /etc/shells for autentisering.
