Установить групповое владение файлами базы данных MySQL?
Когда эта опция включена, для любых новых баз данных MySQL будет установлено владение группой в их файлах данных (обычно в /var/lib/mysql ), чтобы они были включены в ограничения квот Unix своих виртуальных серверов. Если этот параметр не включен, использование диска базы данных не будет ограничено, и для виртуального сервера будет возможно превышение разрешенного объема дискового пространства.
