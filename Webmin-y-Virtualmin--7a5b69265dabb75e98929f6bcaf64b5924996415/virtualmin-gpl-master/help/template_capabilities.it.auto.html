Capacità di modifica predefinite
Queste caselle di controllo determinano quali capacità dei loro server virtuali i proprietari di dominio possono modificare per impostazione predefinita. Una volta creato un server, queste possono essere modificate nella pagina Modifica limiti proprietario.

Se è selezionata l'opzione Automatico , i limiti vengono determinati in base al fatto che al proprietario del server virtuale sia consentito o meno creare sottoserver (controllato dal campo Limite al numero di server virtuali ). In tal caso, avrà accesso a tutte le capacità. Altrimenti, può solo gestire utenti, alias e modificare pagine web.
