Изменить доменное имя
Эта форма может быть использована для изменения доменного имени существующего виртуального сервера. По умолчанию имя пользователя и домашний каталог также будут изменены, чтобы соответствовать новому домену, при условии, что они основаны на нем.

Его также можно использовать для изменения имени пользователя администратора или домашнего каталога, вообще не затрагивая имя домена, например, если вы хотите перейти в новый базовый каталог. Только главный администратор может изменить имя пользователя или переместить дом в произвольное место.

По умолчанию существующие пользователи почтовых ящиков в домене не переименовываются, поэтому их учетные записи IMAP будут работать. Вы можете принудительно внести изменения, установив суффикс Изменить имя почтового ящика? Автоматически выбран, хотя. Это поле также можно использовать для изменения суффикса на что угодно, даже если имя домена не изменилось.

Если домен использует формат username@domain.com для почтовых ящиков, пользователи будут переименовываться только при изменении фактического имени домена.
