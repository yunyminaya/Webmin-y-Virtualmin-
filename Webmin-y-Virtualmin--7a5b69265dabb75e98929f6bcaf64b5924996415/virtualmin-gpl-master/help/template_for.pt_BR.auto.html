Para uso por
Os modelos podem ser aplicados a qualquer tipo de conta de usuário do domínio. Os tipos atuais são: "Servidores virtuais de nível superior", que são contas de host virtual padrão; "Sub-servidores", que são hosts virtuais que pertencem a um servidor virtual de nível superior e aparecem no diretório "domínios" do diretório inicial do servidor de nível superior; "Servidores de alias" são servidores que, geralmente, simplesmente encaminham a Web e o correio para um domínio de nível superior existente; e "Proprietários do servidor" estarão disponíveis para os proprietários do servidor quando eles criarem novos domínios. Para permitir o uso deste modelo pelos proprietários do domínio (por exemplo, proprietários do servidor), você deve habilitar pelo menos um outro tipo de servidor).
