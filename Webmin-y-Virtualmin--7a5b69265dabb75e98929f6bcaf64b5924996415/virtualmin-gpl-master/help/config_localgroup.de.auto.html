Primäre Gruppe für lokale Benutzer
Virtualmin kann auch lokale Benutzer auf Ihrem Server verwalten. Dies kann hilfreich sein, wenn Sie Virtualmin nur zum Verwalten von E-Mail- und Home-Websites verwenden möchten. Mit einer kleinen Neukonfiguration kann Virtualmin eine benutzerfreundliche Oberfläche zum Hinzufügen von Postfach- / Website-Benutzern bereitstellen, deren Website über "http://www.domain.com/~username" erreichbar ist. Dies ist nicht der Hauptfokus von Virtualmin, kann jedoch die Wartung von Postfächern und Websites für ISPs vereinfachen.
