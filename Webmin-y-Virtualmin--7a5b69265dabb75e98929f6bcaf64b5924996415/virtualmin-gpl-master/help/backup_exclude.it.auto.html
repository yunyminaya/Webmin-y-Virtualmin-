File da escludere da ciascun dominio

Questo campo consente di inserire un elenco di percorsi relativi ai file nella home directory di ciascun server virtuale da escludere da questo backup. Per escludere più percorsi, posizionali ciascuno sulla propria riga (ad esempio, separati da avanzamenti di riga).

Esempi

Per escludere un'intera directory, fornire il percorso relativo dalla home directory dell'utente, ad esempio public_html/stats per escludere la directory dei report di Webalizer o homes per escludere le home directory degli utenti.

Per escludere un singolo file, fornire il percorso relativo al nome file, ad esempio public_html/index.html .

I file possono anche essere specificati usando caratteri jolly, come *.log , che escluderà qualsiasi file con estensione .log qualsiasi punto all'interno della home directory del dominio. È inoltre possibile fornire un percorso, ad esempio etc/*.ini per escludere tutti i file che terminano con .ini .
