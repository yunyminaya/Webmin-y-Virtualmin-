Standard-IP-Adresse für DNS-Einträge
DNS-Einträge werden automatisch für neue virtuelle Hosts generiert. Unter normalen Umständen möchten Sie, dass sie mit der oben angegebenen IP-Adresse des virtuellen Servers übereinstimmen. Wenn sich Ihr Server jedoch hinter einem NAT-Router befindet, betreiben Sie einen Proxyserver, einen Lastausgleichsserver oder einen anderen Gerätetyp, der dazu führt, dass Ihr Server mit einer anderen IP-Adresse arbeitet, als die Welt sieht, wenn Sie mit Ihren Domänen sprechen Möglicherweise muss der Wert hier geändert werden. Eine andere Instanz, in der Sie dies möglicherweise ändern möchten, ist das Hinzufügen von Domänen zu einem Entwicklungsserver, der bei Produktionsbeginn unterschiedliche Adressen hat. Sie können hier die Produktions- oder öffentliche IP-Adresse eingeben, anstatt Virtualmin die lokale IP-Adresse verwenden zu lassen.

Wenn die Option Externe Adresse automatisch erkennen ausgewählt ist, versucht Virtualmin, die IP- Adresse beim Erstellen des virtuellen Servers zu ermitteln. Dies erfolgt durch eine HTTP-Anfrage an www.virtualmin.com , die mit der Adresse antwortet, von der die Verbindung stammt.
