È possibile selezionare home directory per gli utenti?
Se abilitato, il proprietario del dominio può scegliere di assegnare agli utenti una home directory diversa da quella predefinita. Può essere utile se i proprietari di domini possono installare servizi aggiuntivi, come un application server (ad es. Zope, Webrick, ecc.) E vorrebbero poter utilizzare un utente diverso per il server delle app. In genere, solo gli utenti con un livello di affidabilità relativamente elevato avranno bisogno di questo tipo di flessibilità.
