Apache SSL protocols to allow
By default, when Virtualmin sets up a new SSL website it configures Apache to reject unsafe SSL protocol versions (TLSv1.1 and below). However, you can specify your own list of protocols by listing them here. Valid protocols are SSLv2, SSLv3, TLSv1, TLSv1.1 and so on. To reject a protocol, put a - before the name. To allow all protocols, enter all.
