Добавить запись NS для этой системы?
Если для этого параметра установлено значение «Да» , как это установлено по умолчанию, запись NS будет добавлена в новые зоны DNS для имени хоста основного DNS-сервера, указанного выше. Значение «Нет» следует устанавливать только в том случае, если вы используете скрытый основной сервер имен.
