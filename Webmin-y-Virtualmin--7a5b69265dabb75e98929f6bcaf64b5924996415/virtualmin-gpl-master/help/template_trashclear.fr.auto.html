Supprimer automatiquement la corbeille ?
Cette option peut être utilisée pour supprimer automatiquement les e-mails présents dans les dossiers poubelle des utilisateurs de messagerie sur les serveurs virtuels créés avec ce modèle dès qu'ils deviennent trop anciens ou dépassent une certaine taille. Seuls les déchets dépassant les critères de taille ou d'âge que vous avez définis ici seront supprimés, et la vérification est effectuée une fois par jour vers minuit.
