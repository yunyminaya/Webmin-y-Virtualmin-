Conchiglie personalizzate
Questa pagina consente di personalizzare le shell Unix disponibili per gli utenti di cassette postali e gli amministratori di server virtuali. La shell viene utilizzata da server come ProFTPd e SSHd per controllare i tipi di login consentiti per ciascun utente, come e-mail, FTP e accesso remoto alla shell.

Quando si selezionano le impostazioni predefinite integrate di Virtualmin , le shell disponibili vengono determinate automaticamente da Virtualmin, che in genere è soddisfacente per la maggior parte dei sistemi. Se si desidera modificarli, selezionare Shell personalizzate di seguito e apportare modifiche alla tabella.

Per ogni shell, è possibile modificare i seguenti campi:

Abilitato?
Questa casella di controllo determina se la shell è visibile durante la creazione o la modifica degli utenti.
Path to shell
Questo deve contenere il percorso completo del comando shell, come /bin/sh o /bin/false .
Descrizione
Questo è il titolo che apparirà nel menu quando viene selezionata la shell.
Admin?
Selezionare questa casella se si desidera che la shell sia resa disponibile per gli utenti Unix dell'amministratore del server virtuale.
Cassetta postale?
Seleziona questa casella se vuoi che la shell sia resa disponibile per gli utenti di posta / FTP.
Predefinito?
Esattamente una shell per gli amministratori e una per le cassette postali devono avere questa voce controllata, a indicare che è l'impostazione predefinita per gli utenti Unix di quel tipo.
Accesso
Questo menu indica a Virtualmin quale tipo di accesso è concesso da questa shell. Quando si aggiunge il proprio, è necessario selezionare se consente FTP e / o SSH o meno.
