Compartilhar certificados SSL entre domínios sempre que possível?
Quando essa opção está definida como Sim (o padrão), o Virtualmin compartilhará certificados SSL entre servidores virtuais que possuem a mesma propriedade, se os nomes de host no certificado corresponderem a todos os domínios. Selecionar Não forçará a criação de um certificado separado para cada novo servidor virtual.
