Processos máximos de servidor Mongrel para Rails
Ao instalar scripts Ruby on Rails, o administrador do servidor pode selecionar como meu servidor Mongrel processa para começar a manipular solicitações ao caminho do script - se o módulo mod_proxy_balancer do Apache estiver instalado. Para impedir que um número excessivo de processos do servidor Mongrel sobrecarregue o sistema, essa opção pode ser usada para limitar o número que os administradores de domínio têm permissão para configurar no Virtualmin.
