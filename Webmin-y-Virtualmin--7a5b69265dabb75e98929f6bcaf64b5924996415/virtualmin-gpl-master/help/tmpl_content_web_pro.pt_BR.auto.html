Página HTML padrão do site
Esta opção permite que você controle a página padrão que será criada e exibida para o novo servidor virtual.

Você pode escolher "Desativado" para pular a criação de uma página ou selecionar "Perguntar no momento da criação" para decidir durante o momento da criação do servidor virtual.

Como alternativa, você pode optar por uma página padrão do site Virtualmin selecionando a opção "Sim, com o modelo de página padrão do site".

Se preferir, escolher "Sim, com conteúdo abaixo" permite que você forneça HTML ou texto personalizado para usar como conteúdo da página da web.
