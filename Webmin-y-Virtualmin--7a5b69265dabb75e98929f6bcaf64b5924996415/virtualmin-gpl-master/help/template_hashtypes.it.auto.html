Tipi di password con hash da archiviare
Per impostazione predefinita, Virtualmin memorizzerà le password del dominio e delle cassette postali in vari formati di hash in modo che la password sia disponibile quando si aggiungono nuove funzionalità al dominio. Tuttavia, non tutti i formati di hash sono ugualmente sicuri, quindi questo campo può essere utilizzato per disabilitare la memorizzazione di alcuni formati. Indipendentemente da ciò che selezioni qui, la password verrà sempre archiviata con l'hash utilizzando il formato predefinito del sistema in /etc/shadow o LDAP.

Tenere presente che la disabilitazione del formato hash MySQL potrebbe impedire a Virtualmin di creare un account di accesso MySQL per gli utenti di cassette postali con la stessa password assegnata in origine alla cassetta postale. La disabilitazione del formato digest Web può impedire l'accesso a una directory Web protetta da parte degli utenti delle cassette postali.
