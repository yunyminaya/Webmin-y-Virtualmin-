Crea sottodominio virtuale
Questo modulo consente di creare un nuovo sottodominio virtuale, di proprietà di un accesso amministrativo esistente. Un sottodominio è simile a un sotto-server, in quanto condivide la quota del disco, i limiti di larghezza di banda e i limiti di utenti, alias e altri oggetti con il suo genitore. Tuttavia, il nome del sottodominio deve essere sotto il nome del dominio principale, utilizza una sottodirectory della directory public_html del genitore per il suo sito Web e i file di registro sono condivisi.

Quando si crea il server, è necessario inserire almeno il nome di dominio e una descrizione. Se sul sistema sono definiti più modelli di server, è possibile selezionarne uno dal menu: i modelli determinano varie impostazioni predefinite per il nuovo server, come la configurazione del sito Web, i record DNS e così via.

Per selezionare quali funzioni sono abilitate per il server, utilizzare le caselle di controllo nella seconda parte della sezione Opzioni avanzate . I sottodomini hanno un set molto più piccolo di funzionalità disponibili rispetto ai sotto-server o ai server di livello superiore.

Nella parte inferiore della sezione Opzioni avanzate è possibile selezionare per inoltrare tutto il traffico Web a questo nuovo sito a un altro URL. Puoi anche scegliere di fare in modo che il nuovo server virtuale utilizzi i propri indirizzi IP privati, richiesti per SSL e hosting FTP anonimo.
