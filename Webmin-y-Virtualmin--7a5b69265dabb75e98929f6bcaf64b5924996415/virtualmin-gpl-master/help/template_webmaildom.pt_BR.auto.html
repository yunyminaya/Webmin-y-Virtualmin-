URL para redirecionamento de webmail
Por padrão, o Virtualmin redirecionará solicitações para http://webmail.thedomain.com para Usermin no mesmo sistema, como https://thedomain.com:20000/ . No entanto, esse campo pode ser usado para alterar o URL de redirecionamento, talvez para um único nome de host para todos os seus domínios para os quais você possui um certificado SSL, como https://webmail.yourcompany.com:20000/ . Ou você pode redirecionar para outro aplicativo de webmail no mesmo domínio, inserindo um URL como http://www.${DOM}/squirrelmail/ .
