Configurar la rotación del archivo de registro?
Si esta opción está habilitada, los archivos de registro web del servidor virtual se rotarán semanalmente para evitar consumir demasiado espacio en disco. Cada semana, el archivo de registro actual se copiará, truncará y comprimirá. De forma predeterminada, los registros rotados se mantendrán durante hasta cinco semanas antes de ser eliminados.
