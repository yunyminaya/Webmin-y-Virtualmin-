Максимальное количество подпроцессов службы PHP
Этот параметр устанавливает ограничение на количество одновременных запросов, которые будут обслуживаться. Когда PHP-скрипты для этого домена запускаются через FPM, количество дочерних процессов будет создано, когда для параметра pm установлено значение static , и максимальное количество дочерних процессов. потенциально может быть создан, если для параметра pm установлено значение dynamic.

Когда сценарии PHP для этого домена настраиваются для запуска через FCGI, управление дочерними процессами PHP всегда должно быть отключено, что будет маршрутизировать только один запрос за раз к процессам приложения, которые он породил. По умолчанию и при настройке переменной среды PHP_FCGI_CHILDREN=0 управление дочерними процессами PHP отключено.
