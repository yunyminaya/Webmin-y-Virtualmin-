Número base para interfaces virtuais
Por padrão, o Virtualmin usará eth0:0 para a primeira interface IP virtual criada, depois eth0:1 e assim por diante. No entanto, esse campo pode ser usado para alterar onde o número da interface após o: é iniciado. Isso pode ser útil se você deseja reservar alguns dos números mais baixos para atribuição manual, mas na maioria dos casos não precisa ser alterado.
