Mode d'exécution des scripts CGI
Cette option détermine la manière dont les scripts CGI sont exécutés pour ce serveur virtuel. Les options possibles sont :
  • Scripts CGI désactivés - les scripts ne peuvent pas être exécutés du tout.
  • Serveur FCGIwrap - les scripts sont exécutés via un serveur en arrière-plan, un par domaine.
  • suEXEC - les scripts sont lancés via le wrapper Apache suexec . Notez que cette option peut ne pas être accessible sur tous les systèmes.
