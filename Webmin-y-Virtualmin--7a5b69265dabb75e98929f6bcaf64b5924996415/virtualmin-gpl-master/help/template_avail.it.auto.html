Moduli Webmin disponibili per gli amministratori di server virtuali
Questa sezione consente di selezionare a quali moduli dei server Webmin avranno accesso i proprietari dei server virtuali, per i server creati utilizzando questo modello. I collegamenti a questi moduli vengono visualizzati nel menu a sinistra di Virtualmin, in genere nella categoria Servizi. È possibile disabilitare alcuni o tutti per limitare i tipi di modifiche alla configurazione più avanzate che i proprietari dei server possono apportare e limitarli invece alla più semplice interfaccia utente di Virtualmin.

Eventuali modifiche apportate qui verranno applicate immediatamente a tutti i server virtuali utilizzando questo modello.
