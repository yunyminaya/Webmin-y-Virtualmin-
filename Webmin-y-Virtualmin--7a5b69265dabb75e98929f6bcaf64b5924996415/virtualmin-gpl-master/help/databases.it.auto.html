Modifica database
Questa pagina elenca tutti i database attualmente di proprietà del server virtuale e consente la creazione di database aggiuntivi fino a qualsiasi limite in vigore per l'utente connesso a Virtualmin. Puoi anche eliminare un altro database selezionandolo dall'elenco e premendo il pulsante Elimina selezionati .

Per visualizzare le tabelle e i dati effettivi in un database, fare clic sul collegamento Gestisci accanto al nome del database nella tabella. Per vedere la dimensione del database, fare clic sul suo nome.

Quando si utilizza Virtualmin Pro, vengono visualizzati due moduli aggiuntivi sotto l'elenco dei database per modificare il login e la password per gli utenti MySQL e PostgreSQL utilizzati da questo server virtuale. Ciò può essere utile se si desidera che l'accesso al database sia diverso dall'accesso dell'amministrazione principale, magari per motivi di sicurezza. Questi dovrebbero essere modificati con attenzione, poiché qualsiasi script PHP (come quelli degli installatori di webapp) che hanno il nome utente o la password nei file di configurazione non verranno aggiornati.
