Comandos personalizados
Esta opción especifica si el propietario del dominio tendrá acceso al módulo Webmin de comandos personalizados. Con el módulo Comandos personalizados puede extender Webmin sin tener que programar un nuevo módulo. Un comando personalizado puede ejecutar cualquier comando en el sistema con las opciones que especifique. El reemplazo de variables también está disponible para que el usuario que ejecuta el comando pueda proporcionar argumentos de entrada u opción al comando, y es posible ejecutar el comando como el usuario registrado de Webmin, o como cualquier usuario en el sistema (incluyendo root, pero sea cuidado con eso! Esta es una característica muy poderosa de Webmin, que le permite agregar nuevos "botones" que realizan funciones específicas para su entorno, o que algunos de sus usuarios pueden necesitar acceso.
