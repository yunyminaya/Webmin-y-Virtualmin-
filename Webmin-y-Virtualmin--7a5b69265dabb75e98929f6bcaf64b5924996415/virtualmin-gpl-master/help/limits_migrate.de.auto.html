Kann Backups migrieren?
Standardmäßig dürfen Domänenbesitzer keine virtuellen Server erstellen, indem sie Sicherungen von anderen Kontrollfeldern (wie cPanel und Plesk) migrieren, da eine schädliche Sicherungsdatei verwendet werden kann, um Root- Zugriff auf das System zu erhalten. Wenn Sie jedoch einem Benutzer vertrauen, können Sie die Berechtigung zum Migrieren von Sicherungen erteilen, indem Sie hier Ja auswählen.
