IPv4-Adresse für namensbasierte Server
Mit dieser Option können virtuelle Server, die für diesen Reseller erstellt wurden, eine andere IP-Adresse als die Standardadresse des Systems verwenden, die normalerweise von allen namenbasierten virtuellen Websites gemeinsam genutzt wird. Dies kann nützlich sein, um Server zu brandmarken, die von verschiedenen Wiederverkäufern erstellt wurden, indem die IP-Adresse jedes Wiederverkäufers in einen anderen Hostnamen aufgelöst wird.

Die von Ihnen eingegebene IP muss bereits auf dem System aktiv sein. Dies kann im Netzwerkkonfigurationsmodul von Webmin erfolgen. Die IP wird nicht für virtuelle Server mit eigenen privaten IPs verwendet, da jeder immer eine eigene Adresse erhält.
