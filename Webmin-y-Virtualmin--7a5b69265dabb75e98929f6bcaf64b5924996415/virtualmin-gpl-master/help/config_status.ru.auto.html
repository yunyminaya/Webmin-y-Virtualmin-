Мониторинг статуса сайта?
Включает или отключает создание мониторов состояния в модуле Webmin System и Server Status. Если этот параметр отключен, монитор службы не будет автоматически создаваться для вашего нового домена. Монитор состояния может уведомить вас по электронной почте, если служба не работает, или он может запустить команду, чтобы выполнить какое-либо действие, чтобы попытаться автоматически устранить проблему.
