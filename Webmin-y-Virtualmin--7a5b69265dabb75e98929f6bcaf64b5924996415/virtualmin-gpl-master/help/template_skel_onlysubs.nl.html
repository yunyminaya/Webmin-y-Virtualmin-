File patronen alleen vervangen
Als de Vervang variabelen in de inhoud? optie is ingesteld op Ja, zal Virtualmin de variabelen vervangen in alle files behalve voor afbeeldingen wanneer hij die kopieert van de skeleton directory. Echter dit kan de PHP of perl scripts kapot maken die hetzelfde formaat gebruiken voor variabelen als de Virtualmin taal template.

Om dit probleem te voorkomen kunt u een lijst met file patronen invullen in dit veld, zoals *.conf. Files die overeenkomen met deze patronen zullen geen last hebben dat hun inhoud word gewijzigd, zelfs als ze overeenkomen in de lijst File patronen niet vervangen.
