Kennwortfeldtyp
Diese Einstellung steuert, wie das Feld zur Eingabe des Kennworts für einen neuen virtuellen Server angezeigt wird. Die verfügbaren Optionen sind:
Zufällig generiertes Passwort
Es wird ein einzelnes sichtbares Feld angezeigt, das ein zufällig generiertes Kennwort enthält, das von Virtualmin ausgewählt wurde. Sie können jedoch ein anderes Passwort eingeben.

Passwort einmal eingeben
Ein einzelnes Feld zur Eingabe Ihres eigenen Passworts wird angezeigt, wobei das tatsächlich eingegebene Passwort ausgeblendet ist.

Geben Sie das zweite Passwort ein
Es werden zwei Felder zur Eingabe Ihres eigenen Passworts angezeigt, wobei das Passwort wieder ausgeblendet ist. Beide Passwörter müssen identisch sein.
