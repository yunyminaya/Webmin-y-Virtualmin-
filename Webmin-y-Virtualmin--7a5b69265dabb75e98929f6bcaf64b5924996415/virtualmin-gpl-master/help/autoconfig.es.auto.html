Configuración del cliente de correo
Esta página puede activar o desactivar la URL de configuración automática del cliente de correo para todos los servidores virtuales con correo electrónico habilitado. Los clientes como Thunderbird pueden usar esta URL para detectar la configuración de SMTP e IMAP basándose solo en una dirección de correo electrónico, incluido el nombre de usuario, el número de puerto y el tipo de cifrado correctos.

Cuando está habilitado, se crea un subdominio llamado autoconfig para cada servidor virtual, y la ruta URL /mail/config-v1.1.xml se asigna a un CGI que sirve XML para proporcionar los parámetros requeridos del cliente de correo. Estos se eliminarán cuando la función esté deshabilitada.
