Esegui solo un processo di spamassasin alla volta?
Quando questo campo è impostato su No come è per impostazione predefinita, Virtualmin eseguirà più processi simultanei di spamassassin se arrivano più messaggi e-mail contemporaneamente. Tuttavia, ciò può utilizzare CPU e RAM eccessive su un sistema di piccole dimensioni. Se si seleziona Sì , tutti i server virtuali con filtro antispam abilitato avranno i file di configurazione Procmail aggiornati per eseguire una sola copia di spamassassin alla volta.

Questa restrizione si applica a tutti i server virtuali, quindi anche se arrivano contemporaneamente due messaggi a domini diversi, ci sarà sempre un solo processo di spamassassin in esecuzione.
