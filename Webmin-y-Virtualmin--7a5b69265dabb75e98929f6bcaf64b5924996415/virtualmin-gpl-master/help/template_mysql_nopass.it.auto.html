Password MySQL per nuovi domini
Questa opzione controlla la password MySQL iniziale per i nuovi server virtuali. Le opzioni sono:
Sincronizzazione con accesso amministrativo
La password per MySQL è sempre la stessa dell'accesso amministrativo del server virtuale.

Come il login di amministrazione iniziale
La password inizialmente è uguale all'account di accesso dell'amministrazione, ma non viene modificata se si modifica la password di amministrazione.

Genera in modo casuale
Quando viene creato il server virtuale, viene scelta una password MySQL generata casualmente e non modificata.
