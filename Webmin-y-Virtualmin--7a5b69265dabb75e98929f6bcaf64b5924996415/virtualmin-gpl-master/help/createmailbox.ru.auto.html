Создать почтовый ящик
Эта страница предназначена для создания локальных почтовых ящиков. Эти почтовые ящики будут предназначены для адресов в домене по умолчанию сервера, а не в виртуальном домене, размещенном на сервере. Почтовые ящики для виртуальных доменов создаются во втором разделе страницы индекса модуля.

При создании нового локального почтового ящика требуются только имя пользователя, настоящее имя и пароль.
