Comando para ejecutar después de hacer cambios a un revendedor
Esta opción se puede usar para especificar una secuencia de comandos o comando que se ejecutará después de que se haya creado, modificado o eliminado un revendedor. Puede hacer uso de las variables de entorno RESELLER_NAME y RESELLER_ACTION para determinar qué revendedor se está cambiando y qué le está sucediendo.
