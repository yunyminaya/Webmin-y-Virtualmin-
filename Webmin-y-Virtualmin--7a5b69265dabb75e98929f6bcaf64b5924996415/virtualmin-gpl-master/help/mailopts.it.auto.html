impostazioni dell 'email
Questa pagina serve per configurare le impostazioni relative alla posta elettronica per un server virtuale. Se il sistema supporta l'e-mail inviata da BCC (ed è abilitata nella pagina Configurazione modulo), è possibile utilizzare la sezione E -mail in uscita BCC per selezionare un indirizzo in cui copiare la posta.

Se il server virtuale è un dominio alias, il campo Modalità alias di posta controlla se gli alias di posta vengono copiati dal dominio di destinazione o se viene creato un singolo alias catchall per inoltrare tutta la posta elettronica alla destinazione. La prima modalità non è disponibile con tutti i server di posta, ma è superiore quando si utilizza Postfix in quanto consente di inviare la posta a indirizzi non validi a livello di conversazione SMTP.
