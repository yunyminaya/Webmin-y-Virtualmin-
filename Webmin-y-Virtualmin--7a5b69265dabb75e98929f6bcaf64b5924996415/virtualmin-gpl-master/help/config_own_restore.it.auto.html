È possibile ripristinare i backup eseguiti dall'amministratore principale?
Per impostazione predefinita, i proprietari di server virtuali possono ripristinare solo i backup eseguiti da soli. Tuttavia, se questa impostazione viene modificata in Sì , i backup effettuati dall'amministratore principale ( root ) possono anche essere ripristinati dai proprietari del dominio. Naturalmente, questo vale solo per i loro domini.

Abilitare questa opzione solo se si è sicuri che tutte le destinazioni di backup siano protette, in quanto il ripristino di un backup eseguito da un utente non attendibile potrebbe compromettere la sicurezza del sistema.
