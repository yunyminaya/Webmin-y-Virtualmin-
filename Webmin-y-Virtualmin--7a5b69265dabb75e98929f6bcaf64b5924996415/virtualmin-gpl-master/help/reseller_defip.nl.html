IP adres voor op naam gebaseerde servers
Deze optie kan worden gebruikt om virtuele servers aan te maken voor deze reseller met gebruik van een IP adres anders dan de standaard IP adres van het systeem, wat normaal gesproken word gedeeld door alle op naam gebaseerde virtuele websites. Dit kan handig zijn voor aparte "merken" servers die aangemaakt worden door verschillende resellers, door voor iedere reseller het IP op te halen naar een andere hostnaam.

Het IP wat u invult moet reeds actief zijn op het systeem, dit kan worden gedaan in de Webmin Netwerk Configuratie module. Het IP adres zal niet worden gebruikt voor virtuele server met hun eigen prive IP adres, omdat een ieder altijd een verschillend adres krijgt.
