Base del directorio de inicio
Este es el directorio de nivel superior donde se crearán los directorios principales. Si su instalación de Virtualmin comparte un servidor con muchos otros servicios o usuarios del sistema estándar, puede ser útil usar algo que no sea / home. El valor predeterminado se obtiene del módulo Webmin de usuarios y grupos, que por defecto es / home. El módulo Usuarios y grupos permite extensiones complejas de esta convención, como "/ home / v / virtualmin", que puede facilitar la navegación de directorios si tiene muchos dominios en un solo servidor. La documentación del módulo Usuarios y Grupos en cualquiera de los libros de Webmin cubre esta característica con más detalle.
