Backup compressie formaat
Standaard gebruikt Virtualmin het Gzip compressie formaat op zijn backup files. U kunt echter deze optie gebruiken om het ruimte besparende (maar CPU intensieve) bzip2 formaat te kiezen. U kunt ook het niet gecomprimeerde tar formaat backup gebruiken wat handig kan zijn indien een snelle backup wilt hebben of indien u naar een tape weg schrijft.

Deze optie word alleen gebruikt wanneer u een virtuele server backup maakt naar een directory met 1 file per domein. Wanneer u een enkele file backup maakt bepaald de file naam het compressie formaat (zoals domains.tar.gz of domains.tar).
