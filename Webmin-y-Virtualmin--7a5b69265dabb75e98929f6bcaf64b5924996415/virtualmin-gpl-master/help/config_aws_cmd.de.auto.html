Vollständiger Pfad zum Befehl aws für S3-Operationen
Wenn der Befehl aws auf Ihrem System installiert ist und dieses Feld den vollständigen Pfad dazu enthält (z. B. /usr/bin/aws ), ruft Virtualmin ihn auf, um S3-Vorgänge für Sicherungen wie das Hochladen und Herunterladen von Dateien auszuführen. Da dieser Befehl von Amazon entwickelt wurde, ist er zuverlässiger als die in Virtualmin integrierte S3-API.
