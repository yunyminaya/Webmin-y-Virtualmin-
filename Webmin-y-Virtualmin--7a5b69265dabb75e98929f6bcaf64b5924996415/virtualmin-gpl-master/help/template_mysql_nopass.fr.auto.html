Mot de passe MySQL pour les nouveaux domaines
Cette option contrôle le mot de passe MySQL initial pour les nouveaux serveurs virtuels. Les options sont les suivantes:
Synchroniser avec la connexion d'administration
Le mot de passe pour MySQL est toujours le même que le login d'administration du serveur virtuel.

Identique à la connexion d'administration initiale
Le mot de passe est initialement identique à la connexion d'administration, mais n'est pas modifié si le mot de passe d'administration est modifié.

Générez au hasard
Lorsque le serveur virtuel est créé, un mot de passe MySQL généré de manière aléatoire est choisi et n'est pas modifié.
