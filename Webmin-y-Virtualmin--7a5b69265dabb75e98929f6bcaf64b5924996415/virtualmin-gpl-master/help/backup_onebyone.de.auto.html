Übertragen Sie jeden virtuellen Server nach dem Sichern
Wenn dieses Kontrollkästchen aktiviert ist und Sie eine Sicherung auf einem Remote-FTP- oder SSH-Server im neuen Format durchführen, wird die Datei für jeden virtuellen Server nach der Erstellung übertragen. Dies spart lokalen Speicherplatz, birgt jedoch das Risiko, eine unvollständige Sicherung zu übertragen, wenn einige virtuelle Server ausfallen.
