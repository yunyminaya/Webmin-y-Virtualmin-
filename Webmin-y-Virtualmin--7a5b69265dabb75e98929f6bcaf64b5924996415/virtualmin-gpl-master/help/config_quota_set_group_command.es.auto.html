Comando para establecer la cuota de un grupo
Cuando se utilizan comandos de cuotas externas, se llamará al programa configurado en esta opción cuando Virtualmin quiera establecer las cuotas para un grupo Unix. Los parámetros con los que se llama son:
  1. El nombre del grupo Unix
  2. La nueva cuota flexible en bloques de 1 kB, o 0 para ilimitado.
  3. La nueva cuota fija en bloques de 1 kB, o 0 para ilimitado.
