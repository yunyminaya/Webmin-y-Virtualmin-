Type de serveur Web
Un serveur Web virtuel peut être basé sur le nom ou sur IP. Dans la grande majorité des cas, un serveur virtuel basé sur le nom suffit. Ce n'est que lorsqu'un site SSL doit être hébergé, ou qu'un répertoire FTP anonyme doit être fourni pour le domaine, qu'il est nécessaire d'utiliser un serveur virtuel basé sur IP.
