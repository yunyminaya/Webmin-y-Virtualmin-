Questa opzione determina se e come gli script CGI vengono eseguiti per impostazione predefinita per i nuovi server virtuali. Le opzioni possibili sono:
  • Script CGI disabilitati : gli script non possono essere eseguiti affatto.
  • Server FCGIwrap : gli script vengono eseguiti tramite un server in background, uno per dominio.
  • suEXEC : gli script vengono avviati tramite il wrapper suexec di Apache. Tieni presente che questa opzione potrebbe non essere accessibile su tutti i sistemi.
