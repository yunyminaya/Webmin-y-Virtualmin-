Interface réseau IPv6
Si vous souhaitez que ce serveur virtuel ait une adresse IPv6 ainsi que IPv4, ce champ peut être utilisé pour avoir une adresse automatiquement allouée, si des plages d'allocation v6 ont été définies. Si aucune plage n'a été configurée dans le modèle de serveur, vous pouvez entrer une adresse IPv6 manuellement.
