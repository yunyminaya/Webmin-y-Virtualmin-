Zusätzlichen Unix-Benutzer auf Postfix-Systemen erstellen, wenn das User @ Domain-Format verwendet wird?
Wenn Sie Unix-Benutzernamen im User @ Domain-Format und den Postfix-Mailserver verwenden, erstellt Virtalmin, wenn diese Option auf yes (Standardeinstellung) gesetzt ist, einen zusätzlichen Unix-Benutzer im User-Domain-Format, um die E-Mail-Zustellung zu erleichtern. Auf einigen Systemen ist dies jedoch nicht erforderlich. Sie können diese Option daher verwenden, um diese Funktion zu deaktivieren. Dies gilt jedoch nur für neu erstellte Benutzer und Domänen.
