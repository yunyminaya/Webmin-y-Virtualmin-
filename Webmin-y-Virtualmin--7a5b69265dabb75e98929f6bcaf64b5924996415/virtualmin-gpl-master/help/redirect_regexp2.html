Redirect sub-directories to
This option controls how paths under the source URL path are redirected. For example, if the source path is /foo and the destination is http://foo.com the meanings of the options are :

  • Same sub-directory under destination URL
    A request to /foo/bar will be redirected to http://foo.com/bar

  • Just destination URL
    A request to /foo/bar will just be redirected to http://foo.com

  • Ignore sub-directories
    A request to /foo/bar will not be redirected at all, only a request to /foo will be.
