Usermin so konfigurieren, dass dasselbe SSL-Zertifikat verwendet wird?
Wenn diese Vorlage ausschließlich für SSL-fähige Websites verwendet wird, können Sie diese Option aktivieren. Usermin verwendet dann dasselbe Zertifikat, das die SSL-Website verwendet. Daher werden keine Zertifikatwarnungen generiert, wenn sich Benutzer mit ihrem eigenen Hostnamen anmelden.
