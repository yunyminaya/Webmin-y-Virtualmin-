Admin.domein omleiden naar Usermin?
Wanneer deze optie is ingesteld op Ja (wat standaard zo is) zal er bij iedere nieuwe virtuele server een admin ingang worden toegevoegd aan zijn DNS domeinen met hetzelfde IP adres als de server website. Wanneer iemand op http://admin.hetdomein.com/ komt zal de browser worden omgeleid naar Virtualmin op poort 10000 van hetzelfde domein, zodat hij toegang krijgt tot de administratie web interface.

Indien u een andere URL wilt gebruiken voor deze Virtualmin toegang (bijv. een URL waar u een SSL certificaat voor heeft) kan de omleiding URL worden geconfigureerd in hiernavolgende URL voor admin omleiding veld.
