SSH/Telnet Login
Deze optie bepaald of de domein eigenaar de SSH/Telnet Login Webmin module kan gebruiken. Deze module is een java applet die inlogt bij de server met gebruik van SSH of Telnet. Het werkt niet goed met alle browsers door Java incompatibiliteit. Let op dat ook al schakelt u dit uit het niet voorkomt dat de domein eigenaar SSH toegang heeft tot de server, alleen de toegang tot de SSH/Telnet Login module van Webmin.
