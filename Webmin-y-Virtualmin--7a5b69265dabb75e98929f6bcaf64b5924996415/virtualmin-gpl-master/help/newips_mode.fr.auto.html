Type d'adresse à modifier
Ce champ détermine si le changement d'adresse IP s'appliquera à la véritable adresse IP des serveurs virtuels (utilisée dans la configuration Apache et dans toutes les interfaces virtuelles), ou à l'adresse IP utilisée dans les enregistrements DNS. Ceux-ci peuvent différer pour les systèmes qui se trouvent derrière une passerelle NAT ou un proxy.
