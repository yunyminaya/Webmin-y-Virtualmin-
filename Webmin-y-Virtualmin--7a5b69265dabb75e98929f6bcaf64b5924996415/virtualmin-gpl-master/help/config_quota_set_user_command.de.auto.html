Befehl zum Festlegen des Benutzerkontingents
Wenn externe Kontingentbefehle verwendet werden, wird das in dieser Option festgelegte Programm aufgerufen, wenn Virtualmin die Kontingente für einen Unix-Benutzer festlegen möchte. Die Parameter, mit denen es aufgerufen wird, sind:
  1. Der Unix-Benutzername
  2. Das neue Soft-Kontingent in 1-kB-Blöcken oder 0 für unbegrenzt.
  3. Das neue harte Kontingent in 1-kB-Blöcken oder 0 für unbegrenzt.
