Команда для перечисления пользовательских квот
Когда используются внешние команды квот, программа, установленная в этом параметре, будет вызываться, когда Virtualmin необходимо будет найти существующие пользовательские квоты. Он не вызывается с какими-либо параметрами, но должен выводить одну строку текста на пользователя, каждая строка содержит разделенные пробелами поля. Поля в каждой строке должны быть:
  1. Имя пользователя Unix.
  2. Текущая квота используется в блоках по 1 кБ.
  3. Мягкая квота в блоках по 1 кБ или 0 для неограниченного количества.
  4. Жесткая квота в блоках по 1 кБ или 0 для неограниченного количества.
