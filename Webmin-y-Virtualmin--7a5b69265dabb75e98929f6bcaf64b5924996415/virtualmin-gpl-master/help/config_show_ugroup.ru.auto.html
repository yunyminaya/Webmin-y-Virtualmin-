Разрешить выбор другой группы для администраторов сервера?
Если для этого параметра установлено значение Да, создатель нового виртуального сервера может выбрать другую группу Unix, членом которой должен быть его владелец, вместо новой группы, созданной для виртуального сервера. Это может быть полезно, если вам необходимо, чтобы все учетные записи владельцев серверов в вашей системе были членами определенной группы, но следует использовать с осторожностью, поскольку это может позволить владельцам серверов обходить групповые квоты.
