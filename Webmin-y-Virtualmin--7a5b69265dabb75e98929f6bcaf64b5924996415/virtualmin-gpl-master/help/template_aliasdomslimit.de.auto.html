Begrenzung der Anzahl der Alias-Server
Wenn der Domänenbesitzer über Berechtigungen zum Erstellen von Subservern verfügt, können Sie die Anzahl der Aliaserver begrenzen, die hier erstellt werden können. Beachten Sie, dass dieses Limit nicht effektiv höher sein kann als das Standardlimit für die Anzahl der virtuellen Server , da das Gesamtlimit für virtuelle Server sowohl für Alias- als auch für Nicht-Alias-Domänen gilt. Ein Limit von 0 verhindert die vollständige Erstellung von Aliasdomänen.
