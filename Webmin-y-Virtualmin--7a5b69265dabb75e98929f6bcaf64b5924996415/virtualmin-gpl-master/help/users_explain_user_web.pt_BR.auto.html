Criar usuário do servidor web
Cria um novo usuário de servidor web dentro dos arquivos htpasswd , especificamente para autenticação segura em diretórios web protegidos. Este usuário é distinto das contas Unix, dedicado exclusivamente ao gerenciamento de acesso a diretórios web. Além disso, esse usuário pode ser perfeitamente integrado mesclando-se com uma conta Unix criada usando o mesmo nome de usuário.
