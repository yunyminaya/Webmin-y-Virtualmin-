Можете выбрать имена баз данных?
Если для этого параметра установлено значение Да , владельцу этого виртуального сервера будет разрешено выбрать имя по умолчанию для баз данных для новых создаваемых им подчиненных серверов. Вам следует установить значение «Нет» только в том случае, если вы хотите полностью контролировать имена баз данных в вашей системе.
