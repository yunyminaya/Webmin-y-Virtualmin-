Configuração Vamos criptografar o certificado para o nome do host?
Quando ativado, o Virtualmin tentará configurar automaticamente um certificado Let's Encrypt SSL válido para o nome do host durante a próxima verificação de configuração.

Se for bem sucedido, este certificado SSL será usado como padrão para todos os serviços que não possuem seu próprio certificado SSL configurado.

Se o nome do host do sistema for alterado, ele será automaticamente detectado e atualizado.
