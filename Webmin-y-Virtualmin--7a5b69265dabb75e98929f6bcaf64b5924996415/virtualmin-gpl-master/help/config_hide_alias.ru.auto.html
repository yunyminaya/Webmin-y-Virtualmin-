Включить псевдоним виртуальных серверов в меню?
По умолчанию левое меню в Virtualmin будет включать все виртуальные серверы, включая псевдонимы. Однако если для этого параметра установлено значение Нет , псевдонимы серверов будут исключены из списка. Это может быть полезно, если в вашей системе размещено большое количество псевдонимов, например, для припаркованных доменов. Вы можете найти эти псевдонимы и управлять ими, используя ссылки со страницы «Редактировать виртуальный сервер» для их родителей.
