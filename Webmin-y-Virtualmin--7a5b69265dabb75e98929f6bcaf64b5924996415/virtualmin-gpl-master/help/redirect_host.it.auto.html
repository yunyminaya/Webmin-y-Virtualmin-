Limita il reindirizzamento al nome host
Per impostazione predefinita, i reindirizzamenti verranno applicati indipendentemente dal nome host dell'URL utilizzato per accedere al sito Web. Tuttavia, questo campo può essere utilizzato per limitare il reindirizzamento a un host specifico. Ciò può essere utile, ad esempio, per reindirizzare www.domain.com a domain.com o per modificare il reindirizzamento per un dominio alias specifico.
