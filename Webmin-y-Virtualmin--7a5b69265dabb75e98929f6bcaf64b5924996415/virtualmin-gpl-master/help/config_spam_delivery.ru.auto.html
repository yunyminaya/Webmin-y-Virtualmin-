Доставка по умолчанию для спама
Если SpamAssassin включен для учетных записей пользователей, сообщения, которые процессор определил как спам, могут быть доставлены (с необязательной маркировкой, настроенной в модуле SpamAssassin), удалены или доставлены в специальную папку с именем Spam в почтовом каталоге пользователей. В то время как ложные срабатывания крайне редки для SpamAssassin в его конфигурации по умолчанию, может быть целесообразно сохранять сообщения в течение некоторого периода времени и разрешать пользователям проверять папку спама с помощью веб-почты Usermin или папок IMAP (если IMAP доступен для ваших пользователей) , По умолчанию спам помещается в папку Spam Maildir.
