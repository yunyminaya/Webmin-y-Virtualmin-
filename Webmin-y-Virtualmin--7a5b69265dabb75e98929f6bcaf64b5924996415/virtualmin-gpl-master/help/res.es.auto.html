Límites de recursos
Esta página le permite definir límites en el uso de CPU y memoria que se aplican a este servidor virtual y a todos los sub-servidores. Los límites se aplican a ambos scripts PHP y los comandos se ejecutan a través de un inicio de sesión SSH. Se pueden usar para evitar que cualquier propietario de un dominio sobrecargue el sistema con una bomba tenedor o comandos que usan una cantidad excesiva de RAM.

Los tipos de límite disponibles son:

Numero maximo de procesos
Intentar lanzar más procesos que se ejecutan como usuario de Unix del servidor que el límite establecido aquí fallará. Esto también se aplica a los scripts PHP concurrentes.

Tamaño máximo por proceso
Cualquier proceso único que intente asignar más memoria que este límite terminará. Sin embargo, la suma total de todos los procesos que posee el servidor aún puede exceder el límite.

Máximo tiempo de CPU por proceso
Cualquier proceso cuyo tiempo dedicado realmente a usar la CPU (sin esperar IO o dormir) exceda este límite se eliminará.
