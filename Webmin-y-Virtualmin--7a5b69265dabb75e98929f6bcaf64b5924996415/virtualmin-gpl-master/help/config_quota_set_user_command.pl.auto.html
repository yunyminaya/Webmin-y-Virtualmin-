Polecenie ustawienia przydziału użytkownika
Gdy używane są zewnętrzne komendy przydziału, program ustawiony w tej opcji zostanie wywołany, gdy Virtualmin będzie chciał ustawić przydziały dla użytkownika systemu Unix. Parametry, które są wywoływane, to:
  1. Nazwa użytkownika systemu Unix
  2. Nowa miękka kwota w blokach 1 kB lub 0 dla nieograniczonej.
  3. Nowa twarda kwota w blokach 1 kB lub 0 dla nieograniczonej.
