Groupe secondaire pour les utilisateurs de messagerie
Si cette option est définie, vous devez entrer le nom d'un groupe Unix auquel seront ajoutés tous les utilisateurs disposant d'un accès e-mail pour les serveurs virtuels sur ce modèle. Si le groupe que vous spécifiez n'existe pas, il sera automatiquement créé si nécessaire.

Cette fonctionnalité est très utile en conjonction avec Usermin, qui peut limiter les modules visibles aux utilisateurs qui sont membres d'un groupe. Vous pouvez donc rendre le module Lire le courrier visible uniquement aux membres du groupe que vous définissez ici.
