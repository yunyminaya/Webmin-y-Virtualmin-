¿Se pueden crear varios servidores desde un archivo por lotes?
La página Batch Create Servers de Virtualmin permite a los usuarios con permisos para agregar servidores crear varios a la vez desde un archivo de texto. Dado que esto podría resultar en la creación rápida de una gran cantidad de dominios, es posible que desee evitar que los administradores del servidor utilicen esta función. Para hacerlo, cambie esta configuración a No ; esto restringirá su uso solo por el administrador maestro y los revendedores.
