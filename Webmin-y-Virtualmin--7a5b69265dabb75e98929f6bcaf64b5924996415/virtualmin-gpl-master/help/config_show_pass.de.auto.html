Können Postfach- und virtuelle Serverkennwörter angezeigt werden?
Wenn diese Option auf " Ja" gesetzt ist , können Serveradministratoren die aktuellen Kennwörter für E-Mail- / FTP-Benutzer und ihren eigenen virtuellen Server anzeigen, wenn sie Virtualmin bekannt sind. Möglicherweise möchten Sie Nein auswählen, um Postfachkennwörter auszublenden, insbesondere solche, die von Postfachbenutzern ausgewählt wurden.

Unabhängig von dieser Einstellung können der Hauptadministrator und die Wiederverkäufer immer die Kennwörter für Postfächer und virtuelle Server anzeigen.
