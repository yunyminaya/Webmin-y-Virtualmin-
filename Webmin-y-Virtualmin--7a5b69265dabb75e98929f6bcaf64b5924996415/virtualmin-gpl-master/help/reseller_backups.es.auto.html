¿Se pueden hacer copias de seguridad de servidores virtuales?
Este campo determina si el revendedor puede usar las funciones de respaldo de Virtualmin. Las opciones disponibles son:

Programar y hacer copias de seguridad
El revendedor puede programar copias de seguridad que se ejecutan regularmente de forma automática y utilizar la interfaz web para ejecutar copias de seguridad manualmente.
Hacer copias de seguridad solamente
Las copias de seguridad solo pueden ejecutarse manualmente utilizando la interfaz web.
Ninguno
El revendedor no tiene acceso a las funciones de respaldo.
