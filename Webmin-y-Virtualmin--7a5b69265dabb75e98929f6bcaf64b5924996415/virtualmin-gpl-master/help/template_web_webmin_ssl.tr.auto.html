Webmin'i aynı SSL sertifikasını kullanacak şekilde yapılandırılsın mı?
Bu şablon yalnızca SSL özellikli web siteleri için kullanılacaksa, bu seçeneği etkinleştirebilirsiniz. Webmin daha sonra SSL web sitesinin kullandığı sertifikayı kullanır, böylece kullanıcılar kendi ana bilgisayar adlarını kullanarak oturum açtıklarında sertifika uyarıları oluşturmazlar.
