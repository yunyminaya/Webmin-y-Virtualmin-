Aktion, wenn das Ziel verwendet wird
Diese Option bestimmt, was passiert, wenn bereits eine Virtualmin-Sicherung auf demselben Ziel ausgeführt wird. Mit der Option Diese Sicherung abbrechen kann die ausgeführte Sicherung abgeschlossen werden, und diese schlägt sofort fehl. Wenn Sie das laufende Backup beenden, wird das bereits ausgeführte Backup beendet, sodass dieses Backup gestartet werden kann. Dies kann nützlich sein, um zu verhindern, dass ein feststeckendes Backup alle anderen blockiert, die darauf folgen würden.
