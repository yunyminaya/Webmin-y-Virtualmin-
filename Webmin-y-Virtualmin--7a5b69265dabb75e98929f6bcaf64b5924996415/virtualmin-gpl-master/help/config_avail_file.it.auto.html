File Manager (solo directory home)
Il File Manager è un'applet Java avanzata che offre una vista a doppio riquadro del file system del server. È una delle funzionalità più potenti di Webmin, che consente anche agli utenti inesperti di gestire file, directory e permessi, modificare file di testo, caricare e scaricare file e altro ancora. Per impostazione predefinita, il proprietario del dominio potrà accedere solo alla propria directory e alle sue sottodirectory.
