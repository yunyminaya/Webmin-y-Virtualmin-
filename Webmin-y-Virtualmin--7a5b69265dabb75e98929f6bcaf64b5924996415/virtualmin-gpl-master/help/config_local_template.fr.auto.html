Fichier modèle pour le courrier électronique aux nouveaux utilisateurs locaux
Les utilisateurs locaux sont des utilisateurs qui n'appartiennent pas à un domaine Virtualmin, par exemple, s'ils ne disposent que du courrier et / ou du service Web dans le domaine natif du système. Vous pouvez envoyer l'e-mail du nouvel utilisateur par défaut, ou vous pouvez spécifier un fichier de modèle différent. Les variables de modèle Virtualmin standard sont disponibles dans le modèle.
