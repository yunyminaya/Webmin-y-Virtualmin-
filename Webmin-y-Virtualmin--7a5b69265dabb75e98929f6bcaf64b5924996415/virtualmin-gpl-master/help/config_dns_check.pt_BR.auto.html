Verifique resolv.conf para este sistema?
Quando estiver definido como Sim (como é padrão), o Virtualmin verificará o arquivo de configuração do cliente DNS do sistema /etc/resolv.conf para garantir que esteja configurado para usar o servidor DNS local na resolução de nomes. Isso é necessário, pois, sem ele, os domínios DNS criados por você podem não ser resolvidos pelo sistema executando o Virtualmin até que sejam totalmente registrados por um registrador DNS.

Se você tiver certeza de que isso não é necessário (talvez porque você tenha uma configuração automática do servidor DNS secundário), isso poderá ser alterado para Não para evitar erros espúrios na verificação da configuração.
