¿Crear clave DNSSEC y firmar nuevos dominios?
Cuando esta opción está habilitada, Virtualmin creará una clave DNSSEC para cualquier dominio DNS nuevo del servidor virtual y la usará para firmar sus registros. Cada vez que se cambian los registros usando Webmin o Virtualmin, las firmas se actualizarán automáticamente para que coincidan.

Para que DNSSEC sea realmente útil, la clave pública para cualquier dominio nuevo debe enviarse al registrador o al propietario del dominio DNS principal.
