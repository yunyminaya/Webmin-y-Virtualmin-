Clonar servidor virtual
Esta página permite duplicar um servidor virtual existente com um novo nome de domínio. Qualquer conteúdo da web, registros DNS, caixas de correio, aliases de correio, bancos de dados e outras configurações associadas ao domínio original serão duplicados, sempre que possível. No entanto, o conteúdo da web não será alterado de forma alguma, portanto, se o servidor virtual contiver um aplicativo da web que tenha sido configurado para usar os bancos de dados ou o diretório inicial do servidor antigo, ele precisará ser reconfigurado manualmente após a clonagem.

Ao duplicar um servidor virtual de nível superior, você também deve inserir um novo nome de usuário e senha para possuir o novo domínio. Caso o servidor possua um endereço IP privado, um novo IP será automaticamente alocado para o clone.

O processo de clonagem copiará todos os bancos de dados associados ao domínio original com novos nomes e duplicará seu conteúdo. No entanto, ele pode não atualizar todos os webapps instalados no domínio original que estão configurados para usar os bancos de dados originais. Isso deve ser feito manualmente.
