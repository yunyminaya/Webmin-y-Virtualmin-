Website Redirects
A redirect maps some URL path like /foo to either a different URL, or a different directory on the same virtual server. This can be used to provide more friendly URL paths on your website, or to cope with the movement of web pages to new locations.

This page lists all existing redirects, and allows you to create a new one by clicking on the Add a new website redirect link. To edit an existing redirect, just click on it's source URL path. To remove multiple redirects, check the boxes next to them and click the Delete button at the bottom of the page.
