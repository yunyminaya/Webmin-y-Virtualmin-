Filtraggio della posta per server SpamAssassin?
Abilita o disabilita la possibilità di abilitare la configurazione di SpamAssassin in base al dominio. Con questa opzione abilitata, è possibile disabilitare il filtro antispam per alcuni domini abilitandolo per altri. È possibile disabilitarlo, se si imporrà il filtro antispam per tutti gli utenti o se si sta utilizzando un sistema di prevenzione dello spam alternativo su base di sistema. Si noti che la configurazione a livello di server non è gestita da Virtualmin. Sarà necessario abilitarlo utilizzando il modulo Webmin SpamAssassin, se si preferisce il filtro antispam a livello di server.
