Virtuellen Server übertragen
Diese Seite kann verwendet werden, um einen virtuellen Server (und alle Subserver oder Aliase) an ein anderes System zu senden, auf dem Virtualmin ausgeführt wird. Der Übertragungsprozess verwendet die Sicherungs- und Wiederherstellungsfunktionen von Virtualmin und umfasst alle Inhalte des Basisverzeichnisses, Datenbanken, DNS-Einträge und andere Konfigurationen.

Der Hostname oder die IP - Adresse des Zielsystems muss in der Hostname des Zielsystems Feld eingegeben werden, und das Root - Passwort für Zielsystem - Feld sollte ausgefüllt werden , es sei denn Sie haben ein Passwort weniger SSH - Authentifizierung Setup.

Standardmäßig wird die Domain einfach auf das neue System kopiert. Das Feld Aktion nach Übertragung kann jedoch verwendet werden, um es stattdessen zu verschieben, indem der Server aus dem Quellsystem gelöscht wird. Dies ist erforderlich, wenn sowohl die Quelle als auch das Ziel einen DNS-Cluster-Server gemeinsam nutzen oder sekundäre Nameserver für einander sind.
