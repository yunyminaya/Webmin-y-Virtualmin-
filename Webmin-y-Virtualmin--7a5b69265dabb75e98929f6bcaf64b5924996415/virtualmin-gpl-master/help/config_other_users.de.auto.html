Andere Module benachrichtigen, wenn Serverbenutzer aktualisiert werden?
Webmin- und Virtualmin-Module können Funktionen bereitstellen, die ausgeführt werden, wenn ein Benutzer geändert wird. Diese Funktion kann von Modulentwicklern verwendet werden, um Kennwörter an verschiedenen Standorten zu synchronisieren, Benutzer von anderen Diensten zu entfernen oder auf andere Weise Daten über den Benutzer in anderen Modulen zu ändern. Wenn diese Option deaktiviert ist, werden andere Module nicht benachrichtigt und Hinzufügungen, Subtraktionen und Änderungen werden nicht auf andere Module repliziert. Diese Option gilt nur für Serverbenutzer, bei denen es sich um Benutzer handelt, die eine oder mehrere Domänen oder virtuelle Server besitzen.
