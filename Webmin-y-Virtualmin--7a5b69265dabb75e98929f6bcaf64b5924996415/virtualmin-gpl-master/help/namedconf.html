Additional named.conf directives for new zones
When a new BIND DNS zone is created, the directives entered into this text box will be added to the zone's section in the named.conf configuration file. For example, you could enter the following to limit zone transfers to a selected IP addresses : 
allow-transfer { 
  1.2.3.4;
  1.2.3.10;
};
This feature should only be used by those already familiar with the BIND configuration format, as a mistake could cause your entire DNS server to stop working.
