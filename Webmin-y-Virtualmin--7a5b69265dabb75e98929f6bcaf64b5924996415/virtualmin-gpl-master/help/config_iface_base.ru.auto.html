Базовый номер для виртуальных интерфейсов
По умолчанию Virtualmin будет использовать eth0:0 для первого создаваемого виртуального IP-интерфейса, затем eth0:1 и так далее. Тем не менее, это поле может быть использовано для изменения того, где начинается номер интерфейса после:. Это может быть полезно, если вы хотите зарезервировать некоторые из нижних номеров для ручного назначения, но в большинстве случаев их не нужно менять.
