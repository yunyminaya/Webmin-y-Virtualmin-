Уведомлять другие модули при обновлении пользователей почтовых ящиков?
Модули Webmin и Virtualmin могут предоставлять функции для запуска в случае смены пользователя. Эта функция может использоваться разработчиками модулей для синхронизации паролей в разных местах, удаления пользователей из других служб или иного изменения данных о пользователе в других модулях. Если эта опция отключена, другие модули не будут уведомлены, а дополнения, вычитания и изменения не будут реплицироваться на другие модули. Этот параметр применяется только к пользователям почтовых ящиков, которые имеют учетные записи почтовых ящиков / FTP / оболочки в виртуальных доменах.
