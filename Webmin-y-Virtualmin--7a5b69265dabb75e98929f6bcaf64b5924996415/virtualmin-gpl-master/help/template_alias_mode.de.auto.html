Ein Domain-Alias, auch als Domain-Parking oder Domain-Weiterleitung bezeichnet, ist eine neue Domain, die Webanfragen und E-Mails einfach an die primäre Domain weiterleitet, deren Alias sie ist. Es stehen verschiedene Arten von Alias zur Verfügung. Derzeit kann der Webdatenverkehr über eine Reihe von ProxyPass-Regeln, eine Redirect-Regel, eine RedirectMatch-Regel oder einfach durch Hinzufügen eines ServerAlias zur übergeordneten Domäne an die primäre Domäne geleitet werden.
