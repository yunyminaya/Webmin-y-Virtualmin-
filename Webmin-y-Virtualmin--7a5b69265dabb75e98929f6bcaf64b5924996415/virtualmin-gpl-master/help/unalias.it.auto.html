Converti Alias Server
Questa pagina può essere utilizzata per convertire un server virtuale alias in un sub-server. Non servirà più le pagine Web del vecchio server di destinazione, ma può invece avere il proprio contenuto Web. Eventuali alias di posta esistenti verranno conservati, ad eccezione del catchall che inoltra la posta al vecchio server di destinazione. I record DNS non verranno effettuati.
