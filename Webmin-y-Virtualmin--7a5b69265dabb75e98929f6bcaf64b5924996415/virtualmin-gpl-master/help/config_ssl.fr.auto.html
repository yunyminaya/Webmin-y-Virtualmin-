Configuration du site Web SSL activée?
S'il est activé, un site Web SSL sera créé pour chaque nouveau domaine. S'il est défini sur "Oui, mais pas par défaut", il sera possible de choisir de créer un site Web SSL lors de la création du domaine. Notez que les sites Web SSL nécessitent une adresse IP pour eux-mêmes, elle n'est donc pas activée pour tous les sites par défaut. De nombreux sites Web n'auront pas besoin de sécurité SSL et peuvent donc partager une seule adresse IP avec des centaines ou des milliers d'autres sites Web.
