¿Redirigir webmail.domain a Usermin?
Cuando este campo se establece en Sí (como lo está por defecto), cualquier servidor virtual nuevo tendrá una entrada de webmail agregada a sus dominios DNS, con la misma dirección IP que el sitio web del servidor. Cuando se accede a http://webmail.thedomain.com/ , el navegador será redirigido a Usermin en el mismo dominio, para acceder al correo electrónico basado en la web.

Si prefiere usar alguna interfaz de correo web que no sea Usermin (como Squirrelmail o Roundcube) o desea usar una URL diferente para el acceso de Usermin, la URL de redireccionamiento se puede configurar en la siguiente URL para el campo de redirección de correo web .
