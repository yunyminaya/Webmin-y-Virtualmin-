¿Permitir el uso de servidor incluye?
Las inclusiones del lado del servidor son una función de servidor web que permite usar directivas macro simples en archivos HTML. Se pueden usar para incluir otros archivos HTML, establecer y mostrar el contenido de variables y controlar qué contenido aparece en la página con sentencias if . Ofrecen una forma más sencilla de crear páginas dinámicas que un lenguaje de script completo como PHP o Perl.

Para obtener más información, consulte la documentación de Apache en http://httpd.apache.org/docs/current/howto/ssi.html
