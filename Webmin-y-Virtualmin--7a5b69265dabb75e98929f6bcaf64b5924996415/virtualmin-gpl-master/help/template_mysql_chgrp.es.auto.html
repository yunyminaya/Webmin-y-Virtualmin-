¿Establecer la propiedad grupal de los archivos de base de datos MySQL?
Cuando esta opción está habilitada, cualquier nueva base de datos MySQL tendrá la propiedad del grupo establecida en sus archivos de datos (generalmente bajo /var/lib/mysql ) para que se incluyan en los límites de cuota de Unix de sus servidores virtuales. Si no está habilitado, el uso del disco de las bases de datos no estará restringido y será posible que un servidor virtual exceda su asignación de disco permitida.
