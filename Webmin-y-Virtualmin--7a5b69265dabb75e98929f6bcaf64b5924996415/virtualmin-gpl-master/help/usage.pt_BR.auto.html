Uso de disco
Esta página exibe o uso do disco para um servidor virtual, dividido em cinco seções com guias:
Visão geral
Esta seção mostra a cota de disco alocada para o servidor virtual, o espaço atualmente usado pelos arquivos pertencentes ao servidor e o tamanho dos bancos de dados pertencentes.

Diretório inicial
Mostra o espaço usado pelos 10 principais diretórios na casa do servidor. Para cada um, o tamanho da cota é o tamanho dos arquivos pertencentes ao grupo Unix do servidor, enquanto o tamanho inclui arquivos pertencentes a outros usuários. O último pode ser maior para diretórios como logs quando tiver arquivos pertencentes à root . O diretório homes que contém usuários de email / FTP não está incluído, como é explicado na próxima seção.

Por usuário
Esta seção mostra o espaço em disco usado pelos 10 principais usuários de caixas de correio nesse domínio e em qualquer sub-servidor. O uso é baseado em arquivos pertencentes ao usuário Unix para cada caixa de correio.

Por sub-servidor
Mostra o espaço em disco usado por cada sub-servidor, com base nos arquivos em seus diretórios pessoais. O espaço usado por qualquer usuário de caixa de correio nesses servidores também está incluído.

Por banco de dados
Mostra o espaço em disco usado por cada banco de dados MySQL ou PostgreSQL neste servidor virtual ou em qualquer sub-servidor. Somente os 10 principais bancos de dados são mostrados.
