Webserver-Zugriffsprotokolldatei
Dieses Feld legt die Datei fest, in die Apache-Zugriffsprotokolle für diesen virtuellen Server geschrieben werden. Jede HTTP-Anfrage, die von der Domain dieser Website bedient wird, wird als separate Zeile in diese Datei geschrieben. Sie sollte nur geändert werden, wenn Sie die Protokollierung in ein anderes Verzeichnis verschieben, z. B. /var/log/virtualmin oder das /var/log/virtualmin des Servers.
