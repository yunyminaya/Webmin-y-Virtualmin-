Immer die Ausgabe von Pre- und Post-Befehlen anzeigen?
Standardmäßig zeigt Virtualmin die Ausgabe von Änderungsbefehlen vor oder nach dem virtuellen Server (siehe oben) nur an, wenn sie fehlschlagen. Wenn dies jedoch in Ja geändert wird, wird immer die Ausgabe angezeigt, die Nachrichten für den Benutzer enthalten kann.
