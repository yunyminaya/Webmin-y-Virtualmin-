Interfaz de red para direcciones virtuales.
Si su servidor tiene múltiples interfaces, puede elegir qué interfaz usar para sus dominios virtuales. Si los hosts virtuales SSL o FTP están habilitados, se requerirá una nueva IP para cada dominio en el que la función esté habilitada. A menos que se configure lo contrario durante la creación del dominio, las nuevas direcciones se crearán en la interfaz especificada aquí.
