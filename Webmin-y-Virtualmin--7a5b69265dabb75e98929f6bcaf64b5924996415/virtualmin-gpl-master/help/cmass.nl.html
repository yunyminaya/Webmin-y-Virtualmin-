Veel servers tegelijk aanmaken
Met deze pagina kunt u in 1 keer meerdere virtuele servers aanmaken via een tekst file. Deze file moet een lijst zijn met 1 server per regel en iedere regel moet in het volgende formaat zijn :

domein:omschrijving:wachtwoord:[gebruikersnaam]:[hoofd-domein]:[ip-adres]:[alias-domein]

De betekenis van de velden zijn :

Domein
De domein naam voor de aan te maken virtuele server, zoals foo.com.
Omschrijving
Een menselijk leesbare omschrijving, zoals Joe's Website.
Wachtwoord
Het wachtwoord voor de server's administratie login. Dit is alleen nodig voor hoofd servers.
Gebruikersnaam
De administratie gebruikersnaam voor de server. Indien dit niet word ingevuld, word er automatisch een gebruikersnaam geselecteerd.
Hoofd-domein
De domein naam van de hoofd server waaronder deze server word aangemaakt. Indien dit niet word ingevuld, zal de nieuwe server zelf worden aangemaakt als een hoofdserver.
ip-adres
Voor virtuele servers die geen eigen prive IP nodig hebben, Dit veld kan leeg gelaten worden. Anders moet u een geldig ongebruikt IP adres invullen, of het woord allocate om Virtualmin er 1 te laten selecteren voor u (indien dit geconfigureerd is).
alias-domein
De domein naam van de virtuele server waar dit een alias voor moet worden. Indien u dit niet invult word de nieuwe server een top-niveau of een sub-server.

Quota, bandbreedte, en andere limieten voor de aangemaakte servers worden ingesteld gebaseerd op de template die u heeft geselecteerd in dit formulier. De opties van de aangemaakte servers worden bepaald door de selectievakjes, net zoals op de normale enkele-server aanmaak pagina.

Indien uw systeem wederverkoop accounts gebruikt kunt u selecteren dat alle virtuele servers worden aangemaakt onder de eigenaarschap van een wederverkoper. Deze optie is alleen beschikbaar voor de hoofd administrator.
