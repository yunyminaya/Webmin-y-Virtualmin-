Crear alias por lotes
Esta página le permite crear múltiples alias de correo electrónico a la vez desde un archivo de texto. Este archivo debe incluir un alias por línea, y cada línea debe tener el siguiente formato:

name:[comment]:destination1:[destination2:...]

Los significados de los campos son:

nombre
El nombre del buzón del alias, con como ventas . Para crear un alias que coincida con todo el correo al dominio, ingrese * aquí.
comentario
Este campo opcional se puede usar para ingresar un comentario descriptivo para el alias, como Ventas corporativas .
destino
Se pueden ingresar uno o más destinos, generalmente en forma de direcciones de correo electrónico para reenviar el correo, como joe@yourdomain.com . También se admiten los siguientes destinos especiales:
  • bounce
    Crea un alias que devuelve el correo electrónico al remitente.
  • autoreply Your autoreply message
    Envía una respuesta automática al remitente, que contiene el texto después de la palabra autoreply .
  • local username
    Entrega correo electrónico directamente al buzón del usuario especificado.
