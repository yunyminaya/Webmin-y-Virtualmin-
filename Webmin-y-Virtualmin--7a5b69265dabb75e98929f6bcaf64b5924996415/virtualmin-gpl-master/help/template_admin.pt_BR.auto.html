Redirecionar admin.domain para Usermin?
Quando esse campo é definido como Sim (como é o padrão), qualquer novo servidor virtual terá uma entrada de admin adicionada aos seus domínios DNS, com o mesmo endereço IP do site do servidor. Quando http://admin.thedomain.com/ for acessado, o navegador será redirecionado para o Virtualmin na porta 10000 do mesmo domínio, para acessar a interface da web de administração.

Se você deseja usar um URL diferente para acesso ao Virtualmin (como o certificado para o qual você possui um certificado SSL), o URL de redirecionamento pode ser configurado no campo URL seguinte para redirecionamento de administrador .
