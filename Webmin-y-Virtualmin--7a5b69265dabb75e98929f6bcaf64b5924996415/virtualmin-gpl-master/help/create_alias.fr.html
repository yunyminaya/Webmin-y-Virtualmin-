Créer un serveur alias
Ce formulaire vous permet de créer un nouveau serveur d'alias, qui est essentiellement un autre nom pour un serveur existant. Alias ne disposent pas de leur propre contenu du site Web, et partagent tous les quotas et les limites avec le serveur cible. Bande passante utilisée par le serveur d'alias est comptée dans le total de la cible.

Lorsque vous créez le serveur, vous devez entrer au moins le nom de domaine et une description. Si plusieurs modèles de serveur sont définies sur le système, vous pouvez sélectionner une dans le menu. Pour sélectionner les options dans le serveur alias qui sont transmis au domaine cible, utilisez les cases à cocher dans la deuxième partie de la section Fonctions activées. Seulement DNS, site Web et e-mail aliasing sont disponibles.
