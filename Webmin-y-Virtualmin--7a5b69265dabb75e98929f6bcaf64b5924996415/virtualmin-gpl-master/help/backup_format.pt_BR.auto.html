Formato de backup
Este campo determina se o backup é gravado como um único arquivo ou se um arquivo é criado no destino para cada servidor virtual selecionado, denominado domínio .tar.gz. A opção final também criará um arquivo por domínio, mas em um novo formato suportado pelo Virtualmin 2.86 e superior, que coloca todas as informações sobre cada servidor no diretório inicial. Esse formato é superior porque não requer que o Virtualmin crie arquivos temporários no diretório / tmp e economize espaço em disco.
