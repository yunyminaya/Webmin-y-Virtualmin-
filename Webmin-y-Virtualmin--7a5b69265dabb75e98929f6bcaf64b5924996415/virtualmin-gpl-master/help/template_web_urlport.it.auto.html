Numero di porta esterno per host virtuali
Questa impostazione può essere utilizzata per modificare il numero di porta utilizzata da Virtualmin negli URL che fanno riferimento a domini creati da questo modello. Può essere utile se il tuo sistema ha un proxy o una cache davanti ad esso in ascolto sulla porta 80 e Virtualmin è stato configurato per creare host virtuali Apache su una porta diversa.
