Chcete nakonfigurovat Usermin tak, aby používal stejný certifikát SSL?
Pokud bude tato šablona použita výhradně pro webové stránky podporující SSL, můžete tuto možnost povolit. Usermin poté použije stejný certifikát, jaký používá web SSL, takže nebude generovat varování certifikátů, když se uživatelé přihlásí pomocí svého vlastního názvu hostitele.
