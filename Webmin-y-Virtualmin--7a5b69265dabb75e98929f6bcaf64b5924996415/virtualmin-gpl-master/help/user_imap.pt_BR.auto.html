Login IMAP/FTP
Este campo mostra o nome de usuário que o usuário da caixa de correio precisará ao efetuar login via IMAP para recuperar e-mails, SMTP para enviar e-mails ou FTP para gerenciar arquivos. Ele difere da primeira parte do endereço de e-mail, pois parte ou todo o nome do domínio é anexado ou prefixado.
