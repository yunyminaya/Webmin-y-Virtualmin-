Log file rotatie opzetten?
Als deze optie is aangezet zullen de virtuele server web log files op wekelijkse basis worden geroteerd, om te voorkomen dat er teveel harddisk ruimte word verbruikt. Iedere week word de huidige log file gekopieerd, ingekort en gecomprimeerd. Standaard worden geroteerde log's vijf weken bewaard voordat ze worden verwijdert.
