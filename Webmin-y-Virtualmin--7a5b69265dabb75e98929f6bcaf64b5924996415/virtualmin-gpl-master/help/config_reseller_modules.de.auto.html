Zusätzliche Module für Wiederverkäufer
Mit dieser Option können Sie Reseller-Benutzern zusätzliche Webmin-Funktionen zur Verfügung stellen, die über das Virtualmin Virtual Servers- Modul hinausgehen. Seien Sie mit dieser Option sehr vorsichtig, da die meisten Webmin-Module standardmäßig eine gefährlich vollständige Kontrolle über die von ihnen verwalteten Dienste bieten. Es gibt jedoch Fälle, in denen dies sehr nützlich sein kann. Wenn Sie beispielsweise über ein benutzerdefiniertes Modul verfügen, das einen für Ihre Umgebung spezifischen Dienst bereitstellt, ist diese Option möglicherweise sehr hilfreich. Darüber hinaus können viele Module standardmäßig mit sehr striktem Zugriff konfiguriert werden, oder Sie können ein Modul klonen und es mit einem ganz bestimmten Satz von Funktionen konfigurieren, um die Sicherheit für alle Wiederverkäufer zu gewährleisten. Mit dem Modul Webmin-Benutzer können Sie einigen Reseller-Benutzern selektiv zusätzliche Berechtigungen gewähren.
