Chemin d'accès complet à la commande aws pour les opérations S3
Si la commande aws est installée sur votre système et que ce champ est défini sur le chemin d'accès complet à celui-ci (comme /usr/bin/aws ), Virtualmin l'appellera pour effectuer des opérations S3 pour les sauvegardes telles que les fichiers téléchargés et téléchargés. Parce que cette commande a été développée par Amazon, elle est plus fiable que l'API S3 intégrée à Virtualmin.
