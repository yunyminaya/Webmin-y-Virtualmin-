Emplacements supplémentaires pour les commandes PHP
Virtualmin détectera automatiquement les principales versions de PHP installées sur votre système, tant qu'elles se trouvent dans des répertoires du chemin de recherche. Cependant, ce champ peut être utilisé pour entrer les chemins d'accès à des exécutables PHP supplémentaires, ce qui peut être utile si vous souhaitez exécuter une version supérieure à celle qui est installée par défaut.
