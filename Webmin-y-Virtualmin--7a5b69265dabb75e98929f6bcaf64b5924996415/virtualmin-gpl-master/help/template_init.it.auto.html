Modello inizialmente selezionato per server di livello superiore?
Quando è impostato su Sì , il menu modello durante la creazione di un nuovo server virtuale di livello superiore avrà questo modello selezionato per impostazione predefinita. Verrà utilizzato anche durante la creazione di un server dalla riga di comando se nessun modello viene esplicitamente specificato con il parametro --template . Solo un modello può essere impostato su Sì alla volta.
