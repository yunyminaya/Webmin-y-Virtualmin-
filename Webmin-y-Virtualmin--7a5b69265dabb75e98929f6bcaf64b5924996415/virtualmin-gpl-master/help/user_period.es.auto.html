Tiempo mínimo entre respuestas automáticas
Cuando las respuestas automáticas están habilitadas, esta opción se puede usar para limitar la velocidad a la que se envían las respuestas a cada remitente. Si llega un segundo mensaje del mismo remitente en el período ingresado aquí, no se enviará una respuesta automática. ¡Esto es útil para evitar bucles de correo en los que dos autorespondedores se envían por correo electrónico!
