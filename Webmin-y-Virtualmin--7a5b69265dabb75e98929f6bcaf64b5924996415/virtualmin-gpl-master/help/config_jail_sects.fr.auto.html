Sections Jailkit supplémentaires à copier
Par défaut, Virtualmin copiera dans les prisons nouvelles ou mises à jour uniquement les fichiers définis dans quelques sections de configuration critiques de Jailkit. Cependant, vous pouvez spécifier des sections supplémentaires à copier (définies dans /etc/jailkit/jk_init.ini ) en les listant ici.
