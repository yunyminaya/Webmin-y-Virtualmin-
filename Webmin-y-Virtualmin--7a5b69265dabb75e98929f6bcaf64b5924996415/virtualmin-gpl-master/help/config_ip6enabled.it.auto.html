Utilizzare l'indirizzo IPv6 predefinito per i nuovi server virtuali?
Per impostazione predefinita, Virtualmin aggiungerà voci DNS e una configurazione del server web ai nuovi server virtuali per utilizzare l'indirizzo IPv6 primario del sistema. Al momento della creazione, un utente può comunque scegliere di utilizzare un indirizzo IPv6 privato o nessuno.
