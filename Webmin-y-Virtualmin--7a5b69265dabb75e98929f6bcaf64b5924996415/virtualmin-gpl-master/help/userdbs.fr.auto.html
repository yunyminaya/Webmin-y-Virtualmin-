Autoriser l'accès aux bases de données
Ce champ vous permet de sélectionner les bases de données appartenant à ce serveur virtuel auxquelles l'utilisateur peut également se connecter. Pour ajouter une base de données, sélectionnez-la dans la liste de gauche et cliquez sur la flèche ->.

Le login et le mot de passe de l'utilisateur seront les mêmes que ceux qu'il utiliserait pour FTP ou IMAP. Par défaut, il aura un accès complet à toutes les tables des bases de données sélectionnées.

MySQL ne prend en charge que les noms d'utilisateur jusqu'à 16 caractères de long, donc pour un utilisateur avec un nom de compte long, seuls les 16 premiers caractères sont nécessaires lors de la connexion à la base de données.
