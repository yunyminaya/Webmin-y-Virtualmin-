Adresse IP externe
Si votre serveur Virtualmin est sur un réseau privé interne, et tout le trafic HTTP est transmis à ce via un pare-feu ou un proxy, vous devez définir ce domaine à l'adresse IP par laquelle il est accessible de l'extérieur, comme celui de votre pare-feu ou un routeur. Lorsque cette option est activée, l'adresse indiquée sera utilisée pour tous les enregistrements DNS qui contiennent normalement l'adresse réelle, tels que www, mail et ftp.

Par défaut, les enregistrements DNS créés automatiquement pour ce serveur virtuel contiendront l'adresse IP qui lui est attribué par Virtualmin. Cependant, dans certains cas, cette adresse ne peut pas réellement être celui à lequel vous voulez connecter Web et clients de courrier.
