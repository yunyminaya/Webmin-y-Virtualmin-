Campos Personalizados
Los campos personalizados permiten al administrador incluir datos adicionales sobre cuentas de dominio. Los campos se mostrarán al editar el dominio. Se pueden usar para casi cualquier tipo de datos. Por ejemplo, puede ser útil tener información de contacto adicional adjunta a la cuenta de dominio, o un campo de notas utilizado para explicar por qué un dominio está deshabilitado o tiene alguna característica adicional o límite más allá de otros dominios creados con la misma plantilla. Si está desarrollando herramientas para interactuar con Virtualmin, estos campos pueden ser útiles para otros fines.

Para cada campo debe ingresar lo siguiente:

Nombre del campo
Un código corto para este campo, que se utiliza para referirse al campo en plantillas y scripts posteriores a la creación. Cada campo debe tener un nombre de campo único.

Etiqueta
El texto que se mostrará al usuario cuando ingrese el valor para este campo. Si la etiqueta contiene un; , todo lo que esté después se usará como información sobre herramientas que aparece al pasar el mouse sobre el campo en la página Editar servidor virtual.

Tipo
Esto determina qué tipos de valores se pueden ingresar para este campo. La mayoría se explica por sí misma, excepto la opción de menú . Cuando se selecciona esto, debe ingresar la ruta a un archivo en el sistema Virtualmin en el cuadro de texto adyacente. Cada línea en el archivo debe contener un código y texto visualizado para una entrada en el menú para este campo personalizado, separados por espacios.

En la lista de servidores?
Si se establece en Sí , se mostrará una columna para este campo personalizado en la página Listar servidores virtuales .
