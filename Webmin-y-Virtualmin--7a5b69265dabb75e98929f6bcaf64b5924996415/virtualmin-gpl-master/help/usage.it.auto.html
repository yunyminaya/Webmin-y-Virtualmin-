Uso del disco
Questa pagina mostra l'utilizzo del disco per un server virtuale, suddiviso in cinque sezioni a schede:
Panoramica
Questa sezione mostra la quota del disco allocata al server virtuale, lo spazio attualmente utilizzato dai file di proprietà del server e la dimensione dei database di proprietà.

Home directory
Mostra lo spazio utilizzato dalle prime 10 directory nella home del server. Per ciascuno, la dimensione della quota è la dimensione dei file di proprietà di Unix grouo del server, mentre la dimensione include i file di proprietà di altri utenti. Quest'ultimo può essere più grande per directory come i logs quando hanno file di proprietà di root . La directory homes che contiene utenti di posta / FTP non è inclusa, come spiegato nella sezione successiva.

Per utente
Questa sezione mostra lo spazio su disco utilizzato dai primi 10 utenti di cassette postali in questo dominio e tutti i server secondari. L'utilizzo si basa sui file di proprietà dell'utente Unix per ciascuna cassetta postale.

Per server secondario
Mostra lo spazio su disco utilizzato da ciascun server secondario, in base ai file nelle loro directory home. È incluso anche lo spazio utilizzato da tutti gli utenti di cassette postali in tali server.

Per database
Mostra lo spazio su disco utilizzato da ciascun database MySQL o PostgreSQL in questo server virtuale o in qualsiasi sub-server. Vengono visualizzati solo i primi 10 database.
