Programma client SpamAssassin
Questa opzione determina se il filtro antispam viene eseguito utilizzando il programma spamassassin autonomo o il client leggero spamc . In quest'ultimo caso, il sistema deve eseguire il server spamd , altrimenti non verrà eseguita alcuna classificazione spam.

L'uso di spamc è in genere più veloce e consuma meno memoria e CPU. Tuttavia, eliminerà la possibilità di avere una configurazione SpamAssassin diversa per ciascun server virtuale, poiché il server spamd può leggere solo un singolo set di file di configurazione.
