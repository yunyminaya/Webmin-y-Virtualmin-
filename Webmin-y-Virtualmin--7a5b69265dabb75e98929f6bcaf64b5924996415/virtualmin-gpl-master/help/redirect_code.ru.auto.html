Тип перенаправления HTTP
По умолчанию Apache будет использовать код состояния HTTP 302 для перенаправления на другие веб-сайты, но это поле можно использовать для настройки кода в зависимости от того, почему добавляется перенаправление. Этот параметр действует только в том случае, если выбран пункт назначения «URL-адрес на другом веб-сайте».
