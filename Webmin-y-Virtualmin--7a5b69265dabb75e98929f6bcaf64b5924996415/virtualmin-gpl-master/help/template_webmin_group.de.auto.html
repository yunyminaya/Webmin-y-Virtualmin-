Webmin-Gruppe für Domaininhaber
In diesem Feld können Sie eine Webmin-Gruppe auswählen, zu der die Webmin-Anmeldung für Besitzer virtueller Server gehört. Gruppen können unter Webmin -> Webmin-Benutzer definiert und verwendet werden, um Virtualmin-Benutzern zusätzliche Module und Berechtigungen zuzuweisen.
