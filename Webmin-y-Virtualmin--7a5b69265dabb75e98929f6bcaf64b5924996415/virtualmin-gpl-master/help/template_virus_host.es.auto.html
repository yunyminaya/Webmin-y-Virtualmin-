Servidor host para escáner remoto
Este campo debe usarse para ingresar el nombre de host del sistema remoto en el que se ejecuta clamd , al que clamdscan o clamd-stream-client enviarán mensajes de correo electrónico cargados de virus para su análisis. No tiene ningún efecto si selecciona los modos de escáner independiente o de servidor local , ya que ambos escanean localmente.
