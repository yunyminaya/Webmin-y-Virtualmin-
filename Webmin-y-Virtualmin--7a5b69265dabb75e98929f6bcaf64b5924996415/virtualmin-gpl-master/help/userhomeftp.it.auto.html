Elenco di casa
Di default, la directory home per un utente FTP del sito web è la stessa della directory web di base del dominio. Tuttavia, puoi cambiarla nella directory home del dominio (utilizzando un valore vuoto) o in una sottodirectory (utilizzando un percorso relativo sotto la directory web).

Ciò presuppone che il server FTP sia configurato per limitare i client alle loro directory home, che dovrebbe essere l'impostazione predefinita nelle installazioni standard di Virtualmin.
