Automatisch aangemaakt alias domein
Deze optie kan worden gebruikt om een extra alias domein aan te maken voor iedere nieuwe virtuele server onder een ander domein. Dit kan handig zijn wanneer u een nieuwe server toevoegt waarvan de DNS domein nog niet is geregistreerd, zodat het bereikt kan worden onder een tijdelijke naam die onder een naam staat die wel al geregistreerd is (zoals uw bedrijfsnaam).
