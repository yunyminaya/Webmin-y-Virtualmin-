Interfaz de red
Si desea configurar un host virtual basado en IP (requerido para sitios web SSL o hosting virtual FTP), esta opción debe configurarse en Interfaz virtual con IP y se debe ingresar una dirección no utilizada. Virtualmin activará esta IP en su sistema cuando se cree el servidor, y lo asociará con el servidor.

Si se selecciona la opción Compartido , la dirección de interfaz de red principal del sistema se utilizará para el sitio web de este servidor y el dominio DNS.
