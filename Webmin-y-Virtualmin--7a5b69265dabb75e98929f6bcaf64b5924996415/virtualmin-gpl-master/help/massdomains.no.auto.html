Oppdater virtuelle servere
Denne siden lar deg oppdatere innstillinger og begrensninger i flere virtuelle servere samtidig, som valgt på forrige side. Som standard vil ingenting bli endret på de valgte serverne før du velger noen attributt eller grense for å endre ved å bruke feltene på denne siden. Innstillinger som kan endres er delt inn i følgende kategorier:

Kvote- og båndbredde endres
Disse feltene kan brukes til å endre diskekvotene for utvalgte servere, eller for å endre båndbreddegrensene. Endring av kvoter påvirker bare servere på toppnivå, ettersom underservere arver kvotene sine fra overordnet domene.

Aktiverte funksjonsendringer
I denne delen kan du velge funksjoner du vil slå av eller på for alle de valgte serverne. Dette er de samme funksjonene som kan aktiveres for individuelle servere på siden Rediger virtuell server .

Endringer i eiergrenser
Disse feltene kan brukes til å endre de forskjellige grensene som gjelder for eiere av virtuelle servere på toppnivå, for eksempel antall postkasser, aliaser og domener de kan opprette. Endringer gjort her har ingen innvirkning på underserverne, som faller under grensene pålagt foreldrene.

Endringer av eierfunksjoner
Feltene i denne delen kontrollerer funksjonene til Virtualmin som er tilgjengelige for administratorene av de valgte virtuelle serverne. Igjen, de er bare relevante for servere på toppnivå.

PHP-alternativet endres
Når den er tilgjengelig, lar denne delen deg endre både standard PHP-versjon og metode for å kjøre PHP-skript på alle valgte servere. Det er imidlertid bare nyttig på servere som har nettsteder.

Konfigurasjon endres
Denne delen er beregnet på å gjøre forskjellige endringer i konfigurasjonen til utvalgte servere, for eksempel metodebruken for spam-rydding eller domeneadministratorens påloggingsskall.

Det er også mulig å deaktivere de valgte serverne midlertidig ved å bruke Deaktiver servere- knappen nederst på siden. Når du bruker dette, kan du også legge inn en grunn til å deaktivere (for eksempel manglende betaling ) i den tilstøtende tekstboksen.

For å aktivere flere servere som er deaktivert på denne siden eller andre steder, bruker du Aktiver servere- knappen.
