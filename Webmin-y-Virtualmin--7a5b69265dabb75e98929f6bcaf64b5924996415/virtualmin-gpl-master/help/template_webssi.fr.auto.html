Autoriser les inclusions côté serveur par défaut?
Les inclusions côté serveur sont une fonctionnalité de serveur Web qui permet d'utiliser de simples directives de macro dans des fichiers HTML. Ils peuvent être utilisés pour inclure d'autres fichiers HTML, définir et afficher le contenu des variables et contrôler le contenu qui apparaît sur la page avec les instructions if . Ils offrent un moyen plus simple de créer des pages dynamiques qu'un langage de script complet comme PHP ou Perl.

Ce paramètre détermine si les nouveaux serveurs virtuels auront SSI activé après la création, désactivé ou laissé pour utiliser les paramètres globaux par défaut de votre serveur Web.

Pour plus d'informations, consultez la documentation Apache à http://httpd.apache.org/docs/current/howto/ssi.html
