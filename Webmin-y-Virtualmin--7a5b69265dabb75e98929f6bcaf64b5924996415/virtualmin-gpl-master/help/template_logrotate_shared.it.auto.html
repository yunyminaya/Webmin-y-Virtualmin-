Condividi la configurazione di logrotate con tutti i domini?
Quando è impostato su No , Virtualmin creerà un blocco separato nel file di configurazione logrotate per ciascun server virtuale. Ciò consente a ciascuno di avere impostazioni diverse, ma causerà il riavvio del server Web dopo la rotazione dei registri di ciascun dominio, interrompendo eventualmente il servizio.

Selezionando Sì , tutti i nuovi domini condivideranno un singolo blocco di configurazione, il che significa che il server Web deve essere riavviato una sola volta. Tuttavia, non è possibile avere impostazioni di rotazione diverse per ciascun dominio.
