Дополнительные параметры только для команды scp
Когда Virtualmin выполняет удаленное резервное копирование или восстановление с помощью SCP, он будет использовать команду scp для передачи файлов. Это поле позволяет указать дополнительные параметры только для команды scp , например -O для использования более старой версии протокола.
