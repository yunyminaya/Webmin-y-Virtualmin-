Indirizzi IP condivisi
Lo scopo principale di questa pagina è l'inserimento di indirizzi IP aggiuntivi che possono essere utilizzati da server virtuali che non dispongono di propri IP privati. Ogni IP inserito deve essere già attivo sul sistema (cosa che può essere fatto nella configurazione di rete del modulo Webmin) e non può essere utilizzato da nessun altro rivenditore o server virtuale IP privato.
