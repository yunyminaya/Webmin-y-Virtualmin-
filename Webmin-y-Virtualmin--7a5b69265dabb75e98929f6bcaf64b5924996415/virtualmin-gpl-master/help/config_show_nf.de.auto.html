Zeigen Sie neue Funktionen an
Wenn Sie ein Thema verwenden, das neue Funktionen in der aktuellen Version von Virtualmin anzeigen kann, bestimmt diese Einstellung, welche Benutzertypen sie sehen können. Benutzer sehen nur Benachrichtigungen über Funktionen, auf die sie Zugriff haben, und können diese jeweils separat ausblenden.
