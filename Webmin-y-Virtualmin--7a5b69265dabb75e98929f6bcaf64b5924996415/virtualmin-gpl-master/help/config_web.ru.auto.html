Настройка и настройка Apache включены?
Этот параметр включает или отключает создание записей Apache VirtualNameHost и настраивает все другие функции виртуального хоста в файле конфигурации Apache. Если вы используете альтернативный веб-сервер, вы можете отключить эту опцию. Может быть возможно использовать интерфейс модуля плагина для создания поддержки для вашего предпочитаемого сервера (хотя в настоящее время другие веб-серверы не поддерживаются), или вы можете вызвать внешний скрипт вместе с переменными, взятыми из информации, предоставленной Virtualmin по адресу время создания домена для настройки вашего предпочтительного веб-сервера.
