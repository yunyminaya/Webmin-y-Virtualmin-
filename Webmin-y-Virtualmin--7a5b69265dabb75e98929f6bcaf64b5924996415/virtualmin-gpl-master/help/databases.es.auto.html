Editar bases de datos
Esta página enumera todas las bases de datos que actualmente pertenecen al servidor virtual y permite la creación de bases de datos adicionales hasta cualquier límite vigente para el usuario que inició sesión en Virtualmin. También puede eliminar una base de datos más seleccionándola de la lista y presionando el botón Eliminar seleccionada .

Para ver las tablas y los datos reales de una base de datos, haga clic en el enlace Administrar junto al nombre de la base de datos en la tabla. Para ver el tamaño de la base de datos, haga clic en su nombre.

Cuando se utiliza Virtualmin Pro, se muestran dos formularios adicionales debajo de la lista de bases de datos para cambiar el nombre de usuario y la contraseña de los usuarios de MySQL y PostgreSQL utilizados por este servidor virtual. Esto puede resultar útil si desea que el inicio de sesión de la base de datos sea diferente del inicio de sesión de administración principal, quizás por motivos de seguridad. Estos deben cambiarse con cuidado, ya que los scripts PHP (como los de los instaladores de aplicaciones web) que tengan el nombre de usuario o la contraseña en sus archivos de configuración no se actualizarán.
