Op IP-gebaseerde virtuele FTP opzetten?
Wanneer deze optie is aangezet zal er een ProFTPd virtuele server worden aangemaakt voor deze Virtualmin virtuele server. Dit kan worden gebruikt om een anonieme FTP site op te zetten die prive is voor deze server. Echter het is wel nodig dat de server zijn eigen IP adres heeft.

Deze functie is alleen nodig indien u een op naam gebaseerde toegang wilt hebben naar een anonieme FTP server. Bijna niemand heeft deze functie nodig en dus hoeft die niet aangezet te worden. U hoeft deze functie dus niet aan te zetten als u FTP toegang wilt verlenen naar het account van de domein eigenaar, omdat iedere gebruiker met een shell account automatisch FTP toegang heeft naar zijn home directory.

Met andere woorden: U heeft deze functie dus niet nodig, zelfs niet als u denkt het nodig te hebben.
