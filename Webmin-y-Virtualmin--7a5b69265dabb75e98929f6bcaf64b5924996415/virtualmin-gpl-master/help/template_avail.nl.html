Beschikbare Webmin modules voor virtuele server admins
Hier kunt u selecteren welke Webmin modules beschikbaar zijn voor virtuele server eigenaars van servers die aangemaakt zijn met deze template. Linkjes naar deze modules verschijnen in het linker menu van Virtualmin, normaal gesproken onder de "Services" categorie. U kunt er een aantal uitschakelen om een limiet te maken op de geavanceerde configuratie instellingen die de server eigenaars kunnen wijzigen en hiermee ook de Virtualmin gebruikers interface simpeler houd.

Alle wijzigingen die u hier maakt worden direct aan alle virtuele servers toegevoegd die deze template gebruiken.
