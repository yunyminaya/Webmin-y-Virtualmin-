Modo de execução de script CGI
Esta opção determina como os scripts CGI são executados para este servidor virtual. As opções possíveis são:
  • Scripts CGI desativados - os scripts não podem ser executados.
  • Servidor FCGIwrap - os scripts são executados através de um servidor em segundo plano, um por domínio.
  • suEXEC - os scripts são iniciados por meio do wrapper suexec do Apache. Observe que esta opção pode não estar acessível em todos os sistemas.
