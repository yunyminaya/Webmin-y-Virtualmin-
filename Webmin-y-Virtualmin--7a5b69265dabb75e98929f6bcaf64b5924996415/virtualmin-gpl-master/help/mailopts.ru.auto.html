Настройки электронной почты
Эта страница предназначена для настройки параметров электронной почты для виртуального сервера. Если ваша система поддерживает отправленную электронную почту BCC (и она включена на странице «Конфигурация модуля»), исходящая электронная почта BCC в разделе может использоваться для выбора адреса для копирования почты.

Если виртуальный сервер является доменом псевдонимов, поле «Режим псевдонимов почты» контролирует, копируются ли псевдонимы почты из целевого домена или создается один псевдоним для переадресации всех писем к цели. Первый режим доступен не для всех почтовых серверов, но он лучше при использовании Postfix, поскольку позволяет пересылать почту на недопустимые адреса на уровне диалога SMTP.
