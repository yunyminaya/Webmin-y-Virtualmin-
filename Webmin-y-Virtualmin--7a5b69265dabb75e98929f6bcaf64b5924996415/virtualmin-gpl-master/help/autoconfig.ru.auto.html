Конфигурация почтового клиента
На этой странице можно включить или отключить URL-адрес автоматической настройки почтового клиента для всех виртуальных серверов с включенной электронной почтой. Такие клиенты, как Thunderbird, могут использовать этот URL-адрес для определения настроек SMTP и IMAP только на основе адреса электронной почты, включая правильное имя пользователя, номер порта и тип шифрования.

Когда этот параметр включен, для каждого виртуального сервера создается поддомен с именем autoconfig , а путь URL /mail/config-v1.1.xml сопоставляется с CGI, который обслуживает XML для предоставления необходимых параметров почтового клиента. Они будут удалены, когда функция отключена.
