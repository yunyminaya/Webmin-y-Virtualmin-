Redirect admin.domain to Usermin?
When this field is set to Yes (as it is by default), any new virtual servers will have an admin entry added to their DNS domains, with the same IP address as the server's website. When http://admin.thedomain.com/ is accessed, the browser will be redirected to Virtualmin on port 10000 of the same domain, to access the administration web interface.
