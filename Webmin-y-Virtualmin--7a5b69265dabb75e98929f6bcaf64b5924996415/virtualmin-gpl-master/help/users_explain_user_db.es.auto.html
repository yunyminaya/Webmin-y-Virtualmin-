Crear usuario de base de datos
Crea un nuevo usuario de base de datos capaz de administrar bases de datos únicamente, aprovechando las cuotas del propietario del servidor virtual para todas las bases de datos administradas. Este usuario opera únicamente dentro del entorno de la base de datos, sin crear una cuenta Unix separada. Además, este usuario se puede integrar perfectamente fusionándose con una cuenta Unix creada con el mismo nombre de usuario.
