Envoyer un e-mail à partir de l'IP du domaine par défaut?
Lorsque cette option est activée, les nouveaux serveurs virtuels qui ont une adresse IP privée seront configurés pour utiliser cette IP comme adresse sortante lors de l'envoi d'e-mails. Cela peut également être activé ou désactivé par domaine sur la page Paramètres de messagerie .
