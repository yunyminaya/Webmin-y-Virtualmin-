Afficher le lien du site Web d'aperçu?
Cette option détermine si l'élément de menu de gauche Aperçu du site Web sous la catégorie Services est visible pour tous, certains ou aucun utilisateur de Virtualmin. Vous voudrez peut-être le cacher aux propriétaires de domaine pour éviter de les confondre avec un élément de menu supplémentaire.
