Можно ли выбирать имена баз данных?
Если это ограничение установлено на Нет , владельцу сервера не будет разрешено выбирать имя базы данных по умолчанию при создании подсервера, на котором включена функция базы данных. Вместо этого по умолчанию всегда будет использоваться первая часть имени домена.

Кроме того, пользователь не сможет удалить базу данных по умолчанию для любого существующего домена, поскольку это позволило бы ему фактически переименовать базу данных по умолчанию после создания домена.
