Другие группы, в которые могут быть назначены пользователи
Это поле можно использовать для ввода групп Unix, которым могут быть назначены пользователи. Это может быть полезно в других приложениях, которые осуществляют контроль доступа по группам, или для предоставления доступа к файлам, которыми пользователи обычно не смогут управлять. Тем не менее, его следует использовать с осторожностью, поскольку предоставление владельцам серверов возможности назначать пользователей в root или bin группы будет представлять серьезную угрозу безопасности.
