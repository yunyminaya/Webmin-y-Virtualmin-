HTTP yönlendirme türü
Apache, varsayılan olarak diğer web sitelerine yönlendirmeler için 302 HTTP durum kodunu kullanır, ancak bu alan, yönlendirmenin neden eklendiğine bağlı olarak kodu özelleştirmek için kullanılabilir. Bu seçenek yalnızca "Başka bir web sitesindeki URL" hedefi seçildiğinde etkili olur.
