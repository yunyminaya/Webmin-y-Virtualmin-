Funzionalità consentite per i server
Queste caselle di controllo controllano quali funzionalità possono essere selezionate durante la creazione di un nuovo server virtuale e quali possono essere abilitate o disabilitate durante la modifica di un server. Ciò può essere utile se si desidera impedire ai proprietari di server di utilizzare determinate funzionalità ad alta intensità di risorse come database, SSL o indirizzi IP privati.
