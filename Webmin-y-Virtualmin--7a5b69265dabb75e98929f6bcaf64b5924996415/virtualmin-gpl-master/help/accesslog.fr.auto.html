Fichier journal d'accès au serveur Web
Ce champ définit le fichier dans lequel les journaux d'accès Apache sont écrits pour ce serveur virtuel. Chaque demande HTTP servie par le domaine de ce site Web sera écrite dans ce fichier sur une ligne distincte. Il ne doit être modifié que si vous déplacez la journalisation vers un répertoire différent, tel que /var/log/virtualmin ou le répertoire personnel du serveur.
