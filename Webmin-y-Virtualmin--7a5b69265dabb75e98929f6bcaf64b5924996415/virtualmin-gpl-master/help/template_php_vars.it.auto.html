Variabili di configurazione di PHP per gli script
Questa tabella può essere utilizzata per immettere le impostazioni di configurazione di PHP che verranno aggiunte all'host virtuale Web di qualsiasi server su cui sia installato uno script di terze parti. Può essere utile per aumentare i limiti di memoria o apportare altre modifiche alla configurazione PHP specificate dal sito per soddisfare i requisiti di script.
