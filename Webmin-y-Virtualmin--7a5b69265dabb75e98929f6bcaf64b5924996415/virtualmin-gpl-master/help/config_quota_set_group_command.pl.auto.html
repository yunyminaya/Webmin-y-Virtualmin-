Polecenie, aby ustawić limit grupy
Gdy używane są zewnętrzne komendy przydziału, program ustawiony w tej opcji zostanie wywołany, gdy Virtualmin będzie chciał ustawić przydziały dla grupy Unix. Parametry, które są wywoływane, to:
  1. Nazwa grupy uniksowej
  2. Nowa miękka kwota w blokach 1 kB lub 0 dla nieograniczonej.
  3. Nowa twarda kwota w blokach 1 kB lub 0 dla nieograniczonej.
