Azione per altri mittenti?
Questo campo controlla il modo in cui il record SPF copre gli host diversi dal server Virtualmin stesso e gli host specificati nelle opzioni precedenti. Le scelte sono:
Consenti (modalità SPF all )
L'indirizzo IP del mittente non dovrebbe essere considerato affatto. Ciò disabilita effettivamente SPF.

Neutro (modalità SPF ?all )
Il record SPF non copre tutti i possibili mittenti. Dovrebbe essere selezionato se gli utenti devono essere in grado di inviare da server di posta arbitrari (come quelli del proprio ISP). Tuttavia, è probabile che porti a sfortunati effetti collaterali in futuro, poiché i server di posta iniziano a bloccare gli host senza record SPF o con record SPF particolarmente lassisti.

Scoraggiare (modalità SPF ~all )
Il record SPF si applicherà solo al server Virtualmin stesso e agli altri host definiti sopra. Questo è il metodo preferito di utilizzo di SPF, poiché limita gli host che possono inviare posta che dichiara di provenire da questo dominio, ma non blocca completamente i messaggi provenienti da altre fonti.

Non consentire (modalità SPF -all )
Come l'opzione precedente, ma i destinatari devono eliminare completamente i messaggi provenienti da altri mittenti.
