Separat bearbeitbare DNS-Einträge für Alias-Domänen?
Standardmäßig werden die DNS-Einträge von Alias-Domänen von der Zieldomäne kopiert, d. h. sie können nicht separat bearbeitet werden. Wenn Sie diese Option auf „Ja“ setzen, können neu erstellte Alias-Domänen über einen eigenen, unabhängigen Datensatzsatz verfügen. Dies dürfte jedoch für die meisten Benutzer nicht von Nutzen sein.
