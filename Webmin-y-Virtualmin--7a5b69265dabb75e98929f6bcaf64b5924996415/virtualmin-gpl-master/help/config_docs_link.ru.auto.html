Ссылка назначения для дополнительной документации
Когда этот параметр установлен, для всех пользователей в правом верхнем углу страницы «Информация о системе» появится дополнительная ссылка на документацию. Это может быть использовано для ссылки на документацию для конкретного сайта, например, на ваше собственное руководство пользователя Virtualmin.
