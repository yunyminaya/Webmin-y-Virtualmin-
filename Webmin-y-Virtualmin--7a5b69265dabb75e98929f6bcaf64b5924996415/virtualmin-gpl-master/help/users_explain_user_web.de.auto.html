Erstellen Sie einen Webserver-Benutzer
Erstellt einen neuen Webserver-Benutzer innerhalb der htpasswd- Dateien, insbesondere für die sichere Authentifizierung bei geschützten Webverzeichnissen. Dieser Benutzer unterscheidet sich von Unix-Konten, die ausschließlich der Verwaltung des Zugriffs auf Webverzeichnisse dienen. Darüber hinaus kann dieser Benutzer nahtlos integriert werden, indem er mit einem Unix-Konto zusammengeführt wird, das mit demselben Benutzernamen erstellt wurde.
