Formato para nombres de usuario que incluyen dominio
Cuando un nombre de usuario tiene el nombre de dominio adjunto, esta opción selecciona el formato. No todas las opciones están disponibles en todas las plataformas de sistema operativo, ya que algunos caracteres no están permitidos en los nombres de usuario. Si el nombre de dominio es el nombre de usuario completo, es decir, "virtualmin.com", o solo la primera parte, es decir, "virtualmin", lo dicta la opción Estilo de nombre de dominio en nombres de usuario en la página de configuración del módulo.
