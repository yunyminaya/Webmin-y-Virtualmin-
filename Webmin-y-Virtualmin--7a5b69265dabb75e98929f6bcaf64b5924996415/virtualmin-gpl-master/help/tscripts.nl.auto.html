Sjabloonwebapp-installatieprogramma's
Op deze pagina kunt u opgeven dat webapps van derden moeten worden geïnstalleerd wanneer een nieuwe virtuele server wordt gemaakt met behulp van deze sjabloon. Voor elke webapp moeten de volgende gegevens worden ingevuld:

Web applicatie
De naam en versie van een webapp die moet worden geïnstalleerd
URL-pad
Een pad onder de website van de virtuele server om de webapp te installeren, zoals /wordpress . Dit bepaalt ook de map onder ~/public_html die de bestanden van de webapp bevat.
DB-naam
De naam van de database die de webapp zal gebruiken om gegevens op te slaan. De opties zijn:
Geen
Dit moet worden geselecteerd voor webapps die geen database nodig hebben, zoals Gallery .
Hoofddatabase
De database van de webapp zal de primaire database van de virtuele server van het gekozen type zijn.
Ander
Als u wilt dat deze webapp een eigen privédatabase gebruikt, kunt u deze optie selecteren en het patroon voor een database invoeren in het aangrenzende tekstveld. U kunt bijvoorbeeld ${DB}_phpnuke invoeren. Als de resulterende database niet bestaat, wordt deze aangemaakt als onderdeel van het installatieproces van de webapp.
DB-type
Het type databasegebruik voor de webapps-gegevens, indien aanwezig, is geselecteerd.
