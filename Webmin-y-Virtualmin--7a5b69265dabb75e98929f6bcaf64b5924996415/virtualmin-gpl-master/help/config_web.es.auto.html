Instalación de Apache y configuración habilitada?
Esta opción habilita o deshabilita la creación de entradas Apache VirtualNameHost y configura todas las demás características del host virtual en su archivo de configuración de Apache. Si está utilizando un servidor web alternativo, puede deshabilitar esta opción. Es posible usar la interfaz del módulo de complemento para crear soporte para su servidor preferido (aunque no se admiten otros servidores web en este momento), o puede elegir llamar a un script externo junto con variables extraídas de la información proporcionada a Virtualmin en tiempo de creación de dominio para configurar su servidor web preferido.
