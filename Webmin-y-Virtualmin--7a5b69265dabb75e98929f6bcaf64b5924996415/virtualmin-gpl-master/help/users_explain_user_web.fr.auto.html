Créer un utilisateur du serveur Web
Crée un nouvel utilisateur de serveur Web dans les fichiers htpasswd , spécifiquement pour une authentification sécurisée sur les répertoires Web protégés. Cet utilisateur est distinct des comptes Unix, dédiés exclusivement à la gestion des accès aux annuaires web. De plus, cet utilisateur peut être intégré de manière transparente en fusionnant avec un compte Unix créé en utilisant le même nom d'utilisateur.
