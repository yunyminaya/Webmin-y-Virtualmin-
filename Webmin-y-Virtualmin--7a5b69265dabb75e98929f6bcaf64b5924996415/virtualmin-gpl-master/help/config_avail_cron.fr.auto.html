Tâches Cron planifiées (tâches Cron de l'utilisateur)
Cette option spécifie si le compte propriétaire du domaine peut utiliser le module Cron Jobs Webmin. Un Cron Job est un événement planifié qui peut être utilisé pour effectuer périodiquement une tâche, comme envoyer un e-mail, sauvegarder ou vider une base de données, vérifier l'état d'un fichier ou d'une boîte aux lettres, etc. Le module Cron Jobs fournit et est facile à utiliser interface avec la syntaxe difficile à retenir (et déroutante pour les débutants) du fichier crontab. Les tâches Cron seront exécutées en tant qu'utilisateur propriétaire du domaine.
