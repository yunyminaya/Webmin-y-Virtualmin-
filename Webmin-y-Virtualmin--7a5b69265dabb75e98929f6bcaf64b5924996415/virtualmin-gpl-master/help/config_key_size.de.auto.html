Standardgröße des SSL-Schlüssels
Standardmäßig erstellt Virtualmin SSL-Schlüssel für virtuelle Server mit einer Länge von 2048 Bit. Da dies jedoch für einige Benutzer oder Anwendungen möglicherweise nicht sicher genug ist, kann diese Einstellung verwendet werden, um eine größere Größe anzugeben, z. B. 4096.
