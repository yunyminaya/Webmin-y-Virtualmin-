Archivos a excluir de cada dominio

Este campo le permite ingresar una lista de rutas relativas a los archivos en el directorio de inicio de cada servidor virtual para excluirlos de esta copia de seguridad. Para excluir múltiples rutas, coloque cada una en su propia línea (por ejemplo, separadas por saltos de línea).

Ejemplos

Para excluir un directorio completo, proporcione la ruta relativa desde el directorio de inicio del usuario, como public_html/stats para excluir el directorio de informes de Webalizer o los homes para excluir los directorios de inicio de los usuarios.

Para excluir un solo archivo, proporcione la ruta relativa al nombre del archivo, como public_html/index.html .

Los archivos también se pueden especificar mediante comodines, como *.log , que excluirá cualquier archivo con la extensión de archivo .log cualquier lugar dentro del directorio de inicio del dominio. También se puede proporcionar una ruta, como etc/*.ini para excluir todos los archivos que terminan en .ini .
