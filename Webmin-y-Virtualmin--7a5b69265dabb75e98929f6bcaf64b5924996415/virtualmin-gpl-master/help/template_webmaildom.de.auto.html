URL für die Weiterleitung von Webmail
Standardmäßig leitet Virtualmin Anforderungen an http://webmail.thedomain.com an Usermin auf demselben System weiter, z. B. https://thedomain.com:20000/ . In diesem Feld können Sie jedoch die Weiterleitungs-URL ändern, möglicherweise in einen einzelnen Hostnamen für alle Ihre Domains, für die Sie ein SSL-Zertifikat haben, z. B. https://webmail.yourcompany.com:20000/ . Sie können auch zu einer anderen Webmail-Anwendung in derselben Domain umleiten, indem Sie eine URL wie http://www.${DOM}/squirrelmail/ .
