¿Configurar el certificado Let's Encrypt para el nombre de host?
Cuando esté habilitado, Virtualmin intentará configurar automáticamente un certificado SSL Let's Encrypt válido para el nombre de host durante la siguiente verificación de configuración.

Si tiene éxito, este certificado SSL se utilizará como predeterminado para todos los servicios que no tengan su propio certificado SSL configurado.

Si el nombre de host del sistema cambia, se detectará y actualizará automáticamente.
