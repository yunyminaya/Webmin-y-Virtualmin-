I sub-server ereditano l'indirizzo IP dal genitore?
Per impostazione predefinita, a tutti i server secondari creati da questo proprietario di dominio verrà assegnato l'indirizzo IP condiviso predefinito del sistema, a meno che il proprietario non disponga delle autorizzazioni per assegnare indirizzi privati. Tuttavia, se si modifica questa opzione su Sì, eventuali server secondari assegneranno invece lo stesso indirizzo del server di livello superiore del proprietario.
