Il sito web corrisponde a tutti i sottodomini?
Quando questa opzione è abilitata, le richieste di qualsiasi URL nel dominio (come http://abc.example.com ) restituiranno il sito Web del server virtuale. Ciò può essere utile se si ospita un'applicazione che può restituire contenuti diversi in base al nome di dominio richiesto, come Wordpress MU.

Se il dominio DNS del server virtuale è ospitato su questo sistema, Virtualmin aggiungerà anche il record DNS jolly * quando si seleziona Sì .
