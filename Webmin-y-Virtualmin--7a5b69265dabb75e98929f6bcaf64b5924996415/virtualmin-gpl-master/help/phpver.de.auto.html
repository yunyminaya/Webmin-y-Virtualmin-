PHP-Versionen
Auf dieser Seite können Sie die Version von PHP auswählen, die standardmäßig auf diesem virtuellen Server verwendet wird, und verschiedene Versionen für verschiedene Web-Unterverzeichnisse auswählen. Dies kann nützlich sein, wenn Sie PHP-Anwendungen haben, die nur mit einer bestimmten Version funktionieren.

Um die standardmäßig verwendete Version zu ändern, wählen Sie sie aus dem Menü neben der Zeile Standard-HTML-Verzeichnis aus und klicken Sie auf Speichern . Um die von einem anderen Verzeichnis verwendete Version zu ändern, wählen Sie sie aus dem Menü neben der Zeile dieses Verzeichnisses aus.

Um eine benutzerdefinierte PHP-Version für ein neues Verzeichnis hinzuzufügen, geben Sie den relativen Pfad in das Textfeld unten in der Tabelle ein, wählen Sie eine Version aus dem Menü unten aus und klicken Sie auf Speichern . Dieser Pfad muss relativ zu public_html , z. B. WordPress oder MediaWiki . Es ist keine führende / oder vollständige URL erforderlich.

Um ein Verzeichnis auf die Standardversion von PHP zurückzusetzen, aktivieren Sie das Kontrollkästchen in der Tabelle daneben und klicken Sie auf die Schaltfläche Ausgewählte entfernen .
