Máximo proceso de servidor mestizo para Rails
Al instalar los scripts de Ruby on Rails, el administrador del servidor puede seleccionar cómo procesa mi servidor Mongrel para comenzar a manejar las solicitudes a la ruta del script, si está instalado el módulo Apache mod_proxy_balancer . Para evitar que un número excesivo de procesos de servidor mestizos sobrecargue el sistema, esta opción se puede usar para limitar el número que los administradores de dominio pueden configurar desde Virtualmin.
