Estilo para o conteúdo inicial
O estilo selecionado neste menu será mesclado com o conteúdo inicial do site para gerar páginas HTML para o novo servidor virtual. Alguns estilos contêm apenas uma única página, enquanto outros têm várias páginas e imagens.

Uma miniatura de cada estilo está incluída no menu. Para ver uma imagem maior, selecione um estilo e clique no link Visualização maior .
