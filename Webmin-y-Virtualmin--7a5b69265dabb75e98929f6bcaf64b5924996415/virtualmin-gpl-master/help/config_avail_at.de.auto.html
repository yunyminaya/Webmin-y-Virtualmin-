Geplante Befehle (Benutzerbefehle)
Diese Option gibt an, ob das Domänenbesitzerkonto das Webmin-Modul "Geplante Befehle" verwenden kann. Ein geplanter Befehl oder eine Aufgabe ist ein geplantes Ereignis, mit dem eine Aufgabe zum angegebenen Zeitpunkt einmal ausgeführt werden kann. Das Modul Geplante Befehle bietet eine benutzerfreundliche Schnittstelle zur schwer zu merkenden (und für Neulinge verwirrenden) Syntax des at-Befehls. Geplante Befehle werden als Benutzer des Domänenbesitzers ausgeführt.
