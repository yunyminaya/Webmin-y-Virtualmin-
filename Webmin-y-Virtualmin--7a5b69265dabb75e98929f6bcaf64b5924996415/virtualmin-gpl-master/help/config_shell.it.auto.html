Shell per utenti di cassette postali
Questa opzione specifica la shell configurata per gli utenti delle cassette postali. L'utente di una cassetta postale è il tipo di utente meno privilegiato e normalmente dovrebbe avere un accesso molto limitato. Tuttavia, se il server Virtualmin viene utilizzato solo da utenti fidati o utenti che devono avere accesso alla shell, è possibile modificarlo dal valore predefinito / dev / null a una delle shell di sistema reali. Nota che / bin / false è una shell speciale che chiuderà gli accessi SSH remoti ma consentirà comunque gli accessi FTP.
