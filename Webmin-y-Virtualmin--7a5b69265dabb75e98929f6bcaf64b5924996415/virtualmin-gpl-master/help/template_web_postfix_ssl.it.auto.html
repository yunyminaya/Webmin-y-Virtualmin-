Copiare i certificati SSL per IP su Postfix?
Quando questa opzione è abilitata e viene creato un server virtuale con SSL abilitato e il proprio IP privato, Dovecot verrà configurato per utilizzare il certificato SSL del server per le connessioni a quell'indirizzo IP. Ciò può essere utile se i client di posta si connettono al nome di dominio in modalità SSL e prevedono un certificato corrispondente.
