Configuração do PHP (para arquivos php.ini do domínio)
Quando ativada, esta opção concede aos proprietários do servidor virtual acesso ao módulo de Configuração do PHP. Isso será restrito para que eles possam gerenciar apenas arquivos php.ini em seus próprios servidores virtuais, para os quais a execução do PHP via CGI está ativada.

Se um domínio usando mod_php para executar scripts PHP, esta opção também controlará se o proprietário do servidor virtual tem permissão para definir configurações relacionadas ao PHP no Apache.
