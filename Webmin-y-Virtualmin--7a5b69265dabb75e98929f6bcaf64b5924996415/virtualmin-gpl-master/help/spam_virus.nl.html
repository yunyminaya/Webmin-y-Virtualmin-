Bestemming voor virus email's
Dit veld bepaald wat er gebeurt met email berichten die zijn geidentificeerd door ClamAV dat er een virus inzit. De beschikbare optie zijn :

Weggooien
Ieder gedetecteerd virus word onmiddellijk verwijdert.
Wegschrijven naar standaard virus file ~/mail/virus
Virussen worden weggeschreven naar de standaard folder voor handmatige analyse en verwijdering.
Wegschrijven naar de email file onder de home directory
Virussen worden opgeslagen naar de email file onder de gebruikers home directory die u heeft ingevuld. Om te schrijven naar een Maildir formaat directory, moet u een / plaatsen aan het einde van de filenaam.
Doorsturen naar email adres
Ieder virus-besmet bericht word doorgestuurd naar het ingevulde adres.
Schrijf naar andere file
Virussen worden weggeschreven naar de opgegeven email file (of directory als het pad eindigt met een /), wat overal kan zijn op de server.
