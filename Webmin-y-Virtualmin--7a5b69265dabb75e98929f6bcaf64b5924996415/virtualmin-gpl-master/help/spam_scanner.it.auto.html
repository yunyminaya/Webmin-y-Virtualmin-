Programma di scansione antivirus
Le possibili opzioni per questo campo sono:
Scanner autonomo ( clamscan )
Questo è in genere il valore predefinito e indica a Virtualmin di utilizzare il programma di scansione ClamAV che esegue tutto il lavoro da solo. Questo funzionerà sicuramente, ma può essere più lento dell'opzione successiva.
Scanner server ( clamdscan )
Questa opzione può essere selezionata se il server di scansione dei virus clamd è in esecuzione sul sistema. È generalmente superiore, poiché il database dei virus ClamAV è piuttosto grande e il server deve solo caricarlo in memoria una volta.
Scanner del server remoto ( clamd-stream-client )
Questa opzione può essere selezionata se il server di scansione dei virus clamd è in esecuzione su un sistema remoto ed è stato configurato per accettare connessioni di rete da altri host. È utile per scaricare l'attività di scansione su un sistema diverso da quello che accetta la posta elettronica.
Altro programma
Questa opzione ti consente di dire a Virtualmin di utilizzare un programma completamente diverso, come il tuo script wrapper per ClamAV.
