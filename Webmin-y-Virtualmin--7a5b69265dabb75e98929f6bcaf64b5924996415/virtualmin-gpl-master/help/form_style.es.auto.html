Estilo para contenido inicial
El estilo seleccionado de este menú se fusionará con el contenido inicial del sitio web para generar páginas HTML para el nuevo servidor virtual. Algunos estilos contienen solo una página, mientras que otros tienen varias páginas e imágenes.

Se incluye una miniatura de cada estilo en el menú. Para ver una imagen más grande, seleccione un estilo y luego haga clic en el enlace Vista previa más grande .
