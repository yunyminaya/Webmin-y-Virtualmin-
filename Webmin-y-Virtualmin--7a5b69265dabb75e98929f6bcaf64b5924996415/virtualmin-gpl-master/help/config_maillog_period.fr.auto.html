Nombre de jours pour conserver les journaux de courrier traités
Virtualmin analysera régulièrement les journaux de messagerie du serveur et stockera les résultats dans un fichier de base de données, pour une recherche plus rapide à l'aide de la page Rechercher les journaux de messagerie . Ce paramètre peut être utilisé pour contrôler la durée de stockage de ces données dans les fichiers /var/webmin/procmail.* . La valeur par défaut est de 30 jours, mais vous souhaiterez peut-être la réduire si les journaux traités occupent trop d'espace disque sur votre système.
