È possibile eseguire il backup di server virtuali?
Questo campo determina se al rivenditore è consentito utilizzare le funzionalità di backup di Virtualmin. Le opzioni disponibili sono:

Pianifica ed esegui backup
Il rivenditore può sia pianificare i backup eseguiti automaticamente su base regolare sia utilizzare l'interfaccia Web per eseguire manualmente i backup.
Effettua solo backup
I backup possono essere eseguiti solo manualmente utilizzando l'interfaccia Web.
Nessuno dei due
Il rivenditore non ha accesso alle funzioni di backup.
