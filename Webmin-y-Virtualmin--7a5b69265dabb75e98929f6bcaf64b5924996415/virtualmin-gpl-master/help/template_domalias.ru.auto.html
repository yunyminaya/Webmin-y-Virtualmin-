Автоматически создавать псевдоним домена
Эта опция может использоваться для создания дополнительного домена псевдонима, созданного для каждого нового виртуального сервера, в каком-либо другом домене. Это может быть полезно при добавлении нового сервера, для которого домен DNS еще не зарегистрирован, чтобы к нему можно было получить доступ под временным именем, которое находится под тем, которое уже зарегистрировано (например, вашей компанией).
