¿Configuración de sitio web SSL habilitada?
Si está habilitado, se creará un sitio web SSL para cada nuevo dominio. Si se establece en "Sí, pero no de manera predeterminada", será posible elegir crear un sitio web SSL después de la creación del dominio. Tenga en cuenta que los sitios web SSL requieren una IP para ellos, por lo que no está habilitada para todos los sitios de forma predeterminada. Muchos sitios web no necesitarán seguridad SSL, por lo que pueden compartir una sola IP con cientos o miles de otros sitios web.
