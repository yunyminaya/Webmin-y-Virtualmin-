URI de informes para informes agregados
De manera predeterminada, el registro DNS de DMARC para nuevos dominios utilizará mailto:postmaster@domain.com como la dirección para los receptores de correo no deseado para informar resúmenes agregados. Sin embargo, puede ingresar un nuevo URI aquí para personalizar la dirección de correo electrónico, por ejemplo, para usar una dirección común en todos los dominios.
