Può modificare il server virtuale
Consente la modifica delle funzionalità del server virtuale, della sua password e della descrizione.
Può gestire gli utenti
Consente di visualizzare, modificare, creare ed eliminare utenti di caselle postali, FTP e database.
Può gestire gli alias
Consente di visualizzare, modificare, creare ed eliminare gli alias email.
Può gestire database
Consente di aggiungere o eliminare da esso ulteriori database MySQL e PostgreSQL a questo server virtuale.
Può installare webapp
Consente l'installazione di app web di terze parti in questo server virtuale.
Può cambiare l'indirizzo IP
Se questo server ha un indirizzo IP privato, questa opzione ne consente la modifica.
Può gestire certificati SSL
Consente il caricamento o la rigenerazione del certificato SSL e della chiave per questo server.
Può modificare il frame di inoltro
Per i siti Web di solo inoltro, questa opzione consente di modificare la destinazione e l'HTML per il frame di inoltro.
Può gestire amministratori aggiuntivi
Consente di visualizzare, creare, modificare ed eliminare ulteriori account amministratore del proprietario del server.
Può configurare la consegna di spam e virus
Consente di modificare le destinazioni e i punteggi di recapito di spam e virus.
Può selezionare le versioni PHP
Per i domini con PHP eseguito tramite CGI o fastCGI, consente di impostare versioni PHP per directory.
Può modificare le impostazioni e-mail
Consente la configurazione del BCC automatico, della modalità di copia degli alias e di altre opzioni relative alla posta elettronica.
Può eseguire backup
Consente di eseguire il backup dei server virtuali di proprietà su un server FTP o SSH remoto.
Può pianificare i backup
Consente di eseguire il backup dei server virtuali con una pianificazione regolare.
Può ripristinare i backup
Consente il ripristino della directory home e del contenuto del database dei server virtuali di proprietà.
Può selezionare IP condivisi
Consente di selezionare indirizzi IP condivisi definiti a livello globale durante la creazione di server virtuali.
Può creare alias catchall
Consente la creazione di alias email che inoltrano tutta la posta indirizzata a un dominio a un dominio o indirizzo diverso.
Può utilizzare l'editor di pagine web
Consente la modifica delle pagine Web tramite l'interfaccia utente di Virtualmin.
Può modificare gli host remoti del database
Consente l'inserimento di sistemi client remoti a cui è consentito connettersi ai database MySQL del dominio.
Può disabilitare i server virtuali
Consente ai server di essere temporaneamente disabilitati.
Può eliminare i server virtuali
Consente l'eliminazione completa dei server di proprietà, inclusi tutti i sottoserver e gli alias.
