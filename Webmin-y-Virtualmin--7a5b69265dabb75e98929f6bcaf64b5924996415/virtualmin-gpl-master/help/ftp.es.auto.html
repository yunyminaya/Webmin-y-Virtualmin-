Configurar FTP virtual basado en IP?
Cuando esta función está habilitada, se creará un servidor virtual ProFTPd para este servidor virtual Virtualmin. Esto se puede usar para configurar un sitio FTP anónimo que sea privado para este servidor. Sin embargo, requiere que el servidor tenga su dirección IP.

Esta función solo es necesaria si desea proporcionar acceso basado en el nombre a un servidor FTP anónimo. Casi nadie necesita que esta función esté habilitada (eso probablemente significa que usted). No es necesario que habilite esta función si solo desea proporcionar acceso FTP para la cuenta del propietario del dominio, ya que cualquier usuario con una cuenta shell tendrá automáticamente acceso FTP a su directorio de inicio.

En otras palabras: casi seguro que no necesitas esto, incluso si crees que tal vez sí.
