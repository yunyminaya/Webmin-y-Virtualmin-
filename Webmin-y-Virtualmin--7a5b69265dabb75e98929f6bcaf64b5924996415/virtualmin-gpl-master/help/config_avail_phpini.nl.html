PHP Configuratie (voor domein php.ini files)
Wanneer dit is aangezet krijgen virtuele server eigenaars toegang tot de PHP Configuratie module. Dit is echter beperkt zodat zij alleen de php.ini files in hun eigen virtuele server kunnen beheren, waarvoor PHP uitvoeren via CGI is aangezet.

Indien een domein mod_php gebruikt om PHP scripts uit te voeren bepaald deze optie ook of het is toegestaan dat de virtuele server eigenaar om PHP gerelateerde instellingen in Apache te configureren.
