Отправить новый почтовый ящик по электронной почте
По умолчанию после создания почтового ящика ему будет отправлено сообщение, содержащее сведения о новой почтовой учетной записи. Это поле можно использовать для отправки этого уведомления на другой адрес, например, на почтовый ящик, который уже есть у владельца.
