Programmes d'installation de modèles d'applications Web
Cette page vous permet de spécifier les applications Web tierces à installer lorsqu'un nouveau serveur virtuel est créé à l'aide de ce modèle. Pour chaque webapp, les informations suivantes doivent être renseignées :

Application Web
Le nom et la version d'une webapp à installer
Chemin de l'URL
Un chemin sous le site Web du serveur virtuel pour installer l'application Web, tel que /wordpress . Cela déterminera également le répertoire sous ~/public_html qui contient les fichiers de l'application Web.
Nom de la base de données
Le nom de la base de données que l'application Web utilisera pour stocker les données. Les options sont :
Aucun
Ceci doit être sélectionné pour les applications Web qui n'ont pas besoin de base de données, telles que Gallery .
Base de données principale
La base de données de la webapp sera la base de données principale du serveur virtuel du type choisi.
Autre
Si vous souhaitez que cette application Web utilise sa propre base de données privée, cette option peut être sélectionnée et le modèle d'une base de données saisi dans le champ de texte adjacent. Par exemple, vous pouvez saisir ${DB}_phpnuke . Si la base de données résultante n'existe pas, elle sera créée dans le cadre du processus d'installation de l'application Web.
Type de base de données
Le type de base de données utilisé pour les données des applications Web, le cas échéant.
