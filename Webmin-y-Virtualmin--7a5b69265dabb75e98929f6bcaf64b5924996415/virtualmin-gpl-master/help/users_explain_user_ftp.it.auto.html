Crea utente FTP
Crea un nuovo utente Unix dedicato all'accesso FTP nella directory del sito web, assegnando automaticamente l'utente e l'ID di gruppo del proprietario del server virtuale. Questo utente specializzato è stato creato pensando all'accesso FTP. Le quote per questo tipo di account sono condivise con il proprietario del server virtuale. In particolare, le funzionalità di posta elettronica non sono disponibili per questo tipo di utente.
