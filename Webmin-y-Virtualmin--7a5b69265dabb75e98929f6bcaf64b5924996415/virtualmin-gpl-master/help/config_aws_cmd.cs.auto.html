Úplná cesta k příkazu aws pro operace S3
Pokud je ve vašem systému nainstalován příkaz aws a toto pole je nastaveno na úplnou cestu k němu (jako /usr/bin/aws ), Virtualmin jej zavolá, aby provedl operace S3 pro zálohy, jako jsou nahrané a stahované soubory. Protože byl tento příkaz vyvinut společností Amazon, je spolehlivější než rozhraní S3 API zabudované do Virtualmin.
