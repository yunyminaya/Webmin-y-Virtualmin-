Apache Setup und Konfiguration aktiviert?
Diese Option aktiviert oder deaktiviert die Erstellung von Apache VirtualNameHost-Einträgen und richtet alle anderen Funktionen des virtuellen Hosts in Ihrer Apache-Konfigurationsdatei ein. Wenn Sie einen alternativen Webserver verwenden, können Sie diese Option deaktivieren. Möglicherweise können Sie die Plugin-Modul-Oberfläche verwenden, um Unterstützung für Ihren bevorzugten Server zu erstellen (obwohl derzeit keine anderen Webserver unterstützt werden), oder Sie können ein externes Skript zusammen mit Variablen aufrufen, die aus den Informationen stammen, die Virtualmin unter bereitgestellt werden Zeit für die Erstellung einer Domain, um Ihren bevorzugten Webserver einzurichten.
