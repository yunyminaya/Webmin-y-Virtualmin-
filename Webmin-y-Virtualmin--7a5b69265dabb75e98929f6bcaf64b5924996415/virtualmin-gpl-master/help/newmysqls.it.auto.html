Server MySQL
Questa pagina consente di definire server MySQL aggiuntivi su sistemi remoti o installazioni aggiuntive sul sistema, per l'utilizzo da parte dei domini Virtualmin. Questo può essere usato per scaricare l'hosting MySQL su un server diverso o per supportare più versioni di MySQL per client diversi.

Per aggiungere un nuovo server MySQL remoto, utilizzare per nella parte inferiore della pagina. Per rimuovere un server precedentemente definito dall'elenco, selezionare la casella accanto ad esso nell'elenco e fare clic sul pulsante Rimuovi selezionati .
