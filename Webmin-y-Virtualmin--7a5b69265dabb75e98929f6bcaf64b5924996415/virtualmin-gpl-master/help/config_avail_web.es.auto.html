Servidor web Apache (para host virtual)
Habilite esta opción si el propietario del dominio debe tener la capacidad de editar la configuración de Apache para su dominio. Si bien este nivel de control puede resultar abrumador para los usuarios sin experiencia, proporciona una gran flexibilidad para los experimentados. También es posible, con modificaciones a la configuración de control de acceso predeterminada, permitir el acceso directo a la sección de dominios virtuales del archivo de configuración de Apache. Aunque existen protecciones para evitar acciones maliciosas, es posible que no sean perfectas. De manera predeterminada, los privilegios otorgados generalmente son seguros para su uso en un entorno de alojamiento compartido.
