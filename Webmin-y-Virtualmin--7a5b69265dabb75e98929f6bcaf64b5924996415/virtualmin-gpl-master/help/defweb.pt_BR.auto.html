Site padrão para o endereço IP?
Quando esse campo é definido como Sim , o site deste servidor virtual será atendido pelo Apache quando receber uma solicitação que não corresponde a nenhum outro servidor virtual no sistema. Isso normalmente acontece se um usuário usa uma URL com um endereço IP ou um nome de host que é resolvido no seu sistema, mas não corresponde a nenhum domínio do Virtualmin.
