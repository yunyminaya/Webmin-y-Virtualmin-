Mode d'exécution du script Ruby
Cette option détermine si les scripts Ruby sont exécutés en tant qu'utilisateur du serveur Web Apache (généralement httpd ou apache ) ou en tant qu'utilisateur Unix propriétaire du serveur virtuel. Le premier est plus rapide car cela signifie qu'aucun processus supplémentaire n'a besoin d'être démarré pour exécuter le code Ruby, mais moins sécurisé car les scripts lisent et écrivent des fichiers en tant qu'utilisateur autre que le propriétaire du domaine.
