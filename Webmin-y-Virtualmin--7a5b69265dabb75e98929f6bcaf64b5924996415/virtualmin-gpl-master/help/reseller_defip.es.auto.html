Dirección IPv4 para servidores basados en nombres
Esta opción se puede usar para que los servidores virtuales creados para este revendedor usen una dirección IP que no sea la predeterminada del sistema, que generalmente es compartida por todos los sitios web virtuales basados en nombres. Esto puede ser útil para los servidores de "creación de marca" creados por diferentes revendedores, al hacer que la IP de cada revendedor resuelva en un nombre de host diferente.

La IP que ingrese ya debe estar activa en el sistema, lo que se puede hacer en el módulo de configuración de red de Webmin. La IP no se usará para servidores virtuales con sus propias IP privadas, ya que cada una de ellas siempre obtiene una dirección distinta.
