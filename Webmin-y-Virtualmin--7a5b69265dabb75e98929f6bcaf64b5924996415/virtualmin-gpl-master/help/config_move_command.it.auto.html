Comando da eseguire per spostare la home directory di un dominio
Per impostazione predefinita, Virtualmin utilizzerà il comando mv per spostare la directory home di un dominio quando necessario, ad esempio quando un dominio viene rinominato. Tuttavia, ciò potrebbe non essere appropriato nei casi in cui ciascun server virtuale si trova sul proprio filesystem, come un mount ZFS.

In casi come questo, è possibile impostare questo campo su un comando o uno script a cui verranno passate le directory home vecchie e nuove come parametri. Avrà anche accesso a tutte le variabili d'ambiente normalmente passate ai comandi pre e post modifica.
