Conexiones máximas para usuarios de buzones
Este campo controla el número de conexiones simultáneas a MySQL que puede hacer un buzón con acceso a la base de datos. El valor predeterminado es permitir un número ilimitado de conexiones, pero dado que cada una consume memoria dentro del proceso del servidor MySQL, es posible que desee limitar el total de conexiones en un sistema ocupado.

El límite establecido aquí solo se aplicará cuando se cree un nuevo buzón o cuando se habilite un inicio de sesión MySQL para un buzón existente.
