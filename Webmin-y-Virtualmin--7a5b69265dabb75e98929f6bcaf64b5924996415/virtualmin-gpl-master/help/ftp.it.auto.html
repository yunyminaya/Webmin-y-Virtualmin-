Configurare FTP virtuale basato su IP?
Quando questa funzione è abilitata, verrà creato un server virtuale ProFTPd per questo server virtuale Virtualmin. Questo può essere usato per configurare un sito FTP anonimo privato per questo server. Tuttavia, richiede che il server abbia il suo indirizzo IP.

Questa funzione è necessaria solo se si desidera fornire l'accesso basato su nomi a un server FTP anonimo. Quasi nessuno ha bisogno di abilitare questa funzione (probabilmente significa che tu). Non è necessario abilitare questa funzione se si desidera solo fornire l'accesso FTP per l'account del proprietario del dominio, poiché qualsiasi utente con un accesso shell avrà automaticamente accesso FTP alla propria directory home.

In altre parole: quasi sicuramente non ti serve questo, anche se pensi che forse lo fai.
