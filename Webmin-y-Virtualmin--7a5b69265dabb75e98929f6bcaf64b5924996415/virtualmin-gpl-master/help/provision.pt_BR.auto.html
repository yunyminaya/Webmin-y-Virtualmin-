Cliente de serviços Cloudmin
Esta página permite configurar o Virtualmin para conectar-se a um servidor dos Serviços Cloudmin, para que os bancos de dados MySQL, zonas DNS e outros serviços possam ser transferidos para sistemas remotos. Idealmente, ele deve ser usado antes da criação de servidores virtuais, pois a mistura de bancos de dados e zonas remotos e servidos localmente é ineficiente.

Os campos que precisam ser preenchidos são:

Mestre de serviços Cloudmin
Este nome de host de uma máquina Cloudmin com o plug-in Cloudmin Services instalado. Isso pode ser executado por você, sua empresa de hospedagem ou provedor de VPS.

Número da porta
Isso quase sempre deve ser definido como 10000, que é a porta padrão Webmin / Cloudmin.

Nome de usuário para o servidor
Este campo deve conter o nome de usuário de uma conta de proprietário do sistema no mestre dos Serviços Cloudmin que tem permissão para criar bancos de dados, zonas DNS ou outros serviços.
Recursos para hospedar
Esta seção permite selecionar quais recursos do Virtualmin serão transferidos para sistemas remotos. Somente aquelas que sua conta no sistema mestre pode provisionar podem ser selecionadas.
