¿El sitio web coincide con todos los subdominios?
Cuando esta opción está habilitada, las solicitudes de cualquier URL bajo el dominio (como http://abc.example.com ) devolverán el sitio web del servidor virtual. Esto puede ser útil si aloja una aplicación que puede devolver contenido diferente según el nombre de dominio solicitado, como Wordpress MU.

Si el dominio DNS del servidor virtual está alojado en este sistema, Virtualmin también agregará el registro DNS comodín * cuando se seleccione Sí .
