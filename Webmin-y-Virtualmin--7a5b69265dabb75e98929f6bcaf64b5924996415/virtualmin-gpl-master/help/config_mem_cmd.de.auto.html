Befehl zum Abrufen der Speichernutzung
Standardmäßig erhält Virtualmin von den entsprechenden Befehlen für Ihr Betriebssystem vollständigen und freien Speicher. Wenn diese jedoch nicht korrekt sind oder nicht die Verwendung widerspiegeln, die Benutzer sehen sollen (z. B. in einer VPS-Umgebung), kann dieses Feld auf einen Befehl festgelegt werden, der die folgenden Werte ausgeben muss, einen pro Zeile:

  1. Gesamter realer Speicher in Bytes
  2. Freier realer Speicher in Bytes
  3. Gesamter Swap-Speicher in Bytes oder 0, wenn keiner
  4. Freier Swap-Speicher in Bytes
