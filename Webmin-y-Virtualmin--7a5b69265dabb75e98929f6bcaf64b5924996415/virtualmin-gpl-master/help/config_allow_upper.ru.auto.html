Принудительно вводить имена пользователей почтовых ящиков в нижнем регистре?
Если для этого поля установлено значение «Да» (как по умолчанию), всем пользователям почты / FTP будут назначаться имена пользователей в нижнем регистре независимо от того, в каком регистре они созданы. Если вы хотите предотвратить такое поведение, выберите «Нет», чтобы иметь почтовые ящики, созданные с точным введенным регистром. Помните, что для входа в систему Usermin и любого сервера POP3, IMAP или FTP необходимо использовать правильный регистр.

Поскольку большинство почтовых серверов не могут обрабатывать доставку пользователям с нестрочными именами, для этого параметра всегда следует устанавливать значение Да, если только вы не создаете учетные записи для использования по FTP, DAV или SVN.
