Können Home-Verzeichnisse für Benutzer ausgewählt werden?
Wenn diese Option aktiviert ist, kann der Domäneninhaber den Benutzern ein anderes Ausgangsverzeichnis als das Standardverzeichnis zuweisen. Dies kann hilfreich sein, wenn Domänenbesitzer zusätzliche Dienste wie einen Anwendungsserver (z. B. Zope, Webrick usw.) installieren können und einen anderen Benutzer für den App-Server verwenden möchten. Im Allgemeinen benötigen nur Benutzer mit einem relativ hohen Maß an Vertrauenswürdigkeit diese Flexibilität.
