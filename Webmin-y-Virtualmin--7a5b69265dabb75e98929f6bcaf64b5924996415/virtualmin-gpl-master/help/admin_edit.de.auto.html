Bearbeitungsfunktionen für virtuelle Server
Diese Kontrollkästchen steuern genau, welche virtuellen Serverfunktionen dieser zusätzliche Administrator verwalten kann. Möglicherweise möchten Sie einige deaktivieren, um einen Administrator zu erstellen, der auf eine bestimmte Aufgabe beschränkt ist, z. B. die Aliasverwaltung.
