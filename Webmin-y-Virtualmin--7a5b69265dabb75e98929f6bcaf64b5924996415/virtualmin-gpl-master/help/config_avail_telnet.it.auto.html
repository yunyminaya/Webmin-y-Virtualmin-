Accesso SSH / Telnet
Questa opzione specifica se l'utente proprietario del dominio può utilizzare il modulo Webmin di accesso SSH / Telnet. Questo modulo è un'applet Java che accede al server tramite SSH o Telnet. Non funziona correttamente con tutti i browser, a causa di incompatibilità Java. La disabilitazione di questo non impedisce l'accesso SSH al server da parte del proprietario del dominio, ma semplicemente l'accesso al modulo Login SSH / Telnet di Webmin.
