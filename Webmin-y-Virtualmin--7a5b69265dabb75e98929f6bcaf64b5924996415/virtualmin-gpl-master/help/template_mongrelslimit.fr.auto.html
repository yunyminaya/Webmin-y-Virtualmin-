Limiter le nombre d'instances de Mongrel
Lors de l'installation des scripts Ruby on Rails, l'administrateur du serveur peut sélectionner la façon dont mon serveur Mongrel traite pour commencer à gérer les requêtes vers le chemin du script - si le module Apache mod_proxy_balancer est installé. Pour empêcher un nombre excessif de processus de serveur Mongrel de surcharger le système, cette option peut être utilisée pour définir la limite par défaut du nombre que les administrateurs de domaine sont autorisés à configurer à partir de Virtualmin.
