¿Redirigir admin.domain a Usermin?
Cuando este campo se establece en Sí (como lo está por defecto), cualquier servidor virtual nuevo tendrá una entrada de admin agregada a sus dominios DNS, con la misma dirección IP que el sitio web del servidor. Cuando se accede a http://admin.thedomain.com/ , el navegador será redirigido a Virtualmin en el puerto 10000 del mismo dominio, para acceder a la interfaz web de administración.
