Plantilla para el directorio de aplicaciones web predeterminado
De forma predeterminada, Virtualmin instalará nuevas aplicaciones web en un subdirectorio con el mismo nombre que el código del script, como wordpress . Este campo le permite ingresar una plantilla para un directorio predeterminado diferente, como webapps/${SCRIPTNAME} . También se pueden utilizar todas las variables de plantilla habituales como ${DOM} y ${USER}.
