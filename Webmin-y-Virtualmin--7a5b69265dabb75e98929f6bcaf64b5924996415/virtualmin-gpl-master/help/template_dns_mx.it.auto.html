Nome host per il record MX
Per impostazione predefinita, il record DNS MX (che controlla dove viene consegnata l'e-mail) per i nuovi domini sarà mail.${DOM}, a cui verrà assegnato lo stesso indirizzo IP del sito Web del server virtuale. Tuttavia, puoi utilizzare questo campo per specificare un nome host del server di posta diverso: in genere viene utilizzato per avere un record MX coerente in tutti i domini. Assicurati di inserire un nome host che si risolva nel tuo sistema Virtualmin, altrimenti l'e-mail non verrà consegnata.

Puoi anche utilizzare le variabili standard di Virtualmin come ${DOM} per specificare un record MX con un nome diverso nel dominio in fase di creazione.
