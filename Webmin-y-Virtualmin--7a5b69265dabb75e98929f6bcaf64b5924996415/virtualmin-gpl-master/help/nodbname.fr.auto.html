Peut choisir des noms de base de données?
Lorsque cette option est définie sur Oui , le propriétaire de ce serveur virtuel sera autorisé à sélectionner le nom par défaut des bases de données pour les nouveaux sous-serveurs qu'il crée. Vous ne devez la définir sur Non que si vous souhaitez un contrôle complet sur les noms des bases de données sur votre système.
