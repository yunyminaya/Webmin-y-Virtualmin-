Limitar el redireccionamiento al nombre de host
De forma predeterminada, las redirecciones se aplicarán independientemente del nombre de host URL utilizado para acceder al sitio web. Sin embargo, este campo se puede utilizar para limitar la redirección a un host específico. Esto puede resultar útil para redirigir www.dominio.com a dominio.com, por ejemplo, o cambiar la redirección para un dominio de alias específico.
