Имя главного сервера DNS
По умолчанию Virtualmin будет использовать имя хоста системы в основных записях NS и SOA для новых создаваемых им доменов DNS. Однако, если имя хоста не является допустимым DNS-именем или оно не подходит для ваших целей, этот параметр можно использовать для ввода другого основного сервера имен. Он должен соответствовать внешне доступному IP-адресу вашей системы или адресу подчиненной системы серверов имен для доменов, созданных с использованием этого шаблона.
