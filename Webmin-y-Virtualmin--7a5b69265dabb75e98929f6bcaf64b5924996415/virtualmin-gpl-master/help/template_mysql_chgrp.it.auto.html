Impostare la proprietà di gruppo dei file di database MySQL?
Quando questa opzione è abilitata, tutti i nuovi database MySQL avranno la proprietà del gruppo impostata sui loro file di dati (in genere in /var/lib/mysql ) in modo da essere inclusi nei limiti di quota Unix dei loro server virtuali. Se non abilitato, l'utilizzo del disco del database non sarà limitato e sarà possibile che un server virtuale superi la sua allocazione del disco consentita.
