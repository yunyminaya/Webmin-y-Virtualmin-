Adicionar usuário Apache ao grupo Unix para novos servidores?
Esta opção determina se o usuário que o Apache executa será adicionado ao novo grupo de hosts virtuais. Quando ativado, as permissões podem ser significativamente reforçadas nos diretórios iniciais dos usuários. Quando permissões mais restritas (ou seja, 750 ) são combinadas com suEXEC, torna impossível para os proprietários de domínio verem os diretórios e arquivos de outros proprietários de domínio. Esta opção é recomendada em qualquer plataforma que suporte grupos secundários.
