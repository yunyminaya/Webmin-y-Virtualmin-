MySQL Servers
This page allows you to define additional MySQL servers on remote systems or additional installs on the system, for use by Virtualmin domains. This can be used to offload MySQL hosting to a different server, or to support multiple versions of MySQL for different clients.

To add a new remote MySQL server, use the for at the bottom of the page. To remove a previously defined server from the list, check the box next to it in the list and click the Remove Selected button.
