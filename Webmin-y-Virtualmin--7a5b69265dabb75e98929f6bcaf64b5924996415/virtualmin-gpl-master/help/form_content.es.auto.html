¿Crear página web inicial?
Para crear una página web inicial para el nuevo servidor virtual, puede elegir no crear ninguna página web seleccionando "No" u optar por una página de sitio web predeterminada de Virtualmin eligiendo la opción "Sí, con la plantilla de página predeterminada del sitio web".

Como alternativa, seleccionar "Sí, con contenido a continuación" le permitirá ingresar HTML o texto personalizado para usar como contenido de la página web.
