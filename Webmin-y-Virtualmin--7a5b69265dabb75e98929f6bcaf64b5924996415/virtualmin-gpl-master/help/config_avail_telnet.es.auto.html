Inicio de sesión SSH / Telnet
Esta opción especifica si el usuario propietario del dominio puede usar el módulo SSH / Telnet Login Webmin. Este módulo es un applet de Java que inicia sesión en el servidor mediante SSH o Telnet. No funciona correctamente con todos los navegadores, debido a las incompatibilidades de Java. Tenga en cuenta que deshabilitar esto no impide el acceso SSH al servidor por parte del propietario del dominio, simplemente el acceso al módulo de inicio de sesión SSH / Telnet de Webmin.
