Home directory aanmaken aanzetten?
Deze optie bepaald of het aanmaken van een home directory voor een nieuwe virtuele server altijd word gedaan (en de optie verstopt is), of dat het optioneel is wanneer een server word aangemaakt. Omdat een home directory bijna altijd nodig is, of u maakt een alleen een DNS domein, dan is de Altijd optie over het algemeen het beste.
