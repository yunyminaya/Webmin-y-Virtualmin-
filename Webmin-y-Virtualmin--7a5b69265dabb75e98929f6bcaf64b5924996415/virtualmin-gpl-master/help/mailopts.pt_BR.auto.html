Configurações de email
Esta página é para definir configurações relacionadas a email para um servidor virtual. Se o seu sistema suportar e-mails enviados em Cco (e está ativado na página Configuração do módulo), a seção E -mail de saída em Cco para pode ser usada para selecionar um endereço para o qual copiar e-mails.

Se o servidor virtual for um domínio de alias, o campo Modo de alias de email controla se os alias de email são copiados do domínio de destino ou se um único alias de catchall é criado para encaminhar todos os emails para o destino. O modo anterior não está disponível em todos os servidores de email, mas é superior ao usar o Postfix, pois permite que emails para endereços inválidos sejam devolvidos no nível da conversa SMTP.
