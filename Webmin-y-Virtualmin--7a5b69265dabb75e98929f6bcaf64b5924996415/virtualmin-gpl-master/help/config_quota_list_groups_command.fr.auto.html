Commande pour répertorier les quotas de groupe
Lorsque des commandes de quota externe sont utilisées, le programme défini dans cette option sera appelé lorsque Virtualmin doit trouver des quotas de groupe existants. Il n'est appelé avec aucun paramètre, mais doit générer une ligne de texte par groupe, chaque ligne contenant des champs séparés par des espaces. Les champs de chaque ligne doivent être:
  1. Le nom du groupe Unix.
  2. Quota actuel utilisé dans des blocs de 1 ko.
  3. Le quota souple en blocs de 1 Ko, ou 0 pour illimité.
  4. Le quota dur en blocs de 1 Ko, ou 0 pour illimité.
