Interface de Rede
Se você deseja configurar um host virtual baseado em IP (necessário para sites SSL ou FTP virtual), essa opção deve ser definida como Interface virtual com IP e um endereço não utilizado inserido. O Virtualmin ativará esse IP no seu sistema quando o servidor for criado e o associará ao servidor.

Se a opção Compartilhado estiver selecionada, o endereço principal da interface de rede do sistema será usado para o site e o domínio DNS deste servidor.
