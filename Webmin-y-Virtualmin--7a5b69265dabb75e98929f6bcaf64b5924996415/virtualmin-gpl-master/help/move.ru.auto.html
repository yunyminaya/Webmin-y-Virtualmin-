Переместить виртуальный сервер
Перемещение виртуального сервера меняет администрацию домена, который контролирует его. Реальные функции сервера не затрагиваются, но теперь он может управляться другим администратором и учитывает любые ограничения на домены, псевдонимы и базы данных для этого нового администратора.

Эта страница может использоваться для выполнения трех различных типов операций, в зависимости от того, какой тип сервера вы редактируете и что вы хотите с ним делать. Они есть :

Переместить субсервер под другого родителя
При редактировании подчиненного сервера вы можете выбрать другой родительский домен в меню, чтобы изменить его владельца. Веб-сайт сервера, домен DNS, почтовые ящики и т. Д. Не будут затронуты, но теперь он будет находиться под контролем владельца нового выбранного родителя. Его домашний каталог будет перемещен под домашний каталог нового родителя, а право собственности на все файлы будет изменено на пользователя и группу нового родителя.

Переместить родительский сервер под другого родителя
При редактировании родительского сервера вы можете преобразовать его в субсервер, используя эту страницу. Это следует делать с осторожностью, поскольку некоторые параметры, которые существуют только для родительских серверов (например, пароль, квоты и ограничения), будут потеряны, поскольку они теперь наследуются от нового родительского сервера. Кроме того, любой пользователь Unix, группа Unix или пользователь Webmin, связанный с доменом, будет удален.

Преобразовать субсервер в родительский сервер
При редактировании подчиненного сервера у вас есть возможность преобразовать его в родительский сервер, принадлежащий новому администратору сервера. Необходимо ввести имя пользователя и пароль для нового администратора. После завершения перемещения все файлы в новом домашнем каталоге сервера будут принадлежать новому пользователю. Дисковые квоты, ограничения полосы пропускания и ограничения владельца сервера для перемещенного сервера будут скопированы из старого родительского объекта, но впоследствии могут быть отредактированы отдельно.

Изменение цели псевдонима
Если выбранный сервер является псевдонимом другого сервера, эту форму можно использовать для указания на другую цель. После завершения перемещения веб-доступ к домену вернет содержимое из новой цели, и электронная почта в домен будет перенаправлена в почтовые ящики новой цели.
