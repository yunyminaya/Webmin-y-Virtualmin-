¿Agregar remitentes de correo no deseado enviados a Hamtrap a la lista blanca?
Por defecto, cuando el alias hamtrap está habilitado para un servidor virtual, cualquier correo electrónico enviado a él se agregará a la base de datos de aprendizaje de SpamAssassin para el usuario que lo reenvió. Sin embargo, esta opción se puede usar para agregar también los remitentes originales de los mensajes a las listas blancas de todo el sistema o de nivel de dominio, para que nunca se clasifiquen como spammers en el futuro.

Esto puede ser útil si tiene remitentes legítimos que se clasifican incorrectamente como spammers.
