Ajustes del correo electrónico
Esta página es para configurar ajustes relacionados con el correo electrónico para un servidor virtual. Si su sistema admite el correo electrónico enviado de BCCing (y está habilitado en la página de Configuración del módulo), la sección de correo electrónico saliente de BCC puede usarse para seleccionar una dirección para copiar el correo.

Si el servidor virtual es un dominio de alias, el campo Modo de alias de correo controla si los alias de correo se copian del dominio de destino, o si se crea un solo alias general para reenviar todo el correo electrónico al destino. El modo anterior no está disponible con todos los servidores de correo, pero es superior cuando se usa Postfix, ya que permite que el correo a direcciones no válidas se devuelva en el nivel de conversación SMTP.
