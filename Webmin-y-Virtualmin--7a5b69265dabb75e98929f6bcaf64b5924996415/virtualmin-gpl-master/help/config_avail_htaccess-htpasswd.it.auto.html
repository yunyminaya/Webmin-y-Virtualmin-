Directory Web protette (nella directory principale)
Questa opzione specifica se il proprietario del dominio avrà accesso al modulo Webmin delle directory Web protette, che consente la creazione di una o più directory protette da password. Questo, ovviamente, può essere fatto utilizzando file .htaccess, ma molti utenti non sanno come creare o utilizzare file .htaccess e questo modulo fornisce un mezzo web per eseguire la stessa attività.
