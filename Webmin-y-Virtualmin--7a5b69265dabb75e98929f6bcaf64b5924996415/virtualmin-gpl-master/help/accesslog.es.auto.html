Archivo de registro de acceso al servidor web
Este campo establece el archivo en el que se escriben los registros de acceso de Apache para este servidor virtual. Cada solicitud HTTP atendida por el dominio de este sitio web se escribirá en este archivo como una línea separada. Solo debe cambiarse si está moviendo el registro a un directorio diferente, como /var/log/virtualmin o el directorio de inicio del servidor.
