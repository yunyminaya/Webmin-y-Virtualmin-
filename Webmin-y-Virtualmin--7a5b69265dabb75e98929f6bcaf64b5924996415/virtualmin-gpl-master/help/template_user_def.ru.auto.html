Добавить пользователя Apache в группу Unix для новых серверов?
Этот параметр определяет, будет ли пользователь, под которым работает Apache, добавлен в новую группу виртуальных хостов. Если этот параметр включен, разрешения могут быть значительно ужесточены для домашних каталогов пользователей. Когда более строгие разрешения (например, 750 ) сочетаются с suEXEC, владельцы доменов не могут видеть каталоги и файлы других владельцев доменов. Этот вариант рекомендуется использовать на любой платформе, поддерживающей вторичные группы.
