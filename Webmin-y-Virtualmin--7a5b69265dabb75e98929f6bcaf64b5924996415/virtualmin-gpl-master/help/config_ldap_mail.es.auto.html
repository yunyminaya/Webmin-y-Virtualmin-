Añadir atributo de correo a los usuarios LDAP?
Al almacenar usuarios de Virtualmin en LDAP, esta configuración también le permite agregar sus direcciones de correo electrónico a la base de datos LDAP. Esto es útil si su servidor de correo o filtro antispam usa la misma base de datos para entregar el correo electrónico.

Las opciones disponibles son:

  • Sí, utilizando solo el atributo de mail
    Las direcciones de correo electrónico principal y adicional de cada usuario se almacenarán en este atributo como valores separados.

  • Sí usando mail y mailAlternateAddress
    Las direcciones principales de los usuarios se almacenarán en el atributo de mail y las direcciones adicionales en mailAlternateAddress .

  • No
    Las direcciones de correo electrónico nunca se almacenarán en LDAP.

Cuando se establece en Sí , Webmin agregará atributos de mail LDAP a cada usuario que cree o modifique para sus direcciones de correo electrónico principal y adicional. Esto es útil si está utilizando la misma base de datos LDAP con un servidor de correo que usa ese atributo para averiguar dónde entregar el correo electrónico.
