Открытый ключ SSH
Эта опция позволяет предоставить доступ по SSH вновь созданному пользователю с использованием аутентификации с открытым ключом. Возможные варианты:

Нет ключа по умолчанию
Не настраивайте ключ SSH. Будет разрешен вход только с паролем.

Используйте открытый ключ ниже
Вход по SSH будет разрешен с использованием закрытого ключа, соответствующего открытому ключу, введенному в текстовое поле ниже.
