Configurazione PHP (per i file php.ini del dominio)
Se abilitata, questa opzione consente ai proprietari di server virtuali di accedere al modulo di configurazione PHP. Questo sarà limitato in modo che possano gestire solo i file php.ini nei propri server virtuali, per i quali è abilitata l'esecuzione di PHP tramite CGI.

Se un dominio utilizza mod_php per eseguire script PHP, questa opzione controlla anche se il proprietario del server virtuale è autorizzato a configurare le impostazioni relative a PHP in Apache.
