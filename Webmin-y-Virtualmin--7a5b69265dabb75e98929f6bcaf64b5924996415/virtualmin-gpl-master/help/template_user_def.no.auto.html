Legge til Apache-bruker til Unix-gruppen for nye servere?
Dette alternativet bestemmer om brukeren som Apache kjører som skal legges til i den nye virtuelle vertsgruppen. Når den er aktivert, kan tillatelsene strammes betraktelig på brukerens hjemmekataloger. Når strammere tillatelser (dvs. 750 ) kombineres med suEXEC, gjør det det umulig for domeneeiere å se andre domeneeiers kataloger og filer. Dette alternativet anbefales på alle plattformer som støtter sekundære grupper.
