Usermin'i aynı SSL sertifikasını kullanacak şekilde yapılandır?
Bu şablon yalnızca SSL özellikli web siteleri için kullanılacaksa, bu seçeneği etkinleştirebilirsiniz. Usermin daha sonra SSL web sitesinin kullandığı sertifikayı kullanacak, böylece kullanıcılar kendi ana bilgisayar adlarını kullanarak oturum açtıklarında sertifika uyarıları oluşturmayacaktır.
