Protocoles SSL Apache pour permettre
Par défaut, lorsque Virtualmin configure un nouveau site Web SSL, il configure Apache pour rejeter les versions de protocole SSL non sécurisées (TLSv1.1 et inférieures). Cependant, vous pouvez spécifier votre propre liste de protocoles en les répertoriant ici. Les protocoles valides sont SSLv2 , SSLv3 , TLSv1 , TLSv1.1 et ainsi de suite. Pour rejeter un protocole, mettez un - avant le nom. Pour autoriser tous les protocoles, entrez all .
