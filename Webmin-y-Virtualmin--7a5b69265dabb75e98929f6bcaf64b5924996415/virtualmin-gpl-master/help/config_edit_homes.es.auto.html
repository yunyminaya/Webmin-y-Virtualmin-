¿Se pueden seleccionar directorios de inicio para los usuarios?
Si está habilitado, el propietario del dominio puede optar por proporcionar a los usuarios un directorio de inicio diferente al predeterminado. Puede ser útil si los propietarios de dominios pueden instalar servicios adicionales, como un servidor de aplicaciones (es decir, Zope, Webrick, etc.), y les gustaría poder usar un usuario diferente para el servidor de aplicaciones. En general, solo los usuarios que tienen un nivel relativamente alto de confiabilidad necesitarán este tipo de flexibilidad.
