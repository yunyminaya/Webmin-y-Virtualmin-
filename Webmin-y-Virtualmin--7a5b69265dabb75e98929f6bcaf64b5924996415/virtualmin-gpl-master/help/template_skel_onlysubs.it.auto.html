Modelli di file da sostituire solo
Se le variabili sostitutive nei contenuti? l'opzione è impostata su Sì , Virtualmin sostituirà le variabili in tutti i file ad eccezione delle immagini quando vengono copiate dalla directory skeleton. Tuttavia, ciò potrebbe interrompere gli script PHP o perl che utilizzano lo stesso formato per le variabili del linguaggio modello di Virtualmin.

Per evitare questo problema, puoi inserire un elenco di modelli di file in questo campo, come *.conf . Solo i file che corrispondono a quei pattern e non i pattern in Pattern file da non sostituire non avranno il loro contenuto modificato.
