Nome de usuário
Este campo define a parte do nome de usuário do login do usuário, que deve ser exclusivo no banco de dados. O login do usuário para acesso ao banco de dados será derivado do nome de usuário aqui inserido, combinado com o nome do domínio. Por exemplo, se você inserir o nome de usuário db-user1 e o domínio for example.com , o login resultante será db-user1@example.com .
