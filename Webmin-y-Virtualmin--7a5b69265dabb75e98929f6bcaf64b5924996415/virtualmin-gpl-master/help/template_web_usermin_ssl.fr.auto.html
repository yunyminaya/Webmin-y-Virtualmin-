Configurer Usermin pour utiliser le même certificat SSL?
Si ce modèle sera utilisé exclusivement pour les sites Web compatibles SSL, vous pouvez activer cette option. Usermin utilisera alors le même certificat que celui utilisé par le site Web SSL, il ne générera donc pas d'avertissements de certificat lorsque les utilisateurs se connecteront en utilisant leur propre nom d'hôte.
