Domainname zur Verwendung in Links
Wenn Virtualmin einen Link zur Website dieses virtuellen Servers anzeigt (z. B. nach der Installation eines Skripts), wird standardmäßig der Domänenname des Servers verwendet. Sie können dieses Feld jedoch stattdessen verwenden, um eine Aliasdomäne auszuwählen, die stattdessen in Links verwendet werden soll. Dies kann nützlich sein, wenn auf den Server nur über den Alias-Domänennamen extern zugegriffen werden kann.
