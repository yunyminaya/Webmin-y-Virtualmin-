API yardımcı komutu oluşturma yolu
Varsayılan olarak, Virtualmin, sisteminizde yazılabilir olana bağlı olarak, /usr/sbin veya /usr/local/sbin dizininde virtualmin komut satırı API yardımcı komut dosyasını oluşturur. Ancak, bu yapılandırma alanı, sitenize uygun /opt/bin/virtualmin gibi farklı bir konum seçmek için kullanılabilir.
