Доступные функции по умолчанию
Если для этого параметра установлено значение «Автоматически» (как по умолчанию), то для новых виртуальных серверов верхнего уровня будут установлены разрешенные функции на основе тех, которые были изначально включены при создании сервера. Если вы хотите предоставить владельцу сервера доступ к дополнительным функциям по умолчанию, выберите переключатель Выбранное ниже и установите флажки для функций, которые вы хотите предоставить.
