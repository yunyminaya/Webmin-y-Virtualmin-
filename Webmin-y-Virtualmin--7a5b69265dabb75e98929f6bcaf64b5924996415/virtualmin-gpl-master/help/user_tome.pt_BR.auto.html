Entregar para este usuário normalmente
Quando esta opção estiver marcada, o e-mail para este usuário será entregue na caixa de correio dele após passar por qualquer filtragem de spam. Normalmente, você não precisa marcar isso, pois acontece automaticamente se nenhum encaminhamento ou resposta automática estiver configurado. No entanto, se você configurar um endereço de encaminhamento ou resposta automática, mas ainda quiser que o usuário mantenha uma cópia do e-mail, esta opção deve ser selecionada.
