Criar usuários em lote
Esta página permite criar vários usuários de email ou FTP de uma vez a partir de um arquivo de texto. Este arquivo deve listar um usuário por linha e cada linha deve ter o seguinte formato:

username : realname : password : ftp-flag : email-flag : quota :[ extras ]:[ forwards ]:[ databases ]

Os significados dos campos são:

nome do usuário
O nome de login curto do usuário, como joe . Se necessário, o prefixo do domínio será anexado para criar o nome de login real.
nome real
O nome real do usuário, como Joe Smith .
senha
A senha inicial do usuário, em texto sem formatação.
ftp-flag
Se estiver definido como 1, o usuário poderá fazer login via FTP. Definir como 0 ou deixá-lo vazio impede o acesso ao FTP. Você também pode usar o caminho completo para um shell válido para usuários de caixas de correio, como /usr/bin/scponly .
sinalizador de email
Se estiver definido como 1, o endereço de email principal do usuário será ativado.
contingente
A cota de correio e diretório inicial do usuário, em blocos de 1 kB.
extras
Uma lista separada por vírgula de endereços de email extras para esse usuário. Eles devem estar nos domínios do Virtualmin que você gerencia.
encaminha
Uma lista separada por vírgula de destinos de encaminhamento de email para esse usuário (geralmente endereços como joe@otherdomain.com ).
bases de dados
Uma lista de bancos de dados separados por vírgula aos quais conceder acesso para esse usuário. Cada um deve ser um tipo de banco de dados ( mysql ou postgres ) e um nome de banco de dados, separado por um espaço.
