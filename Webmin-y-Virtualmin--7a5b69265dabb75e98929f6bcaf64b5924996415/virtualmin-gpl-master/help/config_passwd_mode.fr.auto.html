Type de champ de mot de passe
Ce paramètre contrôle l'apparence du champ de saisie du mot de passe pour un nouveau serveur virtuel. Les options disponibles sont:
Mot de passe généré aléatoirement
Un seul champ visible s'affiche, contenant un mot de passe généré aléatoirement sélectionné par Virtualmin. Vous pouvez cependant entrer un mot de passe différent.

Entrez le mot de passe une fois
Un champ unique pour entrer votre propre mot de passe s'affiche, avec le mot de passe réel caché.

Entrez le mot de passe deux
Deux champs pour saisir votre propre mot de passe s'affichent, toujours avec le mot de passe masqué. Les deux mots de passe doivent être identiques.
