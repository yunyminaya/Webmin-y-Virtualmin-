Connexions maximales pour les propriétaires de domaine
Ce champ contrôle le nombre de connexions simultanées à MySQL que le compte du propriétaire d'un serveur virtuel peut établir. Cela inclut les connexions des applications Web installées par Virtualmin, telles que Wordpress ou Drupal. La valeur par défaut est d'autoriser un nombre illimité de connexions, mais comme chacune consomme de la mémoire dans le processus du serveur MySQL, vous souhaiterez peut-être limiter le nombre total de connexions sur un système occupé.

La limite définie ici ne sera appliquée que lorsqu'un nouveau serveur virtuel est créé ou lorsqu'une connexion MySQL est activée pour un serveur existant.
