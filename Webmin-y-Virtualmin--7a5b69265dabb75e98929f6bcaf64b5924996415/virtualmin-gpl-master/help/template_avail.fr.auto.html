Modules Webmin disponibles pour les administrateurs de serveurs virtuels
Cette section vous permet de sélectionner les modules Webmin auxquels les propriétaires de serveurs virtuels auront accès pour les serveurs créés à l'aide de ce modèle. Les liens vers ces modules apparaissent dans le menu de gauche de Virtualmin, généralement sous la catégorie Services. Vous souhaiterez peut-être désactiver certains d'entre eux ou tous pour limiter les types de modifications de configuration plus avancées que les propriétaires de serveurs peuvent effectuer, et les limiter à la simple interface utilisateur Virtualmin.

Toutes les modifications apportées ici seront appliquées à tous les serveurs virtuels utilisant ce modèle immédiatement.
