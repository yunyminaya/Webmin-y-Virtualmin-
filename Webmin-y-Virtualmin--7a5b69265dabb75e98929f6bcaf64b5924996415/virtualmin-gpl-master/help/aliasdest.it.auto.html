Destinazioni Alias
Questa tabella di campi determina cosa succede all'email inviata a questo alias. Se vengono inserite più destinazioni, la posta verrà inviata a tutte. Ogni destinazione ha un tipo (scelto dal menu), che può essere uno dei seguenti:
  • Indirizzo email
    L'e-mail verrà inoltrata all'indirizzo inserito nel campo adiacente.

  • Indirizzi nel file
    L'e-mail verrà inoltrata a tutti gli indirizzi elencati nel file di testo il cui percorso è inserito nel campo adiacente.

  • Scrivi nel file
    L'email all'alias verrà aggiunta al file specificato. Saranno incluse tutte le intestazioni e il corpo grezzo non decodificato.

  • Feed al programma
    Il programma inserito nel campo adiacente verrà eseguito e inviato via e-mail all'alias passato come input.

  • Risposta automatica dal file
    Quando l'e-mail viene ricevuta da questo alias e il messaggio di risposta automatica verrà rispedito al mittente. Il testo di questa risposta automatica verrà preso dal file inserito nel campo di testo adiacente.

  • Applica file di filtro
    Le e-mail inviate all'alias verranno elaborate in base alle regole nel file di filtro specificato, che potrebbe causare l'inoltro ad altri indirizzi o file a seconda del contenuto effettivo del messaggio.

  • Cassetta postale dell'utente
    L'e-mail verrà recapitata direttamente alla casella di posta dell'utente immessa nel campo di testo adiacente. Questo eviterà qualsiasi indirizzo di inoltro che l'utente ha impostato. Tuttavia, è necessario inserire il nome utente di accesso POP3 completo, come joe.foo anziché solo joe .

Se si scelgono gli indirizzi nel file , Autoreply dal file o Applica i tipi di destinazione del file di filtro , quando si modificano nuovamente i collegamenti alias etichettati Modifica .. verranno visualizzati accanto a tali destinazioni per modificare il contenuto del file specificato.
