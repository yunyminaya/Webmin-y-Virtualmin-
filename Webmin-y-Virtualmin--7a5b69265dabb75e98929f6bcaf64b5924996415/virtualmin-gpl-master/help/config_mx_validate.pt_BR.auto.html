Validar registros MX para domínios com email?
Quando essa opção é definida como Sim (como é o padrão), os registros MX nos servidores virtuais ativados por email serão validados para garantir que apontem para o sistema Virtualmin. Normalmente, é isso que você deseja, a menos que você tenha um filtro de spam ou outro encaminhador que aceite emails para seus domínios antes de encaminhá-los para o sistema Virtualmin. Nesse caso, você deve selecionar Não para evitar erros de validação falsa.
