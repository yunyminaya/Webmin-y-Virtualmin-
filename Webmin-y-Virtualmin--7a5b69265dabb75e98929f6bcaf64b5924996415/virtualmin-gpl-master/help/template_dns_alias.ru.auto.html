Отдельно редактируемые записи DNS для псевдонимов доменов?
По умолчанию записи DNS доменов-псевдонимов копируются из целевого домена, что означает, что их нельзя редактировать отдельно. Установка этого параметра на "Да" позволяет вновь созданным доменам-псевдонимам иметь собственный независимый набор записей. Однако это вряд ли будет полезно для большинства пользователей.
