Autoriser l'utilisation des inclusions côté-serveur?
Inclusions côté-serveur sont une caractéristique de serveur web qui permet aux directives macro simples pour être utilisés dans les fichiers HTML. Ils peuvent être utilisés pour inclure d'autres fichiers HTML, définir et afficher le contenu des variables, et de contrôler ce contenu apparaît sur la page avec déclarations if. Ils offrent un moyen plus simple de créer des pages dynamiques que d'un langage de script complet comme PHP ou Perl.

Pour plus d'informations, consultez la documentation Apache à http://httpd.apache.org/docs/current/howto/ssi.html
