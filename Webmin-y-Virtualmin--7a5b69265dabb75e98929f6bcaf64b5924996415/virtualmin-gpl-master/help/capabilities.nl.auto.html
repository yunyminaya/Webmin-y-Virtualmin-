Kan virtuele server bewerken
Maakt het bewerken van virtuele serverfuncties, het wachtwoord en de beschrijving mogelijk.
Kan gebruikers beheren
Hiermee kunnen mailbox-, FTP- en databasegebruikers worden bekeken, bewerkt, aangemaakt en verwijderd.
Kan aliassen beheren
Hiermee kunnen e-mailaliassen worden bekeken, bewerkt, gemaakt en verwijderd.
Kan databases beheren
Hiermee kunnen extra MySQL- en PostgreSQL-databases aan deze virtuele server worden toegevoegd of ervan worden verwijderd.
Kan webapps installeren
Hiermee kunnen webapps van derden op deze virtuele server worden geïnstalleerd.
Kan IP-adres wijzigen
Als deze server een privé IP-adres heeft, kunt u dit met deze optie wijzigen.
Kan beheerde SSL-certificaten
Hiermee kunnen het SSL-certificaat en de sleutel voor deze server worden geüpload of opnieuw worden gegenereerd.
Kan het doorstuurframe bewerken
Voor websites die alleen doorsturen, kunt u met deze optie de bestemming en HTML voor het doorstuurframe bewerken.
Kan extra beheerders beheren
Hiermee kunnen extra beheerdersaccounts van servereigenaren worden bekeken, aangemaakt, bewerkt en verwijderd.
Kan de bezorging van spam en virussen configureren
Hiermee kunnen bezorgingsbestemmingen en scores voor spam en virussen worden bewerkt.
Kan PHP-versies selecteren
Voor domeinen met PHP die via CGI of fastCGI worden uitgevoerd, kunnen PHP-versies per map worden ingesteld.
Kan e-mailinstellingen bewerken
Maakt configuratie van automatische BCCing, aliaskopiemodus en andere e-mailgerelateerde opties mogelijk.
Kan back-ups maken
Maakt het mogelijk om een back-up te maken van eigen virtuele servers op een externe FTP- of SSH-server.
Kan back-ups plannen
Maakt het mogelijk om op regelmatige basis een back-up te maken van virtuele servers.
Kan back-ups herstellen
Maakt het mogelijk om de homedirectory en database-inhoud van virtuele servers in eigendom te herstellen.
Kan gedeelde IP's selecteren
Maakt het mogelijk om globaal gedefinieerde gedeelde IP-adressen te selecteren bij het maken van virtuele servers.
Kan verzamelaliassen maken
Maakt het aanmaken van e-mailaliassen mogelijk die alle e-mail naar een domein doorsturen naar een ander domein of adres.
Kan webpagina-editor gebruiken
Maakt het mogelijk webpagina's te bewerken via de Virtualmin UI.
Kan database-hosts op afstand bewerken
Maakt toegang mogelijk tot externe clientsystemen die verbinding mogen maken met de MySQL-databases van het domein.
Kan virtuele servers uitschakelen
Hiermee kunnen servers tijdelijk worden uitgeschakeld.
Kan virtuele servers verwijderen
Hiermee kunnen eigen servers volledig worden verwijderd, inclusief alle subservers en aliassen.
