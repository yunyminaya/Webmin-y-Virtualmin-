Сетевой интерфейс для адресов IPv6
Когда Virtualmin добавляет IPv6-адрес для виртуального сервера, он будет добавлен к указанному здесь интерфейсу (например, eth1 ). Если имя интерфейса не указано, будет использоваться автоматически обнаруженный интерфейс IPv4. Обычно это первый интерфейс Ethernet в системе.
