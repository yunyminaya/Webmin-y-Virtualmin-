URL para redireccionamiento de correo web
De manera predeterminada, Virtualmin redirigirá las solicitudes a http://webmail.thedomain.com a Usermin en el mismo sistema, como https://thedomain.com:20000/ . Sin embargo, este campo se puede usar para cambiar la URL de redireccionamiento, quizás a un solo nombre de host para todos sus dominios para los que tiene un certificado SSL, como https://webmail.yourcompany.com:20000/ . O puede redirigir a otra aplicación de correo web en el mismo dominio, ingresando una URL como http://www.${DOM}/squirrelmail/ .
