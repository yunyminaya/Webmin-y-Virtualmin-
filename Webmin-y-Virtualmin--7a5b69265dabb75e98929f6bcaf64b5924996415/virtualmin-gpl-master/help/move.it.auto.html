Sposta server virtuale
Lo spostamento di un server virtuale modifica l'amministrazione del dominio che ha il controllo su di esso. Le funzioni effettive del server non sono interessate, ma ora possono essere gestite da un amministratore diverso e vengono conteggiate per eventuali limiti su domini, alias e database per quel nuovo amministratore.

Questa pagina può essere utilizzata per eseguire tre diversi tipi di operazioni, a seconda del tipo di server che si sta modificando e di ciò che si desidera fare con esso. Loro sono :

Sposta un sub-server in un altro genitore
Quando si modifica un server secondario, è possibile selezionare un dominio padre diverso dal menu per modificarne la proprietà. Il sito Web del server, il dominio DNS, le cassette postali e così via non saranno interessati, ma ora sarà sotto il controllo del proprietario del nuovo genitore selezionato. La sua directory home verrà spostata sotto la home del nuovo genitore e tutti i file avranno la proprietà cambiata in utente e gruppo del nuovo genitore.

Sposta un server padre in un altro padre
Quando si modifica un server di appartenenza, è possibile convertirlo in un server secondario utilizzando questa pagina. Questo dovrebbe essere fatto con cura, poiché alcune impostazioni che esistono solo per i server di appartenenza (come password, quote e limiti) andranno perse, poiché ora sono ereditate dal nuovo genitore. Inoltre, verrà eliminato qualsiasi utente Unix, gruppo Unix o utente Webmin associato al dominio.

Converti un sub-server in un server di appartenenza
Quando si modifica un server secondario, è possibile convertirlo in un server padre di proprietà di un nuovo amministratore del server. È necessario inserire un nome utente e una password per il nuovo amministratore. Al termine dello spostamento, tutti i file nella nuova home directory del server saranno di proprietà del nuovo utente. Le quote del disco, i limiti di larghezza di banda e i limiti del proprietario del server per il server spostato verranno copiati dal vecchio genitore, ma potranno successivamente essere modificati separatamente.

Modifica del target di un alias
Se il server selezionato è un alias di un altro server, questo modulo può essere utilizzato per puntarlo a una destinazione diversa. Una volta completato lo spostamento, l'accesso Web al dominio restituirà il contenuto dalla nuova destinazione e l'e-mail al dominio verrà inoltrata alle cassette postali della nuova destinazione.
