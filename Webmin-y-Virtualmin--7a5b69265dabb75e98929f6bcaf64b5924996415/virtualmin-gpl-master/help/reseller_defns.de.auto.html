Nameserver-Hostnamen
Standardmäßig haben von diesem Reseller erstellte virtuelle Server in ihrer BIND-Konfiguration dieselben Nameserver wie alle anderen Domänen. Da Wiederverkäufer jedoch häufig den Eindruck erwecken möchten, dass sie ihren eigenen Server ausführen, möchten sie möglicherweise nicht, dass Ihre Nameserver-Hosts in den von ihnen erstellten Domänen angezeigt werden.

In dieses Feld können Sie benutzerdefinierte Nameserver-Hosts eingeben, die als NS- Datensätze in jeder vom Reseller erstellten Domäne verwendet werden. Stellen Sie sicher, dass sie bereits in DNS vorhanden sind und von jedem Registrar verwendet werden, bei dem die Domains registriert sind.
