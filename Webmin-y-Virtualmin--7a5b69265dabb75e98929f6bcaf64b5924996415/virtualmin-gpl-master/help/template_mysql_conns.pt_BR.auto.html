Máximo de conexões para proprietários de domínio
Este campo controla o número de conexões simultâneas com o MySQL que a conta do proprietário de um servidor virtual pode fazer. Isso inclui conexões de webapps instalados pelo Virtualmin, como Wordpress ou Drupal. O padrão é permitir um número ilimitado de conexões, mas como cada uma consome memória dentro do processo do servidor MySQL, você pode querer limitar o total de conexões em um sistema ocupado.

O limite definido aqui só será aplicado quando um novo servidor virtual for criado ou quando um login MySQL for habilitado para um servidor existente.
