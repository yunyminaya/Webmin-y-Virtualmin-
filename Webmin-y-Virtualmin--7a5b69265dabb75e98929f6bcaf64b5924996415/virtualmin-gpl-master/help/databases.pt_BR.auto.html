Editar bancos de dados
Esta página lista todos os bancos de dados que atualmente pertencem ao servidor virtual e permite a criação de bancos de dados adicionais até qualquer limite em vigor para o usuário logado no Virtualmin. Você também pode excluir mais um banco de dados selecionando-os na lista e clicando no botão Excluir selecionados .

Para visualizar as tabelas e dados reais em um banco de dados, clique no link Gerenciar próximo ao nome do banco de dados na tabela. Para ver o tamanho do banco de dados, clique em seu nome.

Ao utilizar o Virtualmin Pro, dois formulários adicionais são mostrados abaixo da lista do banco de dados para alterar o login e a senha dos usuários MySQL e PostgreSQL utilizados por este servidor virtual. Isto pode ser útil se você quiser que o login do banco de dados seja diferente do login da administração primária, talvez por motivos de segurança. Eles devem ser alterados com cuidado, pois quaisquer scripts PHP (como aqueles de instaladores de webapp) que tenham o nome de usuário ou a senha em seus arquivos de configuração não serão atualizados.
