Criar subdomínio virtual
Este formulário permite criar um novo subdomínio virtual, que pertence a um logon de administração existente. Um subdomínio é semelhante a um sub-servidor, pois compartilha cota de disco, limites de largura de banda e limites para usuários, aliases e outros objetos com seu pai. No entanto, o nome do subdomínio deve estar sob o nome do domínio pai, ele usa um subdiretório do diretório public_html do pai para seu site e os arquivos de log são compartilhados.

Ao criar o servidor, você deve inserir pelo menos o nome do domínio e uma descrição. Se vários modelos de servidor estiverem definidos no sistema, você poderá selecionar um no menu - os modelos determinam vários padrões para o novo servidor, como a configuração do site, os registros DNS e assim por diante.

Para selecionar quais recursos estão ativados para o servidor, use as caixas de seleção na segunda parte da seção Opções avançadas . Os subdomínios têm um conjunto muito menor de recursos disponíveis em comparação com os sub-servidores ou servidores de nível superior.

Na parte inferior da seção Opções avançadas , você pode optar por encaminhar todo o tráfego da Web para este novo site para outro URL. Você também pode optar por fazer com que o novo servidor virtual use seus próprios endereços IP privados, necessários para hospedagem SSL e FTP anônimo.
