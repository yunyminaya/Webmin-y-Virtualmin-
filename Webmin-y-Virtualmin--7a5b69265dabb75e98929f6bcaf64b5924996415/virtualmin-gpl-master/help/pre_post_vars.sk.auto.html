Konkrétnu akciu Virtualmin, ktorá bola vykonaná, nájdete v premennej VIRTUALSERVER_ACTION, ako napríklad:

CREATE_DOMAIN
Vytvára sa nový virtuálny server.
DELETE_DOMAIN
Existujúci virtuálny server sa odstraňuje.
DISABLE_DOMAIN
Virtuálny server je dočasne vypnutý.
ENABLE_DOMAIN
Opätovne sa aktivuje virtuálny server.
MODIFY_DOMAIN
Virtuálny server sa aktualizuje, napríklad pri pridávaní alebo odstraňovaní funkcií alebo pri zmene vlastníctva.
CLONE_DOMAIN
Virtuálny server sa vytvára kopírovaním existujúceho servera.
