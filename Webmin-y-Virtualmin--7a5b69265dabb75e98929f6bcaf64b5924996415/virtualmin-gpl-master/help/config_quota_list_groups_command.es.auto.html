Comando para enumerar cuotas de grupo
Cuando se utilizan comandos de cuota externos, se llamará al programa configurado en esta opción cuando Virtualmin necesite encontrar cuotas de grupo existentes. No se llama con ningún parámetro, pero debe generar una línea de texto por grupo, y cada línea contiene campos separados por espacios. Los campos en cada fila deben ser:
  1. El nombre del grupo Unix.
  2. Cuota actual utilizada en bloques de 1 kB.
  3. La cuota flexible en bloques de 1 kB, o 0 para ilimitado.
  4. La cuota fija en bloques de 1 kB, o 0 para ilimitado.
