Parametri della riga di comando extra per il comando di compressione
Per impostazione predefinita, Virtualmin esegue i comandi gzip o bzip2 per comprimere i backup senza parametri aggiuntivi. Ciò significa che verranno utilizzati il livello e il formato di compressione predefiniti. Tuttavia, è possibile utilizzare questo campo per inserire parametri aggiuntivi, come --fast per eseguire i backup più velocemente (al costo di richiedere più spazio), --best per ottenere una compressione migliore (al costo del tempo della CPU), o --rsyncable per rendere il file più facilmente replicabile.
