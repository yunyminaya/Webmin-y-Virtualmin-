Включить доменное имя в имена пользователей?
Эта опция выбирает, как будут генерироваться имена пользователей. По умолчанию Virtualmin будет использовать запрошенное базовое имя пользователя, но если оно конфликтует с существующим именем пользователя, оно добавит имя домена способом, указанным в следующем параметре. Если согласованность предпочтительна, установите для этого параметра значение «Всегда», и ко всем именам пользователей в пределах подсерверов будет добавлено имя домена.
