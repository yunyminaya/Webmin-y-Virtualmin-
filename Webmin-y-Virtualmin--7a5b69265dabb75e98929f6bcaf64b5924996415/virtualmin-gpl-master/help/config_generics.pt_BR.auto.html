Também atualizar endereços de saída para caixas de correio?
Se ativado, os arquivos /etc/genericstable (para Sendmail) ou /etc/postfix/canonical (para Postfix) são atualizados ao adicionar e modificar caixas de correio. Eles são usados para mapear nomes de usuário Unix para endereços de email e podem ser úteis se os usuários enviarem emails a partir de scripts.
