Domeinnaam voor gebruik in links
Standaard zal Virtualmin, wanneer een link naar de website van deze virtuele server wordt weergegeven (zoals na het installeren van een script), de domeinnaam van de server gebruiken. U kunt dit veld echter in plaats daarvan gebruiken om een aliasdomein te selecteren om in plaats daarvan in links te gebruiken. Dit kan handig zijn als de server alleen extern toegankelijk is via de alias domeinnaam.
