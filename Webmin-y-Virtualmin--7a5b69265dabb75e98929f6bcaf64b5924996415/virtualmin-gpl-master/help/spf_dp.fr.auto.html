Politique DMARC pour les e-mails qui échouent SPF ou DKIM
Si un message échoue à la validation SPF ou DKIM et que DMARC est activé, ce paramètre détermine comment le serveur de messagerie destinataire doit le gérer. Les options sont les suivantes:
  • Ne rien faire - Le message est toujours livré comme d'habitude.
  • E-mail de quarantaine - Cela dépend du destinataire, mais signifie généralement que le message sera placé dans un dossier de spam.
  • Rejeter l'e - mail - Le message sera supprimé.
