Directorios web protegidos (en el directorio de inicio)
Esta opción especifica si el propietario del dominio tendrá acceso al módulo Webmin de directorios web protegidos, que permite la creación de uno o más directorios protegidos con contraseña. Por supuesto, esto puede hacerse usando archivos .htaccess, pero muchos usuarios no saben cómo crear o usar archivos .htaccess, y este módulo proporciona un medio basado en la web para realizar la misma tarea.
