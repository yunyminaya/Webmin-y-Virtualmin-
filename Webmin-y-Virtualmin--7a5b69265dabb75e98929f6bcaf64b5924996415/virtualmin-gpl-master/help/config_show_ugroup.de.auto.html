Auswahl verschiedener Gruppen für Serveradministratoren zulassen?
Wenn diese Option auf "Ja" gesetzt ist, kann der Ersteller eines neuen virtuellen Servers anstelle der neuen Gruppe, die für den virtuellen Server erstellt wurde, eine andere Unix-Gruppe auswählen, zu der sein Besitzer gehören soll. Dies kann nützlich sein, wenn alle Serverbesitzer-Konten auf Ihrem System Mitglieder einer bestimmten Gruppe sein müssen, sollte jedoch mit Vorsicht verwendet werden, da Serverbesitzer Gruppenkontingenten umgehen können.
