Email Einstellungen
Diese Seite dient zum Konfigurieren von E-Mail-bezogenen Einstellungen für einen virtuellen Server. Wenn Ihr System BCCing-gesendete E-Mails unterstützt (und dies auf der Seite Modulkonfiguration aktiviert ist), kann im Abschnitt Ausgehende BCC-E-Mails an eine Adresse zum Kopieren von E-Mails ausgewählt werden.

Wenn der virtuelle Server eine Aliasdomäne ist, steuert das Feld Mail-Aliase-Modus , ob Mail-Aliase aus der Zieldomäne kopiert werden oder ob ein einzelner Catchall-Alias erstellt wird, um alle E-Mails an das Ziel weiterzuleiten. Der erstere Modus ist nicht für alle Mailserver verfügbar, ist jedoch bei Verwendung von Postfix überlegen, da Mail an ungültige Adressen auf SMTP-Konversationsebene zurückgeschickt werden kann.
