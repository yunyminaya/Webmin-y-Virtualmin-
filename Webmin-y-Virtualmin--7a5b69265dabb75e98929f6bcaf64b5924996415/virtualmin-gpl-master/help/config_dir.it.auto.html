Creazione directory home abilitata?
Questa opzione determina se la creazione di una home directory per un nuovo server virtuale viene sempre eseguita (e l'opzione nascosta) o se è facoltativa durante la creazione di un server. Poiché una directory home è quasi sempre necessaria a meno che non si stia creando un dominio solo DNS, l'opzione Sempre è generalmente la migliore.
