Número base para interfaces virtuales
Por defecto, Virtualmin usará eth0:0 para la primera interfaz IP virtual que cree, luego eth0:1 y así sucesivamente. Sin embargo, este campo se puede usar para cambiar dónde comienza el número de interfaz después de: Esto puede ser útil si desea reservar algunos de los números más bajos para la asignación manual, pero en la mayoría de los casos no es necesario cambiarlo.
