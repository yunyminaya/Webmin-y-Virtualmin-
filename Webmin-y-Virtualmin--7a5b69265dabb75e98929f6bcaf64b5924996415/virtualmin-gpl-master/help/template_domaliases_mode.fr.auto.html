Alias de messagerie pour les nouveaux domaines
Il est souvent utile de créer automatiquement un certain nombre d'alias lorsqu'un nouveau domaine est créé. Par exemple, le RFC recommandé "postmaster", "webmaster", "abuse" et "hostmaster" sont tous créés par défaut par Virtualmin. Vous souhaiterez peut-être ajouter les vôtres, par exemple, "support" ou "bugs", ou toute autre chose, selon l'utilisateur le plus probable des domaines créés à partir de ce modèle.
