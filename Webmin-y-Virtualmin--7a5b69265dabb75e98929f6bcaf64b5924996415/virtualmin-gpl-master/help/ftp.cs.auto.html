Nastavit virtuální FTP založené na IP?
Pokud je tato funkce povolena, bude pro tento virtuální server Virtualmin vytvořen virtuální server ProFTPd. To lze použít k nastavení anonymního serveru FTP, který je pro tento server soukromý. Vyžaduje však, aby server měl svou IP adresu.

Tato funkce je nutná pouze v případě, že chcete poskytnout anonymní FTP server přístup podle jména. Téměř nikdo nepotřebuje tuto funkci povolit (to pravděpodobně znamená, že jste). Tuto funkci nemusíte aktivovat, pokud chcete pouze poskytnout přístup FTP pro účet vlastníka domény, protože každý uživatel s shell accu nt bude mít automaticky přístup FTP do svého domovského adresáře.

Jinými slovy: Téměř určitě to nepotřebujete, i když si myslíte, že možná ano.
