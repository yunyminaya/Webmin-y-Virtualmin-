Base directory home
Questa è la directory di livello superiore in cui verranno create le home directory. Se l'installazione di Virtualmin condivide un server con molti altri servizi o utenti di sistema standard, può essere utile utilizzare qualcosa di diverso da / home. L'impostazione predefinita viene raccolta dal modulo Webmin Utenti e gruppi, che per impostazione predefinita è / home. Il modulo Utenti e gruppi consente estensioni complesse a questa convenzione, come "/ home / v / virtualmin", che può facilitare la navigazione delle directory se si hanno molti domini su un singolo server. La documentazione del modulo Utenti e gruppi in uno dei libri di Webmin tratta questa funzionalità in modo più dettagliato.
