Pagina HTML predefinita del sito web
Questa opzione consente di controllare la pagina predefinita che verrà creata e visualizzata per il nuovo server virtuale.

Puoi scegliere "Disabilitato" per saltare la creazione di una pagina oppure selezionare "Chiedi al momento della creazione" per decidere durante la creazione del server virtuale.

In alternativa, puoi optare per una pagina predefinita del sito web Virtualmin selezionando l'opzione "Sì, con il modello di pagina predefinita del sito web".

Se preferisci, selezionando "Sì, con il contenuto sottostante" puoi fornire codice HTML o testo personalizzato da utilizzare come contenuto della pagina web.
