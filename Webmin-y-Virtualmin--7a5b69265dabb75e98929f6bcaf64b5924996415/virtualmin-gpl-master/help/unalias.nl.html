Converteer Alias Server
Deze pagina kan worden gebruikt om een alias virtuele server te converteren in een sub-server. Hij zal dan niet langer de web pagina's leveren van de oude doel server maar kan vanaf dat moment zijn eigen web inhoud bevatten. Iedere bestaande email alias zal worden bewaard behalve voor de catchall die email doorstuurt naar de oude doel server. DNS records zullen niet worden gewijzigd.
