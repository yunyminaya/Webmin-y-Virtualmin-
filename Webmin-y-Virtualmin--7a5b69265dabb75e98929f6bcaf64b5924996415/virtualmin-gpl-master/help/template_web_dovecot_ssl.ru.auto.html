Копировать SSL-сертификаты для IP в Dovecot?
Когда эта опция включена и виртуальный сервер создан с включенным SSL и его собственным частным IP, Dovecot будет настроен на использование SSL-сертификата сервера для соединений с этим IP-адресом. Это может быть полезно, если почтовые клиенты подключаются к имени домена в режиме SSL и ожидают соответствующий сертификат.
