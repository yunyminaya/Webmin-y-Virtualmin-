Aliases de criação em lote
Esta página permite criar vários aliases de email de uma vez a partir de um arquivo de texto. Este arquivo deve listar um alias por linha e cada linha deve ter o seguinte formato:

name:[comment]:destination1:[destination2:...]

Os significados dos campos são:

nome
O nome da caixa de correio do alias, com as vendas . Para criar um alias que corresponda a todos os emails do domínio, digite * aqui.
Comente
Este campo opcional pode ser usado para inserir um comentário descritivo para o alias, como Vendas corporativas .
destino
É possível digitar um ou mais destinos, normalmente na forma de endereços de email para encaminhar emails, como joe@seudominio.com . Os seguintes destinos especiais também são suportados:
  • bounce
    Cria um alias que retorna o email ao remetente.
  • autoreply Your autoreply message
    Envia uma resposta automática de volta para o remetente, contendo o texto após a palavra autoreply .
  • local username
    Entrega email diretamente para a caixa de correio do usuário especificado.
