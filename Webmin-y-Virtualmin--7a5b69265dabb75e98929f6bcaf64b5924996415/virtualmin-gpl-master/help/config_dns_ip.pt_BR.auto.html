Endereço IP padrão para registros DNS
Os registros DNS são gerados automaticamente para novos hosts virtuais e, em circunstâncias normais, você deseja que eles correspondam ao IP do servidor virtual fornecido acima. No entanto, se o servidor estiver atrás de um roteador NAT, você operará um servidor proxy, um servidor de balanceamento de carga ou algum outro tipo de dispositivo que faça com que o servidor opere em um endereço IP diferente do que o mundo vê ao conversar com seus domínios, pode ser necessário alterar o valor aqui. Outra instância em que você pode alterar isso é se estiver adicionando domínios a um servidor de desenvolvimento que terá endereços diferentes quando entrar em produção. Você pode preencher o endereço IP público ou de produção aqui, em vez de o Virtualmin usar o endereço IP local.

Se a opção Detectar automaticamente endereço externo estiver selecionada, o Virtualmin tentará calcular o IP no momento da criação do servidor virtual. Isso é feito fazendo uma solicitação HTTP para www.virtualmin.com , que responderá com o endereço de onde vem a conexão.
