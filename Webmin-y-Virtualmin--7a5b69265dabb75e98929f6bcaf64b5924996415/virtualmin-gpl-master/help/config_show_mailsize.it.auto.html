Mostra le dimensioni della cassetta postale nell'elenco utenti?
Se si seleziona Sì per questa opzione, le dimensioni della cassetta postale di ciascun utente verranno visualizzate nell'elenco utenti di un server virtuale. È possibile selezionare No se le quote disco sono abilitate, poiché verrà mostrato anche l'utilizzo totale del disco degli utenti.
