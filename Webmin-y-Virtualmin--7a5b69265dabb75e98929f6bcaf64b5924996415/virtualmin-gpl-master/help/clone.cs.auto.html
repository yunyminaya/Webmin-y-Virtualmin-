Klonovat virtuální server
Tato stránka umožňuje duplikovat existující virtuální server s novým názvem domény. Veškerý webový obsah, záznamy DNS, poštovní schránky, poštovní aliasy, databáze a další nastavení související s původní doménou budou pokud možno duplikovány. Webový obsah se však žádným způsobem nezmění , takže pokud virtuální server obsahuje webovou aplikaci, která byla nakonfigurována tak, aby používala databáze nebo domovský adresář starého serveru, bude nutné ji po klonování ručně znovu nakonfigurovat.

Při duplikování virtuálního serveru nejvyšší úrovně musíte také zadat nové uživatelské jméno a heslo pro vlastnictví nové domény. Pokud má server privátní IP adresu, bude klonu automaticky přidělena nová IP.

Proces klonování zkopíruje všechny databáze spojené s původní doménou pod novými názvy a duplikuje jejich obsah. Nemusí však aktualizovat všechny webové aplikace nainstalované v původní doméně, které jsou nakonfigurovány pro použití původních databází. To je nutné provést ručně.
