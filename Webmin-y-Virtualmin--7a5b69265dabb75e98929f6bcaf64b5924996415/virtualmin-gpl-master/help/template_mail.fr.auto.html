Message électronique à envoyer lors de la création du serveur
Si vous souhaitez que les propriétaires de domaines créés à partir de ce modèle reçoivent un e-mail différent de la valeur par défaut de Virtualmin, vous pouvez l'entrer ici. Vous pouvez utiliser les variables de modèle standard pour inclure le répertoire personnel, le nom d'hôte, le mot de passe, etc.
