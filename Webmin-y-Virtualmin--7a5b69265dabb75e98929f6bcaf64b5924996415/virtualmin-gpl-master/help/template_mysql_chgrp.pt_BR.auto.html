Definir a propriedade do grupo dos arquivos de banco de dados MySQL?
Quando essa opção é ativada, qualquer novo banco de dados MySQL terá a propriedade do grupo definida em seus arquivos de dados (normalmente em /var/lib/mysql ), para que sejam incluídos nos limites de cota Unix de seus servidores virtuais. Se não ativado, o uso do disco dos bancos de dados não será restrito e será possível que um servidor virtual exceda sua alocação de disco permitida.
