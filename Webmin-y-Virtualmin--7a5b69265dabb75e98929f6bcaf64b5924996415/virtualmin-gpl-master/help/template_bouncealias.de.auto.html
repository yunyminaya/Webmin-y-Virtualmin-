Bounce E-Mail an ungültige Adressen
Wenn diese Option aktiviert ist, wird jede E-Mail an eine Adresse in einer Domain, die mit dieser Vorlage erstellt wurde und nicht mit einem Benutzer oder Alias übereinstimmt, an den Absender zurückgesendet. Dadurch wird verhindert, dass E-Mails an root @ somedomain an den Root-Benutzer auf Ihrem Server gesendet werden.
