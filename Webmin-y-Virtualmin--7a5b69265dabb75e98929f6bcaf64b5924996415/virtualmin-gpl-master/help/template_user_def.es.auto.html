¿Agregar usuario de Apache al grupo Unix para nuevos servidores?
Esta opción determina si el usuario con el que se ejecuta Apache se agregará al nuevo grupo de host virtual. Cuando está habilitado, los permisos se pueden restringir significativamente en los directorios de inicio de los usuarios. Cuando se combinan permisos más estrictos (es decir, 750 ) con suEXEC, resulta imposible que los propietarios de dominios vean los directorios y archivos de otros propietarios de dominios. Esta opción se recomienda en cualquier plataforma que admita grupos secundarios.
