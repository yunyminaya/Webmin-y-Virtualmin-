Переменные конфигурации PHP для скриптов
Эта таблица может использоваться для ввода параметров конфигурации PHP, которые будут добавлены к виртуальному веб-узлу любого сервера, на котором установлен сторонний скрипт. Это может быть полезно для увеличения пределов памяти или внесения других изменений конфигурации PHP, определяемых сайтом, для удовлетворения требований скрипта.
