Creare la pagina web iniziale?
Per creare una pagina web iniziale per il nuovo server virtuale, puoi scegliere di non creare alcuna pagina web selezionando "No" oppure optare per una pagina predefinita del sito web Virtualmin selezionando l'opzione "Sì, con il modello di pagina predefinita del sito web".

In alternativa, selezionando "Sì, con contenuto sottostante" è possibile immettere codice HTML o testo personalizzato da utilizzare come contenuto della pagina web.
