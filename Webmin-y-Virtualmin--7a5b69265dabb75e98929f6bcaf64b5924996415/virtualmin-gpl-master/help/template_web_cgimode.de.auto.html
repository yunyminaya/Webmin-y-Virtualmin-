CGI-Skriptausführungsmodus
Mit dieser Option legen Sie fest, ob und wie CGI-Skripte standardmäßig für neue virtuelle Server ausgeführt werden. Folgende Optionen sind möglich:
  • CGI-Skripte deaktiviert – Skripte können überhaupt nicht ausgeführt werden.
  • FCGIwrap-Server – Skripte werden über einen Hintergrundserver ausgeführt, eines pro Domäne.
  • suEXEC - Skripte werden über den Apache- Suexec- Wrapper gestartet. Beachten Sie, dass diese Option möglicherweise nicht auf allen Systemen verfügbar ist.
