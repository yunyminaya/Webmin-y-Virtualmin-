Puede editar el servidor virtual
Permite editar las características del servidor virtual, su contraseña y descripción.
Puede administrar usuarios
Permite ver, editar, crear y eliminar usuarios de buzones de correo, FTP y bases de datos.
Puede gestionar alias
Permite ver, editar, crear y eliminar alias de correo electrónico.
Puede gestionar bases de datos
Permite agregar o eliminar bases de datos MySQL y PostgreSQL adicionales a este servidor virtual.
Puede instalar aplicaciones web
Permite instalar aplicaciones web de terceros en este servidor virtual.
Puede cambiar la dirección IP
Si este servidor tiene una dirección IP privada, esta opción permite cambiarla.
Puede gestionar certificados SSL
Permite cargar o volver a generar el certificado SSL y la clave para este servidor.
Puede editar el marco de reenvío
Para sitios web de solo reenvío, esta opción permite editar el destino y el HTML del marco de reenvío.
Puede administrar administradores adicionales
Permite ver, crear, editar y eliminar cuentas adicionales de administrador del propietario del servidor.
Puede configurar la entrega de spam y virus.
Permite editar destinos de entrega de spam y virus y puntuaciones.
Puede seleccionar versiones de PHP
Para dominios con PHP ejecutado a través de CGI o fastCGI, permite configurar versiones de PHP por directorio.
Puede editar la configuración del correo electrónico
Permite la configuración de BCC automático, modo de copia de alias y otras opciones relacionadas con el correo electrónico.
Puede hacer copias de seguridad
Permite realizar copias de seguridad de los servidores virtuales propios en un servidor FTP o SSH remoto.
Puede programar copias de seguridad
Permite realizar copias de seguridad de los servidores virtuales de forma regular.
Puede restaurar copias de seguridad
Permite restaurar el directorio principal y el contenido de la base de datos de los servidores virtuales propios.
Puede seleccionar IP compartidas
Permite seleccionar direcciones IP compartidas definidas globalmente al crear servidores virtuales.
Puede crear alias generales
Permite la creación de alias de correo electrónico que reenvían todo el correo de un dominio a un dominio o dirección diferente.
Puede utilizar el editor de páginas web.
Permite editar páginas web a través de la interfaz de usuario de Virtualmin.
Puede editar hosts remotos de bases de datos
Permite el ingreso de sistemas cliente remotos que pueden conectarse a las bases de datos MySQL del dominio.
Puede desactivar servidores virtuales
Permite deshabilitar servidores temporalmente.
Puede eliminar servidores virtuales
Permite eliminar por completo los servidores propios, incluidos todos los subservidores y alias.
