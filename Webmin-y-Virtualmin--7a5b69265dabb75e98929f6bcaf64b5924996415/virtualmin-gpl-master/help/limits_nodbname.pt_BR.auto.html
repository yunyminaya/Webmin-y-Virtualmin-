É possível escolher nomes de bancos de dados?
Se essa restrição for definida como Não , o proprietário do servidor não poderá selecionar o nome do banco de dados padrão ao criar um subservidor que tenha um recurso de banco de dados habilitado. Em vez disso, ele sempre assumirá como padrão a primeira parte do nome do domínio.

Além disso, o usuário não poderá remover o banco de dados padrão de nenhum domínio existente, pois isso permitiria que ele renomeasse efetivamente o banco de dados padrão após a criação do domínio.
