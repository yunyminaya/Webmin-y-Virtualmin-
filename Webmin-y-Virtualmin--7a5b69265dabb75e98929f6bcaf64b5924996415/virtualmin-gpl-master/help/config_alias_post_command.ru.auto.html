Команда для запуска после внесения изменений в псевдоним
Этот параметр можно использовать для указания сценария или команды, которая будет выполняться после создания, изменения или удаления почтового псевдонима. Он может использовать переменные среды ALIAS_FROM, ALIAS_TO и ALIAS_ACTION, чтобы определить, какой псевдоним был изменен и что с ним произошло.
