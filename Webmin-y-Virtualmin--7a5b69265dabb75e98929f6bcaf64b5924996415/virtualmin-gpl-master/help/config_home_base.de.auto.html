Basis des Basisverzeichnisses
Dies ist das Verzeichnis der obersten Ebene, in dem Basisverzeichnisse erstellt werden. Wenn Ihre Virtualmin-Installation einen Server mit vielen anderen Diensten oder Standardsystembenutzern gemeinsam nutzt, kann es hilfreich sein, etwas anderes als / home zu verwenden. Die Standardeinstellung wird im Webmin-Modul Benutzer und Gruppen erfasst, das standardmäßig / home ist. Das Modul Benutzer und Gruppen ermöglicht komplexe Erweiterungen dieser Konvention, z. B. "/ home / v / virtualmin", wodurch die Navigation in Verzeichnissen vereinfacht wird, wenn sich viele Domänen auf einem einzelnen Server befinden. In der Dokumentation zum Modul Benutzer und Gruppen in einem der Webmin-Bücher wird diese Funktion ausführlicher behandelt.
