Crea Alias Server
Questo modulo consente di creare un nuovo server alias, che è essenzialmente un nome alternativo per un server esistente. Gli alias non hanno il proprio contenuto del sito Web e condividono tutte le quote e i limiti con il server di destinazione. La larghezza di banda utilizzata dal server alias viene conteggiata per il totale del target.

Quando si crea il server, è necessario inserire almeno il nome di dominio e una descrizione. Se sul sistema sono definiti più modelli di server, è possibile selezionarne uno dal menu. Per selezionare quali funzionalità nel server alias vengono inoltrate al dominio di destinazione, utilizzare le caselle di controllo nella seconda parte della sezione Opzioni avanzate . Sono disponibili solo DNS, sito Web e alias email.
