Поделиться SSL сертификатами между доменами, где это возможно?
Если для этого параметра установлено значение «Да» (по умолчанию), Virtualmin будет обмениваться сертификатами SSL между виртуальными серверами, которые имеют одинаковые права владения, если имена хостов в сертификате соответствуют всем доменам. Выбор No приведет к созданию отдельного сертификата для каждого нового виртуального сервера.
