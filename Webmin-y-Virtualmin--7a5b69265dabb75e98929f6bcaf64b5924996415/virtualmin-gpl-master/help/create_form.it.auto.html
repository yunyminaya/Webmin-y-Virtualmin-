Crea server virtuale
Questo modulo consente di creare un nuovo server virtuale di livello superiore, a cui sarà associato un nuovo accesso amministrativo. Quando si crea il server, è necessario inserire almeno il nome dominio, la descrizione e la password di amministrazione. Se sul sistema sono definiti più modelli di server, è possibile selezionarne uno dal menu: i modelli determinano varie impostazioni predefinite per il nuovo server, come la configurazione del sito Web, i record DNS e così via.

Nella sezione Opzioni avanzate è possibile selezionare un nome utente di amministrazione, un nome di gruppo e un nome di database diversi per il server, anziché i valori predefiniti presi dalla prima parte del nome di dominio. È anche possibile inserire un indirizzo e-mail di contatto per il server - idealmente dovrebbe essere l'indirizzo esistente della persona che gestirà il server, in un dominio diverso.

Per selezionare quali funzioni sono abilitate per il server, utilizzare le caselle di controllo nella seconda parte della sezione Opzioni avanzate . Alcune funzionalità dipendono da altre e non possono essere selezionate da sole - ad esempio, i report di Webalizer sulle statistiche dei siti Web dipendono dall'avere effettivamente un sito Web.

Nella parte inferiore della sezione Opzioni avanzate è possibile selezionare per inoltrare tutto il traffico Web a questo nuovo sito a un altro URL e inoltrare tutta la posta elettronica. Puoi anche scegliere di fare in modo che il nuovo server virtuale utilizzi i propri indirizzi IP privati, richiesti per SSL e hosting FTP anonimo.
