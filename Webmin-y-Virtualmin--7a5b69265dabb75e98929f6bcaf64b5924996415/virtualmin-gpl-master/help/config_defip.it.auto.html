Indirizzo IP predefinito del server virtuale
I server virtuali senza funzionalità host virtuale SSL o FTP abilitate risiederanno su un singolo indirizzo IP. Questa opzione specifica quell'indirizzo. Se lasciato al suo valore predefinito, Virtualmin utilizzerà l'indirizzo primario sull'interfaccia specificata dall'interfaccia di rete per gli indirizzi virtuali sopra.
