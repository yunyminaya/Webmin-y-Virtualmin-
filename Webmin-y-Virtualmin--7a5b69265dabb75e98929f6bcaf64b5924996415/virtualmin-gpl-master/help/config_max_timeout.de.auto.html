Wartezeit, wenn das Maximum überschritten wird
Wenn dies auf Halt sofort eingestellt ist und eine Sicherung gestartet wird, die das Limit überschreitet, schlägt dies sofort fehl. Alternativ können Sie eine Anzahl von Sekunden eingeben, um zu warten, bis andere Sicherungen abgeschlossen sind, bevor Sie fortfahren. Wenn dieses Zeitlimit überschritten wird und noch zu viele Sicherungen ausgeführt werden, schlägt die neue fehl.
