Les utilisateurs de Qmail LDAP sont également des utilisateurs Unix?
Si les utilisateurs de la base de données LDAP auront également des comptes d'utilisateur UNIX, sélectionnez oui. Les utilisateurs doivent disposer de comptes système UNIX pour utiliser les services shell ou FTP, bien que dans de nombreux environnements ce niveau d'accès soit inutile et indésirable.
