Основная группа для локальных пользователей
Virtualmin также может управлять локальными пользователями на вашем сервере, что может быть полезно, если вы хотите использовать Virtualmin только для управления почтой и домашними веб-сайтами. С небольшой переконфигурацией Virtualmin может предоставить простой в использовании интерфейс для добавления пользователей почтовых ящиков / веб-сайтов, чей веб-сайт доступен через "http://www.domain.com/~username". Это не основная задача Virtualmin, но она может упростить обслуживание почтовых ящиков и веб-сайтов для интернет-провайдеров.
