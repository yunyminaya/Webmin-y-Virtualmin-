Sottodirectory FTP anonima da creare
Se sono consentiti download FTP anonimi, è possibile specificare la sottodirectory in cui risiederanno i file per il download. Se si modificano le direttive nell'opzione precedente, è necessario includere una sezione anonima e corrispondere alla posizione fornita qui. Si noti che l'accesso anonimo tramite un nome di dominio specifico richiede la configurazione di un host FTP virtuale basato su IP.
