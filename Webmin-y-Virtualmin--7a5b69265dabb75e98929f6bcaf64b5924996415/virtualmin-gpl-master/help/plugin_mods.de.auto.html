Aktivierte Plugin-Module
Über das Mehrfachauswahlfeld können Sie auswählen, welche Module für virtuelle Virtualmin-Server aktiviert werden sollen. Nach der Aktivierung können Sie die Verwendung auf Domänenbasis zulassen. Ausgewählte Plugins werden hervorgehoben. Um mehrere Plugins auszuwählen, halten Sie die STRG-Taste gedrückt, während Sie zusätzliche Plugins auswählen.
