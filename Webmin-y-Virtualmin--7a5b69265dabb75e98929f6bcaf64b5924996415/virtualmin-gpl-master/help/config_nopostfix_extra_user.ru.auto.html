Создать дополнительного пользователя Unix в системах Postfix при использовании формата user @ domain?
При использовании имен пользователей Unix user @ domain-format и почтового сервера Postfix Virtalmin, когда для этого параметра установлено значение yes (по умолчанию), создает дополнительного пользователя Unix в формате пользовательского домена для облегчения доставки почты. Однако в некоторых системах это не требуется, и поэтому вы можете использовать эту опцию, чтобы отключить эту функцию. Это будет относиться только к вновь созданным пользователям и доменам.
