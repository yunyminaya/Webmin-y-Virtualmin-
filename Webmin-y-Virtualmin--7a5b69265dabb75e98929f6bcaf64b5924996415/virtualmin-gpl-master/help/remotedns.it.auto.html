Server DNS remoti
Questa pagina consente di registrare i sistemi che eseguono Webmin come server DNS, consentendo alle zone di essere ospitate su di essi anziché sul sistema Virtualmin primario. I sistemi che aggiungi devono avere già BIND in esecuzione e il modulo Webmin DNS Server configurato correttamente.

Una volta aggiunte, le zone DNS possono essere spostate o create sui sistemi remoti. È anche possibile usare solo DNS remoti e spegnere il server BIND su questo sistema per risparmiare memoria.
