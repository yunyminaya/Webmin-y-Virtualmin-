Prefijo para nombres de usuario de correo?
De manera predeterminada, Virtualmin agregará o antepondrá el nombre de usuario o la primera parte del nombre de dominio a todos los usuarios de buzones creados en este dominio, para garantizar que no haya conflictos de nombres de usuario. Sin embargo, este campo se puede usar para especificar un prefijo o sufijo diferente, que puede ser útil si su sistema tiene una longitud de nombre de usuario corta como (como FreeBSD) y el prefijo predeterminado es demasiado largo.

La forma en que se selecciona el prefijo por defecto está determinada por el estilo de nombre de dominio en la opción de nombre de usuario en la página de configuración del módulo . Si se establece en Nombre de usuario o primera parte del dominio , el prefijo será el mismo que el nombre de usuario y grupo del dominio. Si se establece en Nombre de dominio completo , siempre será el nombre de dominio del servidor virtual. De lo contrario, será la primera parte del nombre de dominio (a menos que haya un conflicto con el prefijo de otro dominio).
