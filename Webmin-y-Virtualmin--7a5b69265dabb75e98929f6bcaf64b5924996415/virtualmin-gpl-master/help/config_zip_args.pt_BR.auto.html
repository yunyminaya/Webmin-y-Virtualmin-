Parâmetros extras da linha de comando para o comando de compactação
Por padrão, o Virtualmin executa os comandos gzip ou bzip2 para compactar backups sem parâmetros adicionais. Isso significa que o nível e o formato padrão de compactação serão usados. No entanto, você pode usar este campo para inserir parâmetros adicionais, como - --fast para executar seus backups mais rapidamente (ao custo de exigir mais espaço), - --best para obter melhor compactação (ao custo do tempo da CPU) ou --rsyncable para tornar o arquivo mais facilmente replicável.
