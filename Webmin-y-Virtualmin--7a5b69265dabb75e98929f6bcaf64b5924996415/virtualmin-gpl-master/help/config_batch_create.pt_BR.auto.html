Pode criar vários servidores a partir do arquivo em lotes?
A página Batch Create Servers do Virtualmin permite que os usuários com permissões para adicionar servidores criem vários de uma vez a partir de um arquivo de texto. Como isso pode resultar na criação rápida de um grande número de domínios, convém impedir que os administradores do servidor usem esse recurso. Para fazer isso, altere essa configuração para Não - isso a restringirá a ser usada apenas pelo administrador principal e pelos revendedores.
