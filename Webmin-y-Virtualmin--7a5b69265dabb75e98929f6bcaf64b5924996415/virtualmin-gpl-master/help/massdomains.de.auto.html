Aktualisieren Sie virtuelle Server
Auf dieser Seite können Sie Einstellungen und Grenzwerte auf mehreren virtuellen Servern gleichzeitig aktualisieren, wie auf der vorherigen Seite ausgewählt. Standardmäßig wird auf den ausgewählten Servern nichts geändert, bis Sie mithilfe der Felder auf dieser Seite ein Attribut oder eine Einschränkung zum Ändern auswählen. Änderbare Einstellungen werden in folgende Kategorien unterteilt:

Kontingent- und Bandbreitenänderungen
Diese Felder können verwendet werden, um die Festplattenkontingente für ausgewählte Server zu ändern oder um ihre Bandbreitenbeschränkungen zu ändern. Das Ändern von Kontingenten wirkt sich nur auf Server der obersten Ebene aus, da Subserver ihre Kontingente von der übergeordneten Domäne erben.

Funktionsänderungen aktiviert
In diesem Abschnitt können Sie Funktionen auswählen, die für alle ausgewählten Server ein- oder ausgeschaltet werden sollen. Dies sind dieselben Funktionen, die für einzelne Server auf der Seite Virtuellen Server bearbeiten aktiviert werden können.

Änderungen der Eigentümergrenzen
Diese Felder können verwendet werden, um die verschiedenen Grenzwerte zu ändern, die für Eigentümer von virtuellen Servern der obersten Ebene gelten, z. B. die Anzahl der Postfächer, Aliase und Domänen, die sie erstellen können. Hier vorgenommene Änderungen wirken sich nicht auf Sub-Server aus, die unter die ihren Eltern auferlegten Grenzen fallen.

Änderungen an den Eigentümerfunktionen
Die Felder in diesem Abschnitt steuern die Funktionen von Virtualmin, die den Administratoren der ausgewählten virtuellen Server zur Verfügung stehen. Auch hier sind sie nur für Server der obersten Ebene relevant.

Änderungen an der PHP-Option
Wenn verfügbar, können Sie in diesem Abschnitt sowohl die Standard-PHP-Version als auch die Methode zum Ausführen von PHP-Skripten auf allen ausgewählten Servern ändern. Es ist jedoch nur auf Servern mit Websites nützlich.

Konfigurationsänderungen
In diesem Abschnitt werden verschiedene Änderungen an der Konfiguration ausgewählter Server vorgenommen, z. B. die Methode zum Löschen von Spam oder die Anmeldeshell des Domänenadministrators.

Es ist auch möglich, vorübergehend die ausgewählten Server zu deaktivieren am unteren Rand der Seite mit der Disable - Server - Taste. Wenn Sie dies verwenden, können Sie auch einen Grund für die Deaktivierung (z. B. Nichtzahlung ) in das nebenstehende Textfeld eingeben.

Verwenden Sie die Schaltfläche Server aktivieren, um mehrere Server wieder zu aktivieren, die auf dieser Seite oder anderswo deaktiviert wurden.
