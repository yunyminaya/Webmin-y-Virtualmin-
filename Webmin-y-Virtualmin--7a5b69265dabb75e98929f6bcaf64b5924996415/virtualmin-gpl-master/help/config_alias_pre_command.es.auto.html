Comando para ejecutar antes de realizar cambios en un alias
Esta opción se puede usar para especificar un script o comando que se ejecutará antes de crear, modificar o eliminar un alias de correo. Puede hacer uso de las variables de entorno ALIAS_FROM, ALIAS_TO y ALIAS_ACTION para determinar qué alias se está cambiando y qué le está sucediendo. Si el comando sale con un estado distinto de cero, se vetará la modificación del alias.
