Arquivo de log de acesso ao servidor da Web
Este campo define o arquivo no qual os logs de acesso do Apache são gravados para este servidor virtual. Cada solicitação HTTP atendida pelo domínio deste site será gravada neste arquivo como uma linha separada. Ele só deve ser alterado se você estiver movendo o log para um diretório diferente, como /var/log/virtualmin ou o diretório inicial do servidor.
