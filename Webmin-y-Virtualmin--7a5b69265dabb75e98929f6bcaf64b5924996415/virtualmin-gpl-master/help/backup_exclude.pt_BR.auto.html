Arquivos a serem excluídos de cada domínio

Esse campo permite inserir uma lista de caminhos relativos aos arquivos no diretório inicial de cada servidor virtual para excluir deste backup. Para excluir vários caminhos, coloque cada um em sua própria linha (por exemplo, separados por linhas).

Exemplos

Para excluir um diretório inteiro, forneça o caminho relativo do diretório inicial do usuário, como public_html/stats para excluir o diretório de relatórios do Webalizer ou os homes para excluir os diretórios pessoais do usuário.

Para excluir um único arquivo, forneça o caminho relativo ao nome do arquivo, como public_html/index.html .

Os arquivos também podem ser especificados usando caracteres curinga, como *.log , que excluirá qualquer arquivo com a extensão .log qualquer lugar do diretório inicial do domínio. Um caminho também pode ser fornecido, como etc/*.ini para excluir todos os arquivos que terminam em .ini .
