IPv4-адрес для серверов на основе имен
Этот параметр можно использовать, чтобы виртуальные серверы, созданные для этого посредника, использовали IP-адрес, отличный от системного по умолчанию, который обычно используется всеми виртуальными веб-сайтами на основе имен. Это может быть полезно для серверов «брендинга», созданных разными посредниками, поскольку IP-адреса каждого посредника разрешаются на разные имена хостов.

Вводимый вами IP-адрес уже должен быть активным в системе, что можно сделать в модуле конфигурации сети Webmin. IP-адрес не будет использоваться для виртуальных серверов с собственными частными IP-адресами, поскольку каждый из них всегда получает отдельный адрес.
