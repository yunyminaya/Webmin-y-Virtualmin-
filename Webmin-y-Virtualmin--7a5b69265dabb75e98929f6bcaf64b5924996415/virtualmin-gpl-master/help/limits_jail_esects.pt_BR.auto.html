Seções extras para o Jailkit ativar
Por padrão, apenas os arquivos essenciais são copiados das seções de configuração do Jailkit para jails novas ou atualizadas. Para incluir arquivos adicionais, liste as seções desejadas de /etc/jailkit/jk_init.ini aqui.

As seções no arquivo /etc/jailkit/jk_init.ini são nomeadas como [git] ou [apacheutils] . Por exemplo, para incluir seções git e apacheutils neste ambiente chroot de domínio, coloque git apacheutils (separados por espaço) neste campo.
