Client Cloudmin Services
Cette page vous permet de configurer Virtualmin pour se connecter à un serveur Cloudmin Services, afin que les bases de données MySQL, les zones DNS et autres services puissent être déchargés sur des systèmes distants. Idéalement, il doit être utilisé avant la création de serveurs virtuels, car le mélange de bases de données et de zones desservies localement et distantes est inefficace.

Les champs à compléter sont:

Maître des services Cloudmin
Ce nom d'hôte d'une machine Cloudmin avec le plugin Cloudmin Services installé. Cela peut être géré par vous, votre hébergeur ou votre fournisseur VPS.

Numéro de port
Cela devrait presque toujours être défini sur 10000, qui est le port Webmin / Cloudmin par défaut.

Nom d'utilisateur pour le serveur
Ce champ doit contenir le nom d'utilisateur d'un compte propriétaire du système sur le maître Cloudmin Services qui est autorisé à créer des bases de données, des zones DNS ou d'autres services.
Fonctionnalités à héberger
Cette section vous permet de sélectionner les fonctionnalités Virtualmin à décharger sur les systèmes distants. Seuls ceux que votre compte sur le système maître est autorisé à approvisionner peuvent être sélectionnés.
