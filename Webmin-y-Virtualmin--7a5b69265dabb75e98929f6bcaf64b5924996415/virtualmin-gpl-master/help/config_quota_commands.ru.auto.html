Использовать внешние команды для получения и установки квот?
Если для этого параметра установлено значение «Да» , Virtualmin будет использовать команды в параметрах конфигурации ниже, чтобы установить и получить квоты пользователей и групп. Это может быть полезно, если квоты на вашем сервере управляются не так, как стандартная система квот Unix, например, с помощью сервера NFS или файловой системы ZFS.

По умолчанию установлено значение «Нет» , что является правильным значением практически для всех систем.
