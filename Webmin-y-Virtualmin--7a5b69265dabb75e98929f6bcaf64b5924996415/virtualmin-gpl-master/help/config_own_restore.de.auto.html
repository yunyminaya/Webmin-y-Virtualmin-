Können vom Master-Administrator erstellte Sicherungen wiederhergestellt werden?
Standardmäßig können Besitzer virtueller Server nur selbst erstellte Sicherungen wiederherstellen. Wenn diese Einstellung jedoch in " Ja" geändert wird, können vom Hauptadministrator ( root ) erstellte Sicherungen auch von Domänenbesitzern wiederhergestellt werden. Dies gilt natürlich nur für ihre eigenen Domains.

Sie sollten diese Option nur aktivieren, wenn Sie sicher sind, dass alle Ihre Sicherungsziele sicher sind, da das Wiederherstellen einer Sicherung durch einen nicht vertrauenswürdigen Benutzer die Systemsicherheit beeinträchtigen kann.
