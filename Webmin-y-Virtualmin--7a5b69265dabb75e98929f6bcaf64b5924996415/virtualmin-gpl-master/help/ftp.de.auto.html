IP-basiertes virtuelles FTP einrichten?
Wenn diese Funktion aktiviert ist, wird ein virtueller ProFTPd-Server für diesen virtuellen Virtualmin-Server erstellt. Dies kann verwendet werden, um eine anonyme FTP-Site einzurichten, die für diesen Server privat ist. Es ist jedoch erforderlich, dass der Server seine IP-Adresse hat.

Diese Funktion wird nur benötigt, wenn Sie einen namenbasierten Zugriff auf einen anonymen FTP-Server gewähren möchten. Fast niemand braucht diese Funktion, um aktiviert zu werden (das bedeutet wahrscheinlich, dass Sie). Sie müssen diese Funktion nicht aktivieren, wenn Sie nur einen FTP-Zugriff für das Domäneninhaberkonto bereitstellen möchten, da jeder Benutzer mit einem Shell-Konto automatisch FTP-Zugriff auf sein Ausgangsverzeichnis hat.

Mit anderen Worten: Sie brauchen das mit ziemlicher Sicherheit nicht, auch wenn Sie denken, dass Sie es vielleicht tun.
