Nome de usuário
Este campo define a parte do nome de usuário do login do usuário, que deve ser exclusivo na lista de usuários do servidor web. O login do usuário para acesso ao servidor web será derivado do nome de usuário aqui inserido, combinado com o nome do domínio. Por exemplo, se você inserir o nome de usuário web-user1 e o domínio for example.com , o login resultante será web-user1@example.com .
