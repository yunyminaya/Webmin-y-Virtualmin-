Intervalo entre as tarefas de coleta de status executadas
O Virtualmin possui um processo em segundo plano chamado collectinfo.pl que é executado por padrão a cada 5 minutos a partir do Cron, para coletar informações usadas para a página Informações do sistema e gráficos históricos. Se isso for muito frequente para o seu sistema, você poderá aumentar o intervalo entre as execuções para até 30 minutos usando este campo.

Como alternativa, selecione Nunca executar para desativar esse processo completamente. Se você raramente faz login na interface da Web do Virtualmin como root e nunca vê os gráficos de carga da CPU e outras estatísticas, essa opção reduz a carga da CPU no seu sistema. No entanto, isso fará com que a página Informações do sistema demore mais para carregar.
