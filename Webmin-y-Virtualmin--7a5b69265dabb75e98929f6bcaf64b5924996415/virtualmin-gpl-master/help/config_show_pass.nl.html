Mag mailbox en virtuele server wachtwoorden bekijken?
Wanneer deze optie is ingesteld op Ja kunnen server administrators het huidige wachtwoord voor email, FTP gebruikers en hun eigen virtuele server zien, als zij bekend zijn bij Virtualmin. U kunt ook Nee selecteren om mailbox wachtwoorden te verstoppen, in het bijzonder welke zijn geselecteerd door mailbox gebruikers.

Ongeacht deze instelling kunnen de hoofd administrator en de resellers altijd het mailbox en virtuele server wachtwoord zien.
