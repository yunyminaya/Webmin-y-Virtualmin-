Alterar nome de domínio
Este formulário pode ser usado para alterar o nome de domínio de um servidor virtual existente. Por padrão, o nome de usuário e o diretório inicial também serão ajustados para corresponder ao novo domínio, supondo que eles sejam baseados nele.

Também pode ser usado para alterar o nome de usuário da administração ou o diretório inicial sem tocar no nome do domínio, por exemplo, se você deseja mover para um novo diretório base. Apenas o administrador principal pode alterar o nome de usuário ou mover a casa para um local de arbitragem artística.

Por padrão, os usuários existentes da caixa de correio no domínio não são renomeados para que seus logins IMAP funcionem. Você pode forçar uma alteração definindo o sufixo Alterar nome da caixa de correio? automaticamente escolhido embora. Este campo também pode ser usado para alterar o sufixo para o que você desejar, mesmo que o nome do domínio não seja alterado.

Se um domínio estiver usando o formato username@domain.com para caixas de correio, os usuários serão renomeados apenas se o nome do domínio real mudar.
