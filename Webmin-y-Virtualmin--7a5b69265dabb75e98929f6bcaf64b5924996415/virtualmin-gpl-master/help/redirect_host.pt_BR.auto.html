Limitar o redirecionamento para o nome do host
Por padrão, os redirecionamentos serão aplicados independentemente do nome de host do URL usado para acessar o site. No entanto, este campo pode ser usado para limitar o redirecionamento para um host específico. Isso pode ser útil para redirecionar www.domain.com para domain.com, por exemplo, ou alterar o redirecionamento para um domínio de alias específico.
