Sekundäre Mailserver
Ein sekundärer Mailserver ist einer, der auch E-Mails für einen der von Virtualmin verwalteten virtuellen Server empfangen kann, jedoch nur als temporäres Relais und nicht als endgültiges Ziel. Es ist ratsam, einen sekundären Server für stark genutzte E-Mail-Domains einzurichten, um als Backup oder Aufbewahrungsort zu fungieren, falls der primäre E-Mail-Server ausfällt.

Auf dieser Seite können Sie andere Server mit installiertem Virtualmin auswählen, die als sekundäre MX-Server für neue Maildomänen verwendet werden sollen, die auf diesem System erstellt werden. Bevor Server angezeigt werden, müssen sie zuerst dem Webmin Servers Index-Modul hinzugefügt werden, in dem Sie den Root-Login und das Passwort für den Remote-Server eingeben müssen.

Für jeden sekundären Mailserver haben Sie die Möglichkeit, anstelle des tatsächlichen Hostnamens des Servers einen anderen Hostnamen zur Verwendung im DNS-MX-Eintrag einzugeben, z. B. secmx.yourdomain.com . Stellen Sie jedoch sicher, dass alles, was Sie eingeben, ein gültiger DNS-Name ist, der auf denselben Server aufgelöst wird.

Wenn auf die Schaltfläche Speichern und anwenden geklickt wird, werden alle vorhandenen virtuellen Server, für die E-Mail aktiviert ist, allen ausgewählten sekundären Mailservern hinzugefügt (oder von diesen entfernt). Darüber hinaus werden DNS-MX-Einträge zu ihren Domains hinzugefügt (oder daraus entfernt), um auf die Mailserver zu verweisen.
