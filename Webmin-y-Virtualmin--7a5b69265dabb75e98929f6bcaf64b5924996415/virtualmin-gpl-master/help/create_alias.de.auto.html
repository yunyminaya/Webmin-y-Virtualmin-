Erstellen Sie einen Alias-Server
Mit diesem Formular können Sie einen neuen Alias-Server erstellen. Dies ist im Wesentlichen ein alternativer Name für einen vorhandenen Server. Aliase haben keinen eigenen Website-Inhalt und teilen alle Kontingente und Limits mit dem Zielserver. Die vom Aliaserver verwendete Bandbreite wird auf die Gesamtsumme des Ziels angerechnet.

Beim Erstellen des Servers müssen Sie mindestens den Domänennamen und eine Beschreibung eingeben. Wenn auf dem System mehrere Servervorlagen definiert sind, können Sie eine aus dem Menü auswählen. Verwenden Sie die Kontrollkästchen im zweiten Teil des Abschnitts Erweiterte Optionen , um auszuwählen, welche Funktionen auf dem Alias-Server an die Zieldomäne weitergeleitet werden. Es ist nur DNS-, Website- und E-Mail-Aliasing verfügbar.
