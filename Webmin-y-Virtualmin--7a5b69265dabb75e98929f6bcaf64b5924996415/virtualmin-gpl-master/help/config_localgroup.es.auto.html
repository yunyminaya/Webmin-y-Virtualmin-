Grupo primario para usuarios locales
Virtualmin también puede administrar usuarios locales en su servidor, lo que puede ser útil si desea usar Virtualmin solo para administrar el correo electrónico y los sitios web domésticos. Con una pequeña reconfiguración, Virtualmin puede proporcionar una interfaz fácil de usar para agregar usuarios de buzones / sitios web a cuyo sitio web se puede acceder a través de "http://www.domain.com/~username". Este no es el enfoque principal de Virtualmin, pero puede facilitar el mantenimiento de buzones y sitios web para los ISP.
