Host do servidor para scanner remoto
Este campo deve ser usado para inserir o nome do host do sistema remoto no qual o clamd é executado, para o qual as mensagens de email carregadas de vírus serão enviadas pelo clamdscan ou pelo clamdscan clamd-stream-client scan para verificação. Não tem efeito se você selecionar os modos Scanner autônomo ou Scanner do servidor local , pois os dois fazem a digitalização localmente.
