Versions PHP
Cette page vous permet de sélectionner la version de PHP utilisée par défaut dans ce serveur virtuel, et également de choisir différentes versions pour différents sous-répertoires web. Cela peut être utile si vous avez des applications PHP qui fonctionnent uniquement avec une version spécifique.

Pour changer la version utilisée par défaut, sélectionnez-le dans le menu à côté de la ligne Répertoire HTML par défaut, et cliquez sur Sauver. De même, pour changer la version utilisée par un autre répertoire, sélectionnez-le dans le menu à côté de la ligne de ce répertoire.

Pour ajouter une version de PHP personnalisé pour un nouveau répertoire, entrez le chemin relatif dans le champ de texte au bas de la table, sélectionnez une version dans le menu en bas, puis cliquez sur Sauver. Ce chemin doit être relatif au public_html, comme wordpress ou mediawiki. Il n'y a pas besoin d'un leader / , ou une URL complète.

Pour réinitialiser un répertoire à utiliser la version par défaut de PHP, cochez la case à côté de lui dans le tableau et cliquez sur le bouton Supprimer la sélection.
