Consentire l'accesso ai database
Questo campo consente di selezionare a quali database di proprietà di questo server virtuale l'utente può anche accedere. Per aggiungere un database, selezionalo dall'elenco a sinistra e fai clic sulla freccia ->.

Il login e la password dell'utente saranno gli stessi che userebbe per FTP o IMAP. Per impostazione predefinita, gli verrà concesso l'accesso completo a tutte le tabelle nei database selezionati.

MySQL supporta solo nomi utente lunghi fino a 16 caratteri, quindi per un utente con un nome account lungo sono necessari solo i primi 16 caratteri quando accede al database.
