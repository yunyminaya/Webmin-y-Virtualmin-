Anfängliche Unix-Shell
In diesem Feld können Sie eine andere Shell für Besitzer virtueller Server definieren, die aus dieser Vorlage erstellt wurden, anstatt den auf der Seite Benutzerdefinierte Shells definierten globalen Standard zu verwenden. Es können jedoch nur Shells ausgewählt werden, die für die Verwendung durch Domaininhaber auf dieser Seite vorgesehen sind.
