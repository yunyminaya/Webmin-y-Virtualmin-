Vorhandene Zone beim Erstellen übernehmen?
Wenn ein Cloud DNS-Anbieter ausgewählt ist, bestimmt diese Option, was passiert, wenn beim Anbieter bereits eine neue Domain vorhanden ist. Wenn Nein ausgewählt ist, schlägt die Domänenerstellung fehl. Aber wenn Ja gewählt wird, wird die vorhandene Domain von Virtualmin verwendet und verwaltet.
