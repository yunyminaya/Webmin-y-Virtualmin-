Begrenzung der Anzahl der Nicht-Alias-Server
Wenn der Domänenbesitzer über Berechtigungen zum Erstellen von Subservern verfügt, können Sie die Anzahl der Nicht-Alias-Server begrenzen, die hier erstellt werden können. Beachten Sie, dass dieses Limit nicht effektiv höher sein kann als das Standardlimit für die Anzahl der virtuellen Server , da das Gesamtlimit für virtuelle Server sowohl für Alias- als auch für Nicht-Alias-Domänen gilt. Ein Limit von 0 verhindert die Erstellung von Nicht-Alias-Servern vollständig.

Dies ist nützlich, wenn Sie einem Domaininhaber erlauben möchten, eine unbegrenzte oder höhere Anzahl von Alias-Servern zu erstellen, aber die teureren Sub-Server, die er erstellen kann, begrenzen möchten.
