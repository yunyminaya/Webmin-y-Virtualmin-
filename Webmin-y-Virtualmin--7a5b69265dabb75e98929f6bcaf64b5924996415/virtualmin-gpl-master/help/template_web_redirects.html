Redirects for new websites
This table allows you to enter website redirects that will be added to all new virtual servers when they are created. For each redirect you must enter a source path like / or /phpadmin, and either a destination URL or a path relative to the domain's public_html directory.

In both fields, virtual server template variables like ${DOM} and ${USER} can be used. You can also select if the redirect applies when the site is accessed via HTTP, HTTPS or both.
