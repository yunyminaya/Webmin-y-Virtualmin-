Senha do MySQL para novos domínios
Esta opção controla a senha inicial do MySQL para novos servidores virtuais. As opções são:
Sincronizar com o login da administração
A senha do MySQL é sempre mantida a mesma do logon de administração do servidor virtual.

Igual ao login de administração inicial
A senha é inicialmente igual ao logon de administração, mas não será modificada se a senha de administração for alterada.

Gerar aleatoriamente
Quando o servidor virtual é criado, uma senha MySQL gerada aleatoriamente é escolhida e não alterada.
