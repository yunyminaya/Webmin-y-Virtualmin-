Remote-DNS-Server
Auf dieser Seite können Sie Systeme, auf denen Webmin läuft, als DNS-Server registrieren, sodass Zonen auf diesen Servern statt auf Ihrem primären Virtualmin-System gehostet werden können. Auf den Systemen, die Sie hinzufügen, muss BIND bereits ausgeführt werden und das Webmin DNS Server-Modul muss ordnungsgemäß konfiguriert sein.

Nach dem Hinzufügen können DNS-Zonen auf die Remote-Systeme verschoben oder dort erstellt werden. Es ist auch möglich, nur Remote-DNS zu verwenden und den BIND-Server auf diesem System herunterzufahren, um Speicher zu sparen.
