Redirect webmail.domain to Usermin?
When this field is set to Yes (as it is by default), any new virtual servers will have a webmail entry added to their DNS domains, with the same IP address as the server's website. When http://webmail.thedomain.com/ is accessed, the browser will be redirected to Usermin on the same domain, to access web-based email.

If you prefer to use some webmail interface other that Usermin (such as Squirrelmail or Roundcube) or want to use a different URL for Usermin access, the redirect URL can be configured in the following URL for webmail redirect field.
