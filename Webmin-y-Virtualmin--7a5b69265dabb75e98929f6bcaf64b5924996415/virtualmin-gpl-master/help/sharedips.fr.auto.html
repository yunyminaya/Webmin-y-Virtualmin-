Adresses IP partagées
Le but principal de cette page est de saisir des adresses IP supplémentaires qui peuvent être utilisées par des serveurs virtuels qui n'ont pas leurs propres IP privées. Chaque IP entrée doit être déjà active sur le système (ce qui peut être fait dans la configuration réseau du module Webmin) et ne peut être utilisée par aucun autre revendeur ou serveur virtuel IP privé.
