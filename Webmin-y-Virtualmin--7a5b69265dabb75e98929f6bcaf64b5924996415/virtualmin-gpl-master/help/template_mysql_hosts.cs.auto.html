Povolení hostitelé MySQL klientů
Pokud by mělo být povoleno dotazovat databázi MySQL dalším hostitelům, můžete je zde specifikovat. To by mohlo být užitečné, pokud je tento systém používán jako databázový server a databáze budou používat další stroje. Měla by být dodržována bezpečnostní opatření, aby bylo zajištěno, že nedovolený přístup není neúmyslně povolen.
