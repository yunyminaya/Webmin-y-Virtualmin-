Indirizzo IP predefinito per i record DNS
I record DNS vengono generati automaticamente per i nuovi host virtuali e, in circostanze normali, si desidera che corrispondano all'IP del server virtuale fornito sopra. Tuttavia, se il tuo server si trova dietro un router NAT, gestisci un server proxy, un server di bilanciamento del carico o un altro tipo di dispositivo che porta il tuo server a operare su un indirizzo IP diverso da quello che il mondo vede quando parla con i tuoi domini, tu potrebbe essere necessario modificare il valore qui. Un'altra istanza in cui potresti voler cambiare questo è se stai aggiungendo domini a un server di sviluppo che avrà indirizzi diversi quando entrerà in produzione. È possibile inserire qui l'indirizzo IP di produzione o pubblico anziché fare in modo che Virtualmin utilizzi l'indirizzo IP locale.

Se l'opzione Rileva automaticamente indirizzo esterno è selezionata, Virtualmin tenterà di elaborare l'IP al momento della creazione del server virtuale. Questo viene fatto inviando una richiesta HTTP a www.virtualmin.com , che risponderà con l'indirizzo da cui proviene la connessione.
