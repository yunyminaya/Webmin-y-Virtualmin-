Nom de domaine à utiliser dans les liens
Par défaut, lorsque Virtualmin affiche un lien vers le site Web de ce serveur virtuel (comme après l'installation d'un script), il utilisera le nom de domaine du serveur. Cependant, vous pouvez utiliser ce champ à la place pour sélectionner un domaine d'alias à utiliser à la place dans les liens. Cela peut être utile si le serveur n'est accessible qu'en externe via le nom de domaine alias.
