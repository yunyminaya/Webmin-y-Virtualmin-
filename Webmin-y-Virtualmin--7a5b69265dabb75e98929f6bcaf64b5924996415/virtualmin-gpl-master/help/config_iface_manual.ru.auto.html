Воспитывать виртуальные интерфейсы
По умолчанию (если выбрано «Да» ), Virtualmin автоматически активирует частные сетевые интерфейсы для серверов, которые в них нуждаются. Если для этого параметра установлено значение Нет , вместо этого будет предполагаться, что вы хотите настроить эти интерфейсы вручную, возможно, используя файлы конфигурации Xen или Zones вместо того, что является стандартным для вашей операционной системы.
