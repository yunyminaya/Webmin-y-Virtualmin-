Создать пользователя базы данных
Создает нового пользователя базы данных, способного только управлять базами данных, используя квоты владельца виртуального сервера для всех управляемых баз данных. Этот пользователь работает исключительно в среде базы данных, не создавая отдельную учетную запись Unix. Кроме того, этого пользователя можно легко интегрировать путем слияния с учетной записью Unix, созданной с тем же именем пользователя.
