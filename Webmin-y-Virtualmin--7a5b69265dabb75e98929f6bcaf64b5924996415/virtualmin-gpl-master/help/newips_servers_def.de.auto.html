Zu aktualisierende Server
Hier können Sie einige oder alle Server im System angeben, die keine dedizierten IP-Adressen haben. Die Option Nur ausgewählt .. ist ein Mehrfachauswahlfeld. Um mehr als einen Eintrag auszuwählen, halten Sie die STRG-Taste gedrückt, während Sie auf einen oder mehrere Domainnamen klicken.
