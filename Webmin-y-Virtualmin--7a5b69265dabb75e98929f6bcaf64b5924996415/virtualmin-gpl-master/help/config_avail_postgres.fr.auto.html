Serveur de base de données PostgreSQL (pour la base de données)
S'il est activé, le compte propriétaire du domaine aura accès au module PostSQL Webmin. Il fournit une interface graphique conviviale aux bases de données qui appartiennent au propriétaire du domaine. Avec ce module et les privilèges appropriés, le propriétaire du domaine peut créer de nouvelles bases de données, modifier des bases de données existantes ou effectuer des requêtes sur les données de la base de données.
