Apache instelling en configuratie aangezet?
Deze optie zet het aanmaken aan of uit van de Apache VirtualNameHost ingangen en stelt alle andere virtuele host functies in van uw Apache configuratie file. Als u een alternatieve webserver gebruikt kunt u deze optie uitschakelen. Het is mogelijk om de plug-in module interface te gebruiken om ondersteuning aan te maken voor uw voorkeurs server (hoewel er op dit moment geen andere web-servers worden ondersteund), of u moet ervoor kiezen dat een extern script samen met de benodigde variabelen word gebruikt die alle informatie geven aan Virtualmin tijdens het aanmaken van een domein om tegelijkertijd de voorkeurs server in te stellen.
