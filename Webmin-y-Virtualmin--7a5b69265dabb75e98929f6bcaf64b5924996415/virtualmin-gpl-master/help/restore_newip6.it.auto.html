Indirizzo IPv6 per server ripristinati
Questa impostazione viene utilizzata solo durante il ripristino di un server virtuale che al momento non esiste sul sistema. Per impostazione predefinita, Virtualmin assegnerà l'indirizzo IPv6 condiviso predefinito del sistema se era originariamente su un IP condiviso, oppure utilizzerà l'indirizzo IPv6 privato dal sistema originale se non si trovava su un indirizzo condiviso.

Funziona bene se stai spostando un dominio tra due sistemi sulla stessa rete, ma in caso contrario è probabile che il dominio ripristinato non sia contattabile tramite la rete. Per evitare ciò, seleziona l'opzione Virtual con IP allocato o Virtual con IP in questo campo e Virtualmin selezionerà un IP gratuito per te o utilizzerà quello che hai inserito.
