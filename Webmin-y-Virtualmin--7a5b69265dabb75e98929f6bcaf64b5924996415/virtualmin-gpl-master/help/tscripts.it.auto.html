Programmi di installazione di Webapp modello
Questa pagina consente di specificare le app Web di terze parti da installare quando viene creato un nuovo server virtuale utilizzando questo modello. Per ogni webapp è necessario inserire le seguenti informazioni:

Webapp
Il nome e la versione di una webapp da installare
Percorso dell'URL
Un percorso sotto il sito Web del server virtuale per installare l'app Web, ad esempio /wordpress . Ciò determinerà anche la directory in ~/public_html che contiene i file della webapp.
Nome del database
Il nome del database che la webapp utilizzerà per archiviare i dati. Le opzioni sono:
Nessuno
Dovrebbe essere selezionato per le app web che non necessitano di un database, come Gallery .
Banca dati principale
Il database della webapp sarà il database primario del server virtuale del tipo scelto.
Altro
Se si desidera che questa webapp utilizzi il proprio database privato, è possibile selezionare questa opzione e inserire il modello per un database nel campo di testo adiacente. Ad esempio, potresti inserire ${DB}_phpnuke . Se il database risultante non esiste, verrà creato come parte del processo di installazione della webapp.
Tipo DB
Il tipo di utilizzo del database per i dati delle webapp, se presente, è selezionato.
