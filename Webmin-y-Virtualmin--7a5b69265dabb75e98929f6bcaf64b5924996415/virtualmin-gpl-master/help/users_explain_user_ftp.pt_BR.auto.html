Criar usuário FTP
Cria um novo usuário Unix dedicado ao acesso FTP no diretório do site, atribuindo automaticamente o usuário e ID de grupo do proprietário do servidor virtual. Este usuário especializado é criado pensando no acesso FTP. As cotas desse tipo de conta são compartilhadas com o proprietário do servidor virtual. Notavelmente, os recursos de e-mail não estão disponíveis para este tipo de usuário.
