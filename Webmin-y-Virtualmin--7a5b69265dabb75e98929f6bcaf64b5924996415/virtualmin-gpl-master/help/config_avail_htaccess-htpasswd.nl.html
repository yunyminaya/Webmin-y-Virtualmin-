Beschermde Web Directory's (onder de home directory)
Deze optie bepaald of de domein eigenaar toegang heeft tot de Beschermde Web Directory's Webmin module, waarmee nog maar directory's kunnen worden aangemaakt die met een wachtwoord beschermt zijn. Dit kan natuurlijk gedaan worden met een .htaccess file, echter veel gebruikers weten niet hoe ze een .htaccess file moeten aanmaken of moeten gebruiken. Deze module geeft ze de mogelijkheid om dezelfde taken uit te voeren op een visuele web-gebaseerde manier.
