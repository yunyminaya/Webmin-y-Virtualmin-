Programa de verificação de vírus
As opções possíveis para este campo são:
Digitalizador Standalone (clamscan)
Normalmente, esse é o padrão e instruirá o Virtualmin a usar o programa de verificação ClamAV que faz todo o trabalho por si só. Isso certamente funcionará, mas pode ser mais lento que a próxima opção.
Scanner de servidor ( clamdscan )
Esta opção pode ser selecionada se o servidor de verificação de vírus clamd estiver sendo executado no seu sistema. Geralmente é superior, pois o banco de dados de vírus ClamAV é bastante grande e o servidor precisa carregá-lo na memória apenas uma vez.
Scanner de servidor remoto ( clamd-stream-client )
Esta opção pode ser selecionada se o servidor de verificação de vírus clamd estiver sendo executado em um sistema remoto e tiver sido configurado para aceitar conexões de rede de outros hosts. É útil para descarregar a tarefa de digitalizar para um sistema diferente daquele que aceita email.
Outro programa
Essa opção permite que o Virtualmin use algum programa completamente diferente, como seu próprio script de wrapper para o ClamAV.
