Überprüfen Sie resolv.conf für dieses System?
Wenn dies auf Ja gesetzt ist (wie standardmäßig), überprüft Virtualmin die DNS-Client-Konfigurationsdatei /etc/resolv.conf des Systems, um sicherzustellen, dass es so eingerichtet ist, dass der lokale DNS-Server für die Namensauflösung verwendet wird. Dies ist erforderlich, da ohne das System und die von Ihnen erstellten DNS-Domänen vom System, auf dem Virtualmin ausgeführt wird, möglicherweise erst aufgelöst werden können, wenn sie vollständig von einem DNS-Registrar registriert wurden.

Wenn Sie sicher sind, dass dies nicht erforderlich ist (möglicherweise, weil Sie ein automatisches sekundäres DNS-Server-Setup haben), kann dies in Nein geändert werden, um falsche Konfigurationsprüfungsfehler zu vermeiden.
