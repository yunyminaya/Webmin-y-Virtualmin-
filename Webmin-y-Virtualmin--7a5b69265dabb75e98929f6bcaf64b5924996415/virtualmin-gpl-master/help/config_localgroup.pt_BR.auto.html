Grupo principal para usuários locais
O Virtualmin também pode gerenciar usuários locais no seu servidor, o que pode ser útil se você desejar usar o Virtualmin apenas para gerenciar emails e sites domésticos. Com um pouco de reconfiguração, o Virtualmin pode fornecer uma interface fácil de usar para adicionar usuários de caixas de correio / sites cujo site pode ser acessado por "http://www.domain.com/~username". Esse não é o foco principal do Virtualmin, mas pode facilitar a manutenção de caixas de correio e sites para os ISPs.
