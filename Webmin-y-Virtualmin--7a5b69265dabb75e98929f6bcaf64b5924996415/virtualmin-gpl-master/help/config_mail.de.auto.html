Mailbox- und Alias-Konfiguration aktiviert?
Diese Option aktiviert oder deaktiviert die Konfiguration von Postfächern. Wenn Sie keine E-Mail-Konten auf Ihrem Server bereitstellen möchten, z. B. wenn Sie einen Hosting-Service nur für Websites betreiben, können Sie die E-Mail-Funktionen deaktivieren. Wenn Sie E-Mails auf einem anderen Server hosten möchten, der LDAP unterstützt, wird diese Funktion nicht deaktiviert. Stattdessen konfigurieren Sie Virtualmin zum Erstellen von LDAP-Benutzern.
