Number of days to keep processed mail logs
Virtualmin will periodically scan server mail logs and store the results in a database file, for quicker searching using the Search Mail Logs page. This setting can be used to control how long this data is stored for, in the /var/webmin/procmail.* files. The default is 30 days, but you may want to decrease this if the processed logs are taking up too much disk space on your system.
