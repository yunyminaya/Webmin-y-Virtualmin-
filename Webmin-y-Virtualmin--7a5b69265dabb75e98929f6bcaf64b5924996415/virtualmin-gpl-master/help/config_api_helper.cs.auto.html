Cesta k vytvoření pomocného příkazu API
Ve výchozím nastavení vytvoří Virtualmin pomocný skript rozhraní API příkazového řádku pro virtualmin příkaz v adresáři /usr/sbin nebo /usr/local/sbin , v závislosti na tom, který je ve vašem systému zapisovatelný. Toto konfigurační pole však lze použít k výběru jiného umístění, například /opt/bin/virtualmin které je vhodné pro váš web.
