Вторичная группа для пользователей базы данных
Если этот параметр установлен, вы должны ввести имя группы Unix, в которую будут добавлены все пользователи с доступом к одной или нескольким базам данных для виртуальных серверов по этому шаблону. Если указанная вами группа не существует, она будет автоматически создана при необходимости.

Эта функция наиболее полезна в сочетании с Usermin, который может ограничивать видимые модули для тех пользователей, которые являются членами какой-либо группы. Таким образом, вы можете сделать модуль базы данных MySQL видимым только членам группы, которую вы здесь установили.
