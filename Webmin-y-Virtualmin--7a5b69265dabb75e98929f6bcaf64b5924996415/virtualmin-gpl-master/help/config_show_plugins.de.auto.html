Andere Plugins in der Benutzerliste anzeigen?
Wenn diese Option auf „ Ja“ gesetzt ist, wird eine Liste der anderen Plugins, auf die jeder Benutzer Zugriff hat, in einer separaten Spalte auf der Seite „Benutzer bearbeiten“ angezeigt. Dies ermöglicht Administratoren eine schnelle Einschätzung des Umfangs des jedem Benutzer gewährten Plugin-Zugriffs. Wenn Sie „Nein“ auswählen, wird diese Spalte mit den Plugin-Informationen ausgeblendet.
