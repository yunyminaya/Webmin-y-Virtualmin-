Запланированные команды (команды пользователя)
Этот параметр указывает, может ли учетная запись владельца домена использовать модуль «Запланированные команды Webmin». Запланированная команда или задание - это запланированное событие, которое можно использовать для выполнения задачи один раз в указанное время. Модуль «Запланированные команды» предоставляет простой в использовании интерфейс для сложного для запоминания (и вводящего в заблуждение новичков) синтаксиса команды at. Запланированные команды будут выполняться от имени владельца домена.
