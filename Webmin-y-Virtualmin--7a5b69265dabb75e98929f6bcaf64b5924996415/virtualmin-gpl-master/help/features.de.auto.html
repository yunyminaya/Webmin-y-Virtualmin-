Funktionen und Plugins
Auf dieser Seite werden alle verfügbaren Virtualmin-Kernfunktionen und Plugin-Module angezeigt und Sie können auswählen, welche für die Erstellung neuer virtueller Server verfügbar sind. Für jedes können Sie die Verfügbarkeit über das Kontrollkästchen in der ersten Spalte steuern, mit Ausnahme des Home-Verzeichnisses und des Administrationsbenutzers, die so kritisch sind, dass sie nicht deaktiviert werden können.

Wenn einige Funktionen beim Erstellen neuer virtueller Server nicht standardmäßig ausgewählt werden sollen, können sie in der Spalte Standard deaktiviert werden. Beispielsweise möchten Sie möglicherweise gelegentlich, aber nicht alle, eine SSL-Website für neue Domänen einrichten die Zeit.

Mehr über Plugins

Plugins sind Webmin-Module, die zusätzliche Server- und Postfachfunktionen für Virtualmin bereitstellen. Im Gegensatz zu den Kernfunktionen können sie separat installiert und von Drittentwicklern geschrieben werden. Wie Webmin und Usermin verfügt Virtualmin über eine API zum Schreiben von Plug-In-Modulen. Es ist jedoch etwas anders, dass Virtualmin-Module auch Webmin-Module sind, aber auch direkt mit Virtualmin-Domänen verknüpft sind, sodass beim Ändern einer Domäne auch die Module von Drittanbietern geändert werden. Virtualmin-Module können bei der Installation auch nach konfigurierten Virtualmin-Funktionen suchen, um die Kompatibilität mit dem System sicherzustellen.
