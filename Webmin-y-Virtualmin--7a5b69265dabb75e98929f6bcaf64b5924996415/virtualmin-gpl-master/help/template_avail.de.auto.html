Webmin-Module, die Administratoren virtueller Server zur Verfügung stehen
In diesem Abschnitt können Sie auswählen, auf welche Webmin-Module Besitzer von virtuellen Servern für Server zugreifen können, die mit dieser Vorlage erstellt wurden. Links zu diesen Modulen werden im linken Menü von Virtualmin angezeigt, normalerweise unter der Kategorie Dienste. Möglicherweise möchten Sie einige oder alle deaktivieren, um die Arten von erweiterten Konfigurationsänderungen einzuschränken, die Serverbesitzer vornehmen können, und sie stattdessen auf die einfachere Virtualmin-Benutzeroberfläche beschränken.

Alle hier vorgenommenen Änderungen werden sofort auf alle virtuellen Server angewendet, die diese Vorlage verwenden.
