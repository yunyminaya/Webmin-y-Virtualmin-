Percorso completo del comando aws per le operazioni S3
Se il comando aws è installato sul tuo sistema e questo campo è impostato sul percorso completo (come /usr/bin/aws ), Virtualmin lo chiamerà per eseguire operazioni S3 per backup come i file caricati e scaricati. Poiché questo comando è stato sviluppato da Amazon, è più affidabile dell'API S3 integrata in Virtualmin.
