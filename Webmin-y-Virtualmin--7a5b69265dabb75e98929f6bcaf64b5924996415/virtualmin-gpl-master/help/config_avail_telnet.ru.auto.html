SSH / Telnet Войти
Этот параметр указывает, может ли пользователь-владелец домена использовать модуль SSH / Telnet Login Webmin. Этот модуль представляет собой Java-апплет, который входит на сервер с использованием SSH или Telnet. Он не работает корректно со всеми браузерами из-за несовместимости Java. Обратите внимание, что отключение этого не препятствует доступу SSH к серверу владельцем домена, просто доступ к модулю SSH / Telnet Login Webmin.
