FTP-Anmeldung aktiviert?
Wenn diese Option auf "Ja" gesetzt ist, erhält der Benutzer die Shell , sodass sie sich über FTP anmelden können. Wenn die Einstellung auf "Nein" gesetzt ist, wird ihre Shell auf stattdessen wird die FTP-Anmeldung verweigert.

Dies setzt voraus, dass die FTP-Shell in der Datei /etc/shells aufgeführt ist oder anderweitig vom auf Ihrem System ausgeführten FTP-Server zugelassen wird.
