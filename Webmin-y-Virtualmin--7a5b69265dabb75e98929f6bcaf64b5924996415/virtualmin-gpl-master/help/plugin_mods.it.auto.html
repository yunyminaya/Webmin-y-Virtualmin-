Moduli plug-in abilitati
La casella di selezione multipla consente di scegliere quali moduli saranno abilitati per i server virtuali Virtualmin. Una volta abilitato, è possibile scegliere di consentire l'utilizzo in base al dominio. I plugin selezionati sono evidenziati. Per selezionare più plug-in tieni premuto il tasto CTRL mentre selezioni plug-in aggiuntivi.
