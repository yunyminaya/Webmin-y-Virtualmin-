Partager la configuration logrotate avec tous les domaines?
Lorsque ce paramètre est défini sur Non , Virtualmin créera un bloc distinct dans le fichier de configuration logrotate pour chaque serveur virtuel. Cela permet à chacun d'avoir des paramètres différents, mais entraînera le redémarrage du serveur Web après la rotation des journaux de chaque domaine - éventuellement en interrompant le service.

Si vous sélectionnez Oui , tous les nouveaux domaines partageront un seul bloc de configuration, ce qui signifie que le serveur Web ne doit être redémarré qu'une seule fois. Cependant, il n'est pas possible d'avoir des paramètres de rotation différents pour chaque domaine.
