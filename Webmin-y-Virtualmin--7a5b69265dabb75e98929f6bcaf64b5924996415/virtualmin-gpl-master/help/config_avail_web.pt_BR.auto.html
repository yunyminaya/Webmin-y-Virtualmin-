Servidor Web Apache (para host virtual)
Habilite isso se o proprietário do domínio deve ter a capacidade de editar a configuração do Apache para seu domínio. Embora esse nível de controle possa ser esmagador para usuários inexperientes, ele fornece grande flexibilidade para os experientes. Também é possível, com modificações na configuração de controle de acesso padrão, permitir acesso direto à seção de domínios virtuais do arquivo de configuração do Apache. Embora proteções estejam em vigor para evitar ações maliciosas, elas podem não ser perfeitas. Por padrão, os privilégios concedidos são geralmente seguros para uso em um ambiente de hospedagem compartilhada.
