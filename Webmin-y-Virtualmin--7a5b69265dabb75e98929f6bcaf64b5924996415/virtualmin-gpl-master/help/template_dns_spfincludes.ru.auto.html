Дополнительные SPF включенные домены
В этом поле перечислены дополнительные имена доменов, чьи записи SPF также считываются для определения разрешенных хостов для новых доменов. Таким образом, любой почтовый сервер, который считается действительным для перечисленных доменов, будет действительным и для нового домена.
