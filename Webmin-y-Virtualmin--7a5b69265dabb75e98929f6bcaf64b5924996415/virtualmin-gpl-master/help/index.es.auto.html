Servidor virtual
El módulo de servidores virtuales de Webmin proporciona un medio para crear, modificar y eliminar buzones de usuario y cuentas de usuario de servidor virtual. Con sus funciones de administración de dominio, puede crear rápida y fácilmente un nuevo sitio web con registros de dominio, configuración de Apache VirtualHost, un directorio de inicio, una cuenta de usuario para administrarlo, configuración del servidor de correo, cuotas de uso de disco y direcciones de correo electrónico y alias.
