Резервный формат
Это поле определяет, будет ли резервная копия записана в виде одного файла или один файл создан в месте назначения для каждого выбранного виртуального сервера с именем domain .tar.gz. Последняя опция также создаст один файл на домен, но в новом формате, поддерживаемом Virtualmin 2.86 и выше, который помещает всю информацию о каждом сервере в его домашнюю директорию. Этот формат лучше, потому что он не требует Virtualmin для создания временных файлов в каталоге / tmp, и, следовательно, экономит место на диске.
