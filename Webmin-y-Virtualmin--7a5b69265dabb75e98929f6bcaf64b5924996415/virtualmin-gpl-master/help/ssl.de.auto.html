SSL-Website auch einrichten?
Wenn diese Option aktiviert ist, wird für diesen virtuellen Server ein zusätzlicher virtueller Apache-Host erstellt, der Port 443 überwacht. Dadurch können Browser mithilfe der SSL-Verschlüsselung eine Verbindung zu https: // domainname.com / sowie zu http: / herstellen / domainname.com / ohne Verschlüsselung.

Ein selbstsignierter SSL-Schlüssel und ein Zertifikat werden im Rahmen des Server-Setup-Prozesses generiert, sodass der virtuelle SSL-Host sofort verwendet werden kann. Sie sollten jedoch so bald wie möglich ein gültiges SSL-Zertifikat für die Domäne erstellen, damit es von den Browsern ordnungsgemäß überprüft werden kann. Dies kann auf der Seite SSL-Zertifikat verwalten erfolgen .
