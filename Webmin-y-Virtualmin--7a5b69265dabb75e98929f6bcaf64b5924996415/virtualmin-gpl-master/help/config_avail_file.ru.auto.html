Файловый менеджер (только домашний каталог)
File Manager — это продвинутый Java-апплет, который предлагает двухпанельный вид файловой системы сервера. Это одна из самых мощных функций Webmin, позволяющая даже неопытным пользователям управлять файлами, каталогами и разрешениями, редактировать текстовые файлы, загружать и скачивать файлы и многое другое. По умолчанию владелец домена сможет получить доступ только к своему собственному каталогу и его подкаталогам.
