Interfaccia di rete per indirizzi virtuali
Se il tuo server ha più interfacce, puoi scegliere quale interfaccia usare per i tuoi domini virtuali. Se gli host virtuali SSL o FTP sono abilitati, sarà richiesto un nuovo IP per ciascun dominio su cui è abilitata la funzione. Salvo diversamente configurato durante la creazione del dominio, i nuovi indirizzi verranno creati sull'interfaccia qui specificata.
