Utworzyć użytkownika Unix dla nowych sprzedawców?
Ta opcja kontroluje, czy nowo utworzone konta sprzedawcy domyślnie otrzymują powiązanego użytkownika Unix (dla dostępu SSH i FTP), czy nie. Można go jednak pominąć w formularzu tworzenia sprzedawcy lub podczas wywoływania polecenia API create-reseller .
