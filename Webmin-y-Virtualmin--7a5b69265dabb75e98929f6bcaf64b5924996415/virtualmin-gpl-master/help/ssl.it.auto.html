Configurare anche il sito Web SSL?
Quando questa opzione è abilitata, verrà creato un host virtuale Apache aggiuntivo per questo server virtuale, in ascolto sulla porta 443. Ciò consentirà ai browser di connettersi a https: // domainname.com / utilizzando la crittografia SSL, nonché a http: / / domainname.com / senza crittografia.

Una chiave e un certificato SSL autofirmati verranno generati come parte del processo di configurazione del server, in modo che l'host virtuale SSL possa essere immediatamente utilizzato. Tuttavia, è necessario generare un certificato SSL valido per il dominio il più presto possibile, in modo che possa essere verificato correttamente dai browser. Questo può essere fatto nella pagina Gestisci certificato SSL .
