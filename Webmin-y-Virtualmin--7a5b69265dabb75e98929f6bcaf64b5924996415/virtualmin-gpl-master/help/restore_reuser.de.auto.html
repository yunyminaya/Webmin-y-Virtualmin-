Benutzer- und Gruppennamen neu zuweisen
Standardmäßig lässt Virtualmin die Wiederherstellung einer Domäne nicht zu, wenn der Administrator-Login bereits auf diesem System verwendet wird, beispielsweise von einer anderen Domäne mit einem ähnlichen Namen. Wenn Sie diese Option jedoch auf „Ja“ ändern, wird eine Neuzuweisung von Benutzer- und Gruppennamen erzwungen, wodurch ein Konflikt verhindert wird. Dies kann nützlich sein, wenn Sie example.com verwenden und eine Sicherung von example.org von einem anderen System wiederherstellen möchten.
