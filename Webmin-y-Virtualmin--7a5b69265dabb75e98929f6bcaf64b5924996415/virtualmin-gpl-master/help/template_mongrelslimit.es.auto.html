Límite en el número de instancias mestizas
Al instalar los scripts de Ruby on Rails, el administrador del servidor puede seleccionar cómo procesa mi servidor Mongrel para comenzar a manejar las solicitudes a la ruta del script, si está instalado el módulo Apache mod_proxy_balancer . Para evitar que una cantidad excesiva de procesos de servidor mestizos sobrecarguen el sistema, esta opción se puede usar para establecer el límite predeterminado en la cantidad que los administradores de dominio pueden configurar desde Virtualmin.
