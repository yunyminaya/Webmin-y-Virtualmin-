Standardverwaltungsgruppe für Domänenbesitzer
Standardmäßig entspricht die Gruppe für den Unix-Benutzer, der für die Dateien dieser Domäne erstellt wurde, der Gruppe für die E-Mail-Benutzer. Sie können jedoch stattdessen eine vorhandene Gruppe auf dem Server auswählen. Dies kann hilfreich sein, wenn für Sie gruppenbasierte Zugriffsbeschränkungen gelten (z. B. für FTP).
