Destino
Este campo determina para onde o caminho da URL acima será redirecionado. Você tem duas opções diferentes:
URL em outro site
Quando essa opção é selecionada, qualquer navegador que acesse o caminho da URL será redirecionado para essa nova URL. Se um subdiretório do caminho for acessado, o redirecionamento também incluirá o mesmo caminho.

Diretório neste sistema
Quando essa opção é escolhida, qualquer navegador que acessar o caminho da URL receberá o conteúdo do diretório especificado. Isso deve estar no diretório inicial do domínio e, de preferência em public_html contrário, é improvável que os scripts PHP funcionem.
