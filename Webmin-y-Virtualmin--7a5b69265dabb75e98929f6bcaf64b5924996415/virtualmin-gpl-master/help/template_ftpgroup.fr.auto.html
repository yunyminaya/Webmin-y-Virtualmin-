Groupe secondaire pour les utilisateurs FTP
Si cette option est définie, vous devez entrer le nom d'un groupe Unix auquel tous les utilisateurs disposant d'un accès FTP pour les serveurs virtuels sur ce modèle seront ajoutés. Si le groupe que vous spécifiez n'existe pas, il sera automatiquement créé si nécessaire.

Cette fonctionnalité est très utile en conjonction avec Usermin, qui peut limiter les modules visibles aux utilisateurs qui sont membres d'un groupe. Vous pouvez donc rendre le module de téléchargement et de téléchargement visible uniquement aux membres du groupe que vous définissez ici.
