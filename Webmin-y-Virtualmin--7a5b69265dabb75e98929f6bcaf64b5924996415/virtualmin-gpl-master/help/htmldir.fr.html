Sous-répertoire Documents du site
Ce champ vous permet de modifier le sous-répertoire d'un serveur virtuel qui contient le contenu Web à partir de la valeur par défaut de public_html. Cela peut être utile si vous avez créé le contenu ou les scripts dans un répertoire sous public_html et vous voulez le rendre visible aux utilisateurs qui accèdent au plus haut niveau URL de votre site web.
