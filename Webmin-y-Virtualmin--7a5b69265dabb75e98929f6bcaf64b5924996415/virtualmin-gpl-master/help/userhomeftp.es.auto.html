Directorio de inicio
De forma predeterminada, el directorio de inicio de un usuario FTP de un sitio web es el mismo que el directorio web base del dominio. Sin embargo, puede cambiarlo al directorio de inicio del dominio (utilizando un valor vacío) o a un subdirectorio (utilizando una ruta relativa debajo del directorio web).

Esto supone que el servidor FTP está configurado para restringir a los clientes a sus directorios de inicio, que deberían ser los predeterminados en las instalaciones estándar de Virtualmin.
