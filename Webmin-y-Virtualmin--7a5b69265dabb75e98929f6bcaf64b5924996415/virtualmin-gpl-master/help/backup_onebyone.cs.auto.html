Po zálohování přeneste každý virtuální server
Pokud je toto políčko zaškrtnuto a provádíte zálohu na vzdálený server FTP nebo SSH v novém formátu, bude soubor pro každý virtuální server přenesen po vytvoření. Tím se šetří místo na místním disku, ale v případě selhání některých virtuálních serverů hrozí riziko přenosu neúplné zálohy.
