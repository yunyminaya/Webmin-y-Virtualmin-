Server virtuale
Il modulo server virtuali Webmin fornisce un mezzo per creare, modificare ed eliminare le cassette postali degli utenti e gli account utente dei server virtuali. Con le sue funzionalità di gestione del dominio è possibile creare rapidamente e facilmente un nuovo sito Web con record di dominio, configurazione di Apache VirtualHost, una home directory, un account utente per gestirlo, configurazione del server di posta, quote di utilizzo del disco e indirizzi e alias e-mail.
