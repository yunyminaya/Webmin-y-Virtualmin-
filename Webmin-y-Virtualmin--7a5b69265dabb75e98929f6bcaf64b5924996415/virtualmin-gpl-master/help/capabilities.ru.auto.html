Можно редактировать виртуальный сервер
Позволяет редактировать функции виртуального сервера, его пароль и описание.
Может управлять пользователями
Позволяет просматривать, редактировать, создавать и удалять пользователей почтового ящика, FTP и базы данных.
Может управлять псевдонимами
Позволяет просматривать, редактировать, создавать и удалять псевдонимы электронной почты.
Может управлять базами данных
Позволяет добавлять или удалять с него дополнительные базы данных MySQL и PostgreSQL.
Можно устанавливать веб-приложения
Позволяет устанавливать сторонние веб-приложения на этот виртуальный сервер.
Можно изменить IP-адрес
Если у этого сервера есть частный IP-адрес, эта опция позволяет его изменить.
Могут ли управляемые сертификаты SSL
Позволяет загружать или повторно генерировать сертификат SSL и ключ для этого сервера.
Можно редактировать фрейм пересылки
Для веб-сайтов, предназначенных только для пересылки, этот параметр позволяет редактировать место назначения и HTML для фрейма пересылки.
Может управлять дополнительными администраторами
Позволяет просматривать, создавать, редактировать и удалять дополнительные учетные записи администратора владельца сервера.
Можно настроить доставку спама и вирусов.
Позволяет редактировать места назначения и оценки для доставки спама и вирусов.
Можно выбрать версии PHP
Для доменов с PHP, запущенным через CGI или fastCGI, позволяет устанавливать версии PHP для каждого каталога.
Можно редактировать настройки электронной почты
Позволяет настраивать автоматическое копирование скрытой копии, режим копирования псевдонимов и другие возможности, связанные с электронной почтой.
Может делать резервные копии
Позволяет создавать резервные копии принадлежащих виртуальным серверам на удаленном FTP- или SSH-сервере.
Может планировать резервное копирование
Позволяет регулярно выполнять резервное копирование виртуальных серверов.
Может восстанавливать резервные копии
Позволяет восстанавливать домашний каталог и содержимое базы данных принадлежащих виртуальных серверов.
Можно выбирать общие IP-адреса
Позволяет выбирать глобально определенные общие IP-адреса при создании виртуальных серверов.
Может создавать общие псевдонимы
Позволяет создавать псевдонимы электронной почты, которые перенаправляют всю почту в домен на другой домен или адрес.
Можно использовать редактор веб-страниц
Позволяет редактировать веб-страницы через пользовательский интерфейс Virtualmin.
Может редактировать удаленные хосты базы данных
Позволяет вводить удаленные клиентские системы, которым разрешено подключение к базам данных MySQL домена.
Можно отключить виртуальные серверы
Позволяет временно отключать серверы.
Можно удалять виртуальные серверы
Позволяет полностью удалить принадлежащие серверы, включая все субсерверы и псевдонимы.
