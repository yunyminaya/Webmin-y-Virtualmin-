Modalità di esecuzione dello script Ruby
Questa opzione determina se gli script Ruby vengono eseguiti come utente del server web Apache (tipicamente httpd o apache ) o come utente Unix che possiede il server virtuale. Il primo è più veloce in quanto significa che non è necessario avviare processi aggiuntivi per eseguire il codice Ruby, ma meno sicuro poiché gli script leggono e scrivono file come utente diverso dal proprietario del dominio.
