Backup formaat
Dit veld bepaald of de backup word weggeschreven als 1 grote file, of dat er 1 file per geselecteerde virtuele server word gemaakt met de naam zoals domein.tar.gz op een bepaalde bestemming. De Laatste optie maakt ook 1 file per domein aan, echter in een nieuw formaat wat ondersteund word in Virtualmin 2.86 en hoger, dit formaat stopt alle informatie van iedere server in zijn home directory. Dit formaat is superieur omdat Virtualmin geen tijdelijke files hoeft aan te maken in de /tmp directory, en dus harddisk ruimte bespaard.
