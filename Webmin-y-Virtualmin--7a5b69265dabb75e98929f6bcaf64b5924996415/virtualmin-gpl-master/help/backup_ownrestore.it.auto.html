Può essere ripristinato dai proprietari di server virtuali?
Se abilitato, i backup creati con questa pianificazione possono essere ripristinati dai proprietari dei loro server virtuali tramite la pagina Log di backup, oltre ai backup creati da loro stessi. Ciò consente all'amministratore principale di garantire il backup di tutto, ma offre ai proprietari di server la flessibilità di eseguire i propri ripristini.
