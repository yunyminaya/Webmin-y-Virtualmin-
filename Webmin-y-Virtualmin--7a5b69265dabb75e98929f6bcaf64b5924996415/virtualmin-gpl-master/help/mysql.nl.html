MySQL login aanmaken?
Indien dit geselecteerd is zal er een MySQL account worden aangemaakt voor dit domein, normaal gesproken met dezelfde naam als de domein administratieve inlog. Als u de standaard template instellingen gebruikt zal er ook meteen een MySQL database worden aangemaakt voor de virtuele server. De database naam zal het eerste gedeelte zijn van de domein naam - dus als u indien u een domein met de naam foo.com aanmaakt zal de database naam foo zijn.
