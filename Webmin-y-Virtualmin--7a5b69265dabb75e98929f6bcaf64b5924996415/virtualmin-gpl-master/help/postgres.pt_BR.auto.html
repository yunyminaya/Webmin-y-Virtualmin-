Criar login no PostgreSQL?
Se selecionada, uma conta do PostgreSQL será criada para esse domínio, normalmente com o mesmo nome que o logon administrativo do domínio. Se você estiver usando as configurações padrão do modelo, um banco de dados PostgreSQL também será criado para o servidor virtual. O nome do banco de dados será a primeira parte do nome do domínio - portanto, se você estiver criando o domínio foo.com , o banco de dados será chamado foo .
