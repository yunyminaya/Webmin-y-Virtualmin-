Конвертировать Alias Server
Эта страница может использоваться для преобразования псевдонима виртуального сервера в субсервер. Он больше не будет обслуживать веб-страницы старого целевого сервера, но вместо этого может иметь собственный веб-контент. Все существующие почтовые псевдонимы будут сохранены, за исключением перехватчика, который пересылает почту на старый целевой сервер. Записи DNS не будут произведены.
