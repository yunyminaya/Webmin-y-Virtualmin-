Website default HTML page
This option allows you to control the default page that will be created and displayed for the new virtual server.

You can choose "Disabled" to skip creating a page or select "Ask at creation time" to decide during virtual server creation time.

Alternatively, you can opt for a Virtualmin website default page by selecting "Yes, with the website default page template" option.

If you prefer, choosing "Yes, with content below" lets you provide custom HTML or text to use as the webpage content.
