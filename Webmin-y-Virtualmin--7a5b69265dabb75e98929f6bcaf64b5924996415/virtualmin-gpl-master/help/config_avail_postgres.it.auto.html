Server database PostgreSQL (per database)
Se abilitato, l'account del proprietario del dominio avrà accesso al modulo Webmin PostSQL. Fornisce una GUI intuitiva per i database che appartengono al proprietario del dominio. Con questo modulo e i privilegi appropriati, il proprietario del dominio può creare nuovi database, modificare database esistenti o eseguire query sui dati all'interno del database.
