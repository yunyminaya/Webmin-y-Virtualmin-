Criar página web inicial?
Para criar uma página inicial para o novo servidor virtual, você pode escolher não criar nenhuma página selecionando "Não" ou optar por uma página padrão do site Virtualmin escolhendo a opção "Sim, com o modelo de página padrão do site".

Como alternativa, selecionar "Sim, com conteúdo abaixo" permite que você insira HTML ou texto personalizado para ser usado como conteúdo da página da web.
