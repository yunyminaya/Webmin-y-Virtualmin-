Maximale FCGI-Ausführungszeit
Standardmäßig legt Virtualmin die maximale Ausführungszeit für FastCGI-Prozesse so fest, dass sie der Ausführungszeit von PHP-Skripten entspricht, die in der php.ini Datei der Domäne festgelegt ist. Dadurch wird sichergestellt, dass PHP-Anwendungen nicht länger ausgeführt werden als vorgesehen.

Sie können dieses Verhalten jedoch deaktivieren, indem Sie in diesem Konfigurationsfeld Nicht festlegen auswählen. Oder Sie können Ihr eigenes Timeout für FastCGI-Skripte festlegen, indem Sie Zeit in Sekunden auswählen und eine Zahl eingeben.

Dies wird durch Festlegen der Apache IPCCommTimeout Direktive oder des Äquivalents in Nginx implementiert.
