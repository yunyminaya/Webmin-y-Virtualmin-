Comando para listar cotas de usuário
Quando comandos de cota externa estão sendo usados, o programa definido nesta opção será chamado quando o Virtualmin precisar localizar cotas de usuário existentes. Não é chamado com nenhum parâmetro, mas deve gerar uma linha de texto por usuário, com cada linha contendo campos separados por espaço. Os campos em cada linha devem ser:
  1. O nome de usuário do Unix.
  2. Cota atual usada em blocos de 1 kB.
  3. A cota flexível em blocos de 1 kB ou 0 para ilimitado.
  4. A cota fixa em blocos de 1 kB ou 0 para ilimitado.
