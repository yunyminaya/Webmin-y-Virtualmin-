Thème pour les nouveaux propriétaires de domaines Webmin
Vous pouvez sélectionner le thème par défaut pour les connexions Webmin du propriétaire du domaine. Il s'agit par défaut de la valeur par défaut globale de Webmin, qui est le thème Virtualmin-Nuvola dans les installations de Virtualmin Professional. Il peut être utile de différencier les schémas de couleurs pour différents niveaux de privilèges, comme le choix d'un thème teinté en rouge pour les comptes administratifs, car ces comptes sont plus dangereux lorsqu'ils sont utilisés avec précaution, tout en utilisant le thème teinté gris-bleu doux standard pour votre domaine propriétaires et revendeurs. Une question de goût.
