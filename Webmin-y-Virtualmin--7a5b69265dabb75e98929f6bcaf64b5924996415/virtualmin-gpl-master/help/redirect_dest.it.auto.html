Destinazione
Questo campo determina dove verrà reindirizzato il percorso URL sopra. Hai due diverse scelte:
URL su altro sito Web
Quando questa opzione è selezionata, qualsiasi browser che accede al percorso URL verrà reindirizzato a questo nuovo URL. Se si accede a una sottodirectory del percorso, il reindirizzamento includerà anche lo stesso percorso.

Directory su questo sistema
Quando viene scelta questa opzione, a qualsiasi browser che accede al percorso URL verrà offerto il contenuto dalla directory specificata. Questo dovrebbe trovarsi nella home directory del dominio, e preferibilmente sotto public_html altrimenti è improbabile che gli script PHP funzionino.
