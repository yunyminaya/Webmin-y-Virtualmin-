Tipo di campo password
Questa impostazione controlla come appare il campo per inserire la password per un nuovo server virtuale. Le opzioni disponibili sono:
Password generata casualmente
Viene visualizzato un singolo campo visibile, contenente una password generata casualmente selezionata da Virtualmin. Tuttavia, puoi inserire una password diversa.

Inserisci la password una volta
Viene visualizzato un singolo campo per l'immissione della propria password, con la password effettiva inserita nascosta.

Inserisci la password due
Vengono visualizzati due campi per l'immissione della propria password, sempre con la password nascosta. Entrambe le password devono essere uguali.
