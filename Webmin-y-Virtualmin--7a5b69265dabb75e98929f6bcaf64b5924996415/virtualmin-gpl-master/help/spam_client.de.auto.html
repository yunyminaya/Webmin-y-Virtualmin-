SpamAssassin-Client-Programm
Diese Option bestimmt, ob die Spam-Filterung mit dem eigenständigen spamassassin Programm oder dem spamc Lightweight-Client durchgeführt wird. Im letzteren Fall muss auf Ihrem System der spamd Server ausgeführt werden. spamd wird keine Spam-Klassifizierung durchgeführt.

Die Verwendung von spamc ist in der Regel schneller und verbraucht weniger CPU. Es wird jedoch nicht mehr möglich sein, für jeden virtuellen Server eine andere SpamAssassin-Konfiguration zu haben, da der spamd Server nur einen einzigen Satz von Konfigurationsdateien lesen kann. Die Verwendung von spamc/spamd verbraucht außerdem mehr Speicher.
