Номер внешнего порта для виртуальных хостов SSL
Этот параметр можно использовать для изменения номера порта, который Virtualmin использует в URL-адресах, которые ссылаются на домены, созданные из этого шаблона. Это может быть полезно, если в вашей системе есть прокси-сервер или кеш, прослушивающий порт 443, и Virtualmin был настроен для создания виртуальных хостов Apache на другом порту.
