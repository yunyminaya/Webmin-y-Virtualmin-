Netzwerkschnittstelle
Wenn Sie einen IP-basierten virtuellen Host einrichten möchten (erforderlich für SSL-Websites oder virtuelles FTP-Hosting), muss diese Option auf Virtuelle Schnittstelle mit IP eingestellt und eine nicht verwendete Adresse eingegeben werden. Virtualmin aktiviert diese IP auf Ihrem System, wenn der Server erstellt wird, und ordnet sie dem Server zu.

Wenn die Option Freigegeben ausgewählt ist, wird stattdessen die primäre Netzwerkschnittstellenadresse des Systems für die Website und die DNS-Domäne dieses Servers verwendet.
