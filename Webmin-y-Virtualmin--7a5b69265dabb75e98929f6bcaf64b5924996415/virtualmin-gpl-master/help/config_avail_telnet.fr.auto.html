Connexion SSH / Telnet
Cette option spécifie si l'utilisateur propriétaire du domaine peut utiliser le module Webmin de connexion SSH / Telnet. Ce module est une applet Java qui se connecte au serveur via SSH ou Telnet. Il ne fonctionne pas correctement avec tous les navigateurs, en raison d'incompatibilités Java. Notez que la désactivation de cela n'empêche pas l'accès SSH au serveur par le propriétaire du domaine, mais simplement l'accès au module de connexion SSH / Telnet de Webmin.
