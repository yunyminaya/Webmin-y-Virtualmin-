Módulos Webmin disponíveis para administradores de servidores virtuais
Esta seção permite selecionar a quais módulos os proprietários do servidor virtual Webmin terão acesso, para servidores criados usando este modelo. Os links para esses módulos aparecem no menu esquerdo do Virtualmin, geralmente na categoria Serviços. Convém desativar alguns ou todos eles para limitar os tipos de alterações de configuração mais avançadas que os proprietários de servidores podem fazer e, em vez disso, restringi-los à interface de usuário mais simples do Virtualmin.

Quaisquer alterações feitas aqui serão aplicadas a todos os servidores virtuais usando este modelo imediatamente.
