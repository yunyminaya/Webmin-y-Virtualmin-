Wenn möglich, SSL-Zertifikate zwischen Domänen teilen?
Wenn diese Option auf Ja (Standardeinstellung) gesetzt ist, gibt Virtualmin SSL-Zertifikate für virtuelle Server mit demselben Besitz frei, wenn die Hostnamen im Zertifikat mit allen Domänen übereinstimmen. Wenn Sie Nein auswählen, wird für jeden neuen virtuellen Server ein separates Zertifikat erstellt.
