Reindirizzare admin.domain su Usermin?
Quando questo campo è impostato su Sì (per impostazione predefinita), tutti i nuovi server virtuali avranno una voce di admin aggiunta ai rispettivi domini DNS, con lo stesso indirizzo IP del sito Web del server. Quando si accede a http://admin.thedomain.com/ , il browser verrà reindirizzato a Virtualmin sulla porta 10000 dello stesso dominio, per accedere all'interfaccia Web di amministrazione.

Se si desidera utilizzare un URL diverso per l'accesso a Virtualmin (come quello per il quale si dispone di un certificato SSL), è possibile configurare l' URL di reindirizzamento nel seguente URL per il campo di reindirizzamento dell'amministratore.
