Crea amministratore del server
Quando questa opzione è impostata su Sì , verrà creato un utente Unix con lo stesso nome dell'amministratore del server virtuale. A seconda della configurazione del sistema, questo può essere usato per accessi SSH e FTP e l'utente sarà proprietario di tutti i file del server virtuale. Tranne quando si crea un server solo per l'hosting DNS, questo dovrebbe essere impostato su Sì .
