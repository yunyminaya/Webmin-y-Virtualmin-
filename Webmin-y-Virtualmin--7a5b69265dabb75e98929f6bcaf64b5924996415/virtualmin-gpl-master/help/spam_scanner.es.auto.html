Programa de escaneo de virus
Las posibles opciones para este campo son:
Escáner independiente ( clamscan )
Este suele ser el valor predeterminado y le indicará a Virtualmin que use el programa de escaneo ClamAV que hace todo el trabajo por sí mismo. Esto seguramente funcionará, pero puede ser más lento que la siguiente opción.
Escáner de servidor ( clamdscan )
Esta opción se puede seleccionar si el servidor de detección de virus clamd se está ejecutando en su sistema. En general, es superior, ya que la base de datos de virus ClamAV es bastante grande y el servidor solo necesita cargarla en la memoria una vez.
Escáner de servidor remoto ( clamd-stream-client )
Esta opción se puede seleccionar si el servidor de detección de virus clamd se está ejecutando en un sistema remoto y se ha configurado para aceptar conexiones de red de otros hosts. Es útil para descargar la tarea de escanear a un sistema diferente del que acepta el correo electrónico.
Otro programa
Esta opción le permite decirle a Virtualmin que use un programa completamente diferente, como su propio script de envoltura para ClamAV.
