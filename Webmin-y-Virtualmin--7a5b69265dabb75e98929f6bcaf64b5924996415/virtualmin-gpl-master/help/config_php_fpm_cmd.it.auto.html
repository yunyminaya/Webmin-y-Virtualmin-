Server PHP FPM aggiuntivo
Per impostazione predefinita, Virtualmin tenterà di trovare il server PHP FPM cercando i pacchetti di sistema, ma se lo hai installato manualmente in una posizione diversa compilando dal sorgente, il percorso del server FPM può essere impostato qui. Ciò è utile anche su altri sistemi operativi come FreeBSD dove Virtualmin non sa come è confezionato PHP-FPM. Se questa opzione è impostata, è necessario impostare anche le due opzioni seguenti.
