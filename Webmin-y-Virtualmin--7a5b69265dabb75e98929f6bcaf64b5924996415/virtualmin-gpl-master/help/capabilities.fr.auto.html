Peut modifier le serveur virtuel
Permet de modifier les fonctionnalités du serveur virtuel, son mot de passe et sa description.
Peut gérer les utilisateurs
Permet aux utilisateurs de boîtes aux lettres, FTP et de bases de données d'être visualisés, modifiés, créés et supprimés.
Peut gérer les alias
Permet d'afficher, de modifier, de créer et de supprimer des alias de courrier électronique.
Peut gérer des bases de données
Permet d'ajouter ou de supprimer des bases de données MySQL et PostgreSQL supplémentaires à ce serveur virtuel.
Peut installer des applications Web
Permet d'installer des applications Web tierces sur ce serveur virtuel.
Peut changer l'adresse IP
Si ce serveur possède une adresse IP privée, cette option permet de la modifier.
Peut gérer les certificats SSL
Permet de télécharger ou de régénérer le certificat SSL et la clé de ce serveur.
Peut modifier le cadre de transfert
Pour les sites Web de transfert uniquement, cette option permet de modifier la destination et le code HTML du cadre de transfert.
Peut gérer des administrateurs supplémentaires
Permet d'afficher, de créer, de modifier et de supprimer des comptes d'administrateur de propriétaire de serveur supplémentaires.
Peut configurer la livraison de spam et de virus
Permet de modifier les destinations et les scores de diffusion du spam et des virus.
Peut sélectionner les versions PHP
Pour les domaines avec PHP exécutés via CGI ou fastCGI, permet de définir des versions PHP par répertoire.
Peut modifier les paramètres de messagerie
Permet la configuration du BCC automatique, du mode de copie d'alias et d'autres options liées au courrier électronique.
Peut faire des sauvegardes
Permet aux serveurs virtuels détenus d'être sauvegardés sur un serveur FTP ou SSH distant.
Peut planifier des sauvegardes
Permet aux serveurs virtuels d'être sauvegardés selon un calendrier régulier.
Peut restaurer les sauvegardes
Permet de restaurer le contenu du répertoire personnel et de la base de données des serveurs virtuels détenus.
Peut sélectionner des adresses IP partagées
Permet de sélectionner des adresses IP partagées définies globalement lors de la création de serveurs virtuels.
Peut créer des alias fourre-tout
Permet la création d'alias de messagerie qui transfèrent tous les messages d'un domaine vers un domaine ou une adresse différente.
Peut utiliser l'éditeur de pages Web
Permet de modifier les pages Web via l'interface utilisateur de Virtualmin.
Peut modifier les hôtes distants de la base de données
Permet d'entrer des systèmes clients distants autorisés à se connecter aux bases de données MySQL du domaine.
Peut désactiver les serveurs virtuels
Permet aux serveurs d'être temporairement désactivés.
Peut supprimer des serveurs virtuels
Permet de supprimer complètement les serveurs détenus, y compris tous les sous-serveurs et alias.
