Redirecionar subdiretórios para
Esta opção controla como os caminhos no caminho do URL de origem são redirecionados. Por exemplo, se o caminho de origem for /foo e o destino for http://foo.com , os significados das opções são:

  • Mesmo subdiretório no URL de destino
    Uma solicitação para /foo/bar será redirecionada para http://foo.com/bar

  • Apenas URL de destino
    Uma solicitação para /foo/bar será redirecionada apenas para http://foo.com

  • Ignorar subdiretórios
    Uma solicitação para /foo/bar não será redirecionada, apenas uma solicitação para /foo será.
