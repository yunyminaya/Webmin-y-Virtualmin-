Buscar registros de correo
Esta página se puede usar para buscar en los registros de correo electrónico mensajes a algún servidor virtual, o todos los servidores si usted es el administrador maestro. Para buscar, puede seleccionar uno o más de los siguientes criterios, todos los cuales deben coincidir:

Fecha de inicio
Se mostrará cualquier correo electrónico recibido en o después de esta fecha. Si se deja vacío, el correo electrónico desde el inicio de los registros del servidor coincidirá.
Fecha final
Se mostrará cualquier correo electrónico recibido en esta fecha o antes. Si se deja vacío, la fecha actual coincidirá.
Dirección de la fuente
Si se completa, se mostrará el correo electrónico para el que la dirección del remitente contiene el nombre de dominio o la dirección que ingresó.
Dirección de destino
Solo se mostrará el correo electrónico que coincida con el servidor virtual seleccionado en este menú. Además, si el campo de nombre de usuario se completa, solo se mostrará el correo a ese buzón.
En la lista resultante, se mostrarán la fecha, la hora, el remitente, la dirección del destinatario, el usuario del buzón local (si corresponde) y la dirección o el archivo de destino final. Para ver más detalles de un mensaje registrado, haga clic en la fecha u hora de recepción.
