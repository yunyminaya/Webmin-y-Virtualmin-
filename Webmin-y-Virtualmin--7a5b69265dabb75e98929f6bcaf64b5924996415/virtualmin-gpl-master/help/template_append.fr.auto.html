Format pour les noms d'utilisateur qui incluent un domaine
Lorsqu'un nom d'utilisateur est accompagné du nom de domaine, cette option sélectionne le format. Toutes les options ne sont pas disponibles sur toutes les plateformes de système d'exploitation, car certains caractères ne sont pas autorisés dans les noms d'utilisateur. Que le nom de domaine soit le nom d'utilisateur complet, c'est-à-dire "virtualmin.com", ou seulement la première partie, c'est-à-dire "virtualmin", est dicté par l'option Style de nom de domaine dans les noms d'utilisateur de la page de configuration du module.
