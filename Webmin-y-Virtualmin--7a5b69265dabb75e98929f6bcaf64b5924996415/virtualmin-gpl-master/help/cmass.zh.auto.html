批量创建服务器
该页面允许您从一个文本文件一次创建多个虚拟服务器。该文件必须每行列出一台服务器，并且每行必须具有以下格式：

domain : description : password :[ username ]:[ parent-domain ]:[ ip-address ]:[ alias-domain ]

这些字段的含义是：

域
要创建的虚拟服务器的域名，例如foo.com 。
描述
易于理解的描述，例如Joe's Website 。
密码
服务器管理登录名的密码。仅父服务器才需要。
用户名
服务器的管理用户名。如果将此留空，则会自动选择一个用户名。
父域
在其下创建此服务器的父服务器的域名。如果将其保留为空，则将新服务器本身创建为父服务器。
IP地址
对于不需要自己的专用IP的虚拟服务器，此字段可以保留为空白。否则，您必须输入一个有效的未使用地址，或使用“ allocate ”一词使Virtualmin为您选择一个（如果已配置）。
别名域
应该成为其别名的虚拟服务器的域名。如果省略，则新服务器将是顶级服务器或子服务器。

已创建服务器的配额，带宽和其他限制是根据您在此表单上选择的模板设置的。就像正常的单服务器创建页面一样，创建的服务器的功能由功能复选框确定。

如果您的系统使用代理商帐户，则可以选择在代理商的所有权下创建所有虚拟服务器。此选项仅对主管理员可用。
