Configurar site SSL também?
Quando essa opção está ativada, um host virtual Apache adicional será criado para esse servidor virtual, escutando na porta 443. Isso permitirá que os navegadores se conectem a https: // domainname.com / usando criptografia SSL, bem como a http: / / domainname.com / sem criptografia.

Uma chave e um certificado SSL autoassinados serão gerados como parte do processo de configuração do servidor, para que o host virtual SSL possa ser usado imediatamente. No entanto, você deve gerar um certificado SSL válido para o domínio o mais rápido possível, para que possa ser verificado corretamente pelos navegadores. Isso pode ser feito na página Gerenciar certificado SSL .
