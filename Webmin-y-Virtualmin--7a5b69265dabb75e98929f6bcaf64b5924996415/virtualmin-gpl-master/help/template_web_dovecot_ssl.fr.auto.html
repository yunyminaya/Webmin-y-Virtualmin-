Copier des certificats SSL par IP dans Dovecot?
Lorsque cette option est activée et qu'un serveur virtuel est créé avec SSL activé et sa propre IP privée, Dovecot sera configuré pour utiliser le certificat SSL du serveur pour les connexions à cette adresse IP. Cela peut être utile si les clients de messagerie se connectent au nom de domaine en mode SSL et attendent un certificat correspondant.
