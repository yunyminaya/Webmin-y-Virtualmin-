Paramètres de messagerie
Cette page sert à configurer les paramètres liés aux e-mails pour un serveur virtuel. Si votre système prend en charge les e-mails envoyés par BCCing (et qu'il est activé sur la page Configuration du module), la section des e-mails sortants BCC vers peut être utilisée pour sélectionner une adresse vers laquelle copier les e-mails.

Si le serveur virtuel est un domaine d'alias, le champ Mode d'alias de messagerie contrôle si les alias de messagerie sont copiés à partir du domaine cible, ou si un seul alias catchall est créé pour transférer tous les e-mails vers la cible. L'ancien mode n'est pas disponible avec tous les serveurs de messagerie, mais il est supérieur lors de l'utilisation de Postfix car il permet de renvoyer le courrier vers des adresses invalides au niveau de la conversation SMTP.
