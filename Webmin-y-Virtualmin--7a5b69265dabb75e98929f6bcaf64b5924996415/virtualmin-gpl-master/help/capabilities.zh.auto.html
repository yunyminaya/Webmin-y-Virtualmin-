可以编辑虚拟服务器
允许编辑虚拟服务器功能、其密码和描述。
可以管理用户
允许查看、编辑、创建和删除邮箱、FTP 和数据库用户。
可以管理别名
允许查看、编辑、创建和删除电子邮件别名。
可以管理数据库
允许向该虚拟服务器添加或删除其他 MySQL 和 PostgreSQL 数据库。
可以安装 webapps
允许第三方 Web 应用程序安装到此虚拟服务器。
可以更改IP地址
如果该服务器具有私有 IP 地址，则此选项允许更改它。
可以管理 SSL 证书
允许上传或重新生成此服务器的 SSL 证书和密钥。
可以编辑转发框架
对于仅转发的网站，此选项允许编辑转发框架的目标和 HTML。
可以管理额外的管理员
允许查看、创建、编辑和删除额外的服务器所有者管理员帐户。
可以配置垃圾邮件和病毒传递
允许编辑垃圾邮件和病毒传递目的地和分数。
可以选择 PHP 版本
对于通过 CGI 或 fastCGI 运行 PHP 的域，允许设置每个目录的 PHP 版本。
可以编辑电子邮件设置
允许配置自动密件抄送、别名复制模式和其他与电子邮件相关的选项。
可以备份
允许将拥有的虚拟服务器备份到远程 FTP 或 SSH 服务器。
可以安排备份
允许定期备份虚拟服务器。
可以恢复备份
允许恢复所拥有的虚拟服务器的主目录和数据库内容。
可以选择共享 IP
允许在创建虚拟服务器时选择全局定义的共享 IP 地址。
可以创建通用别名
允许创建电子邮件别名，将发往一个域的所有邮件转发到不同的域或地址。
可以使用网页编辑器
允许通过 Virtualmin UI 编辑网页。
可以编辑数据库远程主机
允许进入允许连接到域的 MySQL 数据库的远程客户端系统。
可以禁用虚拟服务器
允许暂时禁用服务器。
可以删除虚拟服务器
允许彻底删除拥有的服务器，包括所有子服务器和别名。
