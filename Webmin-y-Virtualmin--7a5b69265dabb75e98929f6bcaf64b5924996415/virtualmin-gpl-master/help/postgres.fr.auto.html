Créer une connexion PostgreSQL?
Si cette option est sélectionnée, un compte PostgreSQL sera créé pour ce domaine, généralement avec le même nom que la connexion administrative du domaine. Si vous utilisez les paramètres de modèle par défaut, une base de données PostgreSQL sera également créée pour le serveur virtuel. Le nom de la base de données sera la première partie du nom de domaine - donc si vous créez le domaine foo.com , la base de données s'appellera foo .
