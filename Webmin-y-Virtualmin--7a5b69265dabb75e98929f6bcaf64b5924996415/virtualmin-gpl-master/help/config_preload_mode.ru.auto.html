Предзагрузить библиотеки Virtualmin при запуске?
Чтобы повысить производительность сценариев CGI, составляющих Virtualmin, Webmin можно настроить для загрузки некоторых из своих библиотек Perl в память во время запуска, чтобы их не приходилось считывать с диска и анализировать для каждого запроса. Однако для этого требуется больше оперативной памяти - примерно 100 МБ в типичной системе Linux.

Этот параметр дает вам больше контроля над тем, сколько предварительной загрузки сделано. Варианты:

Все библиотеки
Загружены библиотеки для всех модулей и плагинов Webmin, используемых Virtualmin. Процесс сервера Webmin будет занимать примерно 100 МБ ОЗУ.
Только ядро
Загружаются только те библиотеки, которые являются частью самого Virtualmin. Процесс сервера Webmin будет использовать около 50 МБ ОЗУ.
нет
Ничего предварительно не загружено - вместо этого все библиотеки загружаются по требованию. Процесс сервера Webmin будет использовать только 15M RAM.
