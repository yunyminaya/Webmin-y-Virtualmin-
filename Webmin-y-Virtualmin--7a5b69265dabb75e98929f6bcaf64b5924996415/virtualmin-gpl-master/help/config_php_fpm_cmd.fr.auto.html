Serveur PHP FPM supplémentaire
Par défaut, Virtualmin tentera de trouver le serveur PHP FPM en recherchant les packages système, mais si vous l'avez installé manuellement dans un emplacement différent en compilant à partir des sources, le chemin d'accès au serveur FPM peut être défini ici. Ceci est également utile sur d'autres systèmes d'exploitation comme FreeBSD où Virtualmin ne sait pas comment PHP-FPM est empaqueté. Si cette option est définie, vous devez également définir les deux options suivantes.
