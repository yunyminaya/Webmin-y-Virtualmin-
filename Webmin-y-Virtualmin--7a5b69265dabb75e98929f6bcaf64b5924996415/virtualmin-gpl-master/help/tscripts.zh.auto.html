模板 Webapp 安装程序
此页面允许您指定使用此模板创建新虚拟服务器时要安装的第三方 Web 应用程序。对于每个 Web 应用程序，必须输入以下信息：

Web 应用程序
要安装的 Web 应用的名称和版本
URL 路径
虚拟服务器网站下安装 webapp 的路径，例如/wordpress 。这还将确定~/public_html下包含 webapp 文件的目录。
数据库名称
Web 应用将用于存储数据的数据库的名称。选项包括：
没有任何
对于不需要数据库的 Web 应用程序（例如Gallery） ，应该选择此选项。
主数据库
Web 应用程序的数据库将是所选类型的虚拟服务器的主数据库。
其他
如果您希望此 Web 应用使用其自己的私有数据库，可以选择此选项并在相邻的文本字段中输入数据库的模式。例如，您可以输入${DB}_phpnuke 。如果生成的数据库不存在，则会在 Web 应用安装过程中创建它。
数据库类型
如果选择了，则表示 Web 应用程序数据使用的数据库类型。
