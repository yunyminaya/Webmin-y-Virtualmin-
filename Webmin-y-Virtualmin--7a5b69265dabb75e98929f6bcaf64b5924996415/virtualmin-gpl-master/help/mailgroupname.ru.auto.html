Группа для пользователей почтовых ящиков
Каждый домен должен иметь связанную группу Unix, которая будет использоваться в качестве основной группы для любых пользователей Unix почтовых ящиков. По умолчанию имя будет выбираться автоматически из первой части имени домена, но вы можете использовать это поле для указания любого имени группы, которое вам нравится, если оно еще не существует.
