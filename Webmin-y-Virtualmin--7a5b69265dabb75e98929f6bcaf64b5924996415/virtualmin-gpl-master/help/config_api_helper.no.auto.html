Sti for å opprette API-hjelperkommando
Som standard oppretter Virtualmin virtualmin kommandolinjen API-hjelpeskript i /usr/sbin eller /usr/local/sbin katalogen, avhengig av hvilken som kan skrives på systemet ditt. Imidlertid kan dette konfigurasjonsfeltet brukes til å velge et annet sted, for eksempel /opt/bin/virtualmin som passer for nettstedet ditt.
