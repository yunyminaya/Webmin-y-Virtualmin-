Types de mot de passe hachés à stocker
Par défaut, Virtualmin stockera les mots de passe de domaine et de boîte aux lettres dans différents formats de hachage afin que le mot de passe soit disponible lors de l'ajout de nouvelles fonctionnalités au domaine. Cependant, tous les formats de hachage ne sont pas également sécurisés, ce champ peut donc être utilisé pour désactiver le stockage de certains formats. Indépendamment de ce que vous sélectionnez ici, le mot de passe sera toujours stocké haché en utilisant le format par défaut du système dans /etc/shadow ou LDAP.

Sachez que la désactivation du format de hachage MySQL peut empêcher Virtualmin de créer une connexion MySQL pour les utilisateurs de boîtes aux lettres avec le même mot de passe que la boîte aux lettres a été initialement attribuée. La désactivation du format de résumé Web peut empêcher l'accès à un répertoire Web protégé par les utilisateurs de boîtes aux lettres.
