Modèle pour le répertoire d'application Web par défaut
Par défaut, Virtualmin installera les nouvelles applications Web dans un sous-répertoire portant le même nom que le code du script, comme wordpress . Ce champ vous permet de saisir un modèle pour un répertoire par défaut différent, comme webapps/${SCRIPTNAME} . Toutes les variables de modèle habituelles telles que ${DOM} et ${USER} peuvent également être utilisées.
