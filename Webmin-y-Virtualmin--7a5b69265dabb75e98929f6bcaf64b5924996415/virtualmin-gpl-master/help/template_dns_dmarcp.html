DMARC policy for emails that fail SPF or DKIM
If a message fails SPF or DKIM validation, this setting determines how the receiving mail server should handle it. The options are :
  • Take no action - The message is still delivered as normal.
  • Quarantine email - This is recipient-dependent, but typically means that the message will be placed into a spam folder.
  • Reject email - The message will be deleted.
