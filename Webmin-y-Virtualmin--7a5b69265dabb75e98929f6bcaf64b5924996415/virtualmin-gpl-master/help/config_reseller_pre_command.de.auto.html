Befehl zum Ausführen, bevor Änderungen an einem Reseller vorgenommen werden
Mit dieser Option können Sie ein Skript oder einen Befehl angeben, der ausgeführt werden soll, bevor ein Reseller erstellt, geändert oder gelöscht wird. Mithilfe der Umgebungsvariablen RESELLER_NAME und RESELLER_ACTION kann ermittelt werden, welcher Reseller geändert wird und was damit geschieht. Wenn der Befehl mit einem Status ungleich Null beendet wird, wird eine Änderung des Wiederverkäufers abgelehnt.
