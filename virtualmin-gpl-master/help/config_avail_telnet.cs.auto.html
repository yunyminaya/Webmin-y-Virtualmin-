Přihlášení SSH / Telnet
Tato volba určuje, zda uživatel vlastníka domény může použít modul WebH Login Přihlášení SSH / Telnet. Tento modul je Java applet, který se přihlašuje k serveru pomocí SSH nebo Telnet. Nefunguje správně u všech prohlížečů kvůli nekompatibilitě Java. Zakázáním to nezabrání přístupu SSH k serveru ze strany vlastníka domény, pouze přístupu k modulu SSH / Telnet Login Webmin.
