URL para redirecionamento de administrador
Por padrão, o Virtualmin redirecionará solicitações para http://admin.thedomain.com para Webmin no mesmo sistema, como https://thedomain.com:10000/ . No entanto, esse campo pode ser usado para alterar o URL de redirecionamento, talvez para um único nome de host para todos os seus domínios para os quais você possui um certificado SSL, como https://yourcompany.com:10000/ .
