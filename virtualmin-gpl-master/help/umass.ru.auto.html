Пакетное создание пользователей
Эта страница позволяет создавать сразу несколько пользователей почты или FTP из текстового файла. Этот файл должен содержать по одному пользователю в каждой строке, и каждая строка должна иметь следующий формат:

username : realname : password : ftp-flag : email-flag : quota :[ extras ]:[ forwards ]:[ databases ]

Значения полей:

имя пользователя
Короткое имя пользователя, например, joe . При необходимости добавляется префикс домена для создания реального имени пользователя.
настоящее имя
Настоящее имя пользователя, как у Джо Смита .
пароль
Исходный пароль для пользователя, в виде простого текста.
FTP-флаг
Если для этого параметра установлено значение 1, пользователю будет разрешено входить через FTP. Если установить значение 0 или оставить его пустым, доступ к FTP невозможен. Вы также можете использовать полный путь к допустимой оболочке для пользователей почтовых ящиков, например /usr/bin/scponly .
электронный флаг
Если установлено значение 1, основной адрес электронной почты пользователя будет включен.
квота
Квота почтового и домашнего каталога пользователя в дисковых блоках 1 КБ.
дополнительные услуги
Разделенный запятыми список дополнительных адресов электронной почты для этого пользователя. Они должны находиться в доменах Virtualmin, которыми вы управляете.
вперед
Разделенный запятыми список адресатов пересылки электронной почты для этого пользователя (обычно адреса, такие как joe@otherdomain.com ).
базы данных
Разделенный запятыми список баз данных, которым должен быть предоставлен доступ для этого пользователя. Каждый должен иметь тип базы данных ( mysql или postgres ) и имя базы данных, разделенные пробелом.
