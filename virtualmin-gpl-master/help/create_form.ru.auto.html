Создать виртуальный сервер
Эта форма позволяет вам создать новый виртуальный сервер верхнего уровня, который будет иметь новый связанный логин администратора. При создании сервера необходимо ввести хотя бы имя домена, описание и пароль администратора. Если в системе определено несколько шаблонов серверов, вы можете выбрать один из них в меню - шаблоны определяют различные значения по умолчанию для нового сервера, такие как конфигурация веб-сайта, записи DNS и т. д.

В разделе «Дополнительные параметры» вы можете выбрать другое имя администратора, имя группы и имя базы данных для сервера, а не значения по умолчанию, которые взяты из первой части имени домена. Также можно ввести контактный адрес электронной почты для сервера - в идеале это должен быть существующий адрес человека, который будет управлять сервером, в другом домене.

Чтобы выбрать, какие функции включены для сервера, используйте флажки во второй части раздела «Дополнительные параметры» . Некоторые функции зависят от других и не могут быть выбраны самостоятельно - например, отчетность Webalizer по статистике веб-сайта зависит от фактического наличия веб-сайта.

В нижней части раздела «Дополнительные параметры» можно выбрать пересылку всего веб-трафика на этот новый сайт по другому URL-адресу и пересылку всей электронной почты. Вы также можете выбрать, чтобы новый виртуальный сервер использовал свои собственные частные IP-адреса, что требуется для SSL и анонимного FTP-хостинга.
