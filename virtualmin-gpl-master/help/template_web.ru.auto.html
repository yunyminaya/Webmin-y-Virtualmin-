Директивы и настройки для новых сайтов
Можно полностью настроить директивы Apache, которые добавляются при создании нового виртуального сервера. Подстановка переменных выполняется для всех стандартных переменных шаблона Virtualmin. Следует соблюдать осторожность с директивами Apache, так как некоторые из них связаны с проблемами безопасности. С документацией Apache и, в меньшей степени, разделами Apache документации Webmin можно ознакомиться, чтобы узнать больше о конфигурации Apache.
