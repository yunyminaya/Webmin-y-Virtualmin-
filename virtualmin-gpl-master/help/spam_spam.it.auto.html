Destinazione per le email di spam
Questo campo determina cosa succede ai messaggi di posta elettronica che sono stati identificati da SpamAssassin come spam, in base al loro punteggio calcolato e alla soglia di spam del destinatario. Le opzioni disponibili sono:

Buttar via
Tutti i messaggi classificati come spam verranno immediatamente eliminati.
Consegna normalmente
Lo spam verrà recapitato nella cassetta postale del destinatario, ma con le intestazioni X-Spam che possono essere utilizzate per il filtro da un client di posta.
Scrivi nel file spam standard ~ / mail / spam
Lo spam viene scritto nella cartella standard per l'analisi manuale e la cancellazione.
Scrivi nel file di posta nella home directory
Lo spam viene salvato nel file di posta nella directory home dell'utente immessa. Per scrivere in una directory di formato Maildir, inserire un / alla fine del nome file.
Inoltra all'indirizzo e-mail
Tutti i messaggi classificati come spam verranno inoltrati all'indirizzo inserito.
Scrivi in un altro file
Lo spam verrà scritto nel file di posta specificato (o nella directory se il percorso termina con /), che può trovarsi ovunque sul server.
