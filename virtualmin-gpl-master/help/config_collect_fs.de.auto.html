Dateisysteme, die in den lokalen Speicherplatz aufgenommen werden sollen
Standardmäßig wird auf der Seite Systeminformationen Speicherplatz auf allen lokalen Dateisystemen angezeigt. Mit dieser Option können Sie jedoch eine durch Leerzeichen getrennte Liste von Einhängepunkten eingeben, die stattdessen nur überprüft werden sollen. Dies kann nützlich sein, wenn Sie sich beispielsweise nur für die Verwendung zu Hause interessieren.

Nachdem Sie diese Einstellung geändert haben, müssen Sie auf der Seite Systeminformationen auf Systeminformationen aktualisieren klicken.
