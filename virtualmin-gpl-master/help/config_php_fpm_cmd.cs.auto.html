Další PHP FPM server
Ve výchozím nastavení se Virtualmin pokusí najít PHP FPM server prohledáním systémových balíčků, ale pokud jste jej ručně nainstalovali do jiného umístění kompilací ze zdroje, cestu k FPM serveru lze nastavit zde. To je také užitečné na jiných operačních systémech, jako je FreeBSD, kde Virtualmin neví, jak je PHP-FPM zabaleno. Pokud je tato možnost nastavena, měli byste také nastavit dvě následující možnosti.
