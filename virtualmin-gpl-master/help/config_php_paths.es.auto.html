Ubicaciones adicionales para comandos PHP
Virtualmin detectará automáticamente las principales versiones de PHP instaladas en su sistema, siempre que estén en directorios en la ruta de búsqueda. Sin embargo, este campo se puede usar para ingresar las rutas a ejecutables PHP adicionales, lo que puede ser útil si desea ejecutar una versión que sea superior a la que viene instalada de manera predeterminada.
