Диапазоны для автоматического назначения IP
Если вы хотите, чтобы Virtualmin автоматически выделял IP-адрес для новых виртуальных серверов, созданных на основе этого шаблона, вы можете ввести один или несколько диапазонов для выбора IP-адресов здесь. В противном случае любой частный виртуальный IP-адрес необходимо вводить вручную при создании домена.
