¿Forzar que el nombre del grupo sea siempre el mismo que el nombre de usuario?
Cuando se establece en Sí , Virtualmin siempre hará que el nombre del grupo Unix para los nuevos dominios sea el mismo que el nombre de usuario de forma predeterminada, a menos que ya esté en el usuario. Cuando se elige No , el nombre del grupo se toma de la primera parte del nombre de dominio o dominio completo.
