MySQL Database Server (für Datenbank)
Wenn aktiviert, hat das Domänenbesitzerkonto Zugriff auf das MySQL Webmin-Modul. Es bietet eine benutzerfreundliche Benutzeroberfläche für die Datenbanken, die dem Domäneninhaber gehören. Mit diesem Modul und den entsprechenden Berechtigungen kann der Domänenbesitzer neue Datenbanken erstellen, vorhandene Datenbanken bearbeiten oder Abfragen für die Daten in der Datenbank durchführen.
