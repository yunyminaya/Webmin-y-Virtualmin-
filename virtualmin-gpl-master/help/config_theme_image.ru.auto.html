URL для изображения логотипа компании
Если для этой опции задан URL-адрес какого-либо изображения, он будет отображаться в верхнем левом углу пользовательского интерфейса Virtualmin при использовании темы в рамке. Вы можете установить его для логотипа вашей хостинговой компании, который при нажатии откроет URL, указанный в параметре Ссылка для логотипа компании .

Хотя точный допустимый размер изображения зависит от браузера, используемого для доступа к Virtualmin, изображение должно иметь ширину не более 220 пикселей. Высота не ограничена, но 120 пикселей кажется разумным размером.
