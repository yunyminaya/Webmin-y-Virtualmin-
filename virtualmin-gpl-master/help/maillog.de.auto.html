Mail-Protokolle durchsuchen
Auf dieser Seite können Sie die E-Mail-Protokolle nach Nachrichten an einen virtuellen Server oder an alle Server durchsuchen, wenn Sie der Hauptadministrator sind. Um zu suchen, können Sie eines oder mehrere der folgenden Kriterien auswählen, die alle übereinstimmen müssen:

Anfangsdatum
Alle E-Mails, die an oder nach diesem Datum eingehen, werden angezeigt. Wenn leer, werden E-Mails vom Anfang der Serverprotokolle abgeglichen.
Endtermin
Alle E-Mails, die an oder vor diesem Datum eingehen, werden angezeigt. Wenn leer, wird das aktuelle Datum per E-Mail abgeglichen.
Quelladresse
Wenn ausgefüllt, wird eine E-Mail angezeigt, für die die Absenderadresse den von Ihnen eingegebenen Domainnamen oder die von Ihnen eingegebene Adresse enthält.
Zieladresse
Es werden nur E-Mails angezeigt, die mit dem in diesem Menü ausgewählten virtuellen Server übereinstimmen. Wenn das Feld Benutzername ausgefüllt ist, werden nur E-Mails an dieses Postfach angezeigt.
In der resultierenden Liste werden Datum, Uhrzeit, Absender, Empfängeradresse, lokaler Postfachbenutzer (falls vorhanden) und endgültige Zieldatei oder Adresse angezeigt. Um weitere Details einer protokollierten Nachricht anzuzeigen, klicken Sie auf das Datum oder die Uhrzeit des Empfangs.
