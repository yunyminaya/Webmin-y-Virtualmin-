Команда для установки квоты пользователя
Когда используются внешние команды квот, программа, установленная в этой опции, будет вызываться, когда Virtualmin захочет установить квоты для пользователя Unix. Параметры, с которыми он вызывается:
  1. Имя пользователя Unix
  2. Новая мягкая квота в блоках по 1 кБ или 0 для неограниченного количества.
  3. Новая жесткая квота в блоках по 1 кБ или 0 для неограниченной.
