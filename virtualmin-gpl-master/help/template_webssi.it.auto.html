Le inclusioni lato server sono una funzione server web che consente di utilizzare semplici direttive macro nei file HTML. Possono essere utilizzati per includere altri file HTML, impostare e visualizzare il contenuto delle variabili e controllare il contenuto visualizzato nella pagina con le istruzioni if .
