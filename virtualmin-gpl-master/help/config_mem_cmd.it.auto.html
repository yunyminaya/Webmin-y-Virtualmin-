Comando per ottenere l'utilizzo della memoria
Per impostazione predefinita, Virtualmin otterrà la memoria totale e libera dai comandi appropriati per il tuo sistema operativo. Tuttavia, se questi non sono precisi o non riflettono l'utilizzo che l'utente desidera vedere (come in un ambiente VPS), questo campo può essere impostato su un comando che deve generare i seguenti valori, uno per riga:

  1. Memoria reale totale in byte
  2. Memoria reale libera in byte
  3. Spazio di scambio totale in byte o 0 se nessuno
  4. Spazio di scambio libero in byte
