Server di posta secondari
Un server di posta secondario è quello che può anche ricevere e-mail per uno dei server virtuali gestiti da Virtualmin, ma solo come inoltro temporaneo piuttosto che come destinazione finale. È consigliabile configurare un server secondario per i domini di posta molto utilizzati, che funga da backup o da luogo di attesa nel caso in cui il server di posta principale sia inattivo.

In questa pagina è possibile selezionare altri server con Virtualmin installato da utilizzare come server MX secondari per i nuovi domini di posta creati su questo sistema. Prima che appaiano i server, devono prima essere aggiunti al modulo Webmin Server Index, in cui è necessario inserire il login e la password root per il server remoto.

Per ogni server di posta secondario, hai la possibilità di inserire un nome host diverso da utilizzare nel record DNS MX, invece del nome host effettivo del server, ad esempio secmx.yourdomain.com . Tuttavia, assicurati che qualunque cosa tu inserisca sia un nome DNS valido che si risolva nello stesso server.

Se si fa clic sul pulsante Salva e applica , tutti i server virtuali esistenti per i quali è abilitata la posta elettronica verranno aggiunti a (o rimossi da) tutti i server di posta secondari selezionati. Inoltre, i record DNS MX verranno aggiunti (o rimossi dai) relativi domini per fare riferimento ai server di posta.
