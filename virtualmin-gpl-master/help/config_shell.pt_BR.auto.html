Shell para usuários de caixa de correio
Esta opção especifica o shell configurado para usuários de caixa de correio. Um usuário de caixa de correio é o tipo de usuário menos privilegiado e normalmente deve ter acesso muito restrito. No entanto, se o servidor Virtualmin for usado apenas por usuários confiáveis ou usuários que precisam ter acesso ao shell, você poderá alterar isso do padrão de / dev / null para um dos shells reais do sistema. Observe que / bin / false é um shell especial que fecha logins SSH remotos, mas ainda permite logins FTP.
