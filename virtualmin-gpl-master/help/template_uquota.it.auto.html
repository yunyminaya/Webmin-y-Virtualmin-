Quota per l'amministratore del server
Questa opzione specifica la quota di utilizzo predefinita del disco per l'account utente Unix del proprietario del server. Si applicherà ai file di proprietà del proprietario del dominio (probabilmente tutto il contenuto Web), alla posta elettronica nella cassetta postale dei proprietari del dominio e, eventualmente, ad altri file all'interno di eventuali sub-server che potrebbero esistere. Potrebbe essere sovrascritto durante la creazione o la modifica del server virtuale.
