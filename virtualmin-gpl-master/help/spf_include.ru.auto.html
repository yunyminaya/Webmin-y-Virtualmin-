Включенные домены, чтобы позволить
В этом поле перечислены дополнительные имена доменов, чьи записи SPF также считываются для определения разрешенных хостов для этого домена. Любой почтовый сервер, который считается действительным для перечисленных доменов, таким образом, будет действительным и для этого домена.
