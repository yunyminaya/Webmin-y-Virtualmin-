Convalidare il certificato SSL?
Questa opzione determina se la convalida del certificato SSL sarà abilitata per i nuovi server virtuali creati da Webmin che hanno SSL abilitato. Puoi scegliere di controllare solo la data di scadenza del certificato o anche di convalidare che il nome di dominio nel certificato corrisponda all'URL del server virtuale.

Quando mancano meno di 7 giorni alla scadenza del certificato di un dominio, Virtualmin invierà un'e-mail di avviso al proprietario del server virtuale, dandogli il tempo di ordinarne il rinnovo.
