Correo electrónico de contacto
Este campo se puede usar para especificar una dirección de correo electrónico a la que se enviarán todos los mensajes relacionados con este servidor virtual. Por defecto, se enviarán al usuario de administración que posee el servidor. El mensaje más común es el que se envía cuando se crea el dominio, que contiene información sobre su dirección, características, inicio de sesión y contraseña.
