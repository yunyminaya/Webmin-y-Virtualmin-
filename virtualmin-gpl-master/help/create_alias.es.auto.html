Crear servidor de alias
Este formulario le permite crear un nuevo servidor de alias, que es esencialmente un nombre alternativo para un servidor existente. Los alias no tienen su propio contenido de sitio web y comparten todas las cuotas y límites con el servidor de destino. El ancho de banda utilizado por el servidor de alias se cuenta para el total del objetivo.

Al crear el servidor, debe ingresar al menos el nombre de dominio y una descripción. Si hay varias plantillas de servidor definidas en el sistema, puede seleccionar una del menú. Para seleccionar qué características del servidor de alias se reenvían al dominio de destino, use las casillas de verificación en la segunda parte de la sección Opciones avanzadas . Solo están disponibles DNS, sitios web y alias de correo electrónico.
