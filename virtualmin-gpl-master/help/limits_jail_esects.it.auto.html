Sezioni extra da abilitare per Jailkit
Per impostazione predefinita, solo i file essenziali vengono copiati dalle sezioni di configurazione di Jailkit in jail nuove o aggiornate. Per includere file aggiuntivi, elenca qui le sezioni desiderate da /etc/jailkit/jk_init.ini .

Le sezioni nel file /etc/jailkit/jk_init.ini sono denominate [git] o [apacheutils] . Ad esempio, per includere le sezioni git e apacheutils in questo ambiente chroot del dominio, inserisci git apacheutils (separato da uno spazio) in questo campo.
