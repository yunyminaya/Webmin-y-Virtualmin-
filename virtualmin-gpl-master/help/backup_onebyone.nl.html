Iedere virtuele server verplaatsen na de backup
Indien u dit heeft aangevinkt en u een backup maakt naar een remote FTP of SSH server in het nieuwe formaat, zal de file van iedere individuele server worden verplaatst nadat die is aangemaakt. Dit bespaart locale harddisk ruimte, echter u loopt het risico om een incomplete backup te verplaatsen indien de backup van een virtuele server mislukt.
