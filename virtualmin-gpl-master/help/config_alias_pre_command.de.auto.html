Befehl zum Ausführen, bevor Änderungen an einem Alias vorgenommen werden
Mit dieser Option können Sie ein Skript oder einen Befehl angeben, der ausgeführt werden soll, bevor ein E-Mail-Alias erstellt, geändert oder gelöscht wird. Mithilfe der Umgebungsvariablen ALIAS_FROM, ALIAS_TO und ALIAS_ACTION kann ermittelt werden, welcher Alias geändert wird und was damit geschieht. Wenn der Befehl mit einem Status ungleich Null beendet wird, wird eine Änderung des Alias abgelehnt.
