Aliasdomäne automatisch erstellen
Mit dieser Option kann für jeden neuen virtuellen Server unter einer anderen Domäne eine zusätzliche Aliasdomäne erstellt werden. Dies kann hilfreich sein, wenn Sie einen neuen Server hinzufügen, für den die DNS-Domäne noch nicht registriert wurde, damit auf sie unter einem temporären Namen zugegriffen werden kann, der unter einem bereits registrierten Namen steht (z. B. Ihrem Unternehmen).
