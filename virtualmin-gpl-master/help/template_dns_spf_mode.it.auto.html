Aggiungi record DNS SPF?
SPF, o Sender Policy Framework , è un mezzo per aiutare a ridurre lo spam applicando alcune restrizioni su ciò che gli host possono inviare e-mail che pretendono di appartenere a quel dominio. Poiché la stragrande maggioranza dello spam ha origine da nomi di dominio falsi, molti ritengono che la prevenzione di nomi di dominio falsi nello spam ridurrà il volume complessivo di spam. Se abilitato, Virtualmin creerà automaticamente un record SPF nel tuo server DNS. Questo record può contenere host specifici a cui è consentito inviare posta utilizzando il nome di dominio in questione (il server stesso è il primo host di questo tipo, ma è possibile elencare un numero qualsiasi di host) oppure un generico catch-all, che significa l'host può inviare la posta dichiarando di appartenere a questo dominio. La seconda opzione annulla efficacemente lo scopo di SPF, ma viene fornita per facilitare agli utenti l'invio di posta "dal" proprio dominio attraverso vari server di posta ISP.
