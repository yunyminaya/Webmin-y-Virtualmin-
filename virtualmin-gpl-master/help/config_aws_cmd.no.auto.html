Full bane til aws-kommando for S3-operasjoner
Hvis aws kommandoen er installert på systemet ditt, og dette feltet er satt til den fullstendige banen til den (som /usr/bin/aws ), vil Virtualmin kalle det for å utføre S3-operasjoner for sikkerhetskopieringer som for eksempel lastet opp og lastet ned filer. Fordi denne kommandoen ble utviklet av Amazon, er den mer pålitelig enn S3 API innebygd i Virtualmin.
