Rotazione del file di registro dell'installazione?
Se questa opzione è abilitata, i file di registro Web del server virtuale verranno ruotati su base settimanale, per evitare di consumare troppo spazio su disco. Ogni settimana il file di registro corrente verrà copiato, troncato e compresso. Per impostazione predefinita, i registri ruotati verranno conservati per un massimo di cinque settimane prima di essere eliminati.
