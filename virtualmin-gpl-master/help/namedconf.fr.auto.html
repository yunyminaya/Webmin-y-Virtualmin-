Directives named.conf supplémentaires pour les nouvelles zones
Lorsqu'une nouvelle zone DNS BIND est créée, les directives entrées dans cette zone de texte seront ajoutées à la section de la zone dans le fichier de configuration named.conf . Par exemple, vous pouvez saisir les informations suivantes pour limiter les transferts de zone vers des adresses IP sélectionnées: 
allow-transfer { 
  1.2.3.4;
  1.2.3.10;
};
Cette fonctionnalité ne doit être utilisée que par ceux qui connaissent déjà le format de configuration BIND, car une erreur pourrait entraîner l'arrêt complet de votre serveur DNS.
