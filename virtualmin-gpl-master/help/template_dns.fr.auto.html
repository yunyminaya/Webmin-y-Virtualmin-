LIER les enregistrements DNS pour les nouveaux domaines
Lors de la création d'un nouveau domaine, une nouvelle zone est créée. Vous pouvez personnaliser les enregistrements créés ici. Ce champ attend des entrées nommées.conf valides, bien que les variables Virtualmin soient traduites dans les valeurs appropriées pour le domaine. Vous pouvez soit ajouter des enregistrements aux enregistrements générés par Virtualmin standard, soit remplacer complètement les enregistrements Virtualmin par les vôtres. Pour plus d'informations sur BIND, consultez le chapitre BIND dans les livres Webmin.

Si vous souhaitez créer votre propre enregistrement SOA, les substitutions ${CURRENT_YEAR} , ${CURRENT_MONTH} et ${CURRENT_DAY} peuvent également être utiles.
