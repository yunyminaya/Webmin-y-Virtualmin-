Modules supplémentaires pour les revendeurs
Cette option vous permet de fournir aux utilisateurs revendeurs des fonctionnalités Webmin supplémentaires au-delà du module Virtualmin Virtual Servers . Soyez très prudent avec cette option, car la plupart des modules Webmin fournissent par défaut un contrôle complètement dangereux sur les services qu'ils gèrent. Il y a cependant des occasions où cela peut être très utile. Par exemple, si vous avez un module personnalisé qui fournit un service spécifique à votre environnement, cette option peut être très utile. De plus, de nombreux modules peuvent être configurés avec un accès très strict par défaut, ou vous pouvez cloner un module et le configurer avec un ensemble très spécifique de fonctions pour le rendre sûr pour une utilisation par tous les revendeurs. Vous pouvez accorder sélectivement des privilèges supplémentaires à certains utilisateurs revendeurs à l'aide du module Utilisateurs Webmin .
