Oorspronkelijk geselecteerde template voor sub-servers?
Wanneer dit is ingesteld op Ja zal het template menu wanneer er een nieuwe virtuele sub-server word aangemaakt deze template standaard geselecteerd hebben. Het zal ook worden gebruikt wanneer er een server word aangemaakt via de opdracht regel indien er niet expliciet een template word opgegeven met de --template parameter. Er kan maar 1 template tegelijkertijd ingesteld staan met deze Ja optie.
