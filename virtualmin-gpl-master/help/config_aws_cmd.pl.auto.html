Pełna ścieżka do komendy aws dla operacji S3
Jeśli polecenie aws jest zainstalowane w systemie, a pole to jest ustawione na pełną ścieżkę do niego (np. /usr/bin/aws Usr /usr/bin/aws ), Virtualmin wywoła je w celu wykonania operacji S3 dla kopii zapasowych, takich jak przesyłanie i pobieranie plików. Ponieważ to polecenie zostało opracowane przez Amazon, jest bardziej niezawodne niż interfejs API S3 wbudowany w Virtualmin.
