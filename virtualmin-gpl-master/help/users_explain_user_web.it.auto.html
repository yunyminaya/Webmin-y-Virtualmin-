Crea utente server web
Crea un nuovo utente del server web all'interno dei file htpasswd , in particolare per l'autenticazione sicura nelle directory web protette. Questo utente è distinto dagli account Unix, dedicati esclusivamente alla gestione dell'accesso alle directory web. Inoltre, questo utente può essere integrato perfettamente unendosi con un account Unix creato utilizzando lo stesso nome utente.
