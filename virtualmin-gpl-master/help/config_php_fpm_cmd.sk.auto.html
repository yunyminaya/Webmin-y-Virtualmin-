Dodatočný PHP FPM server
V predvolenom nastavení sa Virtualmin pokúsi nájsť PHP FPM server prehľadávaním systémových balíkov, ale ak ste ho manuálne nainštalovali na iné miesto kompiláciou zo zdroja, cestu k FPM serveru môžete nastaviť tu. To je tiež užitočné na iných operačných systémoch, ako je FreeBSD, kde Virtualmin nevie, ako je PHP-FPM zabalený. Ak je táto možnosť nastavená, mali by ste nastaviť aj dve nasledujúce možnosti.
