Crear usuario FTP
Crea un nuevo usuario Unix dedicado al acceso FTP en el directorio del sitio web, asignando automáticamente el ID de usuario y grupo del propietario del servidor virtual. Este usuario especializado se crea teniendo en cuenta el acceso FTP. Las cuotas para este tipo de cuenta se comparten con el propietario del servidor virtual. En particular, las funciones de correo electrónico no están disponibles para este tipo de usuario.
