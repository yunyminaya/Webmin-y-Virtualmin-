Подстановочный знак для дополнительных разрешенных баз данных MySQL
Это поле предназначено для ввода шаблона подстановки MySQL, который предоставляет доступ к дополнительным базам данных - возможно, что-то вроде ${DB}_% . Для домена, подобного example.com , это будет расширено до example_% , что позволит получить доступ ко всем базам данных, начиная с example_ .

Поскольку Virtualmin создает правила доступа MySQL для всех баз данных, которые он создает, это поле обычно можно оставить пустым. Заполнять его нужно только в том случае, если вы создаете дополнительные базы данных другим способом, не зависящим от Virtualmin.
