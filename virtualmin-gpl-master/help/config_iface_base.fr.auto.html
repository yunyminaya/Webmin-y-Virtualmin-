Numéro de base pour les interfaces virtuelles
Par défaut, Virtualmin utilise eth0:0 pour la première interface IP virtuelle qu'il crée, puis eth0:1 et ainsi de suite. Cependant, ce champ peut être utilisé pour changer où le numéro d'interface après le: démarre. Cela peut être utile si vous souhaitez réserver certains des nombres inférieurs pour une affectation manuelle, mais dans la plupart des cas, il n'est pas nécessaire de les modifier.
