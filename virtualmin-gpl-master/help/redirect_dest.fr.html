Destination
Ce champ détermine où le chemin URL au-dessus sera redirigé. Vous avez deux choix différents :
URL à d'autres sites
Lorsque cette option est sélectionnée, les navigateurs qui accèdent au chemin de l'URL seront redirigés à cette nouvelle URL. Si un sous-répertoire du chemin d'accès est accessible, la redirection comprendra également cette même chemin.

Répertoire sur ce système
Lorsque cette option est choisie, les navigateurs qui accèdent au chemin URL seront servis contenu dans le répertoire spécifié. Cela devrait être dans le répertoire d'accueil du domaine, et de préférence sous public_html ou autrement les scripts PHP sont peu susceptibles de travailler.
