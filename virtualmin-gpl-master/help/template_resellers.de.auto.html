Verfügbar für Wiederverkäufer
In diesem Feld können Sie steuern, welche Wiederverkäufer diese Vorlage beim Erstellen neuer virtueller Server verwenden dürfen. Dies kann hilfreich sein, wenn Sie zu Testzwecken eine Vorlage mit nicht standardmäßigen Einstellungen erstellt haben oder die auf irgendeine Weise zu viel Leistung gewährt.
