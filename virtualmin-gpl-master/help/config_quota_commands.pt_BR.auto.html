Use comandos externos para obter e definir cotas?
Quando essa opção está definida como Sim , o Virtualmin utilizará os comandos nas opções de configuração abaixo para definir e obter cotas de usuário e grupo. Isso pode ser útil se as cotas no servidor forem gerenciadas de alguma maneira que não seja o sistema de cotas Unix padrão, como por um servidor NFS ou sistema de arquivos ZFS.

Por padrão, é definido como Não , que é a configuração correta para quase todos os sistemas.
