Répertoire squelette des fichiers
Un répertoire squelette contient des fichiers qui seront copiés dans le répertoire personnel nouvellement créé de l'utilisateur du domaine. Il peut contenir d'autres répertoires, qui seront également créés dans le répertoire personnel. Cela peut être utilisé pour fournir un ensemble préconfiguré de scripts ou de contenu Web pour certains ou tous les modèles de serveur.
