Sicherungsformat
Diese Option steuert das Dateiformat, das Virtualmin für diese Sicherung verwendet. Die verfügbaren Optionen sind:
Einzelne Archivdatei
Alle virtuellen Server werden in einer einzigen Datei an dem vom Sicherungsziel angegebenen Speicherort gespeichert.
Eine Datei pro Server
Jeder virtuelle Server wird in einer eigenen Datei in dem vom Sicherungsziel angegebenen Verzeichnis gesichert.
Eine Datei pro Server im neuen Format
Jeder virtuelle Server wird in einer eigenen Datei gesichert, jedoch in einem anderen Format als die vorherige Option. Dieses neue Format ist überlegen, da nicht zuerst eine große Datei im Verzeichnis /tmp , wodurch Speicherplatz gespart wird.
Die letzte Option wird empfohlen, da einzelne virtuelle Server einfacher wiederhergestellt werden können und während der Sicherung temporären Speicherplatz gespart wird.
