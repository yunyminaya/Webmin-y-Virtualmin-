Nombre de host del servidor DNS maestro
De forma predeterminada, Virtualmin usará el nombre de host del sistema en los registros primarios NS y SOA para los nuevos dominios DNS que cree. Sin embargo, si el nombre de host no es un nombre DNS válido o no es adecuado para sus propósitos, esta opción se puede usar para ingresar un servidor de nombres primario diferente. Debe resolver la dirección IP accesible externamente de su sistema, o la dirección de un sistema de servidor de nombres esclavo para dominios creados usando esta plantilla.
