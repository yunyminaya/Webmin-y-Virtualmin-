Capacités d’édition par défaut
Ces cases à cocher déterminent les capacités des serveurs virtuels que les propriétaires de domaine peuvent modifier par défaut. Une fois qu'un serveur est créé, ces options peuvent être modifiées sur la page Modifier les limites du propriétaire.

Si l'option Automatique est sélectionnée, les limites sont déterminées en fonction de la possibilité pour le propriétaire du serveur virtuel de créer des sous-serveurs ou non (contrôlé par le champ Limite du nombre de serveurs virtuels ). Si tel est le cas, il aura accès à toutes les fonctionnalités. Sinon, il ne pourra que gérer les utilisateurs, les alias et modifier les pages Web.
