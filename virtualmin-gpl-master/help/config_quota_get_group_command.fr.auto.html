Commande pour obtenir le quota d'un groupe
Lorsque des commandes de quota externe sont utilisées, le programme défini dans cette option peut être appelé par Virtualmin pour récupérer le quota pour un seul groupe. Il est appelé avec un seul paramètre - le nom du groupe Unix, et doit générer une seule ligne contenant les champs séparés par des espaces suivants:
  1. Quota actuel utilisé dans des blocs de 1 ko.
  2. Le quota souple en blocs de 1 Ko, ou 0 pour illimité.
  3. Le quota dur en blocs de 1 Ko, ou 0 pour illimité.
Si cette commande n'est pas définie sur la page Configuration du module, Virtualmin utilisera la commande pour répertorier les quotas de groupe à la place.
