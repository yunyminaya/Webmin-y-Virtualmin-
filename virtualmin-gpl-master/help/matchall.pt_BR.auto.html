O site corresponde a todos os subdomínios?
Quando essa opção está ativada, as solicitações de qualquer URL no domínio (como http://abc.example.com ) retornam o site do servidor virtual. Isso pode ser útil se você hospedar um aplicativo que possa retornar conteúdo diferente com base no nome de domínio solicitado, como o Wordpress MU.

Se o domínio DNS do servidor virtual estiver hospedado neste sistema, o Virtualmin também adicionará o registro DNS curinga * quando Sim estiver selecionado.
