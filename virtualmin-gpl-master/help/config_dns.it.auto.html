Impostazione e configurazione BIND abilitate?
Ciò abilita o disabilita la configurazione del server dei nomi locale. Se hai già configurato la gestione del dominio altrove o non creerai molti nuovi domini (se, ad esempio, stai utilizzando Virtualmin principalmente per la generazione di account di posta elettronica all'interno di uno o un piccolo set di domini), puoi scegliere di disabilitarlo. Si noti che anche se si desidera ospitare DNS primario e secondario su macchine separate, si desidera comunque abilitare questa opzione, in modo che la funzione Cluster DNS Slaves possa essere utilizzata per propagare automaticamente le nuove zone ai server master.
