Výchozí HTML stránka webu
Tato možnost vám umožňuje ovládat výchozí stránku, která bude vytvořena a zobrazena pro nový virtuální server.

Můžete zvolit "Zakázáno" pro přeskočení vytváření stránky nebo vybrat "Zeptat se při vytvoření" a rozhodnout se během vytváření virtuálního serveru.

Případně se můžete rozhodnout pro výchozí stránku webu Virtualmin výběrem možnosti „Ano, se šablonou výchozí stránky webu“.

Pokud chcete, výběrem možnosti „Ano, s obsahem níže“ můžete zadat vlastní kód HTML nebo text, který se použije jako obsah webové stránky.
