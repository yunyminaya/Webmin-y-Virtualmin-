Virtuele Server aanmaken
Met dit formulier kunt u een nieuwe top-niveau virtuele server aanmaken die een nieuwe bijbehorende administratieve login heeft. Wanneer u een server aanmaakt moet u op zijn minst een domein naam opgeven, omschrijving en een administratie wachtwoord. Als er meerdere server templates zijn gedefinieerd op het systeem kunt u uit het menu selecteren wat u wenst - template bepalen verschillende standaard instellingen voor de nieuwe server, zoals de website configuratie, DNS records enzovoort.

Onder de Geavanceerde opties sectie kunt u een andere administratie gebruikersnaam kiezen, een groepsnaam en database naam voor de server, anders dan die standaard afgenomen worden van het eerste gedeelte van de domein naam. Er kan ook een contact email adres ingevuld worden voor de server - het beste is om een bestaand email adres te gebruiken voor de persoon die de server beheerd, in een ander domein.

Om te selecteren welke functies er aan moeten staan voor de server gebruikt u de selectievakjes in het tweede gedeelte van de Geavanceerde opties sectie. Sommige functies zijn afhankelijk van andere functie en kunnen dus niet afzonderlijk gekozen worden - bijvoorbeeld, Webalizer rapportage voor website statistieken hangt af van werkelijk in het bezit zijn van een website.

Aan de onderkant van de Geavanceerde opties sectie kunt u selecteren om al het web verkeer naar deze nieuwe website moet worden doorgestuurd naar een andere URL, wat ook geld voor alle email. U kunt ook selecteren dat de nieuwe virtuele server zijn eigen IP adres moet gebruiken die nodig is voor SSL en anonieme FTP hosting.
