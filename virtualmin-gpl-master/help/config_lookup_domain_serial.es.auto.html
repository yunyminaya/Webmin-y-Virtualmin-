Buscar dominio para correo electrónico entrante
Por defecto, el proceso de búsqueda-dominio-demonio de Virtualmin lanzará un subproceso separado para encontrar la configuración de dominio y correo no deseado para cada mensaje entrante en su sistema. Sin embargo, esto puede consumir grandes cantidades de tiempo de CPU y memoria si llegan varios mensajes al mismo tiempo.

Para que el correo electrónico se procese uno por uno, seleccione la opción Uno a la vez para este campo. La entrega de correo será un poco más lenta, pero el uso de CPU y memoria será limitado.
