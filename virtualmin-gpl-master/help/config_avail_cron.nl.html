Geplande Cron Taken (gebruikers's Cron taken)
Deze optie bepaald of de domein eigenaar de Cron Taken Webmin module kan gebruiken. Een Cron taak is een geplande actie die kan worden gebruikt om periodiek taken uit te voeren, zoals een email versturen, backup maken of een database ophalen, de status van een file of mailbox controleren enz. De Cron Taak module heeft een gemakkelijke interface om de moeilijk te onthouden (en verwarrend voor nieuwe gebruikers) syntaxis van de crontab file. Cron taken zullen worden uitgevoerd als domein eigenaar gebruiker.
