Heslo MySQL pro nové domény
Tato volba řídí počáteční heslo MySQL pro nové virtuální servery. Možnosti jsou:
Synchronizace s přihlášením správce
Heslo pro MySQL je vždy stejné jako přihlašovací jméno pro správu virtuálního serveru.

Stejné jako počáteční přihlašovací přihlašovací jméno
Heslo je zpočátku stejné jako přihlašovací jméno pro správu, ale při změně hesla pro správu se nezmění.

Generovat náhodně
Při vytvoření virtuálního serveru je vybráno náhodně vygenerované heslo MySQL, které se nemění.
