Comando para ejecutar antes de hacer cambios a un revendedor
Esta opción se puede utilizar para especificar un script o comando que se ejecutará antes de crear, modificar o eliminar un revendedor. Puede hacer uso de las variables de entorno RESELLER_NAME y RESELLER_ACTION para determinar qué revendedor se está cambiando y qué le está sucediendo. Si el comando sale con un estado distinto de cero, la modificación del revendedor será vetada.
