Interfaccia di rete
Se si desidera impostare un host virtuale basato su IP (richiesto per i siti Web SSL o l'hosting FTP virtuale), questa opzione deve essere impostata su Interfaccia virtuale con IP e immesso un indirizzo non utilizzato. Virtualmin attiverà questo IP sul tuo sistema al momento della creazione del server e lo assocerà al server.

Se si seleziona l'opzione Condivisa , verrà utilizzato l'indirizzo di interfaccia di rete principale del sistema per il sito Web e il dominio DNS di questo server.
