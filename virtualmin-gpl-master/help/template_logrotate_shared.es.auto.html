¿Compartir configuración de rotación de registros con todos los dominios?
Cuando esto se establece en No , Virtualmin creará un bloque separado en el archivo de configuración logrotate para cada servidor virtual. Esto permite que cada uno tenga una configuración diferente, pero hará que el servidor web se reinicie después de la rotación de los registros de cada dominio, posiblemente interrumpiendo el servicio.

Si selecciona Sí , todos los dominios nuevos compartirán un solo bloque de configuración, lo que significa que el servidor web solo debe reiniciarse una vez. Sin embargo, no es posible tener configuraciones de rotación diferentes para cada dominio.
