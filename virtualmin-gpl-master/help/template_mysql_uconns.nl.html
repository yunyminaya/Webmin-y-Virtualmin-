Maximum verbindingen voor mailbox gebruikers
Dit veld bepaald het aantal samengaande verbindingen naar MySQL die door een mailbox met database toegang kan maken. Standaard word een ongelimiteerd aantal verbindingen toegestaan, echter omdat iedere verbinding geheugen gebruikt binnen het MySQL server proces kan het zijn dat u het aantal verbindingen wilt beperken op een druk systeem.

De limiet die hier word ingesteld word alleen toegevoegd aan een mailbox die nieuw aangemaakt word, of wanneer er een MySQL login word aangemaakt voor een bestaande mailbox.
