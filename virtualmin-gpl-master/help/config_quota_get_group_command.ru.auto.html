Команда для получения квоты группы
Когда используются внешние команды квот, Virtualmin может вызывать программу, заданную в этой опции, для извлечения квоты для одной группы. Он вызывается с одним параметром - именем группы Unix и должен выводить одну строку, содержащую следующие разделенные пробелами поля:
  1. Текущая квота используется в блоках по 1 кБ.
  2. Мягкая квота в блоках по 1 кБ или 0 для неограниченного количества.
  3. Жесткая квота в блоках по 1 кБ или 0 для неограниченного количества.
Если эта команда не задана на странице «Конфигурация модуля», Virtualmin будет использовать команду для вывода списка групповых квот .
