Comando para obter a cota de um usuário
Quando comandos de cota externa estão sendo usados, o programa definido nesta opção pode ser chamado pelo Virtualmin para buscar a cota para um único usuário. Ele é chamado com um único parâmetro - o nome de usuário completo do Unix e deve gerar uma única linha contendo os seguintes campos separados por espaço:
  1. Cota atual usada em blocos de 1 kB.
  2. A cota flexível em blocos de 1 kB ou 0 para ilimitado.
  3. A cota fixa em blocos de 1 kB ou 0 para ilimitado.
Se esse comando não estiver definido na página Configuração do módulo, o Virtualmin utilizará o comando para listar cotas de usuário .
