Gruppenbesitz von MySQL-Datenbankdateien festlegen?
Wenn diese Option aktiviert ist, wird für alle neuen MySQL-Datenbanken der Gruppenbesitz für ihre Datendateien festgelegt (normalerweise unter /var/lib/mysql ), sodass sie in den Unix-Kontingentgrenzen ihrer virtuellen Server enthalten sind. Wenn diese Option nicht aktiviert ist, wird die Festplattennutzung der Datenbanken nicht eingeschränkt, und ein virtueller Server kann die zulässige Festplattenzuweisung überschreiten.
