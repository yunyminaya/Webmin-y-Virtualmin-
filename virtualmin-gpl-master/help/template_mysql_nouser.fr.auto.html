Garder les noms d'utilisateurs MySQL et d'administration synchronisés?
Par défaut, Virtualmin changera la connexion MySQL pour un domaine lorsque la connexion d'administration change. Cependant, cela entraînera des problèmes pour toutes les applications installées qui ont déjà été configurées avec l'ancien nom d'utilisateur. Pour éviter cela, définissez ce paramètre sur Non .
