Shell für Postfachbenutzer
Diese Option gibt die für Postfachbenutzer konfigurierte Shell an. Ein Postfachbenutzer ist der am wenigsten privilegierte Benutzertyp und sollte normalerweise nur sehr eingeschränkten Zugriff haben. Wenn Ihr Virtualmin-Server jedoch nur von vertrauenswürdigen Benutzern oder Benutzern verwendet wird, die über Shell-Zugriff verfügen müssen, können Sie dies von der Standardeinstellung / dev / null in eine der realen System-Shells ändern. Beachten Sie, dass / bin / false eine spezielle Shell ist, die Remote-SSH-Anmeldungen schließt, aber weiterhin FTP-Anmeldungen zulässt.
