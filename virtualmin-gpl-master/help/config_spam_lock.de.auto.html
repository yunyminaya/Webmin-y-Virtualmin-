Führen Sie immer nur einen Spamassasin-Prozess aus?
Wenn dieses Feld standardmäßig auf Nein gesetzt ist, führt Virtualmin mehrere gleichzeitige Spamassassin- Prozesse aus, wenn mehrere E-Mail-Nachrichten gleichzeitig eingehen . Dies kann jedoch auf einem kleinen System zu viel CPU und RAM beanspruchen. Wenn Sie Ja auswählen, werden bei virtuellen Servern mit aktivierter Spamfilterung die Procmail-Konfigurationsdateien so aktualisiert, dass jeweils nur eine Kopie von Spamassassin ausgeführt wird.

Diese Einschränkung gilt für alle virtuellen Server. Selbst wenn zwei Nachrichten an verschiedene Domänen gleichzeitig eingehen , wird immer nur ein Spamassassin- Prozess ausgeführt.
