Subdiretório FTP anônimo para criar
Se downloads anônimos de FTP forem permitidos, você pode especificar o subdiretório onde os arquivos para download permanecerão. Se alterar as diretivas na opção anterior, você deve incluir uma seção Anônima e ela deve corresponder ao local fornecido aqui. Observe que o acesso anônimo por meio de um nome de domínio específico requer a configuração de um host FTP virtual baseado em IP.
