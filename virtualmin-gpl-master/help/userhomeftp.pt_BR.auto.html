Diretório inicial
Por padrão, o diretório home para um usuário FTP de site é o mesmo que o diretório web base do domínio. No entanto, você pode alterá-lo para o diretório home do domínio (usando um valor vazio) ou para um subdiretório (usando um caminho relativo sob o diretório web).

Isso pressupõe que o servidor FTP esteja configurado para restringir os clientes aos seus diretórios pessoais, o que deve ser o padrão em instalações padrão do Virtualmin.
