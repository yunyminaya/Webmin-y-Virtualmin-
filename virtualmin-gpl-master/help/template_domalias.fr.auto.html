Créer automatiquement un domaine d'alias
Cette option peut être utilisée pour créer un domaine d'alias supplémentaire pour chaque nouveau serveur virtuel, sous un autre domaine. Il peut être utile lors de l'ajout d'un nouveau serveur pour lequel le domaine DNS n'a pas encore été enregistré, afin qu'il soit accessible sous un nom temporaire qui est sous un nom déjà enregistré (tel que celui de votre entreprise).
