Netwerk Interface
Indien u een IP gebaseerde virtuele host wilt opzetten (die nodig is voor een SSL website of een virtuele FTP hosting), moet deze optie worden ingesteld naar Virtuele interface met IP en moet u een niet in gebruik zijnde IP adres invullen. Virtualmin zal dit IP adres activeren zodra de server word aangemaakt en zal het associeeren met de server.

Als de Gedeelde optie is geselecteerd zal de primaire systeem netwerk interface adres worden gebruikt voor deze server zijn website en DNS domein.
