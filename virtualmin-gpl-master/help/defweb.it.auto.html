Sito Web predefinito per l'indirizzo IP?
Quando questo campo è impostato su Sì , il sito Web di questo server virtuale verrà servito da Apache quando riceve una richiesta che non corrisponde a nessun altro server virtuale sul sistema. Questo in genere accade se un utente utilizza un URL con un indirizzo IP al suo interno o un nome host che si risolve nel tuo sistema ma non corrisponde a nessun dominio Virtualmin.
