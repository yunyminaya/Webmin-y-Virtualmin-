Установить функции по умолчанию из плана при создании сервера?
Если для этого параметра установлено значение Да , функции, включенные по умолчанию для новых виртуальных серверов верхнего уровня, устанавливаются на основе тех, которые разрешены выбранным планом. Если выбрано «Нет» , функции по умолчанию определяются на странице «Функции и плагины» .
