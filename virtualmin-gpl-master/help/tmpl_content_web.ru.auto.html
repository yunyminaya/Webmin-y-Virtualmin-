HTML-страница веб-сайта по умолчанию
Эта опция позволяет вам управлять страницей по умолчанию, которая будет создана и отображена для нового виртуального сервера.

Вы можете выбрать «Отключено», чтобы пропустить создание страницы, или выбрать страницу по умолчанию для веб-сайта Virtualmin, выбрав опцию «Да, с шаблоном страницы веб-сайта по умолчанию».
