Можно создать несколько серверов из пакетного файла?
На странице Virtualmin Batch Create Servers пользователи с правами на добавление серверов могут создавать несколько серверов одновременно из текстового файла. Поскольку это может привести к быстрому созданию большого количества доменов, вы можете запретить администраторам сервера использовать эту функцию. Для этого измените этот параметр на Нет - это ограничит его использование только главным администратором и посредниками.
