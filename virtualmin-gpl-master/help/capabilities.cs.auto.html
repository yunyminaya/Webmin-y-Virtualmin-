Může upravovat virtuální server
Umožňuje editaci vlastností virtuálního serveru, jeho hesla a popisu.
Může spravovat uživatele
Umožňuje uživatelům poštovních schránek, FTP a databází prohlížet, upravovat, vytvářet a mazat.
Umí spravovat aliasy
Umožňuje prohlížet, upravovat, vytvářet a mazat e-mailové aliasy.
Umí spravovat databáze
Umožňuje přidat další databáze MySQL a PostgreSQL na tento virtuální server nebo z něj odstranit.
Může instalovat webové aplikace
Umožňuje instalaci webových aplikací třetích stran na tento virtuální server.
Lze změnit IP adresu
Pokud má tento server privátní IP adresu, tato možnost umožňuje její změnu.
Může spravovat certifikáty SSL
Umožňuje nahrát nebo znovu vygenerovat certifikát SSL a klíč pro tento server.
Může upravit rám pro předávání
U webových stránek pouze pro přeposílání tato možnost umožňuje upravit cíl a HTML pro rámec přeposílání.
Může spravovat další správce
Umožňuje prohlížet, vytvářet, upravovat a mazat další administrátorské účty vlastníků serveru.
Může nakonfigurovat doručování spamu a virů
Umožňuje upravovat cíle doručení spamu a virů a skóre.
Lze vybrat verze PHP
Pro domény s PHP běžící přes CGI nebo fastCGI umožňuje nastavení verzí PHP pro jednotlivé adresáře.
Může upravit nastavení e-mailu
Umožňuje konfiguraci automatického BCC, režimu kopírování aliasů a dalších možností souvisejících s e-mailem.
Může dělat zálohy
Umožňuje vlastněné virtuální servery zálohovat na vzdálený server FTP nebo SSH.
Může plánovat zálohování
Umožňuje zálohování virtuálních serverů podle pravidelného plánu.
Může obnovit zálohy
Umožňuje obnovit obsah domovského adresáře a databáze vlastněných virtuálních serverů.
Může vybrat sdílené IP adresy
Umožňuje výběr globálně definovaných sdílených IP adres při vytváření virtuálních serverů.
Může vytvářet catchall aliasy
Umožňuje vytváření e-mailových aliasů, které přeposílají veškerou poštu do domény na jinou doménu nebo adresu.
Může používat editor webových stránek
Umožňuje upravovat webové stránky prostřednictvím uživatelského rozhraní Virtualmin.
Může upravovat vzdálené hostitele databáze
Umožňuje zadávání vzdálených klientských systémů, které mají povoleno připojení k doménovým MySQL databázím.
Může deaktivovat virtuální servery
Umožňuje dočasně deaktivovat servery.
Může odstranit virtuální servery
Umožňuje úplné smazání vlastněných serverů, včetně všech podserverů a aliasů.
