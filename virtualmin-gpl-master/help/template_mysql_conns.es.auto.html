Conexiones máximas para propietarios de dominios
Este campo controla la cantidad de conexiones simultáneas a MySQL que puede realizar la cuenta del propietario de un servidor virtual. Esto incluye conexiones desde aplicaciones web instaladas por Virtualmin, como Wordpress o Drupal. El valor predeterminado es permitir un número ilimitado de conexiones, pero debido a que cada una consume memoria dentro del proceso del servidor MySQL, es posible que desee limitar el total de conexiones en un sistema ocupado.

El límite establecido aquí solo se aplicará cuando se cree un nuevo servidor virtual o cuando se habilite un inicio de sesión MySQL para un servidor existente.
