L'azione Virtualmin specifica che è stata eseguita può essere trovata nella variabile VIRTUALSERVER_ACTION, ad esempio:

CREATE_DOMAIN
Viene creato un nuovo server virtuale.
DELETE_DOMAIN
È in corso la rimozione di un server virtuale esistente.
DISABLE_DOMAIN
Un server virtuale viene temporaneamente disabilitato.
ENABLE_DOMAIN
È in corso la riattivazione di un server virtuale.
MODIFY_DOMAIN
Un server virtuale viene aggiornato, ad esempio quando vengono aggiunte o rimosse funzionalità o viene modificata la proprietà.
CLONE_DOMAIN
Un server virtuale viene creato copiando un server esistente.
