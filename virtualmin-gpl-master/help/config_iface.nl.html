Netwerk interface voor virtuele adressen
Indien uw server meerdere interfaces heeft kunt u kiezen welke interface word gebruikt voor uw virtuele domeinen. Als SSL of FTP virtuele hosts worden aangezet zal er een nieuw IP adres nodig zijn voor ieder domein waarop deze optie word aangezet. Of dit moet anders geconfigureerd zijn bij het aanmaken van het domein, anders zullen de nieuwe adressen op de hier gespecificeerde interface worden aangemaakt.
