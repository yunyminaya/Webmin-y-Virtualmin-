Hashed Passworttypen zum Speichern
Standardmäßig speichert Virtualmin Domänen- und Postfachkennwörter in verschiedenen Hashformaten, sodass das Kennwort verfügbar ist, wenn der Domäne neue Funktionen hinzugefügt werden. Da jedoch nicht alle Hash-Formate gleich sicher sind, kann dieses Feld zum Deaktivieren der Speicherung einiger Formate verwendet werden. Unabhängig davon, was Sie hier auswählen, wird das Kennwort immer gehasht im Standardformat des Systems in /etc/shadow oder LDAP gespeichert.

Beachten Sie, dass das Deaktivieren des MySQL- Hash-Formats Virtualmin möglicherweise daran hindert, ein MySQL-Login für Postfachbenutzer mit demselben Kennwort zu erstellen, das dem Postfach ursprünglich zugewiesen wurde. Durch Deaktivieren des Web Digest- Formats wird möglicherweise der Zugriff von Postfachbenutzern auf ein geschütztes Webverzeichnis verhindert.
