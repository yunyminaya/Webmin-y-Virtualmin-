From: address to email sent to domain owners
If a virtual server is owned by a reseller, you may want email send by Virtualmin (such as bandwidth quota warnings or new script notifications) to come from the reseller's email address, rather than the master sysadmin. To enable this, select Reseller's address (if any) - otherwise, leave Address set above selected.

If a virtual server does not have a reseller or the reseller doesn't have an address, the Reseller's address mode will fall back to using the global From: address instead.
