Instalação e configuração do Apache ativada?
Esta opção ativa ou desativa a criação de entradas do Apache VirtualNameHost e define todos os outros recursos do host virtual no arquivo de configuração do Apache. Se você estiver usando um servidor da Web alternativo, poderá desativar esta opção. Pode ser possível usar a interface do módulo de plug-in para criar suporte para o seu servidor preferido (embora nenhum outro servidor da Web seja suportado no momento) ou você pode optar por chamar um script externo junto com as variáveis extraídas das informações fornecidas ao Virtualmin em hora de criação do domínio para configurar seu servidor da web preferido.
