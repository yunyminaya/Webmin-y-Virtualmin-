Tipo de campo de senha
Essa configuração controla como o campo para inserir a senha para um novo servidor virtual é exibido. As opções disponíveis são:
Senha gerada aleatoriamente
Um único campo visível é exibido, contendo uma senha gerada aleatoriamente selecionada pelo Virtualmin. Você pode digitar uma senha diferente.

Digite a senha uma vez
Um único campo para inserir sua própria senha é exibido, com a senha real digitada oculta.

Digite a senha dois
Dois campos para inserir sua própria senha são exibidos, novamente com a senha oculta. Ambas as senhas devem ser iguais.
