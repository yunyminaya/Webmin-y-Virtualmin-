I modelli possono essere applicati a qualsiasi tipo di account utente di dominio. I tipi attuali sono: "Server virtuali di livello superiore", che sono account host virtuali standard; "Sotto-server", che sono host virtuali che appartengono a un server virtuale di livello superiore e vengono visualizzati nella directory "domini" della directory principale del server di livello superiore; I "server alias" sono server che, in genere, semplicemente inoltrano web e posta a un dominio di primo livello esistente; e "Proprietari di server" saranno disponibili per i proprietari di server quando creano nuovi domini.
