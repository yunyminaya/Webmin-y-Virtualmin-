¿Puedo elegir nombres de bases de datos?
Si esta restricción se establece en No , el propietario del servidor no podrá seleccionar el nombre de base de datos predeterminado al crear un subservidor que tenga habilitada una función de base de datos. En su lugar, siempre se utilizará de manera predeterminada la primera parte del nombre de dominio.

Además, el usuario no podrá eliminar la base de datos predeterminada de ningún dominio existente, ya que esto le permitiría cambiar efectivamente el nombre de la base de datos predeterminada después de crear el dominio.
