Geplante Cron-Jobs (Cron-Jobs des Benutzers)
Diese Option gibt an, ob das Domäneninhaberkonto das Webmin-Modul von Cron Jobs verwenden kann. Ein Cron-Job ist ein geplantes Ereignis, mit dem regelmäßig bestimmte Aufgaben ausgeführt werden können, z. B. das Versenden einer E-Mail, das Sichern oder Staubsaugen einer Datenbank, das Überprüfen des Status einer Datei oder eines Postfachs usw. Das Modul Cron-Jobs ist einfach und benutzerfreundlich Schnittstelle zur schwer zu merkenden (und für Neulinge verwirrenden) Syntax der Crontab-Datei. Cron Jobs werden als Domaininhaber ausgeführt.
