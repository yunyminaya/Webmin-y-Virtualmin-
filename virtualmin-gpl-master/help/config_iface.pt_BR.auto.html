Interface de rede para endereços virtuais
Se o seu servidor tiver várias interfaces, você poderá escolher qual interface usar para seus domínios virtuais. Se os hosts virtuais SSL ou FTP estiverem ativados, será necessário um novo IP para cada domínio no qual o recurso está ativado. A menos que configurado de outra forma durante a criação do domínio, os novos endereços serão criados na interface especificada aqui.
