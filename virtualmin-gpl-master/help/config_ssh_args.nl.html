Extra parameters naar SSH en SCP opdrachten
Wanneer Virtualmin een remote backup maakt of hersteld van een remote server met gebruik van SSH zal hij de ssh en scp opdrachten gebruiken om files te verplaatsen. Dit veld geeft de mogelijkheid om extra parameters te specificeren voor die opdrachten, zoals de -i vlag gevolgd door een pad naar een alternatieve SSH prive sleutel file.
