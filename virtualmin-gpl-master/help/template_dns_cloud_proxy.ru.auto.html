Включить проксирование для новых записей?
Для поставщиков облачных DNS, поддерживающих проксирование, этот параметр определяет, создаются ли записи в новых доменах с включенным проксированием или нет. Вы можете либо проксировать все записи, либо только те, которые используются для хостинга веб-сайтов (domain.com и www.domain.com).
