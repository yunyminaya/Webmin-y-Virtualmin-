Numero massimo di connessioni per utenti di cassette postali
Questo campo controlla il numero di connessioni simultanee a MySQL che può effettuare una cassetta postale con accesso al database. L'impostazione predefinita è consentire un numero illimitato di connessioni, ma poiché ciascuna consuma memoria all'interno del processo del server MySQL, potresti voler limitare le connessioni totali su un sistema occupato.

Il limite impostato qui verrà applicato solo quando viene creata una nuova cassetta postale o quando è abilitato un accesso MySQL per una cassetta postale esistente.
