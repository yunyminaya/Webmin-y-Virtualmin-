¿Usar la dirección IPv6 predeterminada para los nuevos servidores virtuales?
De manera predeterminada, Virtualmin agregará entradas de DNS y una configuración de servidor web a los nuevos servidores virtuales para usar la dirección IPv6 principal del sistema. Sin embargo, en el momento de la creación, un usuario aún puede seleccionar usar una dirección IPv6 privada o ninguna.
