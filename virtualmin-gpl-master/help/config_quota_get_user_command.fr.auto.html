Commande pour obtenir le quota d'un utilisateur
Lorsque des commandes de quota externes sont utilisées, le programme défini dans cette option peut être appelé par Virtualmin pour récupérer le quota pour un seul utilisateur. Il est appelé avec un seul paramètre - le nom d'utilisateur Unix complet et doit générer une seule ligne contenant les champs séparés par des espaces suivants:
  1. Quota actuel utilisé dans des blocs de 1 ko.
  2. Le quota souple en blocs de 1 Ko, ou 0 pour illimité.
  3. Le quota dur en blocs de 1 Ko, ou 0 pour illimité.
Si cette commande n'est pas définie sur la page Configuration du module, Virtualmin utilisera la commande pour répertorier les quotas utilisateur à la place.
