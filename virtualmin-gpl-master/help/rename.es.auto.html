Cambiar nombre de dominio
Este formulario se puede usar para cambiar el nombre de dominio de un servidor virtual existente. Por defecto, el nombre de usuario y el directorio de inicio también se ajustarán para que coincidan con el nuevo dominio, suponiendo que se basen en él.

También se puede usar para cambiar el nombre de usuario de administración o el directorio de inicio sin tocar el nombre de dominio, por ejemplo, si desea pasar a un nuevo directorio base. Sin embargo, solo el administrador maestro puede cambiar el nombre de usuario o mover la casa a una ubicación arbitraria.

De forma predeterminada, los usuarios de buzones existentes en el dominio no cambian de nombre para que sus inicios de sesión IMAP funcionen. ¿Puede forzar un cambio configurando el sufijo Cambiar nombre de buzón? Sin embargo, elegido automáticamente . Este campo también se puede usar para cambiar el sufijo a lo que desee, incluso si no se cambia el nombre de dominio.

Si un dominio usa el formato username@domain.com para los buzones, los usuarios solo cambiarán su nombre si cambia el nombre real del dominio.
