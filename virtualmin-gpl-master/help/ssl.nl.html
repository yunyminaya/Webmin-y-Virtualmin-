Een SSL website opzetten?
Wanneer deze optie is aangezet zal er een extra Apache virtuele host worden aangemaakt voor deze virtuele server, die luistert op poort 443. Hierdoor kunnen browsers verbinding maken via https://domeinnaam.com/ met gebruik van SSL versleuteling, maar ook met http://domeinnaam.com/ zonder versleuteling.

Er word een zelf ondertekend certificaat gegenereerd als onderdeel van het server installatie proces, zodat de SSL virtuele host onmiddellijk kan worden gebruikt. U moet echter zo snel mogelijk een geldig SSL certificaat genereren voor dit domein, zodat het op de juiste manier geverifieerd word door de browsers. Dit kan worden gedaan op de Beheer SSL Certificaten pagina.
