Shell pour les utilisateurs de boîtes aux lettres
Cette option spécifie le shell configuré pour les utilisateurs de boîtes aux lettres. Un utilisateur de boîte aux lettres est le type d'utilisateur le moins privilégié et devrait normalement avoir un accès très restreint. Cependant, si votre serveur Virtualmin n'est utilisé que par des utilisateurs de confiance ou des utilisateurs qui doivent avoir un accès shell, vous pouvez modifier cela de la valeur par défaut de / dev / null à l'un des shells système réels. Notez que / bin / false est un shell spécial qui fermera les connexions SSH distantes mais permettra toujours les connexions FTP.
