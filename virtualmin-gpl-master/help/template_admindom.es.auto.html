URL para redireccionamiento de administrador
De forma predeterminada, Virtualmin redirigirá las solicitudes a http://admin.thedomain.com a Webmin en el mismo sistema, como https://thedomain.com:10000/ . Sin embargo, este campo se puede usar para cambiar la URL de redireccionamiento, quizás a un único nombre de host para todos sus dominios para los que tiene un certificado SSL, como https://yourcompany.com:10000/ .
