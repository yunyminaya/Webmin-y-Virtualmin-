Включить перенаправление для
Для доменов с включенным веб-сайтом SSL эти флажки можно использовать, чтобы выбрать, будет ли перенаправление включено для веб-сайта SSL, сайта без SSL или обоих. Это может быть полезно, например, для создания перенаправления, которое заставляет всех пользователей подключаться в режиме SSL.
