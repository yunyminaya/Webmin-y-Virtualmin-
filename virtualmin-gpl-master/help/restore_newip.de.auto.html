IPv4-Adresse für wiederhergestellte Server
Diese Einstellung wird nur verwendet, wenn ein virtueller Server wiederhergestellt wird, der derzeit nicht auf dem System vorhanden ist. Standardmäßig weist Virtualmin ihm entweder die standardmäßige freigegebene IP-Adresse des Systems zu, wenn es sich ursprünglich auf einer freigegebenen IP befand, oder verwendet die private IP des ursprünglichen Systems, wenn es sich nicht auf einer freigegebenen Adresse befand.

Dies funktioniert einwandfrei, wenn Sie eine Domäne zwischen zwei Systemen im selben Netzwerk verschieben. Andernfalls kann die wiederhergestellte Domäne wahrscheinlich nicht über das Netzwerk kontaktiert werden. Um dies zu vermeiden, wählen Sie in diesem Feld die Option Virtuell mit zugeordneter IP oder Virtuell mit IP aus. Virtualmin wählt entweder eine freie IP für Sie aus oder verwendet die von Ihnen eingegebene.
