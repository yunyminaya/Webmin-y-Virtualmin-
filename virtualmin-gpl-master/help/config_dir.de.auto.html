Erstellung des Home-Verzeichnisses aktiviert?
Diese Option bestimmt, ob die Erstellung eines Basisverzeichnisses für einen neuen virtuellen Server immer abgeschlossen ist (und die Option ausgeblendet ist) oder ob sie beim Erstellen eines Servers optional ist. Da fast immer ein Basisverzeichnis benötigt wird, es sei denn, Sie erstellen eine reine DNS-Domäne, ist die Option Immer im Allgemeinen die beste.
