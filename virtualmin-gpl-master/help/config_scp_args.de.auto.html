Zusätzliche Parameter nur für den scp-Befehl
Wenn Virtualmin eine Remote-Sicherung oder -Wiederherstellung mit SCP durchführt, wird der Befehl scp zum Übertragen von Dateien verwendet. In diesem Feld können Sie zusätzliche Parameter nur für den Befehl scp angeben, z. B. -O, um eine ältere Protokollversion zu verwenden.
