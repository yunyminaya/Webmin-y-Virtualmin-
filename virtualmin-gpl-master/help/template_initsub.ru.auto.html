Изначально выбран шаблон для субсерверов?
Если для этого параметра установлено значение «Да» , в меню шаблонов при создании нового виртуального подсервера этот шаблон будет выбран по умолчанию. Он также будет использоваться при создании сервера из командной строки, если в параметре --template явно не указан шаблон . Только один шаблон может иметь этот набор для Да в то время.
