Criar usuário Unix extra em sistemas Postfix ao usar o formato user @ domain?
Ao usar nomes de usuário Unix no formato user @ domain e o servidor de correio Postfix, o Virtalmin, quando esta opção estiver definida como yes (o padrão), criará um usuário Unix no formato de domínio do usuário adicional para facilitar a entrega de mensagens. No entanto, em alguns sistemas, isso não é necessário e, portanto, você pode usar esta opção para desativar esse recurso. Só será aplicável a usuários e domínios recém-criados.
