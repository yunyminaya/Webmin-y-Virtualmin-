¿Permitir que nombres de usuarios y grupos comiencen con números?
De forma predeterminada, Virtualmin no permite que los nombres de usuario, grupo o base de datos comiencen con un número, ya que esto puede confundir a algunas herramientas Unix. Por lo tanto, un nombre de dominio como 1234.com tendrá el nombre de usuario de administración one234 en lugar de 1234 Sin embargo, puede usar esta opción para permitir nombres de usuario completamente numéricos si funcionan en su sistema.
