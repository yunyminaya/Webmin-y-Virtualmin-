Login SSH / Telnet
Esta opção especifica se o usuário proprietário do domínio pode usar o módulo Webmin de login SSH / Telnet. Este módulo é um applet Java que efetua login no servidor usando SSH ou Telnet. Não funciona corretamente com todos os navegadores, devido a incompatibilidades Java. Observe que desabilitar isso não impede o acesso do SSH ao servidor pelo proprietário do domínio, apenas o acesso ao módulo de logon SSH / Telnet do Webmin.
