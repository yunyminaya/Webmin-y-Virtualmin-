Realocar UIDs e GID?
Quando essa opção está definida como Sim e você está restaurando um servidor que não existe no sistema no momento, os IDs de usuário e grupo do Unix para o administrador do servidor e os usuários de email / FTP serão realocados para usar IDs gratuitos em seu sistema. Normalmente, é isso que você deseja, pois evita conflitos com os IDs existentes no seu sistema.

A seleção de Não informa ao Virtualmin para usar os IDs que estavam em uso quando o backup foi criado. Isso deve ser usado se ainda existirem outros arquivos no sistema com os IDs antigos e se esses IDs não foram alocados a nenhum outro usuário ou grupo desde que o backup foi feito.
