MySQL Server
Auf dieser Seite können Sie zusätzliche MySQL-Server auf Remote-Systemen oder zusätzliche Installationen auf dem System definieren, die von Virtualmin-Domänen verwendet werden. Dies kann verwendet werden, um MySQL-Hosting auf einen anderen Server zu verlagern oder um mehrere Versionen von MySQL für verschiedene Clients zu unterstützen.

Verwenden Sie zum Hinzufügen eines neuen Remote-MySQL-Servers das für am Ende der Seite. Um einen zuvor definierten Server aus der Liste zu entfernen, aktivieren Sie das Kontrollkästchen daneben in der Liste und klicken Sie auf die Schaltfläche Ausgewählte entfernen .
