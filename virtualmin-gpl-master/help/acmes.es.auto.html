Proveedores de SSL
Esta página se puede utilizar para agregar proveedores de SSL compatibles con ACME para utilizarlos al solicitar y renovar certificados. Al agregar un proveedor, puede elegir uno de los servicios conocidos por Virtualmin o ingresar cualquier URL de API de ACME.

Algunos proveedores requieren autenticación mediante una clave y HMAC, que suelen estar disponibles en su sitio web después de crear una cuenta. Otros permiten solicitar certificados sin autenticación.
