URL de redirection de messagerie Web
Par défaut, Virtualmin redirige les demandes vers http://webmail.thedomain.com vers Usermin sur le même système, comme https://thedomain.com:20000/ . Cependant, ce champ peut être utilisé pour modifier l'URL de redirection, peut-être vers un seul nom d'hôte pour tous vos domaines pour lesquels vous avez un certificat SSL, comme https://webmail.yourcompany.com:20000/ . Ou vous pouvez rediriger vers une autre application de messagerie Web sur le même domaine, en saisissant une URL telle que http://www.${DOM}/squirrelmail/ .
