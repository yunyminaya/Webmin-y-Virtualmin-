可以編輯虛擬伺服器
允許編輯虛擬伺服器功能、其密碼和描述。
可以管理用戶
允許查看、編輯、建立和刪除郵箱、FTP 和資料庫使用者。
可以管理別名
允許檢視、編輯、建立和刪除電子郵件別名。
可以管理資料庫
允許將其他 MySQL 和 PostgreSQL 資料庫新增至此虛擬伺服器或從中刪除。
可安裝網頁應用程式
允許將第三方 Web 應用程式安裝到此虛擬伺服器中。
可以更改IP位址
如果該伺服器有私有 IP 位址，則此選項允許變更它。
可以管理 SSL 憑證
允許上傳或重新產生該伺服器的 SSL 憑證和金鑰。
可以編輯轉送幀
對於僅轉送的網站，此選項允許編輯轉送框架的目標和 HTML。
可以管理額外的管理員
允許查看、建立、編輯和刪除額外的伺服器擁有者管理員帳戶。
可以配置垃圾郵件和病毒傳送
允許編輯垃圾郵件和病毒傳送目的地和分數。
可以選擇PHP版本
對於透過 CGI 或 fastCGI 運行 PHP 的網域，允許設定每個目錄的 PHP 版本。
可以編輯電子郵件設定
允許配置自動密件副本、別名複製模式和其他與電子郵件相關的選項。
可以進行備份
允許將擁有的虛擬伺服器備份到遠端 FTP 或 SSH 伺服器。
可以安排備份
允許定期備份虛擬伺服器。
可以恢復備份
允許恢復擁有的虛擬伺服器的主目錄和資料庫內容。
可以選擇共享IP
允許在建立虛擬伺服器時選擇全域定義的共用 IP 位址。
可以創造包羅萬象的別名
允許建立電子郵件別名，將所有郵件轉送到一個網域到不同的網域或位址。
可以使用網頁編輯器
允許透過 Virtualmin UI 編輯網頁。
可以編輯資料庫遠端主機
允許進入允許連接到網域的 MySQL 資料庫的遠端客戶端系統。
可以禁用虛擬伺服器
允許暫時禁用伺服器。
可以刪除虛擬伺服器
允許完全刪除擁有的伺服器，包括所有子伺服器和別名。
