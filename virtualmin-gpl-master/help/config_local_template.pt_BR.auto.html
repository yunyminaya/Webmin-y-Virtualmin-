Arquivo de modelo para email para novos usuários locais
Usuários locais são usuários que não pertencem a um domínio Virtualmin, por exemplo, se eles tiverem apenas correio e / ou serviço da Web no domínio nativo do sistema. Você pode enviar o email do novo usuário padrão ou especificar um arquivo de modelo diferente. As variáveis padrão do modelo Virtualmin estão disponíveis no modelo.
