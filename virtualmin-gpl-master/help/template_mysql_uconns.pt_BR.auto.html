Número máximo de conexões para usuários de caixas de correio
Este campo controla o número de conexões simultâneas ao MySQL que uma caixa de correio com acesso ao banco de dados pode fazer. O padrão é permitir um número ilimitado de conexões, mas como cada uma consome memória no processo do servidor MySQL, você pode limitar o total de conexões em um sistema ocupado.

O limite definido aqui será aplicado somente quando uma nova caixa de correio for criada ou quando um login do MySQL estiver ativado para uma caixa de correio existente.
