Полный путь к команде aws для операций S3
Если в вашей системе установлена команда aws и в этом поле aws полный путь к ней (например, /usr/bin/aws ), Virtualmin будет вызывать ее для выполнения операций S3 для резервного копирования, например загрузки и загрузки файлов. Поскольку эта команда была разработана Amazon, она более надежна, чем S3 API, встроенный в Virtualmin.
