Настройка сертификата Let's Encrypt для имени хоста?
Если этот параметр включен, Virtualmin попытается автоматически настроить действительный SSL-сертификат Let's Encrypt для имени хоста во время следующей проверки конфигурации.

В случае успеха этот сертификат SSL будет использоваться по умолчанию для всех служб, для которых не настроен собственный сертификат SSL.

Если имя хоста системы изменится, оно будет автоматически обнаружено и обновлено.
