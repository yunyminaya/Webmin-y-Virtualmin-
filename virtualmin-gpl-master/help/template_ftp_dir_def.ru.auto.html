Анонимный FTP-подкаталог для создания
Если анонимная загрузка по FTP будет разрешена, вы можете указать подкаталог, в котором будут храниться файлы для скачивания. Если вы изменяете директивы в предыдущем варианте, вы должны включить раздел «Аноним», и он должен соответствовать расположению, указанному вами здесь. Обратите внимание, что анонимный доступ через определенное доменное имя требует настройки виртуального хоста FTP на основе IP.
