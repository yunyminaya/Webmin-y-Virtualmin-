Sitio web predeterminado para la dirección IP?
Cuando este campo se establece en Sí , Apache servirá el sitio web de este servidor virtual cuando reciba una solicitud que no coincida con ningún otro servidor virtual en el sistema. Esto suele suceder si un usuario usa una URL con una dirección IP o un nombre de host que se resuelve en su sistema pero no coincide con ningún dominio de Virtualmin.
