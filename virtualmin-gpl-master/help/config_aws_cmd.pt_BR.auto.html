Caminho completo para o comando aws para operações S3
Se o comando aws estiver instalado no seu sistema e esse campo estiver definido como o caminho completo (como /usr/bin/aws ), o Virtualmin o chamará para executar operações S3 para backups, como upload e download de arquivos. Como esse comando foi desenvolvido pela Amazon, é mais confiável que a API S3 incorporada ao Virtualmin.
