Compresser les sauvegardes MySQL?
Lorsque cette option est définie sur Oui , les vidages de base de données MySQL seront compressés par gzip avant que Virtualmin ne crée le fichier de sauvegarde final, qui peut également être lui-même compressé à nouveau. Cela économise de l'espace lorsque la sauvegarde finale n'est qu'un fichier TAR, mais augmente le temps de sauvegarde global.
