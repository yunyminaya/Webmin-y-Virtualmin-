Заголовок ссылки для дополнительной документации
Этот параметр имеет смысл только в сочетании с пунктом назначения ссылки для дополнительной документации над ним. Его можно использовать для задания текста ссылки на документацию, специфичную для сайта, вместо значения по умолчанию для Дополнительной документации .
