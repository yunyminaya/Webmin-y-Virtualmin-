Kommando for å angi en brukers kvote
Når eksterne kvotekommandoer brukes, vil programmet som er satt i dette alternativet bli kalt når Virtualmin ønsker å stille inn kvoter for en Unix-bruker. Parametrene det heter med er:
  1. Unix-brukernavnet
  2. Den nye myke kvoten i 1 kB blokker, eller 0 for ubegrenset.
  3. Den nye harde kvoten på 1 kB blokkerer, eller 0 for ubegrenset.
