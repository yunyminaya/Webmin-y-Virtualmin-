Directivas y configuraciones para nuevos sitios web
Es posible personalizar completamente las directivas de Apache que se agregan cuando se crea un nuevo servidor virtual. La sustitución de variables se realiza para todas las variables de plantilla de Virtualmin estándar. Se debe tener cuidado con las directivas de Apache, ya que algunos problemas de seguridad actuales. La documentación de Apache y, en menor grado, las secciones de Apache de la documentación de Webmin se pueden consultar para encontrar más detalles sobre la configuración de Apache.
