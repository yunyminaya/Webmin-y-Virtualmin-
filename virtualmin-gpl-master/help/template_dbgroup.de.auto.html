Sekundärgruppe für Datenbankbenutzer
Wenn diese Option aktiviert ist, müssen Sie den Namen einer Unix-Gruppe eingeben, zu der alle Benutzer mit Zugriff auf eine oder mehrere Datenbanken für virtuelle Server in dieser Vorlage hinzugefügt werden. Wenn die von Ihnen angegebene Gruppe nicht vorhanden ist, wird sie bei Bedarf automatisch erstellt.

Diese Funktion ist am nützlichsten in Verbindung mit Usermin, wodurch sichtbare Module auf Benutzer beschränkt werden können, die Mitglieder einer Gruppe sind. Sie können das MySQL-Datenbankmodul also nur für Mitglieder der Gruppe festlegen, die Sie hier festgelegt haben.
