Bearbeitung von Limits beim Erstellen eines Servers zulassen?
Wenn diese Option auf Nein gesetzt ist , werden für alle neuen virtuellen Server Kontingente, Bandbreite und andere Grenzwerte immer auf die im ausgewählten Kontoplan definierten Standardeinstellungen festgelegt. Bei der Einstellung Ja hat der Serverersteller die Möglichkeit, diese Einstellungen zuerst im Erstellungsformular zu ändern, bevor der Server gespeichert wird. Die letztere Option ist flexibler, führt jedoch dazu, dass auf der Erstellungsseite mehrere normalerweise unveränderte Felder angezeigt werden.
