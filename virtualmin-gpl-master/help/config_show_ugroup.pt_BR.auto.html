Permitir seleção de grupo diferente para administradores de servidor?
Quando essa opção é definida como Sim, o criador de um novo servidor virtual pode selecionar um grupo Unix diferente para o qual seu proprietário seja membro, em vez do novo grupo criado para o servidor virtual. Isso pode ser útil se você precisar que todas as contas de proprietário do servidor em seu sistema sejam membros de um grupo específico, mas deve ser usado com cuidado, pois permite que os proprietários do servidor evitem as cotas do grupo.
