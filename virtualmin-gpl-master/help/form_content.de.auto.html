Erste Webseite erstellen?
Um eine erste Webseite für den neuen virtuellen Server zu erstellen, können Sie „Nein“ auswählen und keine Webseite erstellen lassen. Alternativ können Sie sich für eine Standardseite der Virtualmin-Website entscheiden, indem Sie die Option „Ja, mit der Standardseitenvorlage der Website“ auswählen.

Alternativ können Sie durch Auswahl von „Ja, mit Inhalt unten“ benutzerdefiniertes HTML oder Text eingeben, der als Webseiteninhalt verwendet werden soll.
