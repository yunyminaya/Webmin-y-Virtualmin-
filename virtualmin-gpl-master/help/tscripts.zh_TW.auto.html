模板 Web 應用程式安裝程式
此頁面可讓您指定使用此範本建立新虛擬伺服器時要安裝的第三方 Web 應用程式。對於每個網路應用程序，必須輸入以下資訊：

網路應用程式
要安裝的 Web 應用程式的名稱和版本
網址路徑
虛擬伺服器網站下用於安裝 webapp 的路徑，例如/wordpress 。這也將確定~/public_html下包含 web 應用程式檔案的目錄。
資料庫名稱
Web 應用程式將用於儲存資料的資料庫的名稱。選項有：
沒有任何
對於不需要資料庫的 web 應用程序，例如Gallery ，應選擇此選項。
主資料庫
Web 應用程式的資料庫將是所選類型的虛擬伺服器的主資料庫。
其他
如果您希望此 Web 應用程式使用自己的私人資料庫，則可以選擇此選項，並在相鄰的文字欄位中輸入資料庫的模式。例如，您可以輸入${DB}_phpnuke 。如果產生的資料庫不存在，它將作為 web 應用程式安裝過程的一部分建立。
資料庫類型
選擇用於 webapps 資料的資料庫類型（如果有）。
