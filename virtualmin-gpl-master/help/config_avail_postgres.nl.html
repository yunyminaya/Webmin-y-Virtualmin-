PostgreSQL Database Server (voor database)
Indien dit is aangezet zal het account van de domein eigenaar toegang hebben tot de PostgreSQL Webmin module. Hij krijgt een vriendelijke GUI tot de database die behoort aan de domein eigenaar. Met deze module en juiste privileges kan de domein eigenaar nieuwe databases aanmaken, bestaande databases bewerken of zoekacties uitvoeren in de data van de database.
