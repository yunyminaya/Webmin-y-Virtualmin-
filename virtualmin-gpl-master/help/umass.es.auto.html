Crear usuarios por lotes
Esta página le permite crear múltiples usuarios de correo o FTP a la vez desde un archivo de texto. Este archivo debe incluir un usuario por línea, y cada línea debe tener el siguiente formato:

username : realname : password : ftp-flag : email-flag : quota :[ extras ]:[ forwards ]:[ databases ]

Los significados de los campos son:

nombre de usuario
El nombre de inicio de sesión corto del usuario, como joe . Si es necesario, se agregará el prefijo de dominio para crear el nombre de inicio de sesión real.
nombre real
El nombre real del usuario, como Joe Smith .
contraseña
La contraseña inicial para el usuario, en texto plano.
bandera-ftp
Si se establece en 1, el usuario podrá iniciar sesión a través de FTP. Establecerlo en 0 o dejarlo vacío evita el acceso FTP. También puede usar la ruta completa a un shell válido para usuarios de buzones, como /usr/bin/scponly .
bandera de correo electrónico
Si se establece en 1, se habilitará la dirección de correo electrónico principal del usuario.
cuota
El correo del usuario y la cuota del directorio de inicio, en bloques de disco de 1 kB.
extras
Una lista separada por comas de direcciones de correo electrónico adicionales para este usuario. Deben estar en los dominios de Virtualmin que administra.
hacia adelante
Una lista separada por comas de destinos de reenvío de correo electrónico para este usuario (generalmente direcciones como joe@otherdomain.com ).
bases de datos
Una lista de bases de datos separadas por comas para otorgar acceso a este usuario. Cada uno debe ser un tipo de base de datos ( mysql o postgres ) y un nombre de base de datos, separados por un espacio.
