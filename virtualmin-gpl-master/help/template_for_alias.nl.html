Voor gebruik bij
Templates kunnen worden toegevoegd aan ieder soort domein gebruikers account. De huidige types zijn : "Top-niveau virtuele servers", wat standaard virtuele host accounts zijn; "Sub-servers", wat virtuele hosts zijn die behoren tot een top-niveau virtuele server, en verschijnen binnen de "domein" directory van de top-niveau server home directory; "Alias servers" zijn servers die over het algemeen alleen maar web content en email doorsturen naar een top-niveau domein; "Server eigenaars" zijn beschikbaar voor server eigenaars zodra die een nieuw domein aanmaken. Om gebruik van deze template toe te staan aan domein eigenaars (o.a.. server eigenaars) moet u op zijn minst 1 server type aanzetten.
