Создать пользователя веб-сервера
Создает нового пользователя веб-сервера в файлах htpasswd , специально для безопасной аутентификации в защищенных веб-каталогах. Этот пользователь отличается от учетных записей Unix и предназначен исключительно для управления доступом к веб-каталогам. Кроме того, этого пользователя можно легко интегрировать путем слияния с учетной записью Unix, созданной с тем же именем пользователя.
