Changer le nom de domaine
Ce formulaire peut être utilisé pour modifier le nom de domaine d'un serveur virtuel existant. Par défaut, le nom d'utilisateur et le répertoire personnel seront également ajustés pour correspondre au nouveau domaine, en supposant qu'ils sont basés sur celui-ci.

Il peut également être utilisé pour modifier le nom d'utilisateur d'administration ou le répertoire personnel sans toucher du tout au nom de domaine, par exemple si vous souhaitez passer à un nouveau répertoire de base. Seul l'administrateur principal peut modifier le nom d'utilisateur ou déplacer la maison vers un emplacement arbitraire.

Par défaut, les utilisateurs de boîtes aux lettres existants dans le domaine ne sont pas renommés afin que leurs connexions IMAP fonctionnent correctement. Vous pouvez forcer une modification en définissant le suffixe Modifier le nom de la boîte aux lettres? à automatiquement choisi cependant. Ce champ peut également être utilisé pour changer le suffixe en ce que vous voulez, même si le nom de domaine n'est pas modifié.

Si un domaine utilise le format username@domain.com pour les boîtes aux lettres, les utilisateurs ne seront renommés que si le nom de domaine réel change.
