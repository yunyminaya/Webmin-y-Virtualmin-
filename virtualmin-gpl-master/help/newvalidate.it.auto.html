Convalida server virtuali
Questa pagina può essere utilizzata per fare in modo che Virtualmin verifichi che tutti i server virtuali siano nello stato previsto. Confronterà i file Apache, BIND, Postfix e altri file di configurazione sottostanti con le impostazioni per i server selezionati e segnalerà eventuali errori di corrispondenza rilevati. Può essere utile per trovare problemi introdotti modificando manualmente le configurazioni o i bug di Virtualmin.

È possibile scegliere quali server convalidare con i server da controllare e quali funzioni all'interno di quelle da esaminare con l'elenco Funzionalità da controllare .
