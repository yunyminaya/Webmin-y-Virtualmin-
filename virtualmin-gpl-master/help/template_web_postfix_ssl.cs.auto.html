Kopírovat SSL certifikáty na IP do Postfixu?
Pokud je tato volba povolena a virtuální server je vytvořen s povoleným SSL a jeho vlastní privátní IP, bude Dovecot nakonfigurován tak, aby používal SSL certifikát serveru pro připojení k této IP adrese. To může být užitečné, pokud se poštovní klienti připojují k názvu domény v režimu SSL a očekávají odpovídající certifikát.
