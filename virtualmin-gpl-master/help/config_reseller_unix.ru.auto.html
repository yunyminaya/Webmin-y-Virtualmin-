Создать пользователя Unix для новых посредников?
Этот параметр определяет, будут ли вновь создаваемые учетные записи посредников получать ассоциированного пользователя Unix (для доступа по SSH и FTP) по умолчанию или нет. Однако его можно переопределить в форме создания посредника или при вызове команды API create-reseller .
