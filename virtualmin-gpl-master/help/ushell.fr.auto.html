Autorisations de connexion
Ce menu détermine le type d'accès de connexion accordé à l'utilisateur. En général, vous aurez le choix entre l'e-mail uniquement ou l'e-mail et le FTP, mais les options peuvent être personnalisées par l'administrateur principal.

Le niveau d'accès sélectionné détermine le shell de l'utilisateur Unix sous-jacent, qui est ensuite utilisé par les serveurs FTP et SSH pour contrôler s'ils peuvent se connecter ou non.
