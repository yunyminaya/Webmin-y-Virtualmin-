Quota per l'utente amministrativo
Il proprietario di un dominio è proprio come un normale account utente, in quanto può ricevere e-mail e possono essere applicate anche quote di utilizzo dello spazio su disco. I file all'interno del sito Web di dominio saranno normalmente di proprietà dell'utente, pertanto può essere utile specificare le dimensioni del sito Web utilizzando questa quota, consentendo al contempo la quota totale per l'intero server di limitare il totale utilizzato per i dati di posta e sito Web appartenenti a tutti gli utenti all'interno del dominio.
