Alias Server aanmaken
Met dit formulier kunt u een nieuwe alias server aanmaken wat in essentie een alternatieve naam is voor een bestaande server. Aliassen hebben niet hun eigen website inhoud en delen alle quota's en limieten die gelden voor de doel server. Bandbreedte gebruik door de alias server word opgeteld bij het totaal van de doel server.

Wanneer u een server aanmaakt moet u op zijn minst een domein naam en omschrijving opgeven. Als er meerdere template's zijn gedefinieerd op het systeem kunt u dit selecteren vanuit het menu.

Om te selecteren welke functies van de alias server worden doorgestuurd naar het hoofd domein gebruikt u de selectievakjes in het tweede gedeelte van de Geavanceerde opties sectie. Alleen de DNS, website en email aliasing zijn beschikbaar.
