Website-Weiterleitungen
Eine Umleitung ordnet einen URL-Pfad wie / foo entweder einer anderen URL oder einem anderen Verzeichnis auf demselben virtuellen Server zu. Dies kann verwendet werden, um benutzerfreundlichere URL-Pfade auf Ihrer Website bereitzustellen oder um die Verlagerung von Webseiten an neue Speicherorte zu bewältigen.

Auf dieser Seite werden alle vorhandenen Weiterleitungen aufgelistet. Sie können eine neue Weiterleitung erstellen, indem Sie auf den Link Neue Website-Weiterleitung hinzufügen klicken. Um eine vorhandene Weiterleitung zu bearbeiten, klicken Sie einfach auf den Quell-URL-Pfad. Um mehrere Weiterleitungen zu entfernen, aktivieren Sie die Kontrollkästchen daneben und klicken Sie unten auf der Seite auf die Schaltfläche Löschen .
