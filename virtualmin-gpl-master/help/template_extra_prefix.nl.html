Voorvoegsel voor extra administrators
Dit veld kan worden gebruikt om bij extra admin accounts te forceren dat gebruikersnamen beginnen met een voorvoegsel. Dit kan handig zijn om te identificeren bij welke server een bepaalde Webmin gebruiker hoort. Een algemene waarde voor dit veld is ${USER}_ , waardoor extra administrators geforceerd worden te beginnen met dezelfde naam als de virtuele server primaire admin login.
