Limite al numero di server non alias
Se il proprietario del dominio dispone dei privilegi di creazione di un server secondario, è possibile limitare il numero di server non alias che è possibile creare qui. Si noti che questo limite non può essere effettivamente superiore al limite predefinito sul numero di server virtuali , poiché il limite complessivo sui server virtuali si applica sia a domini alias che non alias. Un limite di 0 impedisce la creazione completa di server non alias.

Ciò è utile se si desidera consentire a un proprietario di dominio di creare un numero illimitato o superiore di server alias, ma limitare i sub-server più costosi che può creare.
