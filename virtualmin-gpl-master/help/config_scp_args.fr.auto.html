Paramètres supplémentaires pour la commande scp uniquement
Lorsque Virtualmin effectue une sauvegarde ou une restauration à distance à l'aide de SCP, il utilise la commande scp pour transférer les fichiers. Ce champ vous permet de spécifier des paramètres supplémentaires uniquement pour la commande scp , tels que -O pour utiliser une version de protocole plus ancienne.
