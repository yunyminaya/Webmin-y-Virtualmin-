Şablon Web Uygulaması Yükleyicileri
Bu sayfa, bu şablon kullanılarak yeni bir sanal sunucu oluşturulduğunda yüklenecek üçüncü taraf web uygulamalarını belirtmenize olanak tanır. Her web uygulaması için aşağıdaki bilgilerin girilmesi gerekir:

İnternet uygulaması
Yüklenecek web uygulamasının adı ve sürümü
URL yolu
Web uygulamasını yüklemek için sanal sunucunun web sitesinin altındaki /wordpress gibi bir yol. Bu aynı zamanda web uygulamasının dosyalarını içeren ~/public_html altındaki dizini de belirleyecektir.
Veritabanı adı
Web uygulamasının verileri depolamak için kullanacağı veritabanının adı. Seçenekler şunlardır:
Hiçbiri
Bu, Galeri gibi bir veritabanına ihtiyaç duymayan web uygulamaları için seçilmelidir.
Ana veritabanı
Web uygulamasının veritabanı, sanal sunucunun seçilen türdeki birincil veritabanı olacaktır.
Diğer
Bu web uygulamasının kendi özel veritabanını kullanmasını istiyorsanız, bu seçenek seçilebilir ve veritabanının modeli bitişikteki metin alanına girilebilir. Örneğin ${DB__phpnuke girebilirsiniz. Ortaya çıkan veritabanı mevcut değilse web uygulaması yükleme işleminin bir parçası olarak oluşturulacaktır.
Veritabanı türü
Varsa, webapps verileri için kullanılan veritabanı türü seçilir.
