Создание пользователя администрирования включено?
Эта опция определяет, всегда ли выполняется создание пользователя Unix для нового виртуального сервера верхнего уровня (и опция скрыта), или это необязательно при создании сервера. Как правило, рекомендуется установить для этого параметра значение Всегда , поскольку пользователь Unix необходим, если только не создается домен только для DNS или веб-прокси.
