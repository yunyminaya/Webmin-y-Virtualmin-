Redirecionamentos para novos sites
Esta tabela permite inserir redirecionamentos de sites que serão adicionados a todos os novos servidores virtuais quando forem criados. Para cada redirecionamento, você deve inserir um caminho de origem como / ou /phpadmin e um URL de destino ou um caminho relativo ao diretório public_html do domínio.

Em ambos os campos, variáveis de modelo de servidor virtual como ${DOM} e ${USER} podem ser usadas. Você também pode selecionar se o redirecionamento se aplica quando o site é acessado via HTTP, HTTPS ou ambos.
