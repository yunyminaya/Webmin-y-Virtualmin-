Mise à jour IP dynamique
Cette page vous permet de configurer Virtualmin pour détecter les modifications apportées à l'interface réseau principale du système et mettre à jour une entrée DNS dynamique lorsqu'elle change. Cela est utile si vous exécutez Virtualmin sur un système dont l'adresse change parfois, comme celle qui s'exécute dans un environnement de machine virtuelle comme EC2 d'Amazon. Il est également utile pour les serveurs connectés à Internet via une connexion sans adresse IP fixe, tels que l'ADSL ou le câble.

Pour utiliser cette page, vous devez sélectionner un service DNS dynamique, saisir le nom d'hôte utilisé par votre système sur ce service, ainsi que les informations de connexion du service. Par défaut, Virtualmin examinera l'adresse IP attribuée à l'interface réseau principale (comme eth0 ) pour vérifier les modifications, mais si votre système est derrière un routeur NAT, vous devez sélectionner l'option Adresse visible depuis Internet pour lui faire découvrir l'IP que le reste de l'Internet voit vraiment.

Lorsqu'un changement est détecté, tous les serveurs virtuels avec des adresses IP partagées utilisant l'ancienne adresse seront mis à jour vers la nouvelle. Cette modification sera effectuée à la fois dans la configuration Apache et dans les enregistrements DNS.

Si le service de script externe est sélectionné, vous devez entrer le chemin d'accès complet à un script dans le champ adjacent qui sera appelé lorsque l'adresse IP du système change. La nouvelle adresse IP et le nouveau nom d'hôte seront transmis au script en tant que paramètres de ligne de commande.
