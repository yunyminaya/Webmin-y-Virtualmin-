Bzip2-Komprimierungsbefehl
Wenn auf Ihrem System der Multithread-Komprimierungsbefehl pbzip2 für bzip2 installiert ist, kann diese Option verwendet werden, damit Virtualmin ihn verwendet. Auf einem Multi-CPU- oder Multi-Core-System läuft es schneller als normales bzip2 (und fast so schnell wie gzip), verbraucht aber insgesamt mehr CPU.
