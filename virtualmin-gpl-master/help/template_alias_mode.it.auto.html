Crea siti Web alias di
Un alias di dominio, noto anche come parcheggio di dominio o inoltro di dominio, è un nuovo dominio che inoltra semplicemente richieste Web ed e-mail al dominio principale di cui è alias. Sono disponibili diversi tipi di alias. Attualmente, il traffico Web può essere indirizzato al dominio principale tramite un set di regole ProxyPass, una regola di reindirizzamento, una regola di RedirectMatch o semplicemente aggiungendo un ServerAlias al dominio padre.
