Backup do MySQL em uma única transação?
Quando esta opção estiver definida como Yes , os backups do MySQL serão executados em uma única transação para garantir a consistência. No entanto, isso pode consumir mais RAM ao fazer backup de um banco de dados grande e variável. Ele também pode falhar com o erro Couldn't execute 'FLUSH TABLES WITH READ LOCK' em algumas versões do MySQL ou Amazon RDS. Nesse caso, você deve alterar isso para Não .
