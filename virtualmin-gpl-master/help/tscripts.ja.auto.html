テンプレート Web アプリケーション インストーラー
このページでは、このテンプレートを使用して新しい仮想サーバーを作成するときにインストールするサードパーティの Web アプリケーションを指定できます。各 Web アプリケーションについて、次の情報を入力する必要があります。

ウェブアプリ
インストールするウェブアプリの名前とバージョン
URL パス
/wordpressなど、Web アプリケーションをインストールするための仮想サーバーの Web サイトの下のパス。これにより、Web アプリケーション ファイルが含まれる~/public_htmlの下のディレクトリも決まります。
DB名
Web アプリケーションがデータを保存するために使用するデータベースの名前。オプションは次のとおりです。
なし
Galleryなど、データベースを必要としない Web アプリの場合はこれを選択する必要があります。
メインデータベース
Web アプリケーションのデータベースは、選択したタイプの仮想サーバーのプライマリ データベースになります。
他の
この Web アプリケーションで独自のプライベート データベースを使用する場合は、このオプションを選択し、隣接するテキスト フィールドにデータベースのパターンを入力します。たとえば、 ${DB}_phpnukeと入力します。結果のデータベースが存在しない場合は、Web アプリケーションのインストール プロセスの一環として作成されます。
DBタイプ
Web アプリケーション データに使用するデータベースの種類 (選択されている場合)。
