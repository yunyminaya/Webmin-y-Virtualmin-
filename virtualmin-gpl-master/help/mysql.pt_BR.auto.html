Criar login no MySQL?
Se selecionada, uma conta MySQL será criada para esse domínio, geralmente com o mesmo nome que o logon administrativo do domínio. Se você estiver usando as configurações padrão do modelo, um banco de dados MySQL também será criado para o servidor virtual. O nome do banco de dados será a primeira parte do nome do domínio - portanto, se você estiver criando o domínio foo.com , o banco de dados será chamado foo .
