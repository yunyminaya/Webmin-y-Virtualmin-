Limite al numero di istanze di Mongrel
Durante l'installazione degli script di Ruby on Rails, l'amministratore del server può selezionare il modo in cui il mio server Mongrel elabora per iniziare a gestire le richieste al percorso dello script, se è installato il modulo Apache mod_proxy_balancer . Per evitare che un numero eccessivo di processi del server Mongrel sovraccarichi il sistema, questa opzione può essere utilizzata per impostare il limite predefinito sul numero che gli amministratori di dominio possono configurare all'interno di Virtualmin.
