Дополнительные параметры командной строки для команды сжатия
По умолчанию Virtualmin запускает команды gzip или bzip2 для сжатия резервных копий без дополнительных параметров. Это означает, что будет использоваться уровень сжатия по умолчанию и формат. Тем не менее, вы можете использовать это поле для ввода дополнительных параметров, таких как - --fast для ускорения резервного копирования (за счет увеличения места), --best для лучшего сжатия (за счет времени процессора) или --rsyncable чтобы сделать файл более легко реплицируемым.
