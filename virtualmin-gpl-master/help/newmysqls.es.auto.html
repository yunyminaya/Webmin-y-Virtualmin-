Servidores MySQL
Esta página le permite definir servidores MySQL adicionales en sistemas remotos o instalaciones adicionales en el sistema, para uso de dominios de Virtualmin. Esto se puede usar para descargar el alojamiento de MySQL en un servidor diferente o para admitir múltiples versiones de MySQL para diferentes clientes.

Para agregar un nuevo servidor MySQL remoto, use el para en la parte inferior de la página. Para eliminar un servidor previamente definido de la lista, marque la casilla junto a él en la lista y haga clic en el botón Eliminar Seleccionado .
