Limieten bewerken toestaan wanneer een server word aangemaakt?
Wanneer deze optie is ingesteld op Nee, zullen alle nieuwe virtuele servers altijd hun quota, bandbreedte en andere limieten worden ingesteld op de gedefinieerde standaarden in het geselecteerde account plan. Indien het is ingesteld op Ja zal de persoon die de server aanmaakt de mogelijkheid hebben om de instellingen te wijzigen in het aanmaak formulier voordat de server word opgeslagen. De laatste optie is veel flexibeler maar kan ervoor zorgen dat er velden verschijnen in de aanmaak pagina die niet te wijzigen zijn.
