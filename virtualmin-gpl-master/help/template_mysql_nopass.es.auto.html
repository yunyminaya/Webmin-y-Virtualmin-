Contraseña MySQL para nuevos dominios
Esta opción controla la contraseña inicial de MySQL para nuevos servidores virtuales. Las opciones son:
Sincronización con inicio de sesión de administración
La contraseña para MySQL siempre se mantiene igual que el inicio de sesión de administración del servidor virtual.

Igual que el inicio de sesión de administración inicial
La contraseña es inicialmente la misma que el inicio de sesión de administración, pero no se modifica si se cambia la contraseña de administración.

Generar al azar
Cuando se crea el servidor virtual, se elige una contraseña MySQL generada aleatoriamente y no se cambia.
