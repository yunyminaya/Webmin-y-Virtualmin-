Trasferire ciascun server virtuale dopo averne eseguito il backup
Se questa casella è selezionata e si sta eseguendo un backup su un server FTP o SSH remoto nel nuovo formato, il file per ciascun server virtuale verrà trasferito dopo la sua creazione. Ciò consente di risparmiare spazio su disco locale, ma comporta il rischio di trasferire un backup incompleto in caso di errore di alcuni server virtuali.
