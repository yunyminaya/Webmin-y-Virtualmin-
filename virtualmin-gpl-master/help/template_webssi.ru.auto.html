Разрешить серверные включения по умолчанию?
Включения на стороне сервера - это функция веб-сервера, которая позволяет использовать простые макрокоманды в файлах HTML. Их можно использовать для включения других файлов HTML, установки и отображения содержимого переменных, а также для контроля содержимого, отображаемого на странице, с помощью операторов if . Они предлагают более простой способ создания динамических страниц, чем полноценный язык сценариев, такой как PHP или Perl.

Этот параметр определяет, будет ли на новых виртуальных серверах включен SSI после создания, отключен или оставлен для использования глобальных настроек по умолчанию для вашего веб-сервера.

Для получения дополнительной информации см. Документацию Apache по адресу http://httpd.apache.org/docs/current/howto/ssi.html.
