Aggiornamento IP dinamico
Questa pagina consente di configurare Virtualmin per rilevare le modifiche all'interfaccia di rete primaria del sistema e aggiornare una voce DNS dinamica quando cambia. Ciò è utile se si esegue Virtualmin su un sistema con un indirizzo che cambia di tanto in tanto, come quello in esecuzione in un ambiente di macchina virtuale come EC2 di Amazon. È inoltre utile per i server collegati a Internet tramite una connessione senza un indirizzo IP fisso, come ADSL o via cavo.

Per utilizzare questa pagina, è necessario selezionare un servizio DNS dinamico, immettere il nome host utilizzato dal sistema in quel servizio e anche i dettagli di accesso del servizio. Per impostazione predefinita, Virtualmin esaminerà l'indirizzo IP assegnato all'interfaccia di rete primaria (come eth0 ) per verificare le modifiche, ma se il sistema si trova dietro un router NAT è necessario selezionare l' indirizzo visibile dall'opzione Internet per fare in modo che rilevi l'IP che il resto di Internet vede davvero.

Quando viene rilevata una modifica, tutti i server virtuali con indirizzi IP condivisi che utilizzano il vecchio indirizzo verranno aggiornati a quello nuovo. Questa modifica verrà effettuata sia nella configurazione Apache che nei record DNS.

Se si seleziona il servizio di script esterno , è necessario immettere il percorso completo di uno script nel campo adiacente che verrà chiamato quando l'indirizzo IP del sistema cambia. Il nuovo IP e il nome host verranno passati allo script come parametri della riga di comando.
