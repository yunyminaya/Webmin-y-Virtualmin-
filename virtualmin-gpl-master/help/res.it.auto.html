Limiti delle risorse
Questa pagina consente di definire i limiti di utilizzo della CPU e della memoria applicabili a questo server virtuale e a tutti i server secondari. I limiti si applicano ad entrambi gli script PHP e i comandi vengono eseguiti tramite un accesso SSH. Possono essere utilizzati per impedire a un solo proprietario di dominio di sovraccaricare il sistema con un fork fork o comandi che utilizzano una quantità eccessiva di RAM.

I tipi di limite disponibili sono:

Numero massimo di processi
Tentare di avviare un numero maggiore di processi in esecuzione come utente Unix del server rispetto al limite impostato non avrà esito positivo. Questo vale anche per gli script PHP simultanei.

Dimensione massima per processo
Qualsiasi singolo processo che tenta di allocare più memoria di questo limite verrà terminato. Tuttavia, la somma totale di tutti i processi di proprietà del server può comunque superare il limite.

Tempo massimo della CPU per processo
Qualsiasi processo il cui tempo trascorso effettivamente utilizzando la CPU (non in attesa di IO o in sospensione) superi questo limite verrà interrotto.
