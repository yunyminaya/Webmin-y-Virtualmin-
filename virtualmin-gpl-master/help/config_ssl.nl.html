SSL website instelling aangezet?
Indien dit is aangezet zal er een SSL website worden aangemaakt voor ieder nieuw domein. Als het is ingesteld op "Ja, maar niet standaard" zal het mogelijk zijn om tijdens het aanmaken van het domein te kiezen of het een SSL website moet zijn. Let op dat een SSL website een eigen IP adres moet hebben, dus het kan niet standaard aangezet worden voor alle websites. Normaal gesproken heeft een website geen SSL veiligheid nodig en daarom kunnen honderden of duizenden website een enkel IP adres delen.
