Servidor de banco de dados PostgreSQL (para banco de dados)
Se ativada, a conta do proprietário do domínio terá acesso ao módulo PostSQL Webmin. Ele fornece uma interface gráfica amigável para os bancos de dados que pertencem ao proprietário do domínio. Com este módulo e os privilégios apropriados, o proprietário do domínio pode criar novos bancos de dados, editar bancos de dados existentes ou executar consultas sobre os dados no banco de dados.
