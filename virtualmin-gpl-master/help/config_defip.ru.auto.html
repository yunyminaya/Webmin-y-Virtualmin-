IP-адрес виртуального сервера по умолчанию
Виртуальные серверы без включенных функций виртуального хоста SSL или FTP будут находиться на одном IP-адресе. Эта опция указывает этот адрес. Если оставить значение по умолчанию, Virtualmin будет использовать основной адрес на интерфейсе, указанном сетевым интерфейсом для виртуальных адресов выше.
