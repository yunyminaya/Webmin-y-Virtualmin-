Nome de domínio para uso em links
Por padrão, quando o Virtualmin exibe um link para o site desse servidor virtual (como após a instalação de um script), ele usa o nome de domínio do servidor. No entanto, você pode usar esse campo para selecionar um domínio de alias para usar em links. Isso pode ser útil se o servidor estiver acessível apenas externamente pelo nome de domínio alternativo.
