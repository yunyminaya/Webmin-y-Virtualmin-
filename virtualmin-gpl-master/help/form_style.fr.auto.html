Style pour le contenu initial
Le style sélectionné dans ce menu sera fusionné avec le contenu initial du site Web pour générer des pages HTML pour le nouveau serveur virtuel. Certains styles ne contiennent qu'une seule page, tandis que d'autres ont plusieurs pages et images.

Une vignette de chaque style est incluse dans le menu. Pour voir une image plus grande, sélectionnez un style puis cliquez sur le lien Aperçu plus grand .
