Версии PHP
Эта страница позволяет вам выбрать версию PHP, используемую по умолчанию на этом виртуальном сервере, а также выбрать разные версии для разных веб-подкаталогов. Это может быть полезно, если у вас есть PHP-приложения, которые будут работать только с определенной версией.

Чтобы изменить версию, используемую по умолчанию, выберите ее в меню рядом со строкой каталога HTML по умолчанию и нажмите «Сохранить» . Аналогично, чтобы изменить версию, используемую другим каталогом, выберите его в меню рядом со строкой этого каталога.

Чтобы добавить пользовательскую версию PHP для нового каталога, введите его относительный путь в текстовое поле внизу таблицы, выберите версию в меню внизу и нажмите «Сохранить» . Этот путь должен быть относительно public_html , как WordPress или MediaWiki. Нет необходимости использовать ведущий / или полный URL.

Чтобы сбросить каталог для использования версии PHP по умолчанию, установите флажок рядом с ним в таблице и нажмите кнопку «Удалить выбранное» .
