Copiar certificados SSL por IP para Dovecot?
Quando esta opção está ativada e um servidor virtual é criado com SSL ativado e seu próprio IP privado, o Dovecot será configurado para usar o certificado SSL do servidor para conexões com esse endereço IP. Isso pode ser útil se os clientes de email se conectarem ao nome de domínio no modo SSL e esperarem um certificado correspondente.
