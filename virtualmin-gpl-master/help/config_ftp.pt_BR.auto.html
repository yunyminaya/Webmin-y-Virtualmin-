Servidor virtual baseado em IP ProFTPd ativado?
Ativa ou desativa um novo host virtual de FTP. Observe que o FTP requer um novo IP para hospedagem de domínio virtual anônimo. A grande maioria dos proprietários de sites não precisa desse recurso e não é necessário para o acesso FTP autenticado aos arquivos dos proprietários do domínio. No entanto, se alguns domínios exigirem acesso FTP anônimo virtual hospedado (por exemplo, se eles estiverem operando um servidor de arquivos FTP para downloads de software ou arquivos para usuários não conectados), convém configurá-lo para "Sim, mas não por padrão" .
