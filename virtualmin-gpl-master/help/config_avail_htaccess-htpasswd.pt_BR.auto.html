Diretórios da Web protegidos (no diretório inicial)
Esta opção especifica se o proprietário do domínio terá acesso ao módulo Webmin de Diretórios da Web Protegidos, que permite a criação de um ou mais diretórios protegidos por senha. Obviamente, isso pode ser feito usando arquivos .htaccess, mas muitos usuários não sabem como criar ou usar arquivos .htaccess, e este módulo fornece um meio baseado na Web para executar a mesma tarefa.
