Установщики шаблонов веб-приложений
На этой странице вы можете указать сторонние веб-приложения, которые будут устанавливаться при создании нового виртуального сервера с использованием этого шаблона. Для каждого веб-приложения необходимо ввести следующую информацию:

Веб-приложение
Имя и версия веб-приложения для установки.
URL-путь
Путь под веб-сайтом виртуального сервера для установки веб-приложения, например /wordpress . Это также определит каталог ~/public_html , содержащий файлы веб-приложения.
Имя БД
Имя базы данных, которую веб-приложение будет использовать для хранения данных. Варианты:
Никто
Это следует выбрать для веб-приложений, которым не нужна база данных, таких как Галерея .
Основная база данных
База данных веб-приложения будет основной базой данных виртуального сервера выбранного типа.
Другой
Если вы хотите, чтобы это веб-приложение использовало собственную частную базу данных, можно выбрать этот параметр и ввести шаблон базы данных в соседнее текстовое поле. Например, вы можете ввести ${DB}_phpnuke . Если результирующая база данных не существует, она будет создана в процессе установки веб-приложения.
Тип БД
Тип использования базы данных для данных веб-приложений, если таковой выбран.
