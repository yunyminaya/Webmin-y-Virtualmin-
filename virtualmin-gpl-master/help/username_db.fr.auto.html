Nom d'utilisateur
Ce champ définit la partie nom d'utilisateur de la connexion de l'utilisateur, qui doit être unique dans la base de données. Le login de l'utilisateur pour accéder à la base de données sera dérivé du nom d'utilisateur saisi ici, combiné avec le nom de domaine. Par exemple, si vous entrez le nom d'utilisateur db-user1 et que le domaine est example.com , la connexion résultante sera db-user1@example.com .
