E-Mail Greylisting
Auf dieser Seite können Sie das Greylisting eingehender E-Mails auf Ihrem Mailserver aktivieren, wodurch Spam reduziert wird, indem sendende Mailserver gezwungen werden, das Senden erneut zu versuchen, wenn sie zum ersten Mal angezeigt werden. Echte Mailserver werden nach einer kurzen Verzögerung erneut versucht, solche, die von Spammern betrieben werden, jedoch normalerweise nicht. Somit werden legitime E-Mails immer noch zugestellt, Spam jedoch nicht.

Wenn Greylisting aktiviert ist, wird auf dieser Seite auch eine Liste der Client-Mailserver und Empfänger-E-Mail-Adressen angezeigt, die von Greylisting ausgeschlossen sind. Sie können einen bearbeiten, indem Sie darauf klicken, oder über den Link Hinzufügen einen neuen Eintrag erstellen. Whitelist-Einträge können entfernt werden, indem Sie die Kontrollkästchen daneben aktivieren und unten auf der Seite auf die Schaltfläche Ausgewählte löschen klicken.
