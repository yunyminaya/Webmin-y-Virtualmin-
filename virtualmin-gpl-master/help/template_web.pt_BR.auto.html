Diretrizes e configurações para novos sites
É possível personalizar completamente as diretivas Apache adicionadas quando um novo servidor virtual é criado. A substituição de variáveis é realizada para todas as variáveis de modelo padrão do Virtualmin. Cuidados devem ser tomados com as diretivas Apache, pois alguns apresentam problemas de segurança. A documentação do Apache e, em menor grau, as seções do Apache da documentação do Webmin podem ser consultadas para encontrar mais detalhes sobre a configuração do Apache.
