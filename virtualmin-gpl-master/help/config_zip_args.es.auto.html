Parámetros adicionales de la línea de comandos para el comando de compresión
Por defecto, Virtualmin ejecuta los comandos gzip o bzip2 para comprimir copias de seguridad sin parámetros adicionales. Esto significa que se utilizará el nivel de compresión y el formato predeterminados. Sin embargo, puede usar este campo para ingresar parámetros adicionales, como - --fast para hacer que sus copias de seguridad se ejecuten más rápido (a costa de requerir más espacio), - --best para obtener una mejor compresión (a costa del tiempo de CPU), o --rsyncable para hacer que el archivo sea más fácilmente replicable.
