Définir la propriété de groupe des fichiers de base de données MySQL?
Lorsque cette option est activée, toutes les nouvelles bases de données MySQL auront la propriété de groupe définie sur leurs fichiers de données (généralement sous /var/lib/mysql ) afin qu'elles soient incluses dans les limites de quota Unix de leurs serveurs virtuels. S'il n'est pas activé, l'utilisation du disque des bases de données ne sera pas restreinte et il sera possible pour un serveur virtuel de dépasser son allocation de disque autorisée.
