Использовать адрес IPv6 по умолчанию для новых виртуальных серверов?
По умолчанию Virtualmin добавит записи DNS и конфигурацию веб-сервера на новые виртуальные серверы, чтобы использовать основной IPv6-адрес системы. Во время создания пользователь по-прежнему может выбрать использование частного IPv6-адреса или вообще не использовать его.
