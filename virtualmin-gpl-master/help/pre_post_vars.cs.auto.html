Konkrétní akci Virtualmin, která byla provedena, lze nalézt v proměnné VIRTUALSERVER_ACTION, například:

CREATE_DOMAIN
Vytváří se nový virtuální server.
DELETE_DOMAIN
Probíhá odstraňování existujícího virtuálního serveru.
DISABLE_DOMAIN
Virtuální server je dočasně zakázán.
ENABLE_DOMAIN
Probíhá opětovné povolení virtuálního serveru.
MODIFY_DOMAIN
Virtuální server se aktualizuje, například když se přidávají nebo odebírají funkce nebo se mění vlastnictví.
CLONE_DOMAIN
Virtuální server se vytváří zkopírováním existujícího serveru.
