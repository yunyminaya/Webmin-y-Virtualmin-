Имя хоста для записи MX
По умолчанию запись DNS MX (которая определяет, куда доставляется электронная почта) для новых доменов будет иметь вид mail.${DOM}, которому будет присвоен тот же IP-адрес, что и веб-сайту виртуального сервера. Однако вы можете использовать это поле, чтобы указать другое имя хоста почтового сервера — обычно оно используется для обеспечения согласованности записи MX во всех доменах. Обязательно введите имя хоста, которое соответствует вашей системе Virtualmin, иначе электронная почта не будет доставлена.

Вы также можете использовать стандартные переменные Virtualmin, такие как ${DOM}, чтобы указать запись MX с другим именем в создаваемом домене.
