Secciones adicionales para habilitar Jailkit
De forma predeterminada, sólo se copian los archivos esenciales de las secciones de configuración de Jailkit a cárceles nuevas o actualizadas. Para incluir archivos adicionales, enumere las secciones deseadas de /etc/jailkit/jk_init.ini aquí.

Las secciones del archivo /etc/jailkit/jk_init.ini se denominan [git] o [apacheutils] . Por ejemplo, para incluir secciones git y apacheutils en este entorno chroot de dominio, coloque git apacheutils (separado por espacio) en este campo.
