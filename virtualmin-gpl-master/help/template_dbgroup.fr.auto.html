Groupe secondaire pour les utilisateurs de base de données
Si cette option est définie, vous devez entrer le nom d'un groupe Unix auquel tous les utilisateurs ayant accès à une ou plusieurs bases de données pour les serveurs virtuels sur ce modèle seront ajoutés. Si le groupe que vous spécifiez n'existe pas, il sera automatiquement créé si nécessaire.

Cette fonctionnalité est très utile en conjonction avec Usermin, qui peut limiter les modules visibles aux utilisateurs qui sont membres d'un groupe. Vous pouvez donc rendre le module de base de données MySQL visible uniquement aux membres du groupe que vous définissez ici.
