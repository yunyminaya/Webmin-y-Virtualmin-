Umleitung auf Hostnamen beschränken
Standardmäßig werden Umleitungen unabhängig vom URL-Hostnamen angewendet, der zum Zugriff auf die Website verwendet wird. Dieses Feld kann jedoch stattdessen verwendet werden, um die Umleitung auf einen bestimmten Host zu beschränken. Dies kann beispielsweise nützlich sein, um www.domain.com auf domain.com umzuleiten oder die Umleitung für eine bestimmte Aliasdomäne zu ändern.
