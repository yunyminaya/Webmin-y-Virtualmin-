Arquivo de log de erros do servidor da Web
Este campo define o arquivo no qual os logs de erro do Apache são gravados para este servidor virtual. Quaisquer falhas no fornecimento de conteúdo para este domínio serão gravadas neste arquivo. Ele só deve ser alterado se você estiver movendo o log para um diretório diferente, como /var/log/virtualmin ou o diretório inicial do servidor.
