Página HTML predeterminada del sitio web
Esta opción le permite controlar la página predeterminada que se creará y se mostrará para el nuevo servidor virtual.

Puede elegir "Deshabilitado" para omitir la creación de una página o seleccionar "Preguntar en el momento de la creación" para decidirlo durante el momento de creación del servidor virtual.

Alternativamente, puede optar por una página de sitio web predeterminada de Virtualmin seleccionando la opción "Sí, con la plantilla de página de sitio web predeterminada".

Si lo prefiere, elegir "Sí, con contenido a continuación" le permitirá proporcionar HTML o texto personalizado para usar como contenido de la página web.
