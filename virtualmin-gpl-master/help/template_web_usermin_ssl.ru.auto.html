Настроить Usermin для использования того же сертификата SSL?
Если этот шаблон будет использоваться исключительно для сайтов с поддержкой SSL, вы можете включить эту опцию. Затем Usermin будет использовать тот же сертификат, что и веб-сайт SSL, поэтому он не будет генерировать предупреждения сертификата, когда пользователи входят в систему, используя свое собственное имя хоста.
