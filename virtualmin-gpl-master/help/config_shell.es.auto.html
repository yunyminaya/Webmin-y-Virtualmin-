Shell para usuarios de buzones
Esta opción especifica el shell configurado para los usuarios del buzón. Un usuario de buzón es el tipo de usuario menos privilegiado, y normalmente debería tener un acceso muy restringido. Sin embargo, si su servidor Virtualmin solo es utilizado por usuarios confiables, o usuarios que deben tener acceso a shell, puede alterar esto desde el valor predeterminado de / dev / null a uno de los shells reales del sistema. Tenga en cuenta que / bin / false es un shell especial que cerrará los inicios de sesión SSH remotos pero aún permitirá los inicios de sesión FTP.
