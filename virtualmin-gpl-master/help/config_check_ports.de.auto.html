Kann unerwartete Ports abhören?
Virtualmin kann nach Serverprozessen suchen, die von Domänenbesitzern ausgeführt werden, die TCP-Ports überwachen, die keinem bekannten Skript oder FPM-Pool zugeordnet sind. Diese Option bestimmt, ob diese Server einfach ignoriert werden, ob sie zulässig sind, aber im Validierungsbericht enthalten sind oder ob sie automatisch beendet werden.

Server, die an unerwarteten Ports ausgeführt werden, können gefährlich sein, da sie es einem Rouge-Domänenbesitzer ermöglichen können, den für eine andere Domäne bestimmten Netzwerkverkehr zu erfassen.
