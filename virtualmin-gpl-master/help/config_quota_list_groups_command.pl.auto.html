Polecenie, aby wyświetlić przydziały grup
Gdy używane są zewnętrzne komendy przydziału, program ustawiony w tej opcji zostanie wywołany, gdy Virtualmin będzie musiał znaleźć istniejące przydziały grup. Nie jest wywoływany z żadnymi parametrami, ale musi generować jeden wiersz tekstu na grupę, przy czym każdy wiersz zawiera pola oddzielone spacjami. Pola w każdym rzędzie muszą być:
  1. Nazwa grupy uniksowej.
  2. Obecny przydział stosowany w blokach 1 kB.
  3. Limit miękki w blokach 1 kB lub 0 dla nieograniczonej.
  4. Twardy przydział w blokach 1 kB lub 0 dla nieograniczonego.
