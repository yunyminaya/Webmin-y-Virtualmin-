Показать новые возможности
При использовании темы, которая может отображать новые функции в текущей версии Virtualmin, этот параметр определяет, какие типы пользователей будут видеть их. Пользователи будут видеть только уведомления о функциях, к которым у них есть доступ, и каждый сможет по отдельности их скрыть.
