Использование диска
На этой странице показано использование диска для виртуального сервера, разбитого на пять разделов с вкладками:
обзор
В этом разделе показана дисковая квота, выделенная для виртуального сервера, пространство, в настоящее время используемое файлами, принадлежащими серверу, и размер принадлежащих баз данных.

Домашний каталог
Показывает пространство, используемое топ-10 каталогов под домом сервера. Размер квоты для каждого из них - это размер файлов, принадлежащих серверу Unix grouo, в то время как размер включает файлы, принадлежащие другим пользователям. Последний может быть больше для каталогов, таких как logs когда файлы принадлежат пользователю root . homes каталог, содержащий пользователей почты / FTP, не включен, как это учитывается в следующем разделе.

По пользователю
В этом разделе показано дисковое пространство, используемое 10 лучшими пользователями почтовых ящиков в этом домене и любыми подчиненными серверами. Использование основано на файлах, принадлежащих пользователю Unix для каждого почтового ящика.

По суб-серверу
Показывает дисковое пространство, используемое каждым субсервером, на основе файлов в их домашних каталогах. Пространство, используемое любыми пользователями почтовых ящиков на этих серверах, также включено.

По базе данных
Показывает дисковое пространство, используемое каждой базой данных MySQL или PostgreSQL на этом виртуальном сервере или любых подчиненных серверах. Показаны только 10 лучших баз данных.
