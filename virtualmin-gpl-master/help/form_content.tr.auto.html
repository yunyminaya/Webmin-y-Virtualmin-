İlk web sayfasını oluştur?
Yeni sanal sunucu için bir başlangıç web sayfası oluşturmak için "Hayır" seçeneğini seçerek web sayfası oluşturmamayı seçebilir veya "Evet, web sitesi varsayılan sayfa şablonuyla" seçeneğini seçerek Virtualmin web sitesi varsayılan sayfasını seçebilirsiniz.

Alternatif olarak, "Evet, aşağıdaki içerikle" seçeneğini seçerek web sayfası içeriği olarak kullanılacak özel HTML veya metin girebilirsiniz.
