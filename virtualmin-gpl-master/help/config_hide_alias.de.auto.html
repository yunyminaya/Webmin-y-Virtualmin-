Virtuelle Alias-Server in das Menü aufnehmen?
Standardmäßig enthält das Menü auf der linken Seite in Virtualmin alle virtuellen Server, einschließlich Aliase. Wenn diese Option jedoch auf Nein gesetzt ist , werden Alias-Server von der Liste ausgeschlossen. Dies kann nützlich sein, wenn Ihr System eine große Anzahl von Aliasnamen hostet, z. B. für geparkte Domänen. Sie können diese Aliaserver weiterhin über die Links auf der Seite "Virtuellen Server bearbeiten" für ihre Eltern finden und verwalten.
