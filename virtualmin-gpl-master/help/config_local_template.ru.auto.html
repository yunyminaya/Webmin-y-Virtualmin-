Файл шаблона для электронной почты новым локальным пользователям
Локальные пользователи - это пользователи, которые не принадлежат домену Virtualmin, например, если у них есть только почта и / или веб-служба в собственном домене системы. Вы можете отправить электронное письмо по умолчанию для нового пользователя или указать другой файл шаблона. Стандартные переменные шаблона Virtualmin доступны в шаблоне.
