Tipo de campo de contraseña
Esta configuración controla cómo aparece el campo para ingresar la contraseña para un nuevo servidor virtual. Las opciones disponibles son:
Contraseña generada aleatoriamente
Se muestra un solo campo visible, que contiene una contraseña generada aleatoriamente seleccionada por Virtualmin. Sin embargo, puede ingresar una contraseña diferente.

Ingrese la contraseña una vez
Se muestra un único campo para ingresar su propia contraseña, con la contraseña real ingresada oculta.

Ingrese la contraseña dos
Se muestran dos campos para ingresar su propia contraseña, nuevamente con la contraseña oculta. Ambas contraseñas deben ser iguales.
