Tipos de contraseña hash para almacenar
De forma predeterminada, Virtualmin almacenará las contraseñas de dominio y buzón en varios formatos hash para que la contraseña esté disponible al agregar nuevas funciones al dominio. Sin embargo, no todos los formatos hash son igualmente seguros, por lo que este campo se puede usar para deshabilitar el almacenamiento de algunos formatos. Independientemente de lo que seleccione aquí, la contraseña siempre se almacenará en hash utilizando el formato predeterminado del sistema en /etc/shadow o LDAP.

Tenga en cuenta que deshabilitar el formato hash de MySQL puede evitar que Virtualmin cree un inicio de sesión MySQL para los usuarios del buzón con la misma contraseña que el buzón originalmente asignado. La desactivación del formato de resumen web puede impedir el acceso a un directorio web protegido por parte de los usuarios del buzón.
