Subdirectorio FTP anónimo para crear
Si se permitirán descargas FTP anónimas, puede especificar el subdirectorio donde vivirán los archivos para descargar. Si modifica las directivas en la opción anterior, debe incluir una sección Anónima y debe coincidir con la ubicación que proporcione aquí. Tenga en cuenta que el acceso anónimo a través de un nombre de dominio específico requiere la configuración de un host FTP virtual basado en IP.
