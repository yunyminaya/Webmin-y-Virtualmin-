Ограничить перенаправление на имя хоста
По умолчанию перенаправления будут применяться независимо от имени хоста URL, используемого для доступа к веб-сайту. Однако вместо этого это поле можно использовать для ограничения перенаправления на конкретный хост. Это может быть полезно, например, для перенаправления www.domain.com на домен.com или изменения перенаправления для определенного псевдонима домена.
