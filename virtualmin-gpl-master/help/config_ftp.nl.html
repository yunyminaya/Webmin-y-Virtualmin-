ProFTPd IP gebaseerde virtuele server aanzetten?
Hiermee zet u een FTP virtuele host aan of uit. Let op dat FTP een nieuw IP nodig heeft voor anonieme virtuele domein hosting. De meeste website eigenaars hebben deze optie niet nodig, en het is niet nodig voor FTP authenticatie toegang naar de domein eigenaar files. Echter indien sommige domeinen een virtuele gehoste anonieme FTP toegang nodig hebben (Bijv. als ze een FTP file server hebben voor software of download files voor niet ingelogde gebruikers) kunt u overdenken dit op Ja, maar niet standaard te zetten.
