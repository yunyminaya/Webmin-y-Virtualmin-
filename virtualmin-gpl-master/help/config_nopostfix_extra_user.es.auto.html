¿Crear un usuario adicional de Unix en los sistemas Postfix al usar el formato de usuario @ dominio?
Cuando se utilizan los nombres de usuario de Unix con formato de dominio de usuario @ y el servidor de correo Postfix, Virtalmin cuando esta opción se establece en sí (el valor predeterminado) crea un usuario de Unix con formato de dominio de usuario adicional para facilitar la entrega de correo. Sin embargo, en algunos sistemas esto no es necesario, por lo que puede usar esta opción para desactivar esa función. Sin embargo, solo se aplicará a los usuarios y dominios recién creados.
