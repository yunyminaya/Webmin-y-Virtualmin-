Caracteres para contraseñas aleatorias
De forma predeterminada, las contraseñas generadas aleatoriamente contendrán solo letras mayúsculas y minúsculas y los dígitos del 0 al 9. Sin embargo, este campo se puede usar para ingresar una lista alternativa de caracteres para usar, como abcdefghijklmnopqrstuvwxyz_-.0123456789 .
