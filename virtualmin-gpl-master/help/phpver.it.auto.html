Versioni di PHP
Questa pagina consente di selezionare la versione di PHP utilizzata per impostazione predefinita in questo server virtuale e anche di scegliere versioni diverse per diverse sottodirectory web. Questo può essere utile se hai applicazioni PHP che funzioneranno solo con una versione specifica.

Per modificare la versione utilizzata per impostazione predefinita, selezionala dal menu accanto alla riga della directory HTML predefinita e fai clic su Salva . Allo stesso modo, per modificare la versione utilizzata da un'altra directory, selezionala dal menu accanto alla riga di quella directory.

Per aggiungere una versione PHP personalizzata per una nuova directory, inserisci il relativo percorso nel campo di testo nella parte inferiore della tabella, seleziona una versione dal menu in basso e fai clic su Salva . Questo percorso deve essere relativo a public_html , come wordpress o mediawiki . Non è necessario un URL principale / o completo.

Per ripristinare una directory in modo da utilizzare la versione predefinita di PHP, seleziona la casella di controllo accanto ad essa nella tabella e fai clic sul pulsante Rimuovi selezionati .
