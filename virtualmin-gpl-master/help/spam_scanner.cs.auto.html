Program pro vyhledávání virů
Možné možnosti pro toto pole jsou:
Samostatný skener ( clamscan )
Toto je obvykle výchozí a řekne Virtualmin, aby použil skenovací program ClamAV, který celou práci provádí sám. To jistě funguje, ale může být pomalejší než další možnost.
Serverový skener ( clamdscan )
Tuto možnost lze vybrat, pokud je ve vašem systému spuštěn server pro vyhledávání virů clamd . Obecně je lepší, protože virová databáze ClamAV je poměrně velká a server ji potřebuje načíst do paměti pouze jednou.
Vzdálený serverový skener ( klient clamd-stream )
Tuto možnost lze vybrat, pokud je server pro vyhledávání virů clamd spuštěn ve vzdáleném systému a byl nakonfigurován tak, aby akceptoval síťová připojení od jiných hostitelů. Je to užitečné pro vyložení úlohy skenování do jiného systému, než je ten, který přijímá e-maily.
Jiný program
Tato volba vám umožní říct Virtualmin, aby použil nějaký úplně jiný program, například vlastní skriptový skript pro ClamAV.
