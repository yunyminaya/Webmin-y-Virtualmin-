Agregar propietarios de dominio al grupo secundario
Cuando este campo se establece con el nombre de un grupo existente, todos los usuarios propietarios de servidores Unix se agregarán como miembros secundarios. Esto puede ser útil para configurar otros programas como Samba o ProFTPd para restringir o permitir aún más a los propietarios de servidores.

Asegúrese de seleccionar un grupo que ya exista y que no tenga miembros actuales, ya que su lista de miembros se sobrescribirá con la lista de propietarios de dominios de Virtualmin.
