¿Crear inicio de sesión PostgreSQL?
Si se selecciona, se creará una cuenta PostgreSQL para este dominio, generalmente con el mismo nombre que el inicio de sesión administrativo del dominio. Si está utilizando la configuración de plantilla predeterminada, también se creará una base de datos PostgreSQL para el servidor virtual. El nombre de la base de datos será la primera parte del nombre de dominio, por lo que si está creando el dominio foo.com , la base de datos se llamará foo .
