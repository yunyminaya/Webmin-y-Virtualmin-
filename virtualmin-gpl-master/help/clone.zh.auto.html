克隆虚拟服务器
此页面允许您使用新域名复制现有虚拟服务器。任何与原始域相关的 Web 内容、DNS 记录、邮箱、邮件别名、数据库和其他设置都将被复制（如果可能）。但是，Web 内容不会以任何方式更改，因此如果虚拟服务器包含已配置为使用旧服务器的数据库或主目录的 Web 应用程序，则在克隆后需要手动重新配置。

复制顶级虚拟服务器时，您还必须输入新的用户名和密码才能拥有新域名。如果服务器有私有 IP 地址，则会自动为克隆分配一个新 IP。

克隆过程将以新名称复制与原始域相关的所有数据库，并复制其内容。但是，它可能不会更新原始域中安装的所有配置为使用原始数据库的 Web 应用程序。这必须手动完成。
