Können mehrere Server aus einer Batchdatei erstellt werden?
Auf der Seite "Stapelerstellungsserver" von Virtualmin können Benutzer mit Berechtigungen Server hinzufügen, die mehrere gleichzeitig aus einer Textdatei erstellen. Da dies zur schnellen Erstellung einer großen Anzahl von Domänen führen kann, möchten Sie möglicherweise verhindern, dass Serveradministratoren diese Funktion verwenden. Ändern Sie dazu diese Einstellung in Nein. Dadurch wird sie nur vom Hauptadministrator und von Wiederverkäufern verwendet.
