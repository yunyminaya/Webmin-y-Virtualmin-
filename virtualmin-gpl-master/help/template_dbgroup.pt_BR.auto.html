Grupo secundário para usuários do banco de dados
Se essa opção estiver configurada, você deverá inserir o nome de um grupo unix ao qual todos os usuários com acesso a um ou mais bancos de dados para servidores virtuais neste modelo serão adicionados. Se o grupo que você especificar não existir, ele será criado automaticamente conforme necessário.

Esse recurso é mais útil em conjunto com o Usermin, que pode limitar os módulos visíveis aos usuários que são membros de algum grupo. Assim, você pode tornar o módulo MySQL Database visível apenas para os membros do grupo que você definir aqui.
