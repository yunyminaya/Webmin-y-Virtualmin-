Backup pianificati
Virtualmin consente di pianificare il backup di alcuni di tutti i server virtuali su file locali o server SSH o FTP remoti, da eseguire nei momenti desiderati. Questa pagina elenca tutti i backup che hai le autorizzazioni per gestire e ti consente di crearne di nuovi.

Per ogni backup visualizzato, è possibile modificarne le impostazioni facendo clic sul collegamento nella colonna Destinazione o rimuoverlo selezionando la casella a sinistra e premendo il pulsante Elimina .

Per eseguire subito un backup pianificato, fare clic sul collegamento Backup .. sulla destra per visualizzare il modulo di backup con i dettagli compilati. O per ripristinare il backup più recente eseguito utilizzando questa pianificazione, fare clic sul collegamento Ripristina . In entrambi i casi, avrai la possibilità di confermare cosa fare prima di procedere.

Non è necessario pianificare più backup per gli stessi domini che vengono eseguiti contemporaneamente, poiché potrebbero interferire tra loro o essere eseguiti in un ordine incoerente.
