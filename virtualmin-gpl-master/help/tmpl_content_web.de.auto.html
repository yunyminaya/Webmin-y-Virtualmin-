Standard-HTML-Seite der Website
Mit dieser Option können Sie die Standardseite steuern, die für den neuen virtuellen Server erstellt und angezeigt wird.

Sie können „Deaktiviert“ wählen, um das Erstellen einer Seite zu überspringen, oder sich für eine Standardseite der Virtualmin-Website entscheiden, indem Sie die Option „Ja, mit der Standardseitenvorlage der Website“ auswählen.
