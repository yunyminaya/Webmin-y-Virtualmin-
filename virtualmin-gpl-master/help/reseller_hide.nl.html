Verstop limieten van server eigenaars?
Wanneer deze optie is ingesteld op Ja, zullen alle limieten van deze reseller voor het aantal domeinen, aliassen of andere objecten die zijn virtuele server (eigenaar) gebruikers mogen aanmaken niet worden getoond aan de server eigenaars. Limieten blijven geldig, maar worden alleen getoond op het moment dat een limiet bereikt word.
