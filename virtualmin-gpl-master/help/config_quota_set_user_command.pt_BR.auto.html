Comando para definir a cota de um usuário
Quando comandos de cota externa estão sendo usados, o programa definido nesta opção será chamado quando o Virtualmin desejar definir as cotas para um usuário Unix. Os parâmetros com os quais é chamado são:
  1. O nome de usuário do Unix
  2. A nova cota flexível em blocos de 1 kB ou 0 para ilimitado.
  3. A nova cota fixa em blocos de 1 kB ou 0 para ilimitado.
