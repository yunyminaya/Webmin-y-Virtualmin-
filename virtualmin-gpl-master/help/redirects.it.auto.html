Reindirizzamenti del sito Web
Un reindirizzamento mappa un percorso URL come / foo su un URL diverso o su una directory diversa sullo stesso server virtuale. Questo può essere usato per fornire percorsi URL più amichevoli sul tuo sito Web o per far fronte allo spostamento di pagine Web in nuove posizioni.

Questa pagina elenca tutti i reindirizzamenti esistenti e consente di crearne uno nuovo facendo clic sul collegamento Aggiungi un nuovo reindirizzamento del sito Web . Per modificare un reindirizzamento esistente, fai clic sul relativo percorso URL di origine. Per rimuovere più reindirizzamenti, seleziona le caselle accanto a loro e fai clic sul pulsante Elimina nella parte inferiore della pagina.
