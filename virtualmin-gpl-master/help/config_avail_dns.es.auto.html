Servidor DNS BIND (para dominio DNS)
Habilite esta opción si la cuenta del propietario del dominio debe tener la capacidad de editar registros DNS para su dominio. Según la configuración, el propietario del dominio puede crear nuevos registros de subdominio, modificar registros de dominio existentes, ajustar registros MX y más. Es posible que los usuarios sin experiencia no necesiten o no deseen este nivel de flexibilidad.
