Cloner le serveur virtuel
Cette page vous permet de dupliquer un serveur virtuel existant avec un nouveau nom de domaine. Tout contenu Web, enregistrements DNS, boîtes aux lettres, alias de messagerie, bases de données et autres paramètres associés au domaine d'origine seront dupliqués, dans la mesure du possible. Cependant, le contenu Web ne sera modifié d'aucune façon, donc si le serveur virtuel contient une application Web qui a été configurée pour utiliser les bases de données ou le répertoire personnel de l'ancien serveur, elle devra être reconfigurée manuellement après le clonage.

Lors de la duplication d'un serveur virtuel de niveau supérieur, vous devez également saisir un nouveau nom d'utilisateur et un nouveau mot de passe pour posséder le nouveau domaine. Si le serveur possède une adresse IP privée, une nouvelle IP sera automatiquement attribuée au clone.

Le processus de clonage copiera toutes les bases de données associées au domaine d'origine sous de nouveaux noms et dupliquera leur contenu. Cependant, il se peut qu'il ne mette pas à jour toutes les applications Web installées dans le domaine d'origine et configurées pour utiliser les bases de données d'origine. Cela doit être fait manuellement.
