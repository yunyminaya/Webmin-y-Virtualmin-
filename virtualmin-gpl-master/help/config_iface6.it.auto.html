Interfaccia di rete per indirizzi IPv6
Quando Virtualmin aggiunge un indirizzo IPv6 per un server virtuale, verrà aggiunto all'interfaccia immessa qui (come eth1 ). Se non viene fornito alcun nome di interfaccia, verrà utilizzata l'interfaccia IPv4 rilevata automaticamente. Questa è in genere la prima interfaccia Ethernet sul sistema.
