Impostazione del sito Web SSL abilitata?
Se abilitato, verrà creato un sito Web SSL per ogni nuovo dominio. Se impostato su "Sì, ma non per impostazione predefinita" sarà possibile scegliere di creare un sito Web SSL al momento della creazione del dominio. Si noti che i siti Web SSL richiedono un IP tutto per sé, quindi non è abilitato per tutti i siti per impostazione predefinita. Molti siti Web non avranno bisogno della sicurezza SSL e quindi possono condividere un singolo IP con centinaia o migliaia di altri siti Web.
