Standardmäßig verfügbare Funktionen
Wenn diese Option auf Automatisch eingestellt ist (wie standardmäßig), werden für neue virtuelle Server der obersten Ebene die zulässigen Funktionen basierend auf den Funktionen festgelegt, die beim Erstellen des Servers ursprünglich aktiviert wurden. Wenn Sie dem Serverbesitzer standardmäßig Zugriff auf zusätzliche Funktionen gewähren möchten, aktivieren Sie das Optionsfeld Ausgewählt unten und aktivieren Sie die Kontrollkästchen für die Funktionen, die Sie gewähren möchten.
