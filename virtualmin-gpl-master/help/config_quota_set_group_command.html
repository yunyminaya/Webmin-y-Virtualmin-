Command to set a group's quota
When external quota commands are being used, the program set in this option will be called when Virtualmin wants to set the quotas for a Unix group. The parameters it is called with are :
  1. The Unix group name
  2. The new soft quota in 1 kB blocks, or 0 for unlimited.
  3. The new hard quota in 1 kB blocks, or 0 for unlimited.
