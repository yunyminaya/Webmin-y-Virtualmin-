Enregistrements d'adresses pour les nouveaux domaines
Ce champ vous permet de sélectionner les enregistrements DNS A (adresse) que Virtualmin ajoutera aux nouveaux domaines. La configuration par défaut dans laquelle ils sont tous ajoutés fonctionnera bien dans tous les cas, mais vous pouvez désélectionner certains enregistrements s'ils ne sont pas nécessaires ou en conflit avec des sous-domaines.
