Скопировать SSL-сертификаты для каждого IP в ProFTPd?
Если этот параметр включен и создается виртуальный сервер с включенным SSL и собственным частным IP-адресом, ProFTPd будет настроен на использование сертификата SSL сервера для подключений к этому IP-адресу при использовании расширения протокола TLS FTP.
