Autorisations sur l'annuaire du site Web
Cette option spécifie les autorisations initiales sur les fichiers et répertoires du répertoire du site Web spécifié dans l'option précédente. Si suEXEC a été activé et que l'utilisateur Apache est ajouté au groupe d'utilisateurs du serveur virtuel, ces autorisations peuvent être renforcées pour empêcher les autres utilisateurs de voir le contenu du répertoire du site Web. Cet ensemble renforcé d'autorisations est recommandé pour tout déploiement, à moins que tous les utilisateurs ayant accès au shell soient des utilisateurs de confiance. Les autorisations recommandées sont 750 , ce qui équivaut à rwxr-x--- . En d’autres termes, le propriétaire peut lire, écrire et exécuter, le groupe peut lire et exécuter, et les autres n’ont aucune autorisation pour lire, écrire ou exécuter des fichiers dans le répertoire.
