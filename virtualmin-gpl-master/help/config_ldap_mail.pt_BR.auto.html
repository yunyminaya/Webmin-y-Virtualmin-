Adicionar atributo de email aos usuários LDAP?
Ao armazenar usuários do Virtualmin no LDAP, essa configuração também permite que seus endereços de email sejam adicionados ao banco de dados LDAP. Isso é útil se o seu servidor de email ou filtro de spam usar o mesmo banco de dados para enviar email.

As opções disponíveis são:

  • Sim usando apenas o atributo mail
    Os endereços de email principal e extra de cada usuário serão armazenados neste atributo como valores separados.

  • Sim, usando mail e mailAlternateAddress
    Os endereços principais dos usuários serão armazenados no atributo mail e endereços adicionais em mailAlternateAddress .

  • Não
    Os endereços de email nunca serão armazenados no LDAP.

Quando definido como Sim , o Webmin adiciona atributos de mail LDAP a cada usuário que ele cria ou modifica para seus endereços de email principais e adicionais. Isso é útil se você estiver usando o mesmo banco de dados LDAP com um servidor de correio que usa esse atributo para descobrir onde entregar o email.
