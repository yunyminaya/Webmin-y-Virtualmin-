Website passt zu allen Subdomains?
Wenn diese Option aktiviert ist, geben Anforderungen für eine URL unter der Domain (wie http://abc.example.com ) die Website des virtuellen Servers zurück. Dies kann nützlich sein, wenn Sie eine Anwendung hosten, die basierend auf dem angeforderten Domänennamen unterschiedliche Inhalte zurückgeben kann, z. B. Wordpress MU.

Wenn die DNS-Domäne des virtuellen Servers auf diesem System gehostet wird, fügt Virtualmin auch den Platzhalter * DNS-Eintrag hinzu, wenn Ja ausgewählt ist.
