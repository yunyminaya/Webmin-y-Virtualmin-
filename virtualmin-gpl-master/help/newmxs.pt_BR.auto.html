Servidores de Email Secundários
Um servidor de e-mail secundário é aquele que também pode receber e-mail para um dos servidores virtuais gerenciados pelo Virtualmin, mas apenas como uma retransmissão temporária e não como destino final. É aconselhável configurar um servidor secundário para domínios de e-mail muito usados, para atuar como um backup ou local de espera caso o servidor de e-mail principal esteja inoperante.

Nesta página, você pode selecionar outros servidores com o Virtualmin instalado para serem usados como servidores MX secundários para novos domínios de correio criados neste sistema. Antes que quaisquer servidores apareçam, eles devem primeiro ser adicionados ao módulo Webmin Servers Index, no qual você deve inserir o login root e a senha do servidor remoto.

Para cada servidor de correio secundário, você tem a opção de inserir um nome de host diferente para usar no registro DNS MX, em vez do nome de host real do servidor, como secmx.yourdomain.com . No entanto, certifique-se de que o que você digitar seja um nome DNS válido que resolva para o mesmo servidor.

Se o botão Salvar e aplicar for clicado, todos os servidores virtuais existentes para os quais o e-mail está habilitado serão adicionados (ou removidos) de todos os servidores de e-mail secundários selecionados. Além disso, os registros DNS MX serão adicionados (ou removidos) de seus domínios para se referir aos servidores de correio.
