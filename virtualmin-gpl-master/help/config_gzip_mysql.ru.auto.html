Сжатие резервных копий MySQL?
Если для этого параметра установлено значение «Да» , дампы базы данных MySQL будут сжаты gzip до того, как Virtualmin создаст окончательный файл резервной копии, который также сам может быть снова сжат. Это экономит место, когда конечная резервная копия представляет собой только файл TAR, но увеличивает общее время резервного копирования.
