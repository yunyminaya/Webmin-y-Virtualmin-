Включенные модули плагинов
Поле множественного выбора позволяет вам выбрать, какие модули будут включены для виртуальных серверов Virtualmin. После включения вы можете разрешить использование для каждого домена. Выбранные плагины подсвечиваются. Чтобы выбрать несколько плагинов, удерживайте клавишу CTRL при выборе дополнительных плагинов.
