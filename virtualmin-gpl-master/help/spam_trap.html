Create spamtrap and hamtrap email aliases?
When this option is set to Yes, Virtualmin will create mail aliases named spamtrap and hamtrap for this domain. Users can then forward spam that gets through the SpamAssassin mail filter to the spamtrap address, which will then be used to train SpamAssassin's learning engine. Similarly, email that is incorrectly classified as spam can be forwarded to the hamtrap address.
