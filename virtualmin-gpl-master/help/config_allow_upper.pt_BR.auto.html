Forçar nomes de usuário da caixa de correio para minúsculas?
Quando esse campo é definido como Sim (como é o padrão), todos os usuários de email / FTP terão seus nomes de usuário forçados a minúsculas, independentemente do caso em que são criados. Se você deseja impedir esse comportamento, selecione Não para ter caixas de correio criadas com o caso exato digitado. Esteja ciente de que, para efetuar login no Usermin e em qualquer servidor POP3, IMAP ou FTP, o caso correto deve ser usado.

Como a maioria dos servidores de correio não pode administrar a entrega a usuários com nomes não minúsculos, essa opção sempre deve ser definida como Sim, a menos que você esteja criando contas para uso de FTP, DAV ou SVN.
