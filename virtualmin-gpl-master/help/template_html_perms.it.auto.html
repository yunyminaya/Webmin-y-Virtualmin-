Autorizzazioni sulla directory del sito web
Questa opzione specifica le autorizzazioni iniziali sui file e sulle directory all'interno della directory del sito Web specificata nell'opzione precedente. Se suEXEC è stato abilitato e l'utente Apache viene aggiunto al gruppo utenti del server virtuale, queste autorizzazioni possono essere limitate per impedire ad altri utenti di vedere il contenuto della directory del sito web. Questo insieme ristretto di autorizzazioni è consigliato per qualsiasi distribuzione, a meno che tutti gli utenti con accesso alla shell non siano utenti fidati. Le autorizzazioni consigliate sono 750 , che equivale a rwxr-x--- . In altre parole, il proprietario può leggere, scrivere ed eseguire, il gruppo può leggere ed eseguire e gli altri non hanno il permesso di leggere, scrivere o eseguire file all'interno della directory.
