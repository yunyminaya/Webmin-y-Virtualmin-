Rozsahy pre automatické prideľovanie IP
Ak chcete, aby Virtualmin automaticky pridelil IP adresu pre nové virtuálne servery vytvorené z tejto šablóny, môžete zadať jeden alebo viac rozsahov a vybrať IP adresy odtiaľto. V opačnom prípade musíte pri vytváraní domény manuálne zadať akúkoľvek súkromnú virtuálnu IP.
