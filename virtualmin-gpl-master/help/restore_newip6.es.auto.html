Dirección IPv6 para servidores restaurados
Esta configuración solo se usa cuando se restaura un servidor virtual que actualmente no existe en el sistema. De manera predeterminada, Virtualmin le asignará la dirección IPv6 compartida predeterminada del sistema si originalmente estaba en una IP compartida, o usará la dirección IPv6 privada del sistema original si no estaba en una dirección compartida.

Esto funciona bien si está moviendo un dominio entre dos sistemas en la misma red, pero si no, es probable que el dominio restaurado no sea contactable a través de la red. Para evitar esto, seleccione la opción Virtual con IP asignada o Virtual con IP en este campo, y Virtualmin seleccionará una IP libre para usted o usará la que ingrese.
