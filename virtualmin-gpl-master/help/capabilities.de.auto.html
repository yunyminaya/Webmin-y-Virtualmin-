Kann virtuellen Server bearbeiten
Ermöglicht das Bearbeiten der Funktionen des virtuellen Servers, seines Kennworts und seiner Beschreibung.
Kann Benutzer verwalten
Ermöglicht das Anzeigen, Bearbeiten, Erstellen und Löschen von Postfach-, FTP- und Datenbankbenutzern.
Kann Aliase verwalten
Ermöglicht das Anzeigen, Bearbeiten, Erstellen und Löschen von E-Mail-Aliasen.
Kann Datenbanken verwalten
Ermöglicht das Hinzufügen oder Löschen zusätzlicher MySQL- und PostgreSQL-Datenbanken zu diesem virtuellen Server.
Kann Webanwendungen installieren
Ermöglicht die Installation von Webanwendungen von Drittanbietern auf diesem virtuellen Server.
Kann die IP-Adresse ändern
Wenn dieser Server eine private IP-Adresse hat, kann diese mit dieser Option geändert werden.
Kann SSL-Zertifikate verwalten
Ermöglicht das Hochladen oder Neugenerieren des SSL-Zertifikats und -Schlüssels für diesen Server.
Kann Weiterleitungsrahmen bearbeiten
Bei Nur-Weiterleitungs-Websites ermöglicht diese Option die Bearbeitung des Ziels und des HTML für den Weiterleitungsrahmen.
Kann zusätzliche Administratoren verwalten
Ermöglicht das Anzeigen, Erstellen, Bearbeiten und Löschen zusätzlicher Administratorkonten des Serverbesitzers.
Kann die Übermittlung von Spam und Viren konfigurieren
Ermöglicht die Bearbeitung von Spam- und Virenzustellungszielen und -bewertungen.
Kann PHP-Versionen auswählen
Für Domänen mit PHP, das über CGI oder FastCGI ausgeführt wird, können PHP-Versionen pro Verzeichnis festgelegt werden.
Kann E-Mail-Einstellungen bearbeiten
Ermöglicht die Konfiguration von automatischem BCC, Alias-Kopiermodus und anderen E-Mail-bezogenen Optionen.
Kann Backups erstellen
Ermöglicht die Sicherung eigener virtueller Server auf einem Remote-FTP- oder SSH-Server.
Kann Backups planen
Ermöglicht die regelmäßige Sicherung virtueller Server.
Kann Backups wiederherstellen
Ermöglicht die Wiederherstellung des Home-Verzeichnisses und des Datenbankinhalts eigener virtueller Server.
Kann gemeinsame IPs auswählen
Ermöglicht die Auswahl global definierter gemeinsam genutzter IP-Adressen beim Erstellen virtueller Server.
Kann Catchall-Aliase erstellen
Ermöglicht die Erstellung von E-Mail-Aliasen, die alle E-Mails an eine Domäne an eine andere Domäne oder Adresse weiterleiten.
Kann den Webseiten-Editor verwenden
Ermöglicht die Bearbeitung von Webseiten über die Virtualmin-Benutzeroberfläche.
Kann Datenbank-Remote-Hosts bearbeiten
Ermöglicht die Eingabe von Remote-Client-Systemen, die eine Verbindung zu den MySQL-Datenbanken der Domäne herstellen dürfen.
Kann virtuelle Server deaktivieren
Ermöglicht die vorübergehende Deaktivierung von Servern.
Kann virtuelle Server löschen
Ermöglicht die vollständige Löschung eigener Server, einschließlich aller Subserver und Aliase.
