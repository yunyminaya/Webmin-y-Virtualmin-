Pfad zum Erstellen des API-Hilfsbefehls
Standardmäßig erstellt Virtualmin das virtualmin Befehlszeilen-API von virtualmin im virtualmin /usr/sbin oder /usr/local/sbin , je nachdem, welches auf Ihrem System beschreibbar ist. In diesem Konfigurationsfeld können Sie jedoch einen anderen Speicherort auswählen, z. B. /opt/bin/virtualmin , der für Ihre Site geeignet ist.
