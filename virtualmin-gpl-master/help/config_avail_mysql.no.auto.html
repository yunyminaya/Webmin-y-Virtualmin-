MySQL Database Server (for database)
Hvis dette er aktivert, vil domeneeierkontoen ha tilgang til MySQL Webmin-modulen. Det gir en vennlig GUI til databasene som tilhører domeneeieren. Med denne modulen, og passende rettigheter, kan domeneeieren opprette nye databaser, redigere eksisterende databaser eller utføre spørsmål om dataene i databasen.
