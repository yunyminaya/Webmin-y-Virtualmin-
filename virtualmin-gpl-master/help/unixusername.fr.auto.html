Nom d'utilisateur d'administration
Il s'agit du nom d'utilisateur du propriétaire du domaine. Si vous le laissez défini sur Automatique, le nom sera le même que la première partie du nom de domaine ou le domaine entier. Alternativement, vous pouvez choisir d'entrer un nom d'utilisateur personnalisé de votre choix.

Cette connexion pourra généralement se connecter via SSH et FTP et gérer le serveur virtuel via l'interface Web. Le mot de passe pour tous ces services sera celui qui a été entré dans le champ Mot de passe d'administration . Une adresse e-mail avec ce nom d'utilisateur sera également créée sur le serveur virtuel si la messagerie est activée.
