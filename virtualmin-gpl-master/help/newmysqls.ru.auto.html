MySQL Servers
Эта страница позволяет вам определить дополнительные серверы MySQL на удаленных системах или дополнительные установки в системе для использования доменами Virtualmin. Это можно использовать для разгрузки хостинга MySQL на другой сервер или для поддержки нескольких версий MySQL для разных клиентов.

Чтобы добавить новый удаленный сервер MySQL, используйте для внизу страницы. Чтобы удалить ранее определенный сервер из списка, установите флажок рядом с ним в списке и нажмите кнопку «Удалить выбранное» .
