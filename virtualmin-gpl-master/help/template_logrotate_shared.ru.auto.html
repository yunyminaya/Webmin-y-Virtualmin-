Поделиться конфигурацией logrotate со всеми доменами?
Если для этого параметра установлено значение Нет , Virtualmin создаст отдельный блок в файле конфигурации logrotate для каждого виртуального сервера. Это позволяет каждому иметь разные настройки, но приведет к перезапуску веб-сервера после ротации журналов каждого домена - возможно, к прерыванию службы.

Выбор Да приведет к тому, что все новые домены будут использовать один и тот же блок конфигурации, что означает, что веб-сервер должен быть перезапущен только один раз. Однако невозможно установить разные параметры ротации для каждого домена.
