Menedżer plików (tylko katalog domowy)
File Manager to zaawansowany aplet Java, który oferuje dwupanelowy widok systemu plików serwera. Jest to jedna z najpotężniejszych funkcji Webmin, umożliwiająca nawet niedoświadczonym użytkownikom zarządzanie plikami, katalogami i uprawnieniami, edycję plików tekstowych, przesyłanie i pobieranie plików i wiele więcej. Domyślnie właściciel domeny będzie mógł uzyskać dostęp tylko do własnego katalogu i jego podkatalogów.
