Geplante Sicherungen
Mit Virtualmin können Sie die Sicherung einiger aller virtuellen Server in lokalen Dateien oder Remote-SSH- oder FTP-Servern planen, die zu den von Ihnen gewählten Zeiten ausgeführt werden sollen. Auf dieser Seite werden alle Sicherungen aufgelistet, für deren Verwaltung Sie berechtigt sind, und Sie können neue Sicherungen erstellen.

Für jedes angezeigte Backup können Sie die Einstellungen bearbeiten, indem Sie auf den Link in der Spalte Ziel klicken, oder ihn entfernen, indem Sie das Kontrollkästchen links aktivieren und auf die Schaltfläche Löschen klicken.

Um jetzt eine geplante Sicherung auszuführen, klicken Sie rechts auf den Link Sichern .. , um das Sicherungsformular mit den ausgefüllten Details aufzurufen. Oder klicken Sie auf den Link Wiederherstellen .. , um die zuletzt nach diesem Zeitplan erstellte Sicherung wiederherzustellen. In beiden Fällen haben Sie die Möglichkeit, zu bestätigen, was zu tun ist, bevor es weitergeht.

Sie sollten nicht mehrere Sicherungen für dieselben Domänen planen, die gleichzeitig ausgeführt werden, da diese sich gegenseitig stören oder in einer inkonsistenten Reihenfolge ausgeführt werden können.
