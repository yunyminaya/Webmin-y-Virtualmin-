Percorso per creare il comando helper API
Per impostazione predefinita, Virtualmin creerà lo script helper dell'API della riga di comando virtualmin nella /usr/sbin o /usr/local/sbin , a seconda di quale sia scrivibile sul sistema. Tuttavia, questo campo di configurazione può essere utilizzato per selezionare una posizione diversa, come /opt/bin/virtualmin che è appropriato per il tuo sito.
