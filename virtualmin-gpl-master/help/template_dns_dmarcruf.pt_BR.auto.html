URI de relatórios para relatórios forenses
Por padrão, o registro DNS do DMARC para novos domínios usará mailto:postmaster@domain.com como o endereço para os destinatários de spam reportarem mensagens pessoais. No entanto, você pode inserir um novo URI aqui para personalizar o endereço de email, por exemplo, para usar um endereço comum em todos os domínios.
