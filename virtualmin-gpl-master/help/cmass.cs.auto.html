Dávkové vytvoření serverů
Tato stránka umožňuje vytvořit více virtuálních serverů najednou z textového souboru. Tento soubor musí obsahovat jeden server na řádek a každý řádek musí mít následující formát:

domain : description : password :[ username ]:[ parent-domain ]:[ ip-address ]:[ alias-domain ]

Význam těchto polí je:

doména
Název domény, kterou má virtuální server vytvořit, například foo.com .
popis
Popis čitelný člověkem, jako je web Joe's Website .
Heslo
Heslo pro přihlášení k administraci serveru. To je nutné pouze pro nadřazené servery.
uživatelské jméno
Uživatelské jméno pro správu serveru. Pokud je toto pole prázdné, uživatelské jméno je vybráno automaticky.
rodičovská doména
Název domény nadřazeného serveru, pod nímž se má tento server vytvořit. Pokud je toto ponecháno prázdné, vytvoří se nový server jako samotný nadřazený server.
IP adresa
U virtuálních serverů, které nepotřebují vlastní privátní IP, může být toto pole ponecháno prázdné. V opačném případě musíte zadat platnou nepoužitou adresu nebo slovo allocate aby vám Virtualmin vybral jednu (pokud je nakonfigurována).
alias-doména
Název domény virtuálního serveru, pro který by se toto mělo stát aliasem. Pokud bude vynechán, bude nový server buď na nejvyšší úrovni, nebo na podserver.

Kvóty, šířka pásma a další limity pro vytvořené servery se nastavují na základě šablony, kterou vyberete v tomto formuláři. Funkce vytvořených serverů jsou určeny zaškrtávacími políčky funkcí, stejně jako na normální stránce pro vytváření jednoho serveru.

Pokud váš systém používá účty distributora, můžete si vybrat, aby všechny virtuální servery byly vytvořeny ve vlastnictví distributora. Tato možnost je k dispozici pouze hlavnímu administrátorovi.
