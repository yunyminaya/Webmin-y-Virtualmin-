Unterverzeichnis der Website-Dokumente
In diesem Feld können Sie das Unterverzeichnis eines virtuellen Servers, das Webinhalte enthält, von der Standardeinstellung public_html ändern. Dies kann nützlich sein, wenn Sie Inhalte oder Skripte in einem Verzeichnis unter public_html und diese für Benutzer sichtbar machen möchten, die auf die URL Ihrer Website auf oberster Ebene zugreifen.
