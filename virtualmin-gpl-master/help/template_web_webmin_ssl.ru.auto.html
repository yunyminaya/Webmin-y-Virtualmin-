Настроить Webmin для использования того же SSL-сертификата?
Если этот шаблон будет использоваться исключительно для сайтов с поддержкой SSL, вы можете включить эту опцию. Затем Webmin будет использовать тот же сертификат, что и веб-сайт SSL, поэтому он не будет генерировать предупреждения сертификата, когда пользователи входят в систему, используя свое собственное имя хоста.
