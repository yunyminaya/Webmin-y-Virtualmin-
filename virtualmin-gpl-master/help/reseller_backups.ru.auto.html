Можно ли создавать резервные копии виртуальных серверов?
Это поле определяет, разрешено ли посреднику использовать функции резервного копирования Virtualmin. Доступные варианты:

Расписание и делать резервные копии
Торговый посредник может как автоматически планировать резервные копии, которые регулярно запускаются, так и использовать веб-интерфейс для запуска резервных копий вручную.
Делать только резервные копии
Резервные копии можно запускать только вручную с помощью веб-интерфейса.
ни
Реселлер не имеет доступа к функциям резервного копирования.
