Netzwerkschnittstelle für virtuelle Adressen
Wenn Ihr Server über mehrere Schnittstellen verfügt, können Sie auswählen, welche Schnittstelle für Ihre virtuellen Domänen verwendet werden soll. Wenn virtuelle SSL- oder FTP-Hosts aktiviert sind, ist für jede Domäne, für die die Funktion aktiviert ist, eine neue IP erforderlich. Sofern bei der Domänenerstellung nicht anders konfiguriert, werden die neuen Adressen auf der hier angegebenen Schnittstelle erstellt.
