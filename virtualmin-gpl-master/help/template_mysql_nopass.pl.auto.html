Hasło MySQL dla nowych domen
Ta opcja kontroluje początkowe hasło MySQL dla nowych serwerów wirtualnych. Dostępne są następujące opcje:
Synchronizuj z loginem administratora
Hasło do MySQL jest zawsze takie samo jak login administratora serwera wirtualnego.

To samo, co początkowy login administratora
Hasło początkowo jest takie samo jak login administratora, ale nie jest modyfikowane, jeśli hasło administracyjne zostanie zmienione.

Generuj losowo
Podczas tworzenia serwera wirtualnego losowo generowane hasło MySQL jest wybierane i nie zmieniane.
