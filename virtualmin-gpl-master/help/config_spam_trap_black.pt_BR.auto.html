Adicionar remetentes de spam enviados para spamtrap à lista negra?
Por padrão, quando o alias de spamtrap está ativado para um servidor virtual, qualquer email enviado a ele será adicionado ao banco de dados de aprendizado do SpamAssassin para o usuário que o encaminhou. No entanto, esta opção pode ser usada para adicionar também os remetentes originais de spam às listas negras do sistema ou do domínio.

Isso pode ser útil se você receber muito spam enviado do mesmo endereço. No entanto, como os spammers costumam variar ou falsificar o endereço de origem no email, a lista negra deles normalmente não é muito eficaz.
