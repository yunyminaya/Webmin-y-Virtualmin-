Показать базы данных в списке пользователей?
Если для этого параметра установлено значение «Да» , количество баз данных, к которым имеет доступ каждый пользователь, будет отображаться в отдельном столбце на странице «Редактировать пользователей» . Эта функция обеспечивает быстрый обзор доступности базы данных для каждого пользователя, дополняя информацию, уже имеющуюся в столбце «Доступ для входа». Если выбрать «Нет» , поле «Базы данных?» будет скрыто. столбец из списка пользователей.
