Compartilhar a configuração do logrotate com todos os domínios?
Quando isso estiver definido como Não , o Virtualmin criará um bloco separado no arquivo de configuração de rotação de log para cada servidor virtual. Isso permite que cada um tenha configurações diferentes, mas fará com que o servidor da Web seja reiniciado após a rotação dos logs de cada domínio - possivelmente interrompendo o serviço.

Selecionar Sim fará com que todos os novos domínios compartilhem um único bloco de configuração, o que significa que o servidor da Web precisa ser reiniciado apenas uma vez. No entanto, não é possível ter configurações de rotação diferentes para cada domínio.
