Путь для создания вспомогательной команды API
По умолчанию Virtualmin создает вспомогательный сценарий API командной строки virtualmin в virtualmin /usr/sbin или /usr/local/sbin , в зависимости от того, что доступно для записи в вашей системе. Однако это поле конфигурации можно использовать для выбора другого местоположения, например /opt/bin/virtualmin , которое подходит для вашего сайта.
