Instaladores de Webapp de modelo
Esta página permite que você especifique webapps de terceiros a serem instalados quando um novo servidor virtual for criado usando este modelo. Para cada webapp, as seguintes informações devem ser inseridas:

Aplicativo web
O nome e a versão de um webapp a ser instalado
Caminho do URL
Um caminho no site do servidor virtual para instalar o webapp, como /wordpress . Isso também determinará o diretório em ~/public_html que contém os arquivos do webapp.
Nome do banco de dados
O nome do banco de dados que o webapp usará para armazenar dados. As opções são:
Nenhum
Isto deve ser selecionado para webapps que não precisam de um banco de dados, como Gallery .
Banco de dados principal
O banco de dados do webapp será o banco de dados primário do servidor virtual do tipo escolhido.
Outro
Se você deseja que este webapp use seu próprio banco de dados privado, esta opção pode ser selecionada e o padrão para um banco de dados inserido no campo de texto adjacente. Por exemplo, você pode inserir ${DB}_phpnuke . Se o banco de dados resultante não existir, ele será criado como parte do processo de instalação do webapp.
Tipo de banco de dados
O tipo de banco de dados usado para os dados de aplicativos da web, se houver algum selecionado.
