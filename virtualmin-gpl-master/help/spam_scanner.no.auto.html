Virussøkingsprogram
De mulige alternativene for dette feltet er:
Frittstående skanner ( muslinger )
Dette er vanligvis standard, og vil fortelle Virtualmin å bruke ClamAV-skanneprogrammet som gjør alt arbeidet av seg selv. Dette vil sikkert fungere, men kan være tregere enn neste alternativ.
Server skanner ( clamdscan )
Dette alternativet kan velges hvis skanningsserveren for clamd- virus kjører på systemet ditt. Den er generelt overlegen, siden ClamAV-virusdatabasen er ganske stor, og serveren trenger bare å laste den inn i minnet en gang.
Ekstern server skanner ( clamd-stream-client )
Dette alternativet kan velges hvis skanningsserveren for clamd- virus kjører på et eksternt system, og er konfigurert til å godta nettverkstilkoblinger fra andre verter. Det er nyttig når du skal laste ned oppgaven med å skanne til et annet system enn det som godtar e-post.
Annet program
Dette alternativet lar deg fortelle Virtualmin om å bruke et helt annet program, for eksempel ditt eget innpakningsskript for ClamAV.
