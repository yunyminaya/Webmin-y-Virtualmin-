Incluir domínios de alias nos limites?
Domínios alias são domínios que apontam para um domínio existente para alguns ou todos os serviços. Se você deseja considerar domínios alias como parte do domínio alias para fins de cota, selecione sim aqui. Se os limites da conta forem especificados separadamente, selecione não. Os domínios de alias são normalmente muito pequenos, pois apenas direcionam todo o tráfego para o domínio de que são aliases, mas com privilégios apropriados, o proprietário de um domínio pode usar a conta de domínio de alias para outros fins.
