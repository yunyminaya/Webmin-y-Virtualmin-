De specifieke Virtualmin-actie die is uitgevoerd, is te vinden in de variabele VIRTUALSERVER_ACTION, zoals:

CREATE_DOMAIN
Er wordt een nieuwe virtuele server gemaakt.
VERWIJDEREN_DOMAIN
Er wordt een bestaande virtuele server verwijderd.
DISABLE_DOMAIN
Een virtuele server wordt tijdelijk uitgeschakeld.
ENABLE_DOMAIN
Een virtuele server wordt opnieuw ingeschakeld.
WIJZIG_DOMEIN
Een virtuele server wordt bijgewerkt, bijvoorbeeld wanneer functies worden toegevoegd of verwijderd, of wanneer het eigendom wordt gewijzigd.
CLONE_DOMAIN
