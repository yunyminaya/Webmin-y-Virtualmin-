Web Sunucusu Kullanıcısı Oluştur
Özellikle korumalı web dizinlerinde güvenli kimlik doğrulaması için htpasswd dosyaları içinde yeni bir web sunucusu kullanıcısı oluşturur. Bu kullanıcı, yalnızca web dizinlerine erişimi yönetmeye adanmış Unix hesaplarından farklıdır. Ayrıca bu kullanıcı, aynı kullanıcı adı kullanılarak oluşturulan bir Unix hesabıyla birleştirilerek sorunsuz bir şekilde entegre edilebilir.
