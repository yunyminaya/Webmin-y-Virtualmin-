¿Agregar usuarios sin acceso SSH al grupo denegadosh?
De manera predeterminada, Virtualmin configurará el servidor SSH para evitar que los usuarios inicien sesión cuyos shells no permitirían el acceso SSH de todos modos. Esto se hace al llenar el grupo denegadoshs y agregarlo a la configuración de OpenSSH.

Si desea deshabilitar este comportamiento, tal vez porque no es necesario o hace que la creación del dominio tarde demasiado, cambie este campo a No.
