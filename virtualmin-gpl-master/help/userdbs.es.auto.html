Permitir acceso a bases de datos
Este campo le permite seleccionar las bases de datos propiedad de este servidor virtual a las que el usuario también puede acceder. Para agregar una base de datos, selecciónela de la lista de la izquierda y haga clic en la flecha ->.

El nombre de usuario y la contraseña serán los mismos que utilizaría para FTP o IMAP. De manera predeterminada, se le concederá acceso total a todas las tablas de las bases de datos seleccionadas.

MySQL solo admite nombres de usuario de hasta 16 caracteres, por lo que para un usuario con un nombre de cuenta largo solo se necesitan los primeros 16 caracteres al iniciar sesión en la base de datos.
