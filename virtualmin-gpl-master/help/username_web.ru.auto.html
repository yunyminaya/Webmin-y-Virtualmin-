Имя пользователя
В этом поле задается часть имени пользователя для входа в систему, которая должна быть уникальной в списке пользователей веб-сервера. Логин пользователя для доступа к веб-серверу будет получен из введенного здесь имени пользователя в сочетании с именем домена. Например, если вы введете имя пользователя web-user1 и домен example.com , полученный логин будет web-user1@example.com .
