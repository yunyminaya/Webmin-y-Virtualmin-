Программа сканирования на вирусы
Возможные варианты для этого поля:
Автономный сканер ( clamscan )
Обычно это значение по умолчанию, и Virtualmin будет указывать использовать программу сканирования ClamAV, которая выполняет всю работу самостоятельно. Это обязательно сработает, но может быть медленнее, чем следующий вариант.
Серверный сканер ( clamdscan )
Эту опцию можно выбрать, если в вашей системе работает сервер антивирусного сканирования clamd . Как правило, он лучше, так как вирусная база ClamAV достаточно велика, и сервер должен загрузить ее в память только один раз.
Удаленный серверный сканер ( clamd-stream-client )
Эту опцию можно выбрать, если сервер антивирусного сканирования clamd работает в удаленной системе и настроен на прием сетевых подключений от других хостов. Это полезно для переноса задачи сканирования в другую систему, отличную от той, которая принимает электронную почту.
Другая программа
Эта опция позволяет вам указать Virtualmin использовать совершенно другую программу, например, собственный скрипт-обертку для ClamAV.
