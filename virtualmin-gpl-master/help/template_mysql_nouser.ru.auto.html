Синхронизировать имена пользователей MySQL и администрации?
По умолчанию Virtualmin меняет имя входа MySQL для домена при изменении имени администратора. Однако это вызовет проблемы для любых установленных приложений, которые уже были настроены со старым именем пользователя. Чтобы избежать этого, измените этот параметр на Нет .
