Pode fazer backup de servidores virtuais?
Este campo determina se o revendedor pode usar os recursos de backup do Virtualmin. As opções disponíveis são:

Agende e faça backups
O revendedor pode agendar backups executados regularmente regularmente e usar a interface da Web para executar backups manualmente.
Faça apenas backups
Os backups só podem ser executados manualmente usando a interface da web.
Nem
O revendedor não tem acesso às funções de backup.
