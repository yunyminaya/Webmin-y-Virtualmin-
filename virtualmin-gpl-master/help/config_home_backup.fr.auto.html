Répertoire de sauvegarde des utilisateurs sous home
Lorsqu'un propriétaire de domaine effectue une sauvegarde vers une destination sur le serveur, elle ne peut être enregistrée que dans le sous-répertoire sous le répertoire de base de l'utilisateur défini par cette option de configuration. Vous pouvez le changer si vous n'aimez pas la valeur par défaut de virtualmin-backup .
