Destinations d'e-mails de virus
Ce champ détermine ce qui se passe aux e-mails qui ont été identifiés par ClamAV comme contenant un virus. Les options disponibles sont :

Jeter
Les virus détectés seront immédiatement supprimés.
Écrire dans le fichier de spam standard ~/mail/virus
Les virus sont écrits dans le dossier standard pour l'analyse manuelle et suppression.
Écrire dans le fichier de courrier sous le répertoire d'accueil
Les virus sont enregistrées dans le fichier de courrier sous le répertoire personnel de l'utilisateur que vous entrez. Pour écrire à un répertoire Maildir, mettre un / à la fin du nom de fichier.
Transférer à l'adresse e-mail
Tous les messages chargés de virus seront transmis à l'adresse indiquée.
Ecrire à autre fichier
Les virus seront écrits dans le fichier de messagerie donné (ou un répertoire si le chemin se termine par /), qui peut être n'importe où sur le serveur.
