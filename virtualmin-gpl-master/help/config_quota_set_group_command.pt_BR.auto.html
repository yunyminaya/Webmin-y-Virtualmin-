Comando para definir a cota de um grupo
Quando comandos de cota externa estão sendo usados, o programa definido nesta opção será chamado quando o Virtualmin desejar definir as cotas para um grupo Unix. Os parâmetros com os quais é chamado são:
  1. O nome do grupo Unix
  2. A nova cota flexível em blocos de 1 kB ou 0 para ilimitado.
  3. A nova cota fixa em blocos de 1 kB ou 0 para ilimitado.
