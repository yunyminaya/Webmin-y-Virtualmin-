Actualización dinámica de IP
Esta página le permite configurar Virtualmin para detectar cambios en la interfaz de red primaria del sistema y actualizar una entrada de DNS dinámica cuando cambie. Esto es útil si está ejecutando Virtualmin en un sistema con una dirección que ocasionalmente cambia, como una que se ejecuta en un entorno de máquina virtual como el EC2 de Amazon. También es útil para servidores que están conectados a Internet a través de una conexión sin una dirección IP fija, como ADSL o Cable.

Para usar esta página, debe seleccionar un servicio DNS dinámico, ingresar el nombre de host utilizado por su sistema en ese servicio y también los detalles de inicio de sesión del servicio. De forma predeterminada, Virtualmin buscará la dirección IP asignada a la interfaz de red primaria (como eth0 ) para verificar los cambios, pero si su sistema está detrás de un enrutador NAT, debe seleccionar la dirección visible desde la opción de Internet para que descubra la IP que el resto de Internet realmente ve.

Cuando se detecta un cambio, todos los servidores virtuales con direcciones IP compartidas que usan la dirección anterior se actualizarán a la nueva. Este cambio se realizará tanto en la configuración de Apache como en los registros DNS.

Si se selecciona el servicio de script externo , debe ingresar la ruta completa a un script en el campo adyacente que se llamará cuando cambie la dirección IP del sistema. La nueva IP y el nombre de host se pasarán al script como parámetros de línea de comandos.
