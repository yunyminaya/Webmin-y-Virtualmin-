Trabalhos Cron agendados (trabalhos Cron do usuário)
Esta opção especifica se a conta do proprietário do domínio pode usar o módulo Cron Jobs Webmin. Um Cron Job é um evento agendado que pode ser usado para executar periodicamente alguma tarefa, como enviar um email, fazer backup ou limpar um banco de dados, verificar o status de um arquivo ou caixa de correio etc. O módulo Cron Jobs fornece e é fácil de usar interface com a sintaxe difícil de lembrar (e confusa para iniciantes) do arquivo crontab. Trabalhos Cron serão executados como o usuário proprietário do domínio.
