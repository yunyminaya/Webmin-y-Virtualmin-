Carica e scarica (come utente)
Questa opzione specifica che il proprietario del dominio avrà accesso al modulo Webmin di upload e download. Il modulo di upload e download fornisce agli utenti un meccanismo molto semplice per caricare o scaricare file da altri siti Web sul server, senza richiedere l'uso di DAV, FTP, SSH, FISH o qualsiasi altro protocollo di trasferimento file che potrebbe essere intimidatorio per nuovi utenti. È interamente un meccanismo basato sul Web per ottenere dati nel sistema.
