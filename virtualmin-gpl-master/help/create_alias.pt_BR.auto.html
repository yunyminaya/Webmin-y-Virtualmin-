Criar servidor de alias
Este formulário permite criar um novo servidor de alias, que é essencialmente um nome alternativo para um servidor existente. Os aliases não têm conteúdo próprio no site e compartilham todas as cotas e limites com o servidor de destino. A largura de banda usada pelo servidor de alias é contada no total do destino.

Ao criar o servidor, você deve inserir pelo menos o nome do domínio e uma descrição. Se vários modelos de servidor estiverem definidos no sistema, você poderá selecionar um no menu. Para selecionar quais recursos no servidor de alias são encaminhados para o domínio de destino, use as caixas de seleção na segunda parte da seção Opções avançadas . Somente DNS, site e e-mail estão disponíveis.
