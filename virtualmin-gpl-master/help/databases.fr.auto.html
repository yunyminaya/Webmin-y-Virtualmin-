Modifier les bases de données
Cette page répertorie toutes les bases de données actuellement détenues par le serveur virtuel et permet la création de bases de données supplémentaires jusqu'à toute limite en vigueur pour l'utilisateur connecté à Virtualmin. Vous pouvez également supprimer une base de données supplémentaire en les sélectionnant dans la liste et en appuyant sur le bouton Supprimer la sélection .

Pour afficher les tables et les données réelles d'une base de données, cliquez sur le lien Gérer en regard du nom de la base de données dans la table. Pour voir la taille de la base de données, cliquez sur son nom.

Lors de l'utilisation de Virtualmin Pro, deux formulaires supplémentaires s'affichent sous la liste de bases de données pour modifier le login et le mot de passe des utilisateurs MySQL et PostgreSQL utilisés par ce serveur virtuel. Cela peut être utile si vous souhaitez que la connexion à la base de données soit différente de la connexion à l'administration principale, peut-être pour des raisons de sécurité. Ceux-ci doivent être modifiés avec soin, car tous les scripts PHP (tels que ceux des installateurs d'applications Web) qui ont le nom d'utilisateur ou le mot de passe dans leurs fichiers de configuration ne seront pas mis à jour.
