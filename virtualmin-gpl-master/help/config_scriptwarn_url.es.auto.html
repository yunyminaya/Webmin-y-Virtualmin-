Virtualmin URL para usar en mensajes de correo electrónico
Cuando Virtualmin envía mensajes de correo electrónico al administrador del sistema, a menudo incluirá la URL de Virtualmin en el sistema desde el que se envió el mensaje. De manera predeterminada, esto se basa en el nombre de host del sistema, pero puede establecer una URL personalizada como https://yourdomain.com:10000 usando este campo.
