Benutzerdefinierte Links
Mit benutzerdefinierten Feldern kann der Administrator zusätzliche Links einfügen, die im linken Rahmen angezeigt werden, wenn ein virtueller Server ausgewählt wird. Links können durch Klicken auf ihren Titel in der Liste auf dieser Seite bearbeitet oder mit dem Link Hinzufügen erstellt werden .

Links können mithilfe der Pfeile in der Spalte Verschieben neu angeordnet werden, wenn Sie mehr als einen definiert haben.

Am Ende der Seite befindet sich ein Formular zum Eingeben oder Ändern von Kategorien für benutzerdefinierte Links. Diese werden als erweiterbare Abschnitte in den gerahmten Themen angezeigt, wobei die Links, die für jede Kategorie ausgewählt wurden, beim Öffnen unter der Abschnittsüberschrift angezeigt werden.
