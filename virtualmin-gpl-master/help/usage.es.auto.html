Uso del disco
Esta página muestra el uso del disco para un servidor virtual, dividido en cinco secciones con pestañas:
Visión general
Esta sección muestra la cuota de disco asignada al servidor virtual, el espacio utilizado actualmente por los archivos propiedad del servidor y el tamaño de las bases de datos propias.

Directorio de inicio
Muestra el espacio utilizado por los 10 directorios principales en la página de inicio del servidor. Para cada uno, el Tamaño de cuota es el tamaño de los archivos que pertenecen al grupo Unix del servidor, mientras que el Tamaño incluye los archivos que pertenecen a otros usuarios. Este último puede ser más grande para directorios como logs cuando tienen archivos propiedad de root . El directorio de homes que contiene usuarios de correo / FTP no está incluido, como se explica en la siguiente sección.

Por usuario
Esta sección muestra el espacio en disco utilizado por los 10 principales usuarios de buzones en este dominio y cualquier subservidor. El uso se basa en archivos propiedad del usuario de Unix para cada buzón.

Por sub-servidor
Muestra el espacio en disco utilizado por cada servidor secundario, en función de los archivos en sus directorios de inicio. También se incluye el espacio utilizado por cualquier usuario de buzón en esos servidores.

Por base de datos
Muestra el espacio en disco utilizado por cada base de datos MySQL o PostgreSQL en este servidor virtual o en cualquier servidor secundario. Solo se muestran las 10 principales bases de datos.
