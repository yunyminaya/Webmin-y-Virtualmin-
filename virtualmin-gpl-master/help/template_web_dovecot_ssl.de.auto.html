Per-IP-SSL-Zertifikate nach Dovecot kopieren?
Wenn diese Option aktiviert ist und ein virtueller Server mit aktiviertem SSL und eigener privater IP erstellt wird, wird Dovecot so konfiguriert, dass das SSL-Zertifikat des Servers für Verbindungen zu dieser IP-Adresse verwendet wird. Dies kann hilfreich sein, wenn E-Mail-Clients im SSL-Modus eine Verbindung zum Domänennamen herstellen und ein übereinstimmendes Zertifikat erwarten.
