Visualizza interfacce virtuali
Per impostazione predefinita (quando si seleziona Sì ), Virtualmin attiverà automaticamente le interfacce di rete private per i server che ne hanno bisogno. Se questa opzione è impostata su No , si supporrà invece che si desideri configurare manualmente queste interfacce, magari utilizzando i file di configurazione Xen o Zones anziché quelli standard per il proprio sistema operativo.
