Redirecciones para nuevos sitios web
Esta tabla le permite ingresar redireccionamientos de sitios web que se agregarán a todos los nuevos servidores virtuales cuando se creen. Para cada redireccionamiento, debe ingresar una ruta de origen como / o /phpadmin y una URL de destino o una ruta relativa al directorio public_html del dominio.

En ambos campos, se pueden usar variables de plantilla de servidor virtual como ${DOM} y ${USER}. También puede seleccionar si la redirección se aplica cuando se accede al sitio a través de HTTP, HTTPS o ambos.
