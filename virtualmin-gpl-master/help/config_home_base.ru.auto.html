База домашних каталогов
Это каталог верхнего уровня, в котором будут создаваться домашние каталоги. Если ваша установка Virtualmin совместно использует сервер со многими другими службами или пользователями стандартной системы, может быть полезно использовать что-то отличное от / home. Значение по умолчанию берется из модуля Webmin пользователей и групп, по умолчанию используется / home. Модуль «Пользователи и группы» допускает сложные расширения этого соглашения, такие как «/ home / v / virtualmin», которые могут упростить навигацию по каталогам, если у вас много доменов на одном сервере. Документация по модулю «Пользователи и группы» в любой из книг Webmin описывает эту функцию более подробно.
