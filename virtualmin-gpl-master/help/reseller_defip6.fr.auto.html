Adresse IPv4 pour les serveurs basés sur le nom
Cette option peut être utilisée pour que les serveurs virtuels créés pour ce revendeur utilisent une adresse IPv6 autre que celle par défaut du système, qui est généralement partagée par tous les sites Web virtuels basés sur le nom. Cela peut être utile pour les serveurs de «marque» créés par différents revendeurs, en faisant résoudre l'adresse IPv4 de chaque revendeur par un nom d'hôte différent.

L'adresse IP que vous entrez doit déjà être active sur le système, ce qui peut être fait dans le module de configuration réseau de Webmin. L'adresse IP ne sera pas utilisée pour les serveurs virtuels avec leurs propres adresses IP privées, car chacun obtient toujours une adresse distincte.
