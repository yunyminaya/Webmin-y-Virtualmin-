Interface réseau pour les adresses virtuelles
Si votre serveur possède plusieurs interfaces, vous pouvez choisir l'interface à utiliser pour vos domaines virtuels. Si les hôtes virtuels SSL ou FTP sont activés, une nouvelle IP sera requise pour chaque domaine sur lequel la fonctionnalité est activée. Sauf configuration contraire lors de la création du domaine, les nouvelles adresses seront créées sur l'interface spécifiée ici.
