SSH / Telnet Girişi
Bu seçenek, etki alanı sahibi kullanıcının SSH / Telnet Oturum Açma Webmin modülünü kullanıp kullanamayacağını belirler. Bu modül, SSH veya Telnet kullanarak sunucuya giriş yapan bir Java uygulamasıdır. Java uyumsuzlukları nedeniyle tüm tarayıcılarda düzgün çalışmaz. Bunu devre dışı bırakmanın etki alanı sahibi tarafından sunucuya SSH erişimini engellemediğini, yalnızca Webmin'in SSH / Telnet Oturum Açma modülüne erişimi engellediğini unutmayın.
