MySQL Database Server (для базы данных)
Если включено, учетная запись владельца домена будет иметь доступ к модулю MySQL Webmin. Он предоставляет дружественный графический интерфейс для баз данных, которые принадлежат владельцу домена. С этим модулем и соответствующими привилегиями владелец домена может создавать новые базы данных, редактировать существующие базы данных или выполнять запросы к данным в базе данных.
