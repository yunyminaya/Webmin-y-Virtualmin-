Начальная оболочка Unix
Это поле позволяет вам определить другую оболочку для владельцев виртуальных серверов, созданную из этого шаблона, вместо того, чтобы использовать глобальное значение по умолчанию, определенное на странице пользовательских оболочек . Только из оболочек, предназначенных для использования владельцами доменов на этой странице, можно выбирать.
