Programme d'analyse antivirus
Les options possibles pour ce champ sont:
Scanner autonome ( clamscan )
Il s'agit généralement de la valeur par défaut, et indiquera à Virtualmin d'utiliser le programme de numérisation ClamAV qui effectue tout le travail seul. Cela est certain de fonctionner, mais peut être plus lent que l'option suivante.
Scanner de serveur ( clamdscan )
Cette option peut être sélectionnée si le serveur d'analyse antivirus clamd est en cours d'exécution sur votre système. Il est généralement supérieur, car la base de données du virus ClamAV est assez volumineuse et le serveur n'a besoin de la charger en mémoire qu'une seule fois.
Scanner de serveur distant ( clamd-stream-client )
Cette option peut être sélectionnée si le serveur d’analyse de virus Clamd s’exécute sur un système distant et a été configuré pour accepter les connexions réseau d’autres hôtes. Il est utile pour décharger la tâche de numérisation vers un système différent de celui qui accepte les e-mails.
Autre programme
Cette option vous permet de dire à Virtualmin d'utiliser un programme complètement différent, tel que votre propre script wrapper pour ClamAV.
