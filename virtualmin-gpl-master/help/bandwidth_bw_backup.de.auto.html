Backup-Übertragungen in die Nutzung einbeziehen?
Standardmäßig enthält Virtualmin die von FTP-, SSH- und S3-Sicherungen für einen virtuellen Server verbrauchte Bandbreite in seiner Gesamtnutzung. Wenn Sie diese Option auf Nein ändern, können Sie verhindern, dass Sicherungsdatenverkehr eingeschlossen wird. Dies kann sinnvoll sein, wenn alle Sicherungen auf einem Speicherserver in Ihrem lokalen Netzwerk durchgeführt werden.
