Standaard website voor het IP adres?
Wanneer dit veld is ingesteld op Ja word deze virtuele server website afgehandeld door Apache wanneer er een aanvraag komt die niet overeenkomt met een andere virtuele server op het systeem. Dit gebeurt normaal gesproken zodra een gebruiker een URL gebruikt waar een IP adres inzit of een hostnaam die naar uw systeem verwijst maar niet overeenkomt met een Virtualmin domein.
