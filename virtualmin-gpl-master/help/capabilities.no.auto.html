Kan redigere virtuell server

Kan administrere brukere
Lar postboks-, FTP- og databasebrukere ses, redigeres, opprettes og slettes.
Kan administrere aliaser
Lar e-postaliaser ses, redigeres, opprettes og slettes.
Kan administrere databaser
Lar ytterligere MySQL- og PostgreSQL-databaser legges til denne virtuelle serveren eller slettes fra den.
Kan installere webapper
Lar tredjeparts webapper installeres på denne virtuelle serveren.
Kan endre IP-adresse
Hvis denne serveren har en privat IP-adresse, lar dette alternativet den endres.
Kan administrere SSL-sertifikater
Lar SSL-sertifikatet og nøkkelen for denne serveren lastes opp eller genereres på nytt.
Kan redigere videresendingsramme

Kan administrere ekstra administratorer

Kan velge PHP-versjoner
For domener med PHP som kjøres via CGI eller fastCGI, lar PHP-versjoner per katalog angis.
Kan redigere e-postinnstillinger

Kan ta backup
Lar eide virtuelle servere sikkerhetskopieres til en ekstern FTP- eller SSH-server.
Kan planlegge sikkerhetskopier

Kan gjenopprette sikkerhetskopier

Kan velge delte IP-er

Kan redigere eksterne verter for database

Kan slette virtuelle servere
Lar eide servere slettes fullstendig, inkludert alle underservere og aliaser.
