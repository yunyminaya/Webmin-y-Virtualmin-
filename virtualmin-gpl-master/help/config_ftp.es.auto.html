¿Servidor virtual ProFTPd basado en IP habilitado?
Habilita o deshabilita un nuevo host virtual FTP. Tenga en cuenta que FTP requiere una nueva IP para el alojamiento de dominio virtual anónimo. La gran mayoría de los propietarios de sitios web no necesitan esta función, y no se requiere para el acceso FTP autenticado a los archivos de los propietarios del dominio. Sin embargo, si algunos dominios requieren acceso FTP anónimo alojado virtual (es decir, si están operando un servidor de archivos FTP para software o descargas de archivos para usuarios no registrados), puede configurarlo en "Sí, pero no de manera predeterminada" .
