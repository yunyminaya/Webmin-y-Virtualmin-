La acción específica de Virtualmin que se realizó se puede encontrar en la variable VIRTUALSERVER_ACTION, como:

CREAR_DOMINIO
Se está creando un nuevo servidor virtual.
ELIMINAR_DOMINIO
Se está eliminando un servidor virtual existente.
DESHABILITAR_DOMINIO
Un servidor virtual se está deshabilitando temporalmente.
ENABLE_DOMAIN
Se está volviendo a habilitar un servidor virtual.
MODIFICAR_DOMINIO
Un servidor virtual se está actualizando, como cuando se agregan o eliminan funciones, o se cambia la propiedad.
CLONE_DOMAIN
Se crea un servidor virtual copiando un servidor existente.
