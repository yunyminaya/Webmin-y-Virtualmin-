Comandos programados (comandos del usuario)
Esta opción especifica si la cuenta del propietario del dominio puede usar el módulo Webmin de comandos programados. Un comando programado o en una tarea es un evento programado que se puede utilizar para realizar una tarea una vez a la hora especificada. El módulo de comandos programados proporciona una interfaz fácil de usar para la sintaxis difícil de recordar (y confusa para los novatos) del comando at. Los comandos programados se ejecutarán como el usuario propietario del dominio.
