Cerca nei log di posta
Questa pagina può essere utilizzata per cercare nei registri e-mail i messaggi su alcuni server virtuali o su tutti i server se si è l'amministratore principale. Per cercare, è possibile selezionare uno o più dei seguenti criteri, che devono corrispondere tutti:

Data d'inizio
Verrà visualizzata qualsiasi e-mail ricevuta a partire da questa data. Se lasciato vuoto, la posta elettronica dall'inizio dei registri del server verrà abbinata.
Data di fine
Verrà visualizzata qualsiasi e-mail ricevuta prima o dopo tale data. Se lasciato vuoto, verrà inviata un'e-mail con la data corrente.
Indirizzo di partenza
Se compilato, verrà visualizzata l'e-mail per la quale l'indirizzo del mittente contiene il nome di dominio o l'indirizzo inserito.
Indirizzo di destinazione
Verrà visualizzata solo l'e-mail corrispondente al server virtuale selezionato da questo menu. Inoltre, se il campo del nome utente è compilato, verrà visualizzata solo la posta a quella casella di posta.
Nell'elenco risultante verranno visualizzati la data, l'ora, il mittente, l'indirizzo del destinatario, l'utente della cassetta postale locale (se presente) e il file o l'indirizzo di destinazione finale. Per visualizzare maggiori dettagli di un messaggio registrato, fai clic sulla data o l'ora di ricezione.
