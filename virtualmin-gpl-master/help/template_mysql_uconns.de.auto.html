Maximale Verbindungen für Postfachbenutzer
Dieses Feld steuert die Anzahl der gleichzeitigen Verbindungen zu MySQL, die ein Postfach mit Datenbankzugriff herstellen kann. Standardmäßig ist eine unbegrenzte Anzahl von Verbindungen zulässig. Da jedoch jede innerhalb des MySQL-Serverprozesses Speicher belegt, möchten Sie möglicherweise die Gesamtzahl der Verbindungen auf einem ausgelasteten System begrenzen.

Das hier festgelegte Limit wird nur angewendet, wenn ein neues Postfach erstellt wird oder wenn eine MySQL-Anmeldung für ein vorhandenes Postfach aktiviert ist.
