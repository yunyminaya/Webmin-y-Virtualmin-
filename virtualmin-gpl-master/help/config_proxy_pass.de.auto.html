Erstellen von Nur-Weiterleitungs-Websites zulassen?
Virtualmin unterstützt das "Parken" von Domains mit der Weiterleitung an eine andere Website. Mit dieser Option können Sie auswählen, ob die Weiterleitung mit ProxyPass-Regeln in Apache über einen Frame um den Zielstandort erfolgen soll, oder die Funktion vollständig deaktivieren.
