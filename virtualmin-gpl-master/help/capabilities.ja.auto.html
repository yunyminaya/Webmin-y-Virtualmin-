仮想サーバーを編集できます
仮想サーバーの機能、パスワード、説明を編集できます。
ユーザーを管理できる
メールボックス、FTP、データベース ユーザーを表示、編集、作成、削除できます。
エイリアスを管理できる
電子メールエイリアスを表示、編集、作成、削除できます。
データベースを管理できる
この仮想サーバーに MySQL および PostgreSQL データベースを追加したり、削除したりすることができます。
ウェブアプリをインストールできる
この仮想サーバーにサードパーティの Web アプリケーションをインストールできるようにします。
IPアドレスを変更できる
このサーバーにプライベート IP アドレスがある場合、このオプションを使用してそれを変更できます。
SSL証明書を管理できる
このサーバーの SSL 証明書とキーをアップロードまたは再生成できるようにします。
転送フレームを編集できる
転送専用の Web サイトの場合、このオプションを使用すると、転送フレームの宛先と HTML を編集できます。
追加の管理者を管理できる
追加のサーバー所有者管理者アカウントを表示、作成、編集、削除できるようにします。
スパムやウイルスの配信を設定できます
スパムやウイルスの配信先とスコアを編集できます。
PHPバージョンを選択可能
CGI または fastCGI 経由で PHP を実行するドメインでは、ディレクトリごとに PHP バージョンを設定できます。
メール設定を編集できます
自動 BCC、エイリアス コピー モード、その他の電子メール関連のオプトインを構成できます。
バックアップを作成できる
所有する仮想サーバーをリモート FTP または SSH サーバーにバックアップできるようにします。
バックアップをスケジュールできる
仮想サーバーを定期的にバックアップできます。
バックアップを復元できる
所有する仮想サーバーのホーム ディレクトリとデータベース コンテンツを復元できます。
共有IPを選択可能
仮想サーバーを作成するときに、グローバルに定義された共有 IP アドレスを選択できるようにします。
包括的なエイリアスを作成できる
ドメインへのすべてのメールを別のドメインまたはアドレスに転送する電子メール エイリアスの作成を許可します。
ウェブページエディタを使用できる
Virtualmin UI を介して Web ページを編集できるようにします。
データベースのリモートホストを編集できます
ドメインの MySQL データベースへの接続が許可されているリモート クライアント システムの入力を許可します。
仮想サーバーを無効にできる
サーバーを一時的に無効にすることができます。
仮想サーバーを削除できる
すべてのサブサーバーとエイリアスを含む、所有サーバーを完全に削除できます。
