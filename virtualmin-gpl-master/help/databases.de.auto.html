Datenbanken bearbeiten
Auf dieser Seite sind alle Datenbanken aufgelistet, die derzeit dem virtuellen Server gehören. Sie können zusätzliche Datenbanken erstellen, bis der für den bei Virtualmin angemeldeten Benutzer geltende Grenzwert erreicht ist. Sie können auch weitere Datenbanken löschen, indem Sie sie aus der Liste auswählen und auf die Schaltfläche „Ausgewählte löschen“ klicken.

Um die tatsächlichen Tabellen und Daten in einer Datenbank anzuzeigen, klicken Sie in der Tabelle auf den Link „Verwalten“ neben dem Datenbanknamen. Um die Größe der Datenbank anzuzeigen, klicken Sie auf ihren Namen.

Bei Verwendung von Virtualmin Pro werden unterhalb der Datenbankliste zwei zusätzliche Formulare angezeigt, um den Benutzernamen und das Kennwort für die von diesem virtuellen Server verwendeten MySQL- und PostgreSQL-Benutzer zu ändern. Dies kann nützlich sein, wenn Sie möchten, dass sich der Datenbank-Benutzername vom primären Administrator-Benutzernamen unterscheidet, möglicherweise aus Sicherheitsgründen. Diese sollten mit Vorsicht geändert werden, da PHP-Skripte (z. B. von Webanwendungs-Installern), die den Benutzernamen oder das Kennwort in ihren Konfigurationsdateien haben, nicht aktualisiert werden.
