Commandes planifiées (commandes de l'utilisateur)
Cette option spécifie si le compte propriétaire du domaine peut utiliser le module Webmin de commandes planifiées. Une commande planifiée ou une tâche est un événement planifié qui peut être utilisé pour exécuter une tâche une fois à l'heure spécifiée. Le module Scheduled Commands fournit une interface facile à utiliser avec la syntaxe difficile à retenir (et déroutante pour les débutants) de la commande at. Les commandes planifiées seront exécutées en tant qu'utilisateur propriétaire du domaine.
