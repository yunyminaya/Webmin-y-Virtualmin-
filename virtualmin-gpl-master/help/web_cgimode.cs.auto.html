Režim provádění skriptu CGI
Tato volba určuje, jak jsou skripty CGI spouštěny pro tento virtuální server. Možné možnosti jsou:
  • CGI skripty vypnuty - skripty nelze vůbec spouštět.
  • FCGIwrap server - skripty jsou spouštěny přes server na pozadí, jeden na doménu.
  • suEXEC - skripty se spouštějí přes Apache suexec wrapper. Upozorňujeme, že tato možnost nemusí být dostupná na všech systémech.
