Crea automaticamente un dominio alias
Questa opzione può essere utilizzata per creare un dominio alias aggiuntivo creato per ogni nuovo server virtuale, sotto qualche altro dominio. Può essere utile quando si aggiunge un nuovo server per il quale il dominio DNS non è stato ancora registrato, in modo che sia possibile accedervi con un nome temporaneo con uno già registrato (come la tua azienda).
