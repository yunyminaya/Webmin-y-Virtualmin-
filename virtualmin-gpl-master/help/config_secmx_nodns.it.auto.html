Aggiungere record MX DNS per server di posta secondari?
Quando è impostato su Sì e vengono selezionati server di posta aggiuntivi nella pagina Server di posta secondari , Virtualmin aggiungerà un record MX al dominio DNS per tutti i server virtuali che utilizzano quei secondari. Tuttavia, se hai già record MX alternativi già definiti nel tuo modello DNS, puoi cambiarlo in No.
