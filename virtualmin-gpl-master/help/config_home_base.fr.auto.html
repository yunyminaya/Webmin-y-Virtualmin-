Base du répertoire personnel
Il s'agit du répertoire de niveau supérieur où les répertoires personnels seront créés. Si votre installation Virtualmin partage un serveur avec de nombreux autres services ou utilisateurs système standard, il peut être utile d'utiliser autre chose que / home. La valeur par défaut est collectée à partir du module Webmin Utilisateurs et groupes, qui est par défaut / home. Le module Utilisateurs et groupes permet des extensions complexes à cette convention, telles que "/ home / v / virtualmin", ce qui peut faciliter la navigation dans les répertoires si vous avez plusieurs domaines sur un seul serveur. La documentation du module Utilisateurs et groupes dans l'un des manuels Webmin couvre cette fonctionnalité plus en détail.
