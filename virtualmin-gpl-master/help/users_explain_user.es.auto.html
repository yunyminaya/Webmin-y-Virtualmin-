Crear usuario
Crea una nueva cuenta de usuario Unix, con el mismo grupo que el propietario del servidor virtual. Esta configuración presenta amplias opciones de personalización, como configurar una contraseña, ajustar los permisos de inicio de sesión y los grupos secundarios, y controlar las cuotas. También abarca la configuración del directorio de inicio y ofrece la flexibilidad de habilitar o deshabilitar una variedad de servicios como correo electrónico, bases de datos y acceso web. Además, incluye la capacidad de controlar otras configuraciones de acceso ofrecidas por complementos adicionales.
