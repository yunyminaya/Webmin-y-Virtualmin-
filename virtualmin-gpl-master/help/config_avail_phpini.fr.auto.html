Configuration PHP (pour les fichiers php.ini du domaine)
Lorsqu'elle est activée, cette option accorde aux propriétaires de serveurs virtuels l'accès au module de configuration PHP. Cela sera limité afin qu'ils ne puissent gérer que les fichiers php.ini dans leurs propres serveurs virtuels, pour lesquels l'exécution PHP via CGI est activée.

Si un domaine utilise mod_php pour exécuter des scripts PHP, cette option contrôle également si le propriétaire du serveur virtuel est autorisé à configurer les paramètres liés à PHP dans Apache.
