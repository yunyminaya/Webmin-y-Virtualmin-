仮想サーバーのクローン
このページでは、既存の仮想サーバーを新しいドメイン名で複製できます。元のドメインに関連付けられている Web コンテンツ、DNS レコード、メールボックス、メール エイリアス、データベース、その他の設定は、可能な場合は複製されます。ただし、Web コンテンツは一切変更されないため、仮想サーバーに古いサーバーのデータベースまたはホーム ディレクトリを使用するように構成された Web アプリケーションが含まれている場合は、複製後に手動で再構成する必要があります。

最上位の仮想サーバーを複製する場合、新しいドメインを所有するために新しいユーザー名とパスワードも入力する必要があります。サーバーにプライベート IP アドレスがある場合は、クローンに対して新しい IP が自動的に割り当てられます。

クローン作成プロセスでは、元のドメインに関連付けられているすべてのデータベースが新しい名前でコピーされ、その内容が複製されます。ただし、元のドメインにインストールされ、元のデータベースを使用するように設定されているすべての Web アプリケーションが更新されない可能性があります。これは手動で行う必要があります。
