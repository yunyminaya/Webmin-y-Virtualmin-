Kann FTP-Benutzer erstellen?
FTP-Benutzer sind Benutzer, die sich über FTP anmelden und Dateien auf den Server hochladen oder von diesem herunterladen können. Wenn virtuelle Server häufig von einem Benutzer verwaltet werden und der Website-Inhalt von einem anderen Benutzer verwaltet wird, kann es hilfreich sein, mehrere FTP-Benutzerkonten zuzulassen.
