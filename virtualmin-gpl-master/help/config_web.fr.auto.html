Installation et configuration d'Apache activées?
Cette option active ou désactive la création d'entrées Apache VirtualNameHost et configure toutes les autres fonctionnalités d'hôte virtuel dans votre fichier de configuration Apache. Si vous utilisez un autre serveur Web, vous pouvez désactiver cette option. Il peut être possible d'utiliser l'interface du module d'extension pour créer une prise en charge de votre serveur préféré (bien qu'aucun autre serveur Web ne soit pris en charge pour le moment), ou vous pouvez choisir d'appeler un script externe avec des variables tirées des informations fournies à Virtualmin à temps de création de domaine pour configurer votre serveur Web préféré.
