Configurer un site Web SSL également?
Lorsque cette option est activée, un hôte virtuel Apache supplémentaire sera créé pour ce serveur virtuel, à l'écoute sur le port 443. Cela permettra aux navigateurs de se connecter à https: // nom_domaine.com / en utilisant le cryptage SSL, ainsi qu'à http: / / domainname.com / sans cryptage.

Une clé et un certificat SSL auto-signés seront générés dans le cadre du processus de configuration du serveur, afin que l'hôte virtuel SSL puisse être immédiatement utilisé. Cependant, vous devez générer un certificat SSL valide pour le domaine dès que possible, afin qu'il puisse être correctement vérifié par les navigateurs. Cela peut être fait sur la page Gérer le certificat SSL .
