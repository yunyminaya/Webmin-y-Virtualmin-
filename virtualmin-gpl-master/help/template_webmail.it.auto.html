Reindirizzare webmail.domain su Usermin?
Quando questo campo è impostato su Sì (come per impostazione predefinita), tutti i nuovi server virtuali avranno una voce webmail aggiunta ai rispettivi domini DNS, con lo stesso indirizzo IP del sito Web del server. Quando si accede a http://webmail.thedomain.com/ , il browser verrà reindirizzato a Usermin sullo stesso dominio, per accedere alla posta elettronica basata sul Web.

Se si preferisce utilizzare un'interfaccia webmail diversa da Usermin (come Squirrelmail o Roundcube) o si desidera utilizzare un URL diverso per l'accesso Usermin, l'URL di reindirizzamento può essere configurato nel seguente URL per il campo di reindirizzamento di webmail .
