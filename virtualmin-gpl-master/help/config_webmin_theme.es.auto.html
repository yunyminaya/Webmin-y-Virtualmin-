Tema para nuevos propietarios de dominios de Webmin
Puede seleccionar el tema predeterminado para los inicios de sesión de Webmin del propietario del dominio. Su valor predeterminado es el valor predeterminado global de Webmin, que es el tema Virtualmin-Nuvola en las instalaciones de Virtualmin Professional. Puede resultarle útil diferenciar entre esquemas de color para varios niveles de privilegios, como elegir un tema teñido de rojo para las cuentas administrativas, ya que esas cuentas son más peligrosas cuando se usan descuidadamente, mientras se usa el tema teñido gris-azul suave estándar para su dominio propietarios y revendedores. Puramente una cuestión de gustos.
