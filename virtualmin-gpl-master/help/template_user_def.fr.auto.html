Ajouter un utilisateur Apache au groupe Unix pour les nouveaux serveurs ?
Cette option détermine si l'utilisateur sous lequel Apache s'exécute sera ajouté au nouveau groupe d'hôtes virtuels. Lorsqu'elles sont activées, les autorisations peuvent être considérablement renforcées sur les répertoires personnels des utilisateurs. Lorsque des autorisations plus strictes (c'est-à-dire 750 ) sont combinées avec suEXEC, il est impossible pour les propriétaires de domaine de voir les répertoires et fichiers des autres propriétaires de domaine. Cette option est recommandée sur toute plateforme prenant en charge les groupes secondaires.
