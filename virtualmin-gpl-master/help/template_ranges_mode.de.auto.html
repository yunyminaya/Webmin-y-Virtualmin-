Bereiche für die automatische IP-Zuweisung
Wenn Sie möchten, dass Virtualmin neuen virtuellen Servern, die aus dieser Vorlage erstellt wurden, automatisch eine IP-Adresse zuweist, können Sie hier einen oder mehrere Bereiche eingeben, um IPs daraus auszuwählen. Andernfalls muss beim Erstellen einer Domain eine private virtuelle IP manuell eingegeben werden.
