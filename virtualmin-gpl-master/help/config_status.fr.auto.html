Surveillance de l'état du site Web?
Active ou désactive la création de moniteurs d'état dans le module Webmin System and Server Status. S'il est désactivé, un moniteur de service ne sera pas créé automatiquement pour votre nouveau domaine. Un moniteur d'état peut vous informer par e-mail si un service tombe en panne, ou il peut exécuter une commande pour effectuer une action pour essayer de corriger automatiquement le problème.
