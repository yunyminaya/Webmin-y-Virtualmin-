Connexion FTP activée ?
Lorsque cette option est définie sur « Oui », l'utilisateur recevra le shell , leur permettant de se connecter via FTP. Lorsqu'il est défini sur « Non », leur shell sera défini sur au lieu de cela, les connexions FTP seront refusées.

Cela suppose que le shell FTP est répertorié dans le fichier /etc/shells ou autorisé par le serveur FTP exécuté sur votre système.
