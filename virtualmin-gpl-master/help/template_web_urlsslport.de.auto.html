Externe Portnummer für virtuelle SSL-Hosts
Mit dieser Einstellung können Sie die Portnummer ändern, die Virtualmin in URLs verwendet, die auf Domänen verweisen, die aus dieser Vorlage erstellt wurden. Dies kann nützlich sein, wenn Ihr System einen Proxy oder Cache vor sich hat, der Port 443 überwacht, und Virtualmin so konfiguriert wurde, dass virtuelle Apache-Hosts an einem anderen Port erstellt werden.
