Befehl zum Auflisten von Benutzerkontingenten
Wenn externe Kontingentbefehle verwendet werden, wird das in dieser Option festgelegte Programm aufgerufen, wenn Virtualmin vorhandene Benutzerkontingente suchen muss. Es wird nicht mit Parametern aufgerufen, sondern muss eine Textzeile pro Benutzer ausgeben, wobei jede Zeile durch Leerzeichen getrennte Felder enthält. Die Felder in jeder Zeile müssen sein:
  1. Der Unix-Benutzername.
  2. Aktuelles Kontingent in 1-kB-Blöcken.
  3. Das weiche Kontingent in 1-kB-Blöcken oder 0 für unbegrenzt.
  4. Das harte Kontingent in 1-kB-Blöcken oder 0 für unbegrenzt.
