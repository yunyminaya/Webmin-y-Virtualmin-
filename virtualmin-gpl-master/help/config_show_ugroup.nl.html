Selectie toestaan van een andere groep voor server administrators?
Wanneer deze optie is ingesteld op Ja kan de persoon die deze nieuwe virtuele server aanmaakt een andere Unix groep kiezen voor zijn eigenaar inplaats van dat er een nieuwe groep word aangemaakt voor de virtuele server. Dit kan handig zijn indien het nodig is dat alle server account eigenaars op uw systeem lid moeten zijn van een bepaalde groep, u moet echter ook voorzichtig zijn omdat hiermee server eigenaars de groep quota kunnen omzeilen.
