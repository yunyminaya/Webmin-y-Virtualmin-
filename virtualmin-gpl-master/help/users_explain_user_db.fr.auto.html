Créer un utilisateur de base de données
Crée un nouvel utilisateur de base de données capable de gérer uniquement les bases de données, en exploitant les quotas du propriétaire du serveur virtuel pour toutes les bases de données gérées. Cet utilisateur opère uniquement dans l'environnement de base de données, sans créer de compte Unix distinct. De plus, cet utilisateur peut être intégré de manière transparente en fusionnant avec un compte Unix créé en utilisant le même nom d'utilisateur.
