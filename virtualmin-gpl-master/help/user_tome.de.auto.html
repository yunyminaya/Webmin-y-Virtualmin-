An diesen Benutzer normal liefern
Wenn diese Option aktiviert ist, werden E-Mails an diesen Benutzer nach Durchlaufen aller Spamfilter in dessen Postfach zugestellt. Normalerweise müssen Sie diese Option nicht aktivieren, da dies automatisch geschieht, wenn keine Weiterleitung oder automatische Antwort eingerichtet ist. Wenn Sie jedoch eine Weiterleitungsadresse oder automatische Antwort einrichten, aber dennoch möchten, dass der Benutzer eine Kopie seiner E-Mails behält, sollte diese Option ausgewählt werden.
