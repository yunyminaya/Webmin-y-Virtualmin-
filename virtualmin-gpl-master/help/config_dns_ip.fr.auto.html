Adresse IP par défaut pour les enregistrements DNS
Les enregistrements DNS sont générés automatiquement pour les nouveaux hôtes virtuels et, dans des circonstances normales, vous souhaiterez qu'ils correspondent à l'adresse IP du serveur virtuel fournie ci-dessus. Cependant, si votre serveur se trouve derrière un routeur NAT, vous utilisez un serveur proxy, un serveur d'équilibrage de charge ou un autre type de périphérique qui conduit votre serveur à fonctionner avec une adresse IP différente de celle que le monde voit lorsque vous parlez à vos domaines, vous il peut être nécessaire de modifier la valeur ici. Vous pouvez également changer cela si vous ajoutez des domaines à un serveur de développement qui auront des adresses différentes lors de sa mise en production. Vous pouvez saisir l'adresse IP de production ou publique ici plutôt que de demander à Virtualmin d'utiliser l'adresse IP locale.

Si l'option Détecter automatiquement l'adresse externe est sélectionnée, Virtualmin tentera de déterminer l'adresse IP au moment de la création du serveur virtuel. Cela se fait en faisant une demande HTTP à www.virtualmin.com , qui répondra avec l'adresse dont il voit la connexion venir.
