Permisos en el directorio del sitio web
Esta opción especifica los permisos iniciales sobre los archivos y directorios dentro del directorio del sitio web especificado en la opción anterior. Si se ha habilitado suEXEC y el usuario de Apache se agrega al grupo de usuarios del servidor virtual, estos permisos se pueden reforzar para evitar que otros usuarios vean el contenido del directorio del sitio web. Este conjunto estricto de permisos se recomienda para cualquier implementación, a menos que todos los usuarios con acceso al shell sean usuarios confiables. Los permisos recomendados son 750 , lo que equivale a rwxr-x--- . En otras palabras, el propietario puede leer, escribir y ejecutar, el grupo puede leer y ejecutar y otros no tienen permisos para leer, escribir o ejecutar archivos dentro del directorio.
