Trabajos Cron programados (trabajos Cron del usuario)
Esta opción especifica si la cuenta del propietario del dominio puede usar el módulo Cron Jobs Webmin. Un trabajo Cron es un evento programado que se puede utilizar para realizar periódicamente alguna tarea, como enviar un correo electrónico, hacer una copia de seguridad o aspirar una base de datos, verificar el estado de un archivo o buzón, etc. El módulo Cron Jobs proporciona y es fácil de usar interfaz con la sintaxis difícil de recordar (y confusa para los novatos) del archivo crontab. Cron Jobs se ejecutará como el usuario propietario del dominio.
