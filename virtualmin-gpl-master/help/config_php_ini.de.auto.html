Vollständiger Pfad zur Master-PHP-Konfigurationsdatei
Dieses Feld muss auf den vollständigen Pfad zur php.ini Datei auf Ihrem System eingestellt sein. Es wird nur verwendet, wenn Virtualmin PHP-Skripte als Besitzer des virtuellen Servers über suEXEC ausführt, und wird zur Änderung pro Server in das Home-Verzeichnis jeder Domäne kopiert.
