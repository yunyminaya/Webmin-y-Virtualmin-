¿Incluir nombre de dominio en nombres de usuario?
Esta opción selecciona cómo se generarán los nombres de usuario. De manera predeterminada, Virtualmin usará el nombre de usuario base solicitado, pero si choca con un nombre de usuario existente, agregará el nombre de dominio de la manera especificada en la siguiente opción. Si es preferible la coherencia, configúrelo en "Siempre", y se agregará el nombre de dominio a todos los nombres de usuario dentro de los servidores secundarios.
