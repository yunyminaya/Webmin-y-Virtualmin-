Redirecionamentos de site
Um redirecionamento mapeia algum caminho de URL como / foo para um URL diferente ou para um diretório diferente no mesmo servidor virtual. Isso pode ser usado para fornecer caminhos de URL mais amigáveis no seu site ou para lidar com o movimento de páginas da web para novos locais.

Esta página lista todos os redirecionamentos existentes e permite criar um novo clicando no link Adicionar um novo redirecionamento de site . Para editar um redirecionamento existente, basta clicar no caminho do URL de origem. Para remover vários redirecionamentos, marque as caixas ao lado deles e clique no botão Excluir na parte inferior da página.
