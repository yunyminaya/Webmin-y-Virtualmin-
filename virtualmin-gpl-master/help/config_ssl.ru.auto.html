Настройка SSL сайта включена?
Если включено, сайт SSL будет создан для каждого нового домена. Если установлено «Да, но не по умолчанию», можно будет создать сайт SSL при создании домена. Обратите внимание, что веб-сайтам SSL требуется отдельный IP-адрес, поэтому по умолчанию он не включен для всех сайтов. Многим веб-сайтам не требуется безопасность SSL, поэтому они могут использовать один IP-адрес для сотен или тысяч других веб-сайтов.
