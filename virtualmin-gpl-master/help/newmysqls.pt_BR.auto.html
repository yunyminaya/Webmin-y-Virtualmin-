Servidores MySQL
Esta página permite definir servidores MySQL adicionais em sistemas remotos ou instalações adicionais no sistema, para uso pelos domínios do Virtualmin. Isso pode ser usado para descarregar a hospedagem MySQL para um servidor diferente ou para suportar várias versões do MySQL para diferentes clientes.

Para adicionar um novo servidor MySQL remoto, use o for na parte inferior da página. Para remover um servidor definido anteriormente da lista, marque a caixa ao lado dele na lista e clique no botão Remover selecionado .
