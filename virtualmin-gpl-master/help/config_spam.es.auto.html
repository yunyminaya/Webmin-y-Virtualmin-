¿Filtro de correo SpamAssassin por servidor?
Habilita o deshabilita la capacidad de habilitar la configuración de SpamAssassin por dominio. Con esto habilitado, puede deshabilitar el filtro de spam para algunos dominios mientras lo habilita para otros. Es posible que desee deshabilitar esto, si va a exigir el filtrado de spam para todos los usuarios, o si está utilizando un sistema alternativo preventivo de spam en todo el sistema. Tenga en cuenta que Virtualmin no maneja la configuración de todo el servidor. Deberá habilitarlo utilizando el módulo SpamAssassin Webmin, si prefiere el filtrado de spam en todo el servidor.
