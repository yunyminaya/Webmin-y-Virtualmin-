Admin.domain nach Usermin umleiten?
Wenn dieses Feld auf " Ja" gesetzt ist (wie standardmäßig), wird allen neuen virtuellen Servern ein admin zu ihren DNS-Domänen hinzugefügt, der dieselbe IP-Adresse wie die Website des Servers aufweist. Wenn auf http://admin.thedomain.com/ zugegriffen wird, wird der Browser an Port 10000 derselben Domäne zu Virtualmin umgeleitet, um auf die Administrations-Weboberfläche zuzugreifen.

Wenn Sie eine andere URL für den Virtualmin-Zugriff verwenden möchten (z. B. eine, für die Sie ein SSL-Zertifikat haben), kann die Umleitungs-URL in der folgenden URL für das Admin-Umleitungsfeld konfiguriert werden.
