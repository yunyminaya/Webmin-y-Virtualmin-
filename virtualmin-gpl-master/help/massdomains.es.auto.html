Actualizar servidores virtuales
Esta página le permite actualizar la configuración y los límites en varios servidores virtuales a la vez, como se seleccionó en la página anterior. De forma predeterminada, nada se cambiará en los servidores seleccionados hasta que elija algún atributo o límite para modificar utilizando los campos de esta página. La configuración modificable se divide en las siguientes categorías:

Cambios de cuota y ancho de banda
Estos campos se pueden usar para modificar las cuotas de disco para los servidores seleccionados o para cambiar sus límites de ancho de banda. Cambiar las cuotas solo tiene un efecto en los servidores de nivel superior, ya que los sub-servidores heredan sus cuotas del dominio principal.

Cambios de funciones habilitados
En esta sección, puede seleccionar funciones para activar o desactivar todos los servidores seleccionados. Estas son las mismas características que se pueden activar para servidores individuales en la página Editar servidor virtual .

Cambios a los límites del propietario
Estos campos se pueden usar para cambiar los diversos límites que se aplican a los propietarios de servidores virtuales de nivel superior, como la cantidad de buzones, alias y dominios que pueden crear. Los cambios realizados aquí no tienen ningún efecto en los servidores secundarios, que caen dentro de los límites impuestos a sus padres.

Cambios en las capacidades del propietario
Los campos en esta sección controlan las capacidades de Virtualmin que están disponibles para los administradores de los servidores virtuales seleccionados. Nuevamente, solo son relevantes para los servidores de nivel superior.

Cambios de opciones de PHP
Cuando esté disponible, esta sección le permite cambiar tanto la versión predeterminada de PHP como el método para ejecutar scripts PHP en todos los servidores seleccionados. Sin embargo, solo es útil en servidores que tienen sitios web.

Cambios de configuración
Esta sección es para realizar varios cambios en la configuración de los servidores seleccionados, como el uso del método para eliminar el correo no deseado o el shell de inicio de sesión del administrador del dominio.

También es posible deshabilitar temporalmente los servidores seleccionados utilizando el botón Deshabilitar servidores en la parte inferior de la página. Al usar esto, también puede ingresar una razón para deshabilitar (como falta de pago ) en el cuadro de texto adyacente.

Para volver a habilitar varios servidores que se han deshabilitado en esta página o en otra parte, use el botón Habilitar servidores .
