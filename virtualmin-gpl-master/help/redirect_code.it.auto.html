Tipo di reindirizzamento HTTP
Per impostazione predefinita, Apache utilizzerà il codice di stato HTTP 302 per i reindirizzamenti ad altri siti Web, ma questo campo può essere utilizzato per personalizzare il codice in base al motivo per cui il reindirizzamento viene aggiunto. Questa opzione ha effetto solo quando è selezionata la destinazione "URL su altro sito Web".
