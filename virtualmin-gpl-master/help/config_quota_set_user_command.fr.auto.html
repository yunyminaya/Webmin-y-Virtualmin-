Commande pour définir le quota d'un utilisateur
Lorsque des commandes de quota externe sont utilisées, le programme défini dans cette option sera appelé lorsque Virtualmin souhaite définir les quotas pour un utilisateur Unix. Les paramètres avec lesquels il est appelé sont:
  1. Le nom d'utilisateur Unix
  2. Le nouveau quota souple en blocs de 1 Ko, ou 0 pour illimité.
  3. Le nouveau quota dur en blocs de 1 Ko, ou 0 pour illimité.
