Comandos agendados (comandos do usuário)
Esta opção especifica se a conta do proprietário do domínio pode usar o módulo Webmin de comandos agendados. Um comando agendado ou na tarefa é um evento agendado que pode ser usado para executar uma tarefa uma vez no horário especificado. O módulo de comandos agendados fornece uma interface fácil de usar para a sintaxe difícil de lembrar (e confusa para iniciantes) do comando at. Comandos agendados serão executados como o usuário proprietário do domínio.
