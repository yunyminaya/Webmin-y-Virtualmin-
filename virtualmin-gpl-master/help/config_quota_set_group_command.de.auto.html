Befehl zum Festlegen des Kontingents einer Gruppe
Wenn externe Kontingentbefehle verwendet werden, wird das in dieser Option festgelegte Programm aufgerufen, wenn Virtualmin die Kontingente für eine Unix-Gruppe festlegen möchte. Die Parameter, mit denen es aufgerufen wird, sind:
  1. Der Unix-Gruppenname
  2. Das neue Soft-Kontingent in 1-kB-Blöcken oder 0 für unbegrenzt.
  3. Das neue harte Kontingent in 1-kB-Blöcken oder 0 für unbegrenzt.
