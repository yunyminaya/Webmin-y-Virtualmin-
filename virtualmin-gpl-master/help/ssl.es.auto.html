Configurar sitio web SSL también?
Cuando esta opción está habilitada, se creará un host virtual Apache adicional para este servidor virtual, escuchando en el puerto 443. Esto permitirá que los navegadores se conecten a https: // domainname.com / usando encriptación SSL, así como a http: / / domainname.com / sin cifrado.

Se generará una clave y un certificado SSL autofirmados como parte del proceso de configuración del servidor, de modo que el host virtual SSL pueda usarse inmediatamente. Sin embargo, debe generar un certificado SSL válido para el dominio lo antes posible, para que los navegadores puedan verificarlo correctamente. Esto se puede hacer en la página Administrar certificado SSL .
