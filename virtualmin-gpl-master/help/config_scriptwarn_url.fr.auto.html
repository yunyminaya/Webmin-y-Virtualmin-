URL Virtualmin à utiliser dans les e-mails
Lorsque Virtualmin envoie des e-mails à l'administrateur système, il inclut souvent l'URL de Virtualmin sur le système à partir duquel le message a été envoyé. Par défaut, cela est basé sur le nom d'hôte du système, mais vous pouvez définir une URL personnalisée comme https://yourdomain.com:10000 aide de ce champ.
