SSH / Telnet-Login
Diese Option gibt an, ob der Benutzer des Domänenbesitzers das SSH / Telnet Login Webmin-Modul verwenden kann. Dieses Modul ist ein Java-Applet, das sich mit SSH oder Telnet beim Server anmeldet. Aufgrund von Java-Inkompatibilitäten funktioniert es nicht mit allen Browsern ordnungsgemäß. Beachten Sie, dass das Deaktivieren dies nicht den SSH-Zugriff des Domänenbesitzers auf den Server verhindert, sondern lediglich den Zugriff auf das SSH / Telnet-Anmeldemodul von Webmin.
