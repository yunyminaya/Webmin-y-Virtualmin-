Data di inizio della risposta automatica
Quando si seleziona Qualsiasi ora (per impostazione predefinita), Virtualmin invierà una risposta automatica indipendentemente dalla data di ricezione dell'email. Tuttavia, è possibile selezionare Data di inserimento e inserire una data nei campi adiacenti per impedire l'invio di risposte automatiche prima di quel giorno. Questo può essere utile quando si imposta un autoreponder per qualcuno che sta andando in vacanza in una data nota.
