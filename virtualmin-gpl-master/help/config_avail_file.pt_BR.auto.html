Gerenciador de arquivos (somente diretório inicial)
O File Manager é um applet Java avançado que oferece uma visão de painel duplo do sistema de arquivos do servidor. É um dos recursos mais poderosos do Webmin, permitindo que até mesmo usuários inexperientes gerenciem arquivos, diretórios e permissões, editem arquivos de texto, carreguem e baixem arquivos e muito mais. Por padrão, o proprietário do domínio só poderá acessar seu próprio diretório e seus subdiretórios.
