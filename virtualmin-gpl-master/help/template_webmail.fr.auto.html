Redirigez webmail.domain vers Usermin?
Lorsque ce champ est défini sur Oui (comme c'est le cas par défaut), tous les nouveaux serveurs virtuels auront une entrée de webmail ajoutée à leurs domaines DNS, avec la même adresse IP que le site Web du serveur. Lorsque http://webmail.thedomain.com/ est accessible, le navigateur sera redirigé vers Usermin sur le même domaine, pour accéder à la messagerie Web.

Si vous préférez utiliser une interface de messagerie Web autre que Usermin (telle que Squirrelmail ou Roundcube) ou si vous souhaitez utiliser une URL différente pour l'accès Usermin, l'URL de redirection peut être configurée dans l' URL suivante pour le champ de redirection de messagerie Web .
