Erzwingen Sie, dass sich virtuelle Server unter einer Domäne befinden
Standardmäßig ist dies auf „Jede Domäne zulassen“ eingestellt, sodass der Wiederverkäufer beliebige Domänennamen erstellen kann. Sie können diese Option jedoch verwenden, um sie auf das Erstellen von Domänen unter einer bestimmten übergeordneten Domäne zu beschränken, normalerweise einer Domäne, die ihnen gehört.
