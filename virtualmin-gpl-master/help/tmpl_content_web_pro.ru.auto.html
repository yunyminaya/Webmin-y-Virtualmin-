HTML-страница веб-сайта по умолчанию
Эта опция позволяет вам управлять страницей по умолчанию, которая будет создана и отображена для нового виртуального сервера.

Вы можете выбрать «Отключено», чтобы пропустить создание страницы, или «Спросить во время создания», чтобы принять решение во время создания виртуального сервера.

Кроме того, вы можете выбрать страницу по умолчанию для веб-сайта Virtualmin, выбрав опцию «Да, с шаблоном страницы по умолчанию для веб-сайта».

При желании можно выбрать вариант «Да, с содержимым ниже», чтобы предоставить пользовательский HTML-код или текст для использования в качестве содержимого веб-страницы.
