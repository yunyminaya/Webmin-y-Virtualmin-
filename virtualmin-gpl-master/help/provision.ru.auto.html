Клиент Cloudmin Services
Эта страница позволяет настроить Virtualmin для подключения к серверу Cloudmin Services, чтобы базы данных MySQL, зоны DNS и другие службы могли быть выгружены на удаленные системы. В идеале его следует использовать до создания виртуальных серверов, поскольку смешивание локально обслуживаемых и удаленных баз данных и зон неэффективно.

Поля, которые необходимо заполнить:

Мастер облачных сервисов
Это имя хоста компьютера Cloudmin с установленным плагином Cloudmin Services. Это может выполняться вами, вашей хостинговой компанией или провайдером VPS.

Номер порта
Это почти всегда должно быть установлено на 10000, который является портом Webmin / Cloudmin по умолчанию.

Имя пользователя для сервера
Это поле должно содержать имя пользователя учетной записи владельца системы на главном сервере Cloudmin Services, которая имеет разрешение на создание баз данных, зон DNS или других служб.
Особенности для размещения
В этом разделе вы можете выбрать, какие функции Virtualmin будут выгружены на удаленные системы. Могут быть выбраны только те, для которых ваша учетная запись в основной системе разрешена.
