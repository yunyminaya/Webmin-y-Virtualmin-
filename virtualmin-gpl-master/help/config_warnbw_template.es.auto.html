Archivo de plantilla para correo electrónico a servidores que se acerca al límite de ancho de banda
Si se configuran los límites y la supervisión del ancho de banda, Virtualmin puede notificar al propietario de un dominio si el uso del ancho de banda para el dominio se acerca a la cuota. Puede usar el mensaje predeterminado o crear el suyo propio y especificarlo aquí. Las variables de plantilla estándar de Virtualmin están disponibles dentro de la plantilla.
