Configuration du client de messagerie
Cette page peut activer ou désactiver l'URL de configuration du client de messagerie automatique pour tous les serveurs virtuels avec la messagerie activée. Des clients comme Thunderbird peuvent utiliser cette URL pour détecter les paramètres SMTP et IMAP en fonction uniquement d'une adresse e-mail, y compris le nom d'utilisateur, le numéro de port et le type de cryptage corrects.

Lorsqu'il est activé, un sous-domaine nommé autoconfig est créé pour chaque serveur virtuel et le chemin URL /mail/config-v1.1.xml est mappé sur un CGI qui sert XML pour fournir les paramètres requis du client de messagerie. Ceux-ci seront supprimés lorsque la fonctionnalité est désactivée.
