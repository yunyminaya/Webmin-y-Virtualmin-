Répertoires Web protégés (sous le répertoire personnel)
Cette option spécifie si le propriétaire du domaine aura accès au module Webmin des répertoires Web protégés, qui permet la création d'un ou de plusieurs répertoires protégés par mot de passe. Cela peut, bien sûr, être fait en utilisant des fichiers .htaccess, mais de nombreux utilisateurs ne savent pas comment créer ou utiliser des fichiers .htaccess, et ce module fournit un moyen basé sur le Web pour effectuer la même tâche.
