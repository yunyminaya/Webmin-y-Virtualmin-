ProFTPd IP-basierter virtueller Server aktiviert?
Aktiviert oder deaktiviert einen neuen virtuellen FTP-Host. Beachten Sie, dass für FTP eine neue IP für das anonyme Hosting virtueller Domänen erforderlich ist. Die überwiegende Mehrheit der Websitebesitzer benötigt diese Funktion nicht und sie ist nicht für den authentifizierten FTP-Zugriff auf die Dateien der Domänenbesitzer erforderlich. Wenn für einige Domänen ein virtuell gehosteter anonymer FTP-Zugriff erforderlich ist (dh wenn sie einen FTP-Dateiserver für Software oder Dateidownloads für nicht angemeldete Benutzer betreiben), möchten Sie ihn möglicherweise auf "Ja, aber nicht standardmäßig" setzen. .
