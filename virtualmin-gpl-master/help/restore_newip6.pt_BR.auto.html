Endereço IPv6 para servidores restaurados
Essa configuração é usada apenas ao restaurar um servidor virtual que não existe atualmente no sistema. Por padrão, o Virtualmin atribui a ele o endereço IPv6 compartilhado padrão do sistema, se ele estava em um IP compartilhado originalmente, ou usa o endereço IPv6 privado do sistema original, se não estiver em um endereço compartilhado.

Isso funciona bem se você estiver movendo um domínio entre dois sistemas na mesma rede, mas, caso contrário, o domínio restaurado provavelmente não poderá ser contatado pela rede. Para evitar isso, selecione a opção Virtual com IP alocado ou Virtual com IP neste campo, e o Virtualmin selecionará um IP gratuito para você ou usará o que você digitar.
