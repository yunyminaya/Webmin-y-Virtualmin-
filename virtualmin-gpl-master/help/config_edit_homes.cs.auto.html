Lze vybrat domovské adresáře pro uživatele?
Pokud je povoleno, majitel domény se může rozhodnout dát uživatelům jiný domovský adresář než výchozí. Může být užitečné, pokud majitelé domén mohou instalovat další služby, jako je aplikační server (tj. Zope, Webrick atd.), A chtěli by pro aplikační server použít jiného uživatele. Obecně bude tento druh flexibility potřebovat pouze uživatel, který má relativně vysokou úroveň důvěryhodnosti.
