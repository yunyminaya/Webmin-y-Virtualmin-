Доменное имя для использования в ссылках
По умолчанию, когда Virtualmin отображает ссылку на веб-сайт этого виртуального сервера (например, после установки скрипта), он использует доменное имя сервера. Однако вместо этого вы можете использовать это поле, чтобы выбрать псевдоним для использования в ссылках. Это может быть полезно, если сервер доступен только снаружи через псевдоним домена.
