Befehl zum Abrufen des Kontingents einer Gruppe
Wenn externe Kontingentbefehle verwendet werden, kann das in dieser Option festgelegte Programm von Virtualmin aufgerufen werden, um das Kontingent für eine einzelne Gruppe abzurufen. Es wird mit einem einzelnen Parameter aufgerufen - dem Unix-Gruppennamen - und muss eine einzelne Zeile mit den folgenden durch Leerzeichen getrennten Feldern ausgeben:
  1. Aktuelles Kontingent in 1-kB-Blöcken.
  2. Das weiche Kontingent in 1-kB-Blöcken oder 0 für unbegrenzt.
  3. Das harte Kontingent in 1-kB-Blöcken oder 0 für unbegrenzt.
Wenn dieser Befehl nicht auf der Seite "Modulkonfiguration" festgelegt ist, verwendet Virtualmin den Befehl, um stattdessen Gruppenkontingente aufzulisten .
