Перенаправить admin.domain в Usermin?
Если для этого поля установлено значение Да (как по умолчанию), любые новые виртуальные серверы будут иметь добавленную в свои домены DNS запись admin с тем же IP-адресом, что и на веб-сайте сервера. При доступе по http://admin.thedomain.com/ браузер будет перенаправлен на Virtualmin через порт 10000 того же домена для доступа к веб-интерфейсу администрирования.

Если вы хотите использовать другой URL-адрес для доступа к Virtualmin (например, тот, для которого у вас есть сертификат SSL), URL-адрес перенаправления можно настроить в следующем URL-адресе для поля перенаправления администратора .
