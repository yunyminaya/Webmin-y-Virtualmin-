Pagina HTML predefinita del sito web
Questa opzione consente di controllare la pagina predefinita che verrà creata e visualizzata per il nuovo server virtuale.

Puoi scegliere "Disabilitato" per saltare la creazione di una pagina oppure optare per una pagina predefinita del sito web Virtualmin selezionando l'opzione "Sì, con il modello di pagina predefinita del sito web".
