Команда для получения квоты пользователя
Когда используются внешние команды квот, Virtualmin может вызывать программу, заданную в этом параметре, для получения квоты для одного пользователя. Он вызывается с одним параметром - полным именем пользователя Unix и должен выводить одну строку, содержащую следующие разделенные пробелами поля:
  1. Текущая квота используется в блоках по 1 кБ.
  2. Мягкая квота в блоках по 1 кБ или 0 для неограниченного количества.
  3. Жесткая квота в блоках по 1 кБ или 0 для неограниченного количества.
Если эта команда не задана на странице «Конфигурация модуля», Virtualmin будет использовать команду вместо списка пользовательских квот .
