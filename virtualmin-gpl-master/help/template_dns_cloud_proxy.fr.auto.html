Activer le proxy sur les nouveaux enregistrements ?
Pour les fournisseurs Cloud DNS qui prennent en charge le proxy, cette option détermine si les enregistrements dans les nouveaux domaines sont créés avec le proxy activé ou non. Vous pouvez choisir de proxy tous les enregistrements ou uniquement ceux utilisés pour l'hébergement du site Web (domaine.com et www.domaine.com).
