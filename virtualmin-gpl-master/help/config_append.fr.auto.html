Inclure le nom de domaine dans les noms d'utilisateur?
Cette option sélectionne la façon dont les noms d'utilisateur seront générés. Par défaut, Virtualmin utilisera le nom d'utilisateur de base demandé, mais s'il se heurte à un nom d'utilisateur existant, il ajoutera le nom de domaine de la manière spécifiée dans l'option suivante. Si la cohérence est préférable, définissez-la sur "Toujours" et tous les noms d'utilisateur au sein des sous-serveurs auront le nom de domaine ajouté.
