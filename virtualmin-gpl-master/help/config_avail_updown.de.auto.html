Hochladen und Herunterladen (als Benutzer)
Diese Option gibt an, dass der Domäneninhaber Zugriff auf das Webmin-Modul zum Hochladen und Herunterladen hat. Das Upload- und Download-Modul bietet Benutzern einen sehr einfachen Mechanismus zum Hochladen von Dateien auf oder Herunterladen von Dateien von anderen Websites auf den Server, ohne dass DAV, FTP, SSH, FISH oder ein anderes möglicherweise einschüchterndes Dateiübertragungsprotokoll verwendet werden müssen neue Nutzer. Es handelt sich ausschließlich um einen webbasierten Mechanismus zum Abrufen von Daten auf das System.
