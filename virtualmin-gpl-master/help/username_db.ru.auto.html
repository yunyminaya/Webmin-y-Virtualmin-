Имя пользователя
В этом поле задается часть имени пользователя для входа в систему, которая должна быть уникальной в базе данных. Логин пользователя для доступа к базе данных будет получен из введенного здесь имени пользователя в сочетании с именем домена. Например, если вы введете имя пользователя db-user1 и домен example.com , полученный логин будет db-user1@example.com .
