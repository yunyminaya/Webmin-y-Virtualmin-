Creazione utente amministrazione abilitata?
Questa opzione determina se la creazione di un utente Unix per un nuovo server virtuale di livello superiore viene sempre eseguita (e l'opzione nascosta) o se è facoltativa durante la creazione di un server. È generalmente una buona idea impostare questa opzione su Sempre , poiché è necessario un utente Unix a meno che non si crei un dominio solo DNS o solo proxy Web.
