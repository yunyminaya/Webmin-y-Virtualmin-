Výchozí adresa IP pro záznamy DNS
Záznamy DNS jsou automaticky generovány pro nové virtuální počítače a za normálních okolností budete chtít, aby odpovídaly výše uvedené adrese IP virtuálního serveru. Pokud však váš server stojí za směrovačem NAT, provozujete server proxy, server vyrovnávající zatížení nebo jiné zařízení, které vede k tomu, že váš server pracuje na jiné adrese IP, než jakou vidí svět při rozhovoru s vašimi doménami, může být nutné změnit hodnotu zde. Dalším případem, kdy byste to mohli chtít změnit, je přidání domén na vývojový server, který bude mít jinou adresu, pokud jde o produkci. Zde můžete zadat produkční nebo veřejnou IP adresu, místo aby Virtualmin používal lokální IP adresu.

Pokud je vybrána možnost Automaticky detekovat externí adresu , pokusí se Virtualmin pokusit zjistit IP v době vytvoření virtuálního serveru. To se provádí zadáním požadavku HTTP na adresu www.virtualmin.com , která odpoví na adresu, ze které vidí připojení přicházející.
