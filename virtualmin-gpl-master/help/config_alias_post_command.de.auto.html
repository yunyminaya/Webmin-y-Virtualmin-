Befehl zum Ausführen, nachdem Änderungen an einem Alias vorgenommen wurden
Mit dieser Option können Sie ein Skript oder einen Befehl angeben, der ausgeführt werden soll, nachdem ein E-Mail-Alias erstellt, geändert oder gelöscht wurde. Mithilfe der Umgebungsvariablen ALIAS_FROM, ALIAS_TO und ALIAS_ACTION kann ermittelt werden, welcher Alias geändert wurde und was damit passiert ist.
