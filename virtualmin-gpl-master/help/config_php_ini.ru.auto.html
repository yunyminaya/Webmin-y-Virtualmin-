Полный путь к основному файлу конфигурации PHP
В этом поле должен быть указан полный путь к файлу php.ini в вашей системе. Он используется только тогда, когда Virtualmin запускает сценарии PHP от имени владельца виртуального сервера через suEXEC, и копируется в домашний каталог каждого домена для модификации каждого сервера.
