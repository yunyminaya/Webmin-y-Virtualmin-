Groep eigenaarschap instellen van MySQL database files?
Wanneer deze optie is aangezet zal bij iedere nieuwe MySQL database de groep eigenaarschap worden ingesteld op zijn data files (typisch onder /var/lib/mysql) waardoor zij worden toegevoegd in hun virtuele server Unix quote limiet. Indien dit niet is aangezet is het database gebruik niet gelimiteerd en zal het mogelijk zijn voor een virtuele server om over zijn toegestane harddisk gebruik te gaan.
