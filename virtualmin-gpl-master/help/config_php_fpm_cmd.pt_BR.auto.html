Servidor PHP FPM adicional
Por padrão, o Virtualmin tentará encontrar o servidor PHP FPM pesquisando os pacotes do sistema, mas se você o instalou manualmente em um local diferente compilando a partir do código-fonte, o caminho para o servidor FPM pode ser definido aqui. Isso também é útil em outros sistemas operacionais como o FreeBSD, onde o Virtualmin não sabe como o PHP-FPM é empacotado. Se esta opção estiver definida, você também deverá definir as duas opções a seguir.
