Comando para ejecutar para mover el directorio de inicio de un dominio
De forma predeterminada, Virtualmin solo usará el comando mv para mover el directorio de inicio de un dominio cuando sea necesario, como cuando se cambia el nombre de un dominio. Sin embargo, esto puede no ser apropiado en los casos en que cada servidor virtual esté en su propio sistema de archivos, como un montaje ZFS.

En casos como este, puede establecer este campo en un comando o script que se pasará a los directorios de inicio antiguos y nuevos como parámetros. También tendrá acceso a todas las variables de entorno que normalmente se pasan a los comandos previos y posteriores a la modificación.
