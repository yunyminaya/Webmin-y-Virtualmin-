Limite al numero di server alias
Se il proprietario del dominio dispone dei privilegi di creazione di un sub-server, è possibile limitare il numero di server alias che è possibile creare qui. Si noti che questo limite non può essere effettivamente superiore al limite predefinito sul numero di server virtuali , poiché il limite complessivo sui server virtuali si applica sia a domini alias che non alias. Un limite di 0 impedisce completamente la creazione di domini alias.
