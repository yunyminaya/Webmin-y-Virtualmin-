Sicherungsformat
Dieses Feld bestimmt, ob die Sicherung als einzelne Datei geschrieben wird oder ob für jeden ausgewählten virtuellen Server mit dem Namen domain .tar.gz eine Datei im Ziel erstellt wird. Mit der letzten Option wird auch eine Datei pro Domäne erstellt, jedoch in einem neuen Format, das von Virtualmin 2.86 und höher unterstützt wird und alle Informationen zu jedem Server in seinem Ausgangsverzeichnis ablegt. Dieses Format ist überlegen, da Virtualmin keine temporären Dateien im Verzeichnis / tmp erstellen muss und somit Speicherplatz spart.
