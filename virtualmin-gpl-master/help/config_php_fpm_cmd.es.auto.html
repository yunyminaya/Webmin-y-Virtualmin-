Servidor PHP FPM adicional
De forma predeterminada, Virtualmin intentará encontrar el servidor PHP FPM buscando paquetes del sistema, pero si lo instaló manualmente en una ubicación diferente al compilar desde el código fuente, la ruta al servidor FPM se puede configurar aquí. Esto también es útil en otros sistemas operativos como FreeBSD donde Virtualmin no sabe cómo está empaquetado PHP-FPM. Si esta opción está configurada, también debe configurar las dos opciones siguientes.
