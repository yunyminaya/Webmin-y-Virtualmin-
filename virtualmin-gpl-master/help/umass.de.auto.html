Batch Benutzer erstellen
Auf dieser Seite können Sie mehrere E-Mail- oder FTP-Benutzer gleichzeitig aus einer Textdatei erstellen. Diese Datei muss einen Benutzer pro Zeile auflisten und jede Zeile muss das folgende Format haben:

username : realname : password : ftp-flag : email-flag : quota :[ extras ]:[ forwards ]:[ databases ]

Die Bedeutungen der Felder sind:

Nutzername
Der kurze Anmeldename des Benutzers, z. B. Joe . Bei Bedarf wird das Domänenpräfix angehängt, um den tatsächlichen Anmeldenamen zu erstellen.
echter Name
Der wirkliche Name des Benutzers, wie Joe Smith .
Passwort
Das anfängliche Passwort für den Benutzer im Klartext.
FTP-Flagge
Wenn dies auf 1 gesetzt ist, kann sich der Benutzer über FTP anmelden. Wenn Sie den Wert auf 0 setzen oder leer lassen, wird der FTP-Zugriff verhindert. Sie können auch den vollständigen Pfad zu einer gültigen Shell für Postfachbenutzer wie /usr/bin/scponly .
E-Mail-Flag
Wenn dies auf 1 gesetzt ist, wird die primäre E-Mail-Adresse des Benutzers aktiviert.
Quote
Das E-Mail- und Home-Verzeichnis-Kontingent des Benutzers in 1-KB-Festplattenblöcken.
Extras
Eine durch Kommas getrennte Liste zusätzlicher E-Mail-Adressen für diesen Benutzer. Diese müssen sich in von Ihnen verwalteten Virtualmin-Domänen befinden.
vorwärts
Eine durch Kommas getrennte Liste von E-Mail-Weiterleitungszielen für diesen Benutzer (normalerweise Adressen wie joe@otherdomain.com ).
Datenbanken
Eine durch Kommas getrennte Liste von Datenbanken, auf die dieser Benutzer zugreifen kann. Jeder muss ein Datenbanktyp ( MySQL oder Postgres ) und ein Datenbankname sein, die durch ein Leerzeichen getrennt sind.
