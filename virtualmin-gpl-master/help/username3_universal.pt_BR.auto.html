Nome de usuário de login
Este campo define a parte do nome de usuário do login do usuário, que deve ser exclusivo no domínio. O login do usuário para acessar serviços como SSH, FTP, e-mail, banco de dados e autenticação de servidor web será derivado do nome de usuário inserido aqui, combinado com o nome de domínio. Por exemplo, se você inserir o nome de usuário user1 e o domínio for example.com , o login resultante será user1@example.com .
