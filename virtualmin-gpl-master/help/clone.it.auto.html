Clona server virtuale
Questa pagina ti consente di duplicare un server virtuale esistente con un nuovo nome di dominio. Qualsiasi contenuto web, record DNS, caselle di posta, alias di posta, database e altre impostazioni associate al dominio originale verranno duplicati, ove possibile. Tuttavia, il contenuto web non verrà modificato in alcun modo, quindi se il server virtuale contiene un'applicazione web che è stata configurata per utilizzare i database o la home directory del vecchio server, sarà necessario riconfigurarla manualmente dopo la clonazione.

Quando duplichi un server virtuale di primo livello devi anche inserire un nuovo nome utente e una nuova password per possedere il nuovo dominio. Se il server ha un indirizzo IP privato, verrà assegnato automaticamente un nuovo IP per il clone.

Il processo di clonazione copierà tutti i database associati al dominio originale con nuovi nomi e ne duplicherà i contenuti. Tuttavia, potrebbe non aggiornare tutte le webapp installate nel dominio originale configurate per utilizzare i database originali. Questo deve essere fatto manualmente.
