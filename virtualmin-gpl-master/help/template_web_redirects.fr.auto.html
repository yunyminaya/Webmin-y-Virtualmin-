Redirections vers de nouveaux sites Web
Ce tableau vous permet de saisir les redirections de sites Web qui seront ajoutées à tous les nouveaux serveurs virtuels lors de leur création. Pour chaque redirection, vous devez entrer un chemin source comme / ou /phpadmin , et soit une URL de destination, soit un chemin relatif au répertoire public_html du domaine.

Dans les deux champs, des variables de modèle de serveur virtuel telles que ${DOM} et ${USER} peuvent être utilisées. Vous pouvez également sélectionner si la redirection s'applique lorsque le site est accessible via HTTP, HTTPS ou les deux.
