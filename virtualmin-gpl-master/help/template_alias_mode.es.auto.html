Crear sitios web de alias por
Un alias de dominio, también conocido como estacionamiento de dominio o reenvío de dominio, es un nuevo dominio que simplemente reenvía las solicitudes web y el correo electrónico al dominio principal del que es un alias. Hay varios tipos de alias disponibles. Actualmente, el tráfico web se puede dirigir al dominio principal a través de un conjunto de reglas de ProxyPass, una regla de redireccionamiento, una regla de redireccionamiento o simplemente agregando un ServerAlias al dominio principal.
