Serveur virtuel basé sur IP ProFTPd activé?
Active ou désactive un nouvel hôte virtuel FTP. Notez que FTP nécessite une nouvelle IP pour l'hébergement de domaine virtuel anonyme. La grande majorité des propriétaires de sites Web n'ont pas besoin de cette fonctionnalité, et elle n'est pas requise pour un accès FTP authentifié aux fichiers des propriétaires de domaine. Cependant, si certains domaines nécessitent un accès FTP anonyme hébergé virtuel (c'est-à-dire s'ils exploitent un serveur de fichiers FTP pour les logiciels ou les téléchargements de fichiers pour les utilisateurs non connectés), vous pouvez le définir sur "Oui, mais pas par défaut". .
