Webalizer Cron-Job für jeden virtuellen Server einrichten?
Wenn diese Option auf Ja gesetzt ist (wie standardmäßig), erstellt Virtualmin für jeden virtuellen Server, für den Webalizer aktiviert ist, einen separaten Cron-Job, um täglich einen Bericht aus seinen Protokollen zu generieren.

Auf einem Server mit einer großen Anzahl von Domänen möchten Sie diese Option möglicherweise stattdessen in Nein ändern und stattdessen den Befehl /usr/libexec/webmin/virtual-server/run-all-webalizer.pl mithilfe von Cron manuell planen. Dadurch wird sichergestellt, dass Webalizer für jede Domäne nacheinander ausgeführt wird, anstatt dass möglicherweise mehrere Domänen gleichzeitig verarbeitet werden, und Sie haben mehr Kontrolle darüber, wann genau die Verarbeitung abgeschlossen ist.
