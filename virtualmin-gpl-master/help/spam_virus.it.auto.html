Destinazione per email di virus
Questo campo determina cosa succede ai messaggi di posta elettronica che sono stati identificati da ClamAV come contenenti un virus. Le opzioni disponibili sono:

Buttar via
Eventuali virus rilevati verranno immediatamente eliminati.
Scrivi nel file virus standard ~ / mail / virus
I virus vengono scritti nella cartella standard per l'analisi manuale e la cancellazione.
Scrivi nel file di posta nella home directory
I virus vengono salvati nel file di posta nella home directory dell'utente immessa. Per scrivere in una directory di formato Maildir, inserire un / alla fine del nome file.
Inoltra all'indirizzo e-mail
Eventuali messaggi carichi di virus verranno inoltrati all'indirizzo inserito.
Scrivi in un altro file
