Grupo de administração padrão para proprietários de domínio
Por padrão, o grupo para o usuário Unix criado para possuir os arquivos deste domínio será o mesmo que o grupo para seus usuários de email. No entanto, você pode selecionar um grupo existente no servidor, o que pode ser útil se você tiver algum tipo de restrição de acesso baseado em grupo em vigor (como no FTP).
