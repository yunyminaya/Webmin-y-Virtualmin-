Formato di backup
Questo campo determina se il backup è scritto come un singolo file o se viene creato un file nella destinazione per ciascun server virtuale selezionato, denominato domain .tar.gz. L'opzione finale creerà anche un file per dominio, ma in un nuovo formato supportato da Virtualmin 2.86 e versioni successive che mette tutte le informazioni su ciascun server nella sua home directory. Questo formato è superiore perché non richiede Virtualmin per creare file temporanei nella directory / tmp, risparmiando così spazio su disco.
