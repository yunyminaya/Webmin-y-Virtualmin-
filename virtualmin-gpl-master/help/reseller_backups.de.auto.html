Können virtuelle Server gesichert werden?
Dieses Feld bestimmt, ob der Reseller die Sicherungsfunktionen von Virtualmin nutzen darf. Die verfügbaren Optionen sind:

Planen und erstellen Sie Backups
Der Reseller kann sowohl Backups planen, die regelmäßig automatisch ausgeführt werden, als auch die Weboberfläche verwenden, um Backups manuell auszuführen.
Erstellen Sie nur Backups
Backups können nur manuell über die Weboberfläche ausgeführt werden.
Weder
Der Reseller hat keinen Zugriff auf Backup-Funktionen.
