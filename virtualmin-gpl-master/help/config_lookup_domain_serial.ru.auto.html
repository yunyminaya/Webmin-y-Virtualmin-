Поиск домена для входящей электронной почты
По умолчанию процесс VirtualMup lookup-domain-daemon запускает отдельный подпроцесс, чтобы найти настройки домена и спама для каждого входящего сообщения в вашей системе. Однако это может потребовать большого количества процессорного времени и памяти, если несколько сообщений поступают одновременно.

Чтобы вместо этого обрабатывать электронную почту один за другим, выберите параметр «По одному за раз» для этого поля. Доставка почты будет немного медленнее, но использование процессора и памяти будет ограничено.
