Группа для Администрации Пользователя
По умолчанию группа для пользователя Unix, созданного для владения файлами этого домена, будет такой же, как группа для его пользователей почты. Однако вместо этого вы можете выбрать существующую группу на сервере, что может быть полезно, если у вас есть какие-то действующие ограничения доступа на основе групп (например, для FTP).
