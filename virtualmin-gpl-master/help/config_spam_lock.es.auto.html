¿Solo ejecuta un proceso de spamassasin a la vez?
Cuando este campo se establece en No como está por defecto, Virtualmin ejecutará múltiples procesos concurrentes de spamassassin si llegan varios mensajes de correo electrónico al mismo tiempo. Sin embargo, esto puede usar CPU y RAM excesivas en un sistema pequeño. Si selecciona Sí , cualquier servidor virtual con filtro de spam habilitado tendrá sus archivos de configuración de Procmail actualizados para ejecutar solo una copia de spamassassin a la vez.

Esta restricción se aplica a todos los servidores virtuales, por lo que incluso si llegan dos mensajes a dominios diferentes al mismo tiempo, solo se ejecutará un proceso de spamassassin .
