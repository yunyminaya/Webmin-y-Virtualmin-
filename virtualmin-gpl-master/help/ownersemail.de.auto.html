Kontakt-E-Mailadresse
In diesem Feld können Sie eine E-Mail-Adresse angeben, an die alle Nachrichten zu diesem virtuellen Server gesendet werden. Standardmäßig werden sie an den Administrationsbenutzer gesendet, dem der Server gehört. Die häufigste Nachricht wird beim Erstellen der Domain gesendet und enthält Informationen zu Adresse, Funktionen, Anmeldung und Kennwort.
