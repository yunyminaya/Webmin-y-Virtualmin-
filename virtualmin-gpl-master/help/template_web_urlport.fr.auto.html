Numéro de port externe pour les hôtes virtuels
Ce paramètre peut être utilisé pour modifier le numéro de port utilisé par Virtualmin dans les URL qui référencent les domaines créés à partir de ce modèle. Cela peut être utile si votre système a un proxy ou un cache en face d'écouter sur le port 80, et Virtualmin a été configuré pour créer des hôtes virtuels Apache sur un port différent.
