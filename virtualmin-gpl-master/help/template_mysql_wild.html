Wildcard for additional allowed MySQL databases
This field is for entering a MySQL wildcard pattern that grants access to additional databases - perhaps something like ${DB}_%. For a domain like example.com, this would be expanded to example_%, which would allow access to all databases starting with example_.

Because Virtualmin creates MySQL access rules for all databases it creates, this field can usually be left blank. It only needs to be filled in if you are creating additional databases via some other method outside of Virtualmin's control.
