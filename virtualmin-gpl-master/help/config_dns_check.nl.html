Controleer resolv.conf voor dit systeem?
Wanneer dit is ingesteld op Ja (wat het standaard is) zal Virtualmin de systeem configuratie file /etc/resolv.conf controleren om er zeker van te zijn dat het correct is ingesteld om de locale DNS server te gebruiken voor naam resolutie. Dit is nodig omdat zonder deze functie de DNS domeinen die u aanmaakt niet op te halen zijn door het systeem waar Virtualmin op werkt totdat deze volledig geregistreerd zijn door een DNS registrator.

Indien u er zeker van bent dat dit niet nodig is (misschien omdat u een automatische secundaire DNS server heeft opgezet) kan dit verandert worden naar Nee om te voorkomen dat er configuratie fouten gaan optreden.
