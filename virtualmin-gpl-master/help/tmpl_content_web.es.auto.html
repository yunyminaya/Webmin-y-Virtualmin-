Página HTML predeterminada del sitio web
Esta opción le permite controlar la página predeterminada que se creará y se mostrará para el nuevo servidor virtual.

Puede elegir "Deshabilitado" para omitir la creación de una página u optar por una página de sitio web predeterminada de Virtualmin seleccionando la opción "Sí, con la plantilla de página predeterminada del sitio web".
