Дополнительные места для команд PHP
Virtualmin автоматически определяет основные версии PHP, установленные в вашей системе, если они находятся в каталогах в пути поиска. Однако это поле можно использовать для ввода путей к дополнительным исполняемым файлам PHP, что может быть полезно, если вы хотите запустить версию, которая выше, чем та, которая установлена по умолчанию.
