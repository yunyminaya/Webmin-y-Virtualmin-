Inclure les domaines d'alias dans les limites?
Les domaines d'alias sont des domaines qui pointent vers un domaine existant pour certains ou tous les services. Si vous souhaitez considérer les domaines d'alias comme faisant partie du domaine qu'ils alias à des fins de quota, sélectionnez oui ici. Si les limites de compte seront spécifiées séparément, sélectionnez non. Les domaines d'alias sont normalement assez petits, car ils dirigent simplement tout le trafic vers le domaine dont ils sont des alias, mais avec les privilèges appropriés, un propriétaire de domaine peut utiliser le compte de domaine d'alias à d'autres fins.
