Valider les enregistrements MX pour les domaines avec e-mail?
Lorsque cette option est définie sur Oui (comme c'est le cas par défaut), les enregistrements MX sur les serveurs virtuels activés par courrier électronique seront validés pour garantir qu'ils pointent vers le système Virtualmin. C'est normalement ce que vous voulez, sauf si vous disposez d'un filtre anti-spam ou d'un autre redirecteur qui accepte le courrier de vos domaines avant de le transférer vers le système Virtualmin. Dans ce cas, vous devez sélectionner Non pour éviter les fausses erreurs de validation.
