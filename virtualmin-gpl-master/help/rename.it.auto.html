Cambia nome dominio
Questo modulo può essere utilizzato per modificare il nome di dominio di un server virtuale esistente. Per impostazione predefinita, anche il nome utente e la directory home verranno adattati per corrispondere al nuovo dominio, supponendo che siano basati su di esso.

Può anche essere utilizzato per modificare il nome utente di amministrazione o la directory principale senza toccare affatto il nome del dominio, ad esempio se si desidera passare a una nuova directory di base. Tuttavia, solo l'amministratore principale può modificare il nome utente o spostare la casa in una posizione arbitraria.

Per impostazione predefinita, gli utenti di cassette postali esistenti nel dominio non vengono rinominati in modo che i loro accessi IMAP funzionino. È possibile forzare una modifica impostando il suffisso Modifica nome cassetta postale? a Scelto automaticamente però. Questo campo può anche essere usato per cambiare il suffisso in qualsiasi cosa tu voglia, anche se il nome di dominio non è cambiato.

Se un dominio utilizza il formato username@domain.com per le cassette postali, gli utenti verranno rinominati solo se il nome di dominio effettivo cambia.
