Domini di posta mittente consentiti
Questo campo elenca i nomi di dominio i cui server di posta (identificati dai record MX DNS) sono autorizzati a inviare e-mail da questo dominio. Poiché i server di posta per un dominio cambiano spesso, si consiglia di inserire qui i nomi di dominio anziché i nomi host del server di posta nel campo Nomi host mittente consentiti , ove appropriato.
