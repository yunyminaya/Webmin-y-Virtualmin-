Comando para obtener la cuota de un usuario
Cuando se utilizan comandos de cuota externos, Virtualmin puede llamar al programa configurado en esta opción para obtener la cuota para un solo usuario. Se llama con un solo parámetro: el nombre de usuario completo de Unix, y debe generar una sola línea que contenga los siguientes campos separados por espacios:
  1. Cuota actual utilizada en bloques de 1 kB.
  2. La cuota flexible en bloques de 1 kB, o 0 para ilimitado.
  3. La cuota fija en bloques de 1 kB, o 0 para ilimitado.
Si este comando no está configurado en la página de configuración del módulo, Virtualmin usará el comando para enumerar las cuotas de los usuarios .
