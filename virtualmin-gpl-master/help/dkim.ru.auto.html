DomainKeys Идентифицированная Почта
DomainKeys Identified Mail (DKIM) - это протокол для подписи исходящих сообщений электронной почты с помощью подписи, подтверждающей, что она действительно исходит из домена отправителя. Это может использоваться почтовыми серверами получателей или фильтрами спама для обнаружения подделки отправителя, что обычно делается спамерами, чтобы избежать попадания своего домена в черный список.

Когда DKIM включен с помощью Virtualmin, ваш почтовый сервер будет настроен на добавление подписи DKIM ко всем исходящим сообщениям, и открытый ключ будет добавлен к каждому из доменов DNS в вашей системе. Другие почтовые серверы могут затем использовать этот ключ для проверки подписи на ретранслируемых сообщениях, таким образом гарантируя, что адрес отправителя не был подделан.

Если у вас есть виртуальные серверы, для которых DNS размещен где-то еще, а почта размещена в этой системе, вы можете ввести их доменные имена в Дополнительные домены, чтобы подписать их . Записи в поле DNS-записи для дополнительных доменов могут быть затем вручную добавлены в эти зоны.
