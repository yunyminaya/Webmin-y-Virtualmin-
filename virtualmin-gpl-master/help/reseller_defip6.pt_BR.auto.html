Endereço IPv4 para servidores baseados em nome
Essa opção pode ser usada para que os servidores virtuais criados para este revendedor usem um endereço IPv6 diferente do padrão do sistema, que geralmente é compartilhado por todos os sites virtuais baseados em nome. Isso pode ser útil para servidores de "criação de marca" criados por revendedores diferentes, fazendo com que o endereço IPv4 de cada revendedor resolva para um nome de host diferente.

O IP digitado já deve estar ativo no sistema, o que pode ser feito no módulo de configuração de rede do Webmin. O IP não será usado para servidores virtuais com seus próprios IPs privados, pois cada um sempre obtém um endereço distinto.
