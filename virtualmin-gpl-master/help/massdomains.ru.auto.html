Обновить виртуальные серверы
Эта страница позволяет обновлять настройки и ограничения на нескольких виртуальных серверах одновременно, как было выбрано на предыдущей странице. По умолчанию ничего не будет изменено на выбранных серверах, пока вы не выберете какой-либо атрибут или ограничение для изменения, используя поля на этой странице. Изменяемые настройки делятся на следующие категории:

Изменение квоты и пропускной способности
Эти поля можно использовать для изменения дисковых квот для выбранных серверов или для изменения их ограничений пропускной способности. Изменение квот влияет только на серверы верхнего уровня, так как подсерверы наследуют свои квоты от родительского домена.

Включены изменения функций
В этом разделе вы можете выбрать функции для включения или выключения для всех выбранных серверов. Это те же функции, которые можно активировать для отдельных серверов на странице «Редактировать виртуальный сервер» .

Изменения в пределах владельца
Эти поля можно использовать для изменения различных ограничений, которые применяются к владельцам виртуальных серверов верхнего уровня, таких как количество почтовых ящиков, псевдонимов и доменов, которые они могут создать. Внесенные здесь изменения не влияют на субсерверы, которые подпадают под ограничения, установленные для их родителей.

Изменения в возможностях владельца
Поля в этом разделе управляют возможностями Virtualmin, которые доступны администраторам выбранных виртуальных серверов. Опять же, они актуальны только для серверов верхнего уровня.

Изменения в параметрах PHP
По возможности этот раздел позволяет изменить как версию PHP по умолчанию, так и способ запуска сценариев PHP на всех выбранных серверах. Это полезно только на серверах, которые имеют веб-сайты.

Изменения конфигурации
Этот раздел предназначен для внесения различных изменений в конфигурацию выбранных серверов, таких как метод, используемый для очистки от спама, или оболочка входа администратора домена.

Также можно временно отключить выбранные серверы с помощью кнопки «Отключить серверы» в нижней части страницы. Используя это, вы также можете ввести причину отключения (например, не оплата ) в соседнем текстовом поле.

Чтобы повторно включить несколько серверов, которые были отключены на этой странице или в другом месте, используйте кнопку «Включить серверы» .
