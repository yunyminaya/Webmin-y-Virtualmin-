Destinations de transfert de courrier électronique
Ce tableau de champs détermine ce qui arrive aux e-mails envoyés à cet utilisateur. Normalement, ils sont envoyés directement dans la boîte aux lettres de l'utilisateur, mais si des destinations sont définies ici, elles seront utilisées à la place. Cela peut être utilisé pour configurer un répondeur automatique si l'utilisateur est absent ou pour transférer son e-mail vers une adresse sur un autre serveur.

Chaque destination possède un type (choisi dans le menu), qui peut être l'un des suivants :

  • Adresse email
    Le courrier électronique sera transmis à l'adresse indiquée dans le champ adjacent.

  • Adresses dans le fichier
    Le courrier électronique sera transmis à toutes les adresses répertoriées dans le fichier texte dont le chemin est saisi dans le champ adjacent.

  • Écrire dans un fichier
    L'e-mail envoyé à l'alias sera ajouté au fichier spécifié. Tous les en-têtes et le corps brut non décodé seront inclus.

  • Nourrir le programme
    Le programme saisi dans le champ adjacent sera exécuté et envoyé par courrier électronique à l'alias qui lui a été transmis en entrée.

  • Réponse automatique à partir du fichier
    Lorsque l'utilisateur reçoit un e-mail, un message de réponse automatique est envoyé à l'expéditeur. Le texte de cette réponse automatique est extrait du fichier saisi dans le champ de texte adjacent.

  • Appliquer le fichier de filtre
    Les courriers électroniques envoyés à l'utilisateur seront traités conformément aux règles du fichier de filtre spécifié, ce qui peut entraîner leur transfert vers d'autres adresses ou fichiers en fonction du contenu réel du message.

  • Boîte mail de l'utilisateur
    L'e-mail sera envoyé directement dans la boîte aux lettres de l'utilisateur indiqué dans le champ de texte adjacent. En règle générale, vous entrez le nom de l'utilisateur lui-même, afin qu'il reçoive une copie de l'e-mail en plus de le transférer vers d'autres destinations. Cependant, vous devez entrer le nom d'utilisateur de connexion POP3 complet, comme joe.foo au lieu de simplement joe .

Si vous choisissez les types de destination Adresses dans le fichier , Réponse automatique à partir du fichier ou Appliquer le filtre de fichier , lorsque vous rééditez les utilisateurs, des liens intitulés Modifier... apparaîtront à côté de ces destinations pour modifier le contenu du fichier spécifié.
