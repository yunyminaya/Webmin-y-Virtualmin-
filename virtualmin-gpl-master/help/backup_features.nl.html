Backup functies
Deze selectievakjes bepalen welke informatie van iedere virtuele server word toegevoegd aan de backup. Normaal gesproken wilt u alles hebben, dus alle selectievakjes moeten dan aangevinkt zijn.

Sommige functie's hebben nog eens extra opties ernaast staan om preciezer te bepalen wat word toegevoegd aan de backup. Bijvoorbeeld, u kunt kiezen of de log's directory onder iedere server home directory moet worden toegevoegd, omdat log files altijd een grote hoeveelheid harddisk ruimte gebruiken kan dit niet handig zijn en vaak zijn de log files niet belangrijk genoeg om er een backup van te hebben.

Let op dat bij een backup maken van een server er geen informatie over een sub-server word toegevoegd als u dit niet expliciet heeft geselecteerd.
