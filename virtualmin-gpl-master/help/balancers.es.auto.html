Caminos Proxy
Proxying le permite asignar una ruta URL en el sitio web de un servidor virtual a una o más URL de destino. Esto normalmente se usa para hacer que las aplicaciones de Ruby on Rails que se sirven en varias instancias de Mongrel estén disponibles bajo una ruta URL en el sitio web del servidor. También se puede utilizar para hacer que otros servidores web como Tomcat estén disponibles a través de una ruta URL en su sitio web.

Por ejemplo, puede asignar la ruta /radiant a las URL http://localhost:3001 y http://localhost:3002 , que son los puertos para dos instancias mestizas que sirven la misma aplicación que se ejecuta en su sistema Virtualmin.

Solo los sistemas que tienen instalado el módulo mod_proxy_balancer Apache pueden asignar una ruta a múltiples URL. Otros que solo admiten el módulo proxy normal de Apache pueden reenviar una ruta a una sola URL.
