Forcer le nom du groupe à être toujours le même que le nom d'utilisateur?
Lorsque ce paramètre est défini sur Oui , Virtualmin définit toujours le nom de groupe Unix pour les nouveaux domaines comme le nom d'utilisateur par défaut, sauf s'il est déjà dans l'utilisateur. Lorsque Non est choisi, le nom de groupe est extrait de la première partie du nom de domaine ou du domaine complet.
