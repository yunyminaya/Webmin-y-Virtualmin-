Crear usuario del servidor web
Crea un nuevo usuario del servidor web dentro de los archivos htpasswd , específicamente para la autenticación segura en directorios web protegidos. Este usuario es distinto de las cuentas Unix, dedicado exclusivamente a gestionar el acceso a directorios web. Además, este usuario se puede integrar perfectamente fusionándose con una cuenta Unix creada con el mismo nombre de usuario.
