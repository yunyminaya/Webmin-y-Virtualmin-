Directives et paramètres pour les nouveaux sites Web
Il est possible de personnaliser complètement les directives Apache qui sont ajoutées lors de la création d'un nouveau serveur virtuel. La substitution de variable est effectuée pour toutes les variables de modèle Virtualmin standard. Des précautions doivent être prises avec les directives Apache, car certains problèmes de sécurité actuels. La documentation Apache et dans une moindre mesure les sections Apache de la documentation Webmin peuvent être consultées pour trouver plus de détails sur la configuration d'Apache.
