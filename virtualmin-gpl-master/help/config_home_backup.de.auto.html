Sicherungsverzeichnis der Benutzer unter home
Wenn ein Domänenbesitzer eine Sicherungskopie an einem Ziel auf dem Server erstellt, kann diese nur im Unterverzeichnis unter der durch diese Konfigurationsoption festgelegten Startseite des Benutzers gespeichert werden. Sie können es ändern, wenn Ihnen die Standardeinstellung für virtualmin-backup nicht gefällt.
