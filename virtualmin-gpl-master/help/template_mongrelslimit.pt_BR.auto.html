Limite no número de instâncias do Mongrel
Ao instalar scripts Ruby on Rails, o administrador do servidor pode selecionar como meu servidor Mongrel processa para começar a manipular solicitações ao caminho do script - se o módulo mod_proxy_balancer do Apache estiver instalado. Para impedir que um número excessivo de processos do servidor Mongrel sobrecarregue o sistema, esta opção pode ser usada para definir o limite padrão do número que os administradores de domínio podem configurar no Virtualmin.
