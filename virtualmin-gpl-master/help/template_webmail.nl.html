Webmail.domain doorsturen naar Usermin?
Wanneer dit veld is ingesteld op Ja (zoals het is standaard is) zal iedere nieuwe virtuele server een webmail ingang krijgen naar hun DNS domeinen, met hetzelfde IP adres als de server website. Wanneer iemand de http://webmail.hetdomein.com/ benaderd zal de browser worden doorgestuurd naar Usermin op hetzelfde domein, om daar toegang te krijgen tot de op web gebaseerde email.

Indien u een andere webmail interface wilt gebruiken dan die van Usermin, zoals Squirrelmail of Roundcube, of een andere URL wilt gebruiken voor Usermin toegang kan de URL om doorgestuurd te worden geconfigureerd worden in het URL voor webmail doorsturen veld.
