Configuración PHP (para archivos php.ini del dominio)
Cuando está habilitada, esta opción otorga a los propietarios de servidores virtuales acceso al módulo de configuración de PHP. Esto estará restringido para que solo puedan administrar archivos php.ini en sus propios servidores virtuales, para lo cual la ejecución de PHP a través de CGI está habilitada.

Si un dominio usa mod_php para ejecutar scripts PHP, esta opción también controla si el propietario del servidor virtual puede configurar ajustes relacionados con PHP en Apache.
