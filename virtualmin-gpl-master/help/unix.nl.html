Server administrator aanmaken
Wanneer deze optie is ingesteld op Ja zal er een Unix gebruiker worden aangemaakt met dezelfde naam als de virtuele server administrator. Afhankelijk van de systeem configuratie kan dit worden gebruikt voor SSH en FTP login, en de gebruiker is meteen eigenaar van alle virtuele server files. Behalve wanneer u een server wilt hebben alleen voor DNS hosting moet u deze optie op Ja laten staan.
