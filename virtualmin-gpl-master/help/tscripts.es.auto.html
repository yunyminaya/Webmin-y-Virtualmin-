Instaladores de plantillas de aplicaciones web
Esta página le permite especificar aplicaciones web de terceros que se instalarán cuando se cree un nuevo servidor virtual utilizando esta plantilla. Para cada aplicación web, se debe ingresar la siguiente información:

Aplicación Web
El nombre y la versión de una aplicación web para instalar.
ruta URL
Una ruta en el sitio web del servidor virtual para instalar la aplicación web, como /wordpress . Esto también determinará el directorio en ~/public_html que contiene los archivos de la aplicación web.
nombre de base de datos
El nombre de la base de datos que utilizará la aplicación web para almacenar datos. Las opciones son:
Ninguno
Esto debe seleccionarse para aplicaciones web que no necesitan una base de datos, como Galería .
Base de datos principal
La base de datos de la aplicación web será la base de datos principal del servidor virtual del tipo elegido.
Otro
Si desea que esta aplicación web utilice su propia base de datos privada, puede seleccionar esta opción e ingresar el patrón de una base de datos en el campo de texto adyacente. Por ejemplo, podrías ingresar ${DB}_phpnuke . Si la base de datos resultante no existe, se creará como parte del proceso de instalación de la aplicación web.
tipo de base de datos
Se selecciona el tipo de uso de la base de datos para los datos de las aplicaciones web, si se selecciona alguno.
