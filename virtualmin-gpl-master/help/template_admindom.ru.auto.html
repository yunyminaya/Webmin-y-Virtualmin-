URL для перенаправления администратора
По умолчанию Virtualmin перенаправляет запросы на http://admin.thedomain.com на Webmin в той же системе, например https://thedomain.com:10000/ . Однако это поле можно использовать для изменения URL-адреса перенаправления, возможно, на одно имя хоста для всех ваших доменов, для которых у вас есть сертификат SSL, например https://yourcompany.com:10000/ .
