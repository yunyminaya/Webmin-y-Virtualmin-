Campi personalizzati
I campi personalizzati consentono all'amministratore di includere dati aggiuntivi sugli account di dominio. I campi verranno visualizzati durante la modifica del dominio. Possono essere utilizzati per quasi tutti i tipi di dati. Ad esempio, può essere utile disporre di ulteriori informazioni di contatto associate all'account di dominio o di un campo note utilizzato per spiegare perché un dominio è disabilitato o ha qualche funzione aggiuntiva o limite oltre altri domini creati con lo stesso modello. Se stai sviluppando strumenti per interagire con Virtualmin, questi campi potrebbero essere utili per altri scopi.

Per ogni campo è necessario inserire quanto segue:

Nome del campo
Un codice breve per questo campo, che viene utilizzato per fare riferimento al campo nei modelli e negli script di post-creazione. Ogni campo deve avere un nome campo univoco.

Etichetta
Il testo che verrà visualizzato all'utente durante l'immissione del valore per questo campo. Se l'etichetta contiene un; , qualsiasi cosa dopo verrà utilizzata come descrizione comandi che viene visualizzata quando si passa il mouse sul campo nella pagina Modifica server virtuale.

genere
Questo determina quali tipi di valori possono essere inseriti per questo campo. La maggior parte sono autoesplicativi, ad eccezione dell'opzione Menu . Quando è selezionato, è necessario immettere il percorso di un file sul sistema Virtualmin nella casella di testo adiacente. Ogni riga nel file deve contenere un codice e il testo visualizzato per una voce nel menu per questo campo personalizzato, separati da spazi.

Nell'elenco dei server?
Se impostato su Sì , verrà visualizzata una colonna per questo campo personalizzato nella pagina Elenco server virtuali .
