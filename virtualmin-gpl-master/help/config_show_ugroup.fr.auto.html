Autoriser la sélection d'un groupe différent pour les administrateurs de serveur?
Lorsque cette option est définie sur Oui, le créateur d'un nouveau serveur virtuel peut sélectionner un groupe Unix différent dont son propriétaire sera membre, au lieu du nouveau groupe créé pour le serveur virtuel. Cela peut être utile si vous avez besoin que tous les comptes de propriétaire de serveur sur votre système soient membres d'un groupe particulier, mais doit être utilisé avec précaution car il peut permettre aux propriétaires de serveur d'échapper aux quotas de groupe.
