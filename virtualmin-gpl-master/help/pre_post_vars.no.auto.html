Den spesifikke Virtualmin-handlingen som ble utført finner du i VIRTUALSERVER_ACTION-variabelen, for eksempel:

CREATE_DOMAIN
En ny virtuell server blir opprettet.
DELETE_DOMAIN
En eksisterende virtuell server blir fjernet.
DISABLE_DOMAIN
En virtuell server blir midlertidig deaktivert.
ENABLE_DOMAIN
En virtuell server blir reaktivert.
MODIFY_DOMAIN
En virtuell server blir oppdatert, for eksempel når funksjoner legges til eller fjernes, eller eierskap endres.
CLONE_DOMAIN
En virtuell server blir opprettet ved å kopiere en eksisterende server.
