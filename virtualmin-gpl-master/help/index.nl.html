Virtuele Server
De Webmin virtuele server module geeft de mogelijkheid voor het aanmaken, modificeren en verwijderen van gebruikers mailboxen en virtuele server accounts. Met zijn domein beheer functies kunt u makkelijk en snel nieuwe website met domein records aanmaken, Apache virtuelehost configureren, een home directory aanmaken, een gebruikers account beheren, een email server configureren, harddisk gebruik quota's en email adressen en aliassen aanmaken en beheren.
