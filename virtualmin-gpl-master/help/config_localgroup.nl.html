Primaire groep voor locale gebruikers
Virtualmin kan ook locale gebruikers beheren op uw server, wat erg handig kan zijn indien u wenst dat Virtualmin alleen de email en home websites beheerd. Met een kleine her-configuratie kan Virtualmin een makkelijke te gebruiken interface geven voor het toevoegen van mailbox en website gebruikers waarvan de website bereikt kan worden via "http://www.domain.com/~username". Dit is natuurlijk niet het primaire doel van Virtualmin maar het kan mailbox en website onderhoud erg makkelijk maken voor de ISP.
