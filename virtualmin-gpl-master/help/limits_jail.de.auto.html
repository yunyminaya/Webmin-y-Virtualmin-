Jailkit für die Domäne aktivieren
Mit dieser Option können Administratoren die Benutzerumgebung einer bestimmten Domäne mithilfe von Jailkit einschränken und so möglicherweise die Sicherheit durch Isolierung vom Gesamtsystem verbessern.

Wenn diese Option aktiviert ist, wird der mit der Domäne verknüpfte Unix-Benutzer in einer Chroot- Umgebung mit Jailkit platziert. Dies bedeutet, dass der Benutzer nur auf eine begrenzte Anzahl von Ressourcen zugreifen kann, die durch die Jailkit-Konfiguration definiert sind.

Die Jailkit-Konfiguration verwendet die Einstellungen aus der Konfigurationsdatei jk_init.ini , um zu bestimmen, auf welche Dateien, Binärdateien und Bibliotheken in der Chroot- Umgebung zugegriffen werden kann. Bei Bedarf können auch zusätzliche Konfigurationen wie zusätzliche Abschnitte oder Befehle definiert werden.

Bevor Sie Jailkit für eine Domäne aktivieren, stellen Sie sicher, dass Sie die Auswirkungen verstehen. Benutzer können beim Zugriff auf bestimmte Systemressourcen oder bei der Ausführung bestimmter Befehle möglicherweise Einschränkungen feststellen.
