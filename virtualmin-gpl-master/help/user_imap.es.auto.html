Inicio de sesión IMAP/FTP
Este campo muestra el nombre de usuario que necesitará el usuario del buzón de correo cuando inicie sesión a través de IMAP para recuperar correo electrónico, SMTP para enviar correo o FTP para administrar archivos. Se diferencia de la primera parte de su dirección de correo electrónico, ya que se agrega o antepone parte o la totalidad del nombre de dominio.
