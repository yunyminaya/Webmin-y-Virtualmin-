Trazer interfaces virtuais
Por padrão (quando Sim é escolhido), o Virtualmin ativará automaticamente as interfaces de rede privadas para os servidores que precisam delas. Se essa opção estiver configurada como Não , será assumido que você deseja configurar essas interfaces manualmente, talvez usando arquivos de configuração do Xen ou Zones em vez do padrão para o seu sistema operacional.
