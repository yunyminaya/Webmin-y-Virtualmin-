Groupe principal pour les utilisateurs locaux
Virtualmin peut également gérer les utilisateurs locaux sur votre serveur, ce qui peut être utile si vous souhaitez utiliser Virtualmin uniquement pour gérer le courrier et les sites Web personnels. Avec une petite reconfiguration, Virtualmin peut fournir une interface facile à utiliser pour ajouter des utilisateurs de boîte aux lettres / site Web dont le site Web peut être atteint via "http://www.domain.com/~username". Ce n'est pas l'objectif principal de Virtualmin, mais cela peut faciliter la maintenance des boîtes aux lettres et des sites Web pour les FAI.
