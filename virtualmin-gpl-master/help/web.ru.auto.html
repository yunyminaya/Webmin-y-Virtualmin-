Настроить сайт для домена?
Модуль может создать любую или все следующие службы для нового домена: почта, служба имен, веб-служба. Если для этого параметра установлено значение Да, новый раздел конфигурации VirtualHost будет добавлен в конфигурацию Apache, и будет создан новый домашний каталог веб-сайта.
