Параметр белого списка спама по умолчанию
Если для этого параметра установлено значение «Включено», то для всех новых виртуальных серверов, созданных с включенной фильтрацией спама, все их почтовые ящики будут автоматически добавляться в белый список SpamAssassin домена. Это гарантирует, что электронная почта из одного почтового ящика в другой никогда не будет классифицироваться как спам.
