È possibile visualizzare le password delle cassette postali e del server virtuale?
Quando questa opzione è impostata su Sì , gli amministratori del server saranno in grado di vedere le password correnti per gli utenti di posta / FTP e il proprio server virtuale, quando sono noti a Virtualmin. È possibile selezionare No per nascondere le password delle cassette postali, in particolare quelle che sono state selezionate dagli utenti delle cassette postali.

Indipendentemente da questa impostazione, l'amministratore principale e i rivenditori saranno sempre in grado di vedere le password delle cassette postali e dei server virtuali.
