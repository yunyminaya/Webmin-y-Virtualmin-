MySQL Database Server (voor database)
Indien dit is aangezet heeft het account van de domein eigenaar toegang tot de MySQL Webmin module. hij krijgt een vriendelijke GUI tot de database die behoort aan de domein eigenaar. Met deze module en de juiste privileges kan de domein eigenaar nieuwe databases aanmaken, bestaande databases bewerken of het uitvoeren van zoekopdrachten op de data in de database.
