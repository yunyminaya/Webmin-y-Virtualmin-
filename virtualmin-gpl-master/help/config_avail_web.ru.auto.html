Веб-сервер Apache (для виртуального хоста)
Включите это, если владелец домена должен иметь возможность редактировать конфигурацию Apache для своего домена. Хотя этот уровень контроля может быть подавляющим для неопытных пользователей, он обеспечивает большую гибкость для опытных пользователей. Также возможно, с изменениями в конфигурации управления доступом по умолчанию, разрешить прямой доступ к разделу виртуальных доменов файла конфигурации Apache. Хотя существуют средства защиты для предотвращения вредоносных действий, они могут быть неидеальными. По умолчанию предоставленные привилегии, как правило, безопасны для использования в среде общего хостинга.
