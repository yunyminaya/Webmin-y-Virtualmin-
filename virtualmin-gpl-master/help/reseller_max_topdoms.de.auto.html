Maximale Server der obersten Ebene
In diesem Feld können Sie die Anzahl der virtuellen Server der obersten Ebene, die der Reseller erstellen kann, separat begrenzen. Hierbei handelt es sich um Domänen mit eigener Administratoranmeldung, nicht unbedingt um DNS-Domänen der obersten Ebene. Eine Beschränkung der Gesamtzahl der virtuellen Server gilt ebenfalls.
