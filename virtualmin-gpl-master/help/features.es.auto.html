Características y complementos
Esta página muestra todas las funciones principales y módulos de complementos de Virtualmin disponibles, y le permite seleccionar cuáles están disponibles para su uso al crear nuevos servidores virtuales. Para cada uno, puede controlar su disponibilidad utilizando la casilla de verificación en la primera columna, a excepción del directorio de inicio y el usuario de administración, que son tan críticos que no se pueden desactivar.

Si hay algunas características que no desea seleccionar de forma predeterminada al crear nuevos servidores virtuales, se pueden anular la selección en la columna Predeterminado ; por ejemplo, es posible que desee configurar un sitio web SSL para nuevos dominios ocasionalmente, pero no todos el tiempo.

Más información sobre complementos

Los complementos son módulos de Webmin que proporcionan funciones adicionales de servidor y buzón para Virtualmin. A diferencia de las funciones principales, pueden ser instaladas por separado y escritas por desarrolladores externos. Al igual que Webmin y Usermin, Virtualmin tiene una API para escribir módulos de complementos. Sin embargo, es algo diferente, ya que los módulos de Virtualmin también son módulos de Webmin, pero también se vinculan directamente con los dominios de Virtualmin, de modo que cuando se modifica un dominio, los módulos de terceros también se modificarán. Los módulos de Virtualmin también tienen la capacidad de verificar las características de Virtualmin configuradas cuando se instalan, para garantizar la compatibilidad con el sistema.
