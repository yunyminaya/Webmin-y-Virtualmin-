Server virtuale basato su IP ProFTPd abilitato?
Abilita o disabilita un nuovo host virtuale FTP. Si noti che FTP richiede un nuovo IP per l'hosting anonimo del dominio virtuale. La stragrande maggioranza dei proprietari di siti Web non ha bisogno di questa funzione e non è richiesta per l'accesso FTP autenticato ai file dei proprietari di dominio. Tuttavia, se alcuni domini richiedono l'accesso FTP anonimo ospitato virtuale (ovvero se utilizzano un file server FTP per scaricare software o file per utenti non connessi), è possibile impostarlo su "Sì, ma non per impostazione predefinita" .
