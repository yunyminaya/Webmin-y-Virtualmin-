Tipo de login permitido
Esta configuração determina como o administrador deste domínio tem permissão para efetuar login, modificando seu shell Unix. As opções padrão disponíveis são:

Somente e-mail
Logins FTP e SSH são bloqueados, mas POP3 e IMAP funcionarão.
E-mail e FTP
O usuário pode se conectar via FTP, mas não via SSH ou telnet.
E-mail, FTP e SSH
Logins via FTP, SSH e telnet são permitidos.
No entanto, eles podem ser personalizados pelo administrador mestre na página Shells Personalizados .

Como esse recurso funciona alterando o shell do usuário Unix do servidor virtual, ele pressupõe que seu servidor FTP usa o arquivo /etc/shells para autenticação.
