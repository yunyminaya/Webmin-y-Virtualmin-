Pad om een API helper opdracht aan te maken
Standaard zal Virtualmin de virtualmin opdracht-regel API helper script in de /usr/sbin of /usr/local/sbin directory aanmaken, afhankelijk welke beschrijfbaar is op uw systeem. Echter dit configuratie veld kan worden gebruikt om een andere locatie te selecteren, zoals /opt/bin/virtualmin die geschikt is voor uw website.
