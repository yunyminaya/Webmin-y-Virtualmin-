Serveur FTP virtuel activé?
Lorsque cette fonction est activée, un serveur virtuel ProFTPd sera créé pour ce serveur virtuel Virtualmin. Cela peut être utilisé pour mettre en place un site FTP anonyme qui est privé de ce serveur. Cependant, il faut que le serveur dispose de son adresse IP.

Cette fonction est nécessaire uniquement si vous voulez fournir un accès basé sur le nom à un serveur FTP anonyme. Presque personne n'a besoin de cette fonction doit être activé (cela signifie que vous probablement). Vous ne devez pas activer cette fonction si vous voulez juste pour fournir un accès FTP pour le compte du propriétaire de domaine, tous les utilisateurs qui disposant d'un compte shell auront automatiquement accès FTP à leur répertoire personnel.

En d'autres termes: vous n'avez presque certainement pas besoin de ce, même si vous croyez.
