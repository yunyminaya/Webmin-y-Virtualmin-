Kann von Besitzern virtueller Server wiederhergestellt werden?
Wenn diese Option aktiviert ist, können nach diesem Zeitplan erstellte Sicherungen von den Eigentümern ihrer virtuellen Server zusätzlich zu den selbst erstellten Sicherungen über die Seite Sicherungsprotokolle wiederhergestellt werden. Auf diese Weise kann der Hauptadministrator sicherstellen, dass alles gesichert ist, aber Serverbesitzer können flexibel ihre eigenen Wiederherstellungen durchführen.
