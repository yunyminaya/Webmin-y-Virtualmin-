Posizioni aggiuntive per i comandi PHP
Virtualmin rileverà automaticamente le principali versioni di PHP installate sul tuo sistema, purché si trovino in directory nel percorso di ricerca. Tuttavia, questo campo può essere utilizzato per inserire i percorsi di ulteriori eseguibili PHP, che può essere utile se si desidera eseguire una versione superiore a quella installata per impostazione predefinita.
