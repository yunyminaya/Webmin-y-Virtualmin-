Curinga para bancos de dados MySQL adicionais permitidos
Este campo é para inserir um padrão curinga do MySQL que concede acesso a bancos de dados adicionais - talvez algo como ${DB}_% . Para um domínio como example.com , isso seria expandido para example_% , o que permitiria o acesso a todos os bancos de dados começando com example_ .

Como o Virtualmin cria regras de acesso do MySQL para todos os bancos de dados criados, esse campo geralmente pode ser deixado em branco. Ele só precisa ser preenchido se você estiver criando bancos de dados adicionais por meio de outro método fora do controle do Virtualmin.
