Servidores DNS remotos
Esta página le permite registrar sistemas que ejecutan Webmin como servidores DNS, lo que permite alojar zonas en ellos en lugar de en su sistema Virtualmin principal. Los sistemas que agregue ya deben tener BIND en ejecución y el módulo Servidor DNS de Webmin configurado correctamente.

Una vez agregadas, las zonas DNS se pueden mover o crear en los sistemas remotos. También es posible utilizar solo DNS remoto y apagar el servidor BIND en este sistema para ahorrar memoria.
