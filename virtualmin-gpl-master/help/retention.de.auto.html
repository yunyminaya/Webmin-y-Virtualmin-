Mailbox-Bereinigung
Auf dieser Seite können Sie eine Postfachbereinigungsrichtlinie einrichten, die für einige oder alle virtuellen Server auf Ihrem System gilt. Im Gegensatz zu den Einstellungen für die Bereinigung von Spam- und Virenordnern pro Domäne gilt die hier festgelegte Richtlinie für alle Ordner aller Benutzer in jeder ausgewählten Domäne - einschließlich des Posteingangs.

Das Löschen von Nachrichten kann entweder auf einem Zeitschwellenwert oder einer maximalen Postfachgröße basieren. Das Löschen basierend auf dem Nachrichtenalter ist eine einfache Möglichkeit, eine Aufbewahrungsrichtlinie einzurichten, um das maximale Alter gespeicherter Nachrichten zu begrenzen. E-Mails, die bereits auf die E-Mail-Clients der Benutzer (wie Outlook und Thunderbird) heruntergeladen wurden, werden jedoch nicht entfernt.
