Comodín para bases de datos MySQL adicionales permitidas
Este campo es para ingresar un patrón comodín MySQL que otorga acceso a bases de datos adicionales, tal vez algo como ${DB}_% . Para un dominio como example.com , esto se expandiría a example_% , lo que permitiría el acceso a todas las bases de datos comenzando con example_ .

Debido a que Virtualmin crea reglas de acceso MySQL para todas las bases de datos que crea, este campo generalmente puede dejarse en blanco. Solo debe completarse si está creando bases de datos adicionales a través de algún otro método fuera del control de Virtualmin.
