Pode editar servidor virtual
Permite a edição das funcionalidades do servidor virtual, sua senha e descrição.
Pode gerenciar usuários
Permite que usuários de caixas de correio, FTP e bancos de dados sejam visualizados, editados, criados e excluídos.
Pode gerenciar aliases
Permite que aliases de e-mail sejam visualizados, editados, criados e excluídos.
Pode gerenciar bancos de dados
Permite que bancos de dados MySQL e PostgreSQL adicionais sejam adicionados a este servidor virtual ou excluídos dele.
Pode instalar aplicativos da web
Permite que aplicativos web de terceiros sejam instalados neste servidor virtual.
Pode alterar o endereço IP
Se este servidor possuir um endereço IP privado, esta opção permite alterá-lo.
Pode gerenciar certificados SSL
Permite que o certificado SSL e a chave deste servidor sejam carregados ou gerados novamente.
Pode editar o quadro de encaminhamento
Para sites somente de encaminhamento, esta opção permite que o destino e o HTML do quadro de encaminhamento sejam editados.
Pode gerenciar administradores extras
Permite que contas extras de administrador proprietário do servidor sejam visualizadas, criadas, editadas e excluídas.
Pode configurar a entrega de spam e vírus
Permite que destinos e pontuações de entrega de spam e vírus sejam editados.
Pode selecionar versões PHP
Para domínios com PHP executado via CGI ou fastCGI, permite definir versões de PHP por diretório.
Pode editar configurações de e-mail
Permite a configuração de CCO automático, modo de cópia de alias e outras opções relacionadas a e-mail.
Pode fazer backups
Permite fazer backup de servidores virtuais próprios em um servidor FTP ou SSH remoto.
Pode agendar backups
Permite que seja feito backup de servidores virtuais em uma programação regular.
Pode restaurar backups
Permite que o diretório inicial e o conteúdo do banco de dados de servidores virtuais de propriedade sejam restaurados.
Pode selecionar IPs compartilhados
Permite que endereços IP compartilhados definidos globalmente sejam selecionados ao criar servidores virtuais.
Pode criar aliases genéricos
Permite a criação de aliases de e-mail que encaminham todos os e-mails de um domínio para um domínio ou endereço diferente.
Pode usar o editor de páginas da web
Permite que páginas da web sejam editadas por meio da UI Virtualmin.
Pode editar hosts remotos de banco de dados
Permite a entrada de sistemas clientes remotos que têm permissão para se conectar aos bancos de dados MySQL do domínio.
Pode desabilitar servidores virtuais
Permite que os servidores sejam desativados temporariamente.
Pode excluir servidores virtuais
Permite que servidores próprios sejam completamente excluídos, incluindo todos os subservidores e aliases.
