Vorlagen-Webanwendungsinstallateure
Auf dieser Seite können Sie Webanwendungen von Drittanbietern angeben, die installiert werden sollen, wenn mit dieser Vorlage ein neuer virtueller Server erstellt wird. Für jede Webanwendung müssen die folgenden Informationen eingegeben werden:

Web-App
Der Name und die Version einer zu installierenden Webanwendung
URL-Pfad
Ein Pfad unter der Website des virtuellen Servers, um die Webanwendung zu installieren, z. B. /wordpress . Dadurch wird auch das Verzeichnis unter ~/public_html bestimmt, das die Dateien der Webanwendung enthält.
DB-Name
Der Name der Datenbank, die die Webanwendung zum Speichern von Daten verwendet. Die Optionen sind:
Keiner
Dies sollte für Webanwendungen ausgewählt werden, die keine Datenbank benötigen, wie z. B. Gallery .
Hauptdatenbank
Die Datenbank der Webanwendung wird die primäre Datenbank des virtuellen Servers des gewählten Typs sein.
Andere
Wenn diese Webanwendung eine eigene private Datenbank verwenden soll, können Sie diese Option auswählen und das Muster für eine Datenbank in das angrenzende Textfeld eingeben. Sie können beispielsweise ${DB}_phpnuke eingeben. Wenn die resultierende Datenbank nicht existiert, wird sie im Rahmen des Installationsprozesses der Webanwendung erstellt.
DB-Typ
Der Typ der Datenbankverwendung für die Webanwendungsdaten (sofern ausgewählt)
