Comandos personalizados
Esta opção especifica se o proprietário do domínio terá acesso ao módulo Webmin de comandos personalizados. Com o módulo Comandos personalizados, você pode estender o Webmin sem precisar programar um novo módulo. Um comando personalizado pode executar qualquer comando no sistema com qualquer opção especificada. A substituição de variáveis também está disponível para que o usuário que está executando o comando possa fornecer argumentos de entrada ou opção para o comando, e é possível executar o comando como o usuário Webmin conectado ou como qualquer usuário no sistema (incluindo raiz, mas seja cuidado com isso!). Esse é um recurso muito poderoso do Webmin, permitindo adicionar novos "botões" que executam funções específicas ao seu ambiente ou aos quais alguns usuários podem precisar acessar.
