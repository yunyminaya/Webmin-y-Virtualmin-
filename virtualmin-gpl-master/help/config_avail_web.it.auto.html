Apache Webserver (per host virtuale)
Abilita questa opzione se il proprietario del dominio deve avere la possibilità di modificare la configurazione di Apache per il proprio dominio. Mentre questo livello di controllo può essere opprimente per gli utenti inesperti, fornisce grande flessibilità per quelli esperti. È anche possibile, con modifiche alla configurazione predefinita del controllo di accesso, consentire l'accesso diretto alla sezione dei domini virtuali del file di configurazione di Apache. Sebbene siano in atto delle protezioni per prevenire azioni dannose, potrebbero non essere perfette. Per impostazione predefinita, i privilegi concessi sono generalmente sicuri per l'uso in un ambiente di hosting condiviso.
