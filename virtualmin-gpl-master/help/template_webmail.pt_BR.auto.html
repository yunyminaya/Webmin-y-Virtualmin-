Redirecionar webmail.domínio para Usermin?
Quando esse campo é definido como Sim (como é o padrão), qualquer novo servidor virtual terá uma entrada de webmail adicionada aos seus domínios DNS, com o mesmo endereço IP do site do servidor. Quando http://webmail.thedomain.com/ for acessado, o navegador será redirecionado para o Usermin no mesmo domínio, para acessar o email baseado na Web.

Se você preferir usar alguma interface de webmail diferente do Usermin (como Squirrelmail ou Roundcube) ou desejar usar um URL diferente para acesso ao Usermin, o URL de redirecionamento poderá ser configurado no campo URL a seguir para redirecionamento de webmail .
