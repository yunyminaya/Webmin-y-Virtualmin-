Adresse IPv4 pour les serveurs restaurés
Ce paramètre n'est utilisé que lors de la restauration d'un serveur virtuel qui n'existe pas actuellement sur le système. Par défaut, Virtualmin lui attribuera l'adresse IP partagée par défaut du système si elle se trouvait à l'origine sur une IP partagée, ou utilisera l'adresse IP privée du système d'origine si elle ne se trouvait pas sur une adresse partagée.

Cela fonctionne bien si vous déplacez un domaine entre deux systèmes sur le même réseau, mais si ce n'est pas le cas, le domaine restauré ne sera probablement pas joignable via le réseau. Pour éviter cela, sélectionnez l'option Virtuel avec IP allouée ou Virtuel avec IP dans ce champ, et Virtualmin sélectionnera une IP gratuite pour vous ou utilisera celle que vous entrez.
