Grupo secundario para usuarios de bases de datos.
Si se establece esta opción, debe ingresar el nombre de un grupo unix al que se agregarán todos los usuarios con acceso a una o más bases de datos para servidores virtuales en esta plantilla. Si el grupo que especifica no existe, se creará automáticamente según sea necesario.

Esta característica es más útil junto con Usermin, que puede limitar los módulos visibles a aquellos usuarios que son miembros de algún grupo. Por lo tanto, puede hacer que el módulo de base de datos MySQL sea visible solo para los miembros del grupo que establezca aquí.
