Benutzerdefinierte Felder
Mit benutzerdefinierten Feldern kann der Administrator zusätzliche Daten zu Domänenkonten einfügen. Die Felder werden beim Bearbeiten der Domain angezeigt. Sie können für nahezu jede Art von Daten verwendet werden. Beispielsweise kann es nützlich sein, zusätzliche Kontaktinformationen an das Domänenkonto anzuhängen oder ein Notizfeld zu verwenden, um zu erklären, warum eine Domäne deaktiviert ist oder über eine andere Funktion oder Einschränkung hinausgeht, die über andere Domänen hinausgeht, die mit derselben Vorlage erstellt wurden. Wenn Sie Tools für die Interaktion mit Virtualmin entwickeln, können diese Felder für andere Zwecke nützlich sein.

Für jedes Feld müssen Sie Folgendes eingeben:

Feldname
Ein Funktionscode für dieses Feld, mit dem auf das Feld in Vorlagen und Skripten nach der Erstellung verwiesen wird. Jedes Feld muss einen eindeutigen Feldnamen haben.

Etikette
Der Text, der dem Benutzer bei der Eingabe des Werts für dieses Feld angezeigt wird. Wenn das Etikett a enthält; Alles, was danach passiert, wird als Tooltip verwendet, der angezeigt wird, wenn Sie mit der Maus über das Feld auf der Seite "Virtuellen Server bearbeiten" fahren.

Art
Dies bestimmt, welche Arten von Werten für dieses Feld eingegeben werden können. Die meisten sind selbsterklärend, mit Ausnahme der Menüoption . Wenn diese Option ausgewählt ist, müssen Sie den Pfad zu einer Datei auf dem Virtualmin-System in das nebenstehende Textfeld eingeben. Jede Zeile in der Datei muss einen Code und angezeigten Text für einen Eintrag im Menü für dieses benutzerdefinierte Feld enthalten, der durch Leerzeichen getrennt ist.

In der Serverliste?
Wenn dies auf Ja gesetzt ist , wird auf der Seite Virtuelle Server auflisten eine Spalte für dieses benutzerdefinierte Feld angezeigt.
