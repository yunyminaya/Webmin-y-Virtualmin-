Standard-HTML-Seite der Website
Mit dieser Option können Sie die Standardseite steuern, die für den neuen virtuellen Server erstellt und angezeigt wird.

Sie können „Deaktiviert“ wählen, um das Erstellen einer Seite zu überspringen, oder „Bei Erstellung fragen“ auswählen, um dies bei der Erstellung des virtuellen Servers zu entscheiden.

Alternativ können Sie sich für eine Standardseite der Virtualmin-Website entscheiden, indem Sie die Option „Ja, mit der Standardseitenvorlage der Website“ auswählen.

Wenn Sie möchten, können Sie durch Auswahl von „Ja, mit Inhalt unten“ benutzerdefiniertes HTML oder Text angeben, der als Webseiteninhalt verwendet werden soll.
