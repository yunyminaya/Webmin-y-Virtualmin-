Provedores SSL
Esta página pode ser usada para adicionar provedores SSL compatíveis com ACME para uso ao solicitar e renovar certificados. Ao adicionar um provedor, você pode escolher um dos serviços conhecidos pelo Virtualmin ou inserir qualquer URL de API ACME.

Alguns provedores exigem autenticação usando uma chave e HMAC, que normalmente estão disponíveis em seus sites após você criar uma conta. Outros permitem que certificados sejam solicitados sem autenticação.
