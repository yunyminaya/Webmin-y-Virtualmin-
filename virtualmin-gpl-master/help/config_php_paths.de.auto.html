Zusätzliche Speicherorte für PHP-Befehle
Virtualmin erkennt automatisch wichtige PHP-Versionen, die auf Ihrem System installiert sind, sofern sie sich in Verzeichnissen im Suchpfad befinden. In dieses Feld können jedoch die Pfade zu zusätzlichen ausführbaren PHP-Dateien eingegeben werden. Dies kann hilfreich sein, wenn Sie eine Version ausführen möchten, die höher ist als die standardmäßig installierte.
