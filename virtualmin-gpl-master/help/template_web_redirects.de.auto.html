Weiterleitungen für neue Websites
In dieser Tabelle können Sie Website-Umleitungen eingeben, die allen neuen virtuellen Servern hinzugefügt werden, wenn sie erstellt werden. Für jede Weiterleitung müssen Sie einen Quellpfad wie / oder /phpadmin und entweder eine Ziel-URL oder einen Pfad relativ zum public_html -Verzeichnis der Domain eingeben.

In beiden Feldern können virtuelle Servervorlagenvariablen wie ${DOM} und ${USER} verwendet werden. Sie können auch auswählen, ob die Weiterleitung gilt, wenn auf die Website über HTTP, HTTPS oder beides zugegriffen wird.
