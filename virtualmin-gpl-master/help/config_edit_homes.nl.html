Mag home directory's selecteren voor gebruikers?
Indien dit is aangezet kan de domein eigenaar ervoor kiezen om gebruikers een andere home directory te geven dan de standaard directory. Dit kan erg handig zijn zodat domein eigenaars extra services kunnen installeren zoals een applicatie server (Bijv. Zope, Webrick, enz.), en zij het fijn vinden om de mogelijkheid te hebben een andere gebruiker voor de applicatie server te gebruiken. Over het algemeen geeft u alleen gebruikers die u goed vertrouwd dit soort flexibiliteit.
