Consentire la creazione di siti Web di solo inoltro?
Virtualmin supporta il "parcheggio" del dominio con l'inoltro a un altro sito Web. Con questa opzione è possibile scegliere se l'inoltro verrà eseguito con le regole ProxyPass in Apache, tramite un frame attorno al sito di destinazione, oppure disabilitare completamente la funzione.
