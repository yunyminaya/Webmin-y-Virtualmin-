Tipos de senha com hash para armazenar
Por padrão, o Virtualmin armazenará senhas de domínio e caixa de correio em vários formatos de hash, para que a senha esteja disponível ao adicionar novos recursos ao domínio. No entanto, nem todos os formatos de hash são igualmente seguros, portanto, esse campo pode ser usado para desativar o armazenamento de alguns formatos. Independentemente do que você selecionar aqui, a senha será sempre armazenada em hash usando o formato padrão do sistema em /etc/shadow ou LDAP.

Esteja ciente de que desabilitar o formato de hash do MySQL pode impedir o Virtualmin de criar um logon do MySQL para usuários de caixas de correio com a mesma senha que a caixa de correio foi originalmente atribuída. Desabilitar o formato de resumo da Web pode impedir o acesso a um diretório da Web protegido pelos usuários da caixa de correio.
