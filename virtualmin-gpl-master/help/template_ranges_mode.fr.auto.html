Plages pour l'attribution automatique d'adresses IP
Si vous souhaitez que Virtualmin alloue automatiquement une adresse IP aux nouveaux serveurs virtuels créés à partir de ce modèle, vous pouvez entrer une ou plusieurs plages pour choisir les adresses IP à partir d'ici. Sinon, toute adresse IP virtuelle privée doit être saisie manuellement lors de la création d'un domaine.
