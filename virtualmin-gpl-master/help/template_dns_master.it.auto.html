Nome host del server DNS principale
Per impostazione predefinita, Virtualmin utilizzerà il nome host del sistema nei record NS e SOA primari per i nuovi domini DNS che crea. Tuttavia, se il nome host non è un nome DNS valido o non è adatto ai tuoi scopi, questa opzione può essere utilizzata per inserire un nameserver primario diverso. Deve risolvere l'indirizzo IP accessibile dall'esterno del sistema o l'indirizzo di un sistema di nameserver slave per i domini creati utilizzando questo modello.
