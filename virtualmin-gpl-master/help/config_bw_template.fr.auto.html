Fichier modèle pour les e-mails aux serveurs dépassant la limite de bande passante
Si la surveillance et les limites de la bande passante sont configurées, Virtualmin peut informer un propriétaire de domaine si l'utilisation de la bande passante pour le domaine a dépassé le quota. Vous pouvez utiliser le message par défaut, ou créer le vôtre et le spécifier ici. Les variables de modèle Virtualmin standard sont disponibles dans le modèle.
