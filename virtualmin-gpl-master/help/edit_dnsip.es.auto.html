Dirección IP externa
Si su servidor Virtualmin está en una red privada interna, y todo el tráfico HTTP se reenvía a través de un firewall o proxy, debe configurar este campo en la dirección IP a la que se accede externamente, como la de su firewall o enrutador. Cuando se establece esta opción, la dirección ingresada se utilizará para todos los registros DNS que normalmente contendrían la dirección real, como www , mail y ftp .

Por defecto, los registros DNS generados automáticamente para este servidor virtual contendrán la dirección IP que Virtualmin le asigna. Sin embargo, en algunos casos, esta dirección puede no ser la que desea que se conecten los clientes web y de correo en Internet.
