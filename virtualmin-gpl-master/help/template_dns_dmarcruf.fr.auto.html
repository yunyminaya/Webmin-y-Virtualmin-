URI de rapport pour les rapports médico-légaux
Par défaut, l'enregistrement DNS DMARC pour les nouveaux domaines utilisera mailto:postmaster@domain.com comme adresse pour les destinataires de spam pour signaler les messages personnels. Cependant, vous pouvez saisir ici un nouvel URI pour personnaliser l'adresse e-mail, par exemple pour utiliser une adresse commune à tous les domaines.
