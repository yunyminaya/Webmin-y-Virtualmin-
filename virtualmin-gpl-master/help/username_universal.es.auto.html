Dirección de correo electrónico
Este campo establece la parte del nombre de usuario del inicio de sesión del usuario, que debe ser única dentro del dominio. El inicio de sesión del usuario para acceder a servicios como SSH, FTP, correo electrónico, base de datos y autenticación de servidor web se derivará del nombre de usuario ingresado aquí, combinado con el nombre de dominio. Por ejemplo, si ingresa el nombre de usuario user1 y el dominio es example.com , el inicio de sesión resultante será user1@example.com .
