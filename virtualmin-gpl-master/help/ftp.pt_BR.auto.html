Configurar o FTP virtual baseado em IP?
Quando esse recurso está ativado, um servidor virtual ProFTPd será criado para esse servidor virtual Virtualmin. Isso pode ser usado para configurar um site FTP anônimo que é privado para este servidor. No entanto, exige que o servidor tenha seu endereço IP.

Esse recurso é necessário apenas se você deseja fornecer acesso baseado em nome a um servidor FTP anônimo. Quase ninguém precisa que esse recurso seja ativado (o que provavelmente significa você). Você não precisa habilitar esse recurso se desejar apenas fornecer acesso FTP à conta do proprietário do domínio, pois qualquer usuário com um shell shell terá automaticamente acesso FTP ao diretório inicial.

Em outras palavras: você quase certamente não precisa disso, mesmo que pense que precisa.
