Hôtes clients MySQL autorisés
Si des hôtes supplémentaires doivent être autorisés à interroger la base de données MySQL, vous pouvez les spécifier ici. Cela pourrait être utile si ce système est utilisé comme serveur de base de données et que d'autres machines utiliseront la base de données. Des précautions de sécurité doivent être suivies pour garantir que l'accès illicite n'est pas involontairement autorisé.
