Chemin pour créer la commande d'assistance API
Par défaut, Virtualmin créera le script d'assistance de l'API de ligne de commande virtualmin dans le /usr/sbin ou /usr/local/sbin , selon ce qui est accessible en écriture sur votre système. Cependant, ce champ de configuration peut être utilisé pour sélectionner un emplacement différent, tel que /opt/bin/virtualmin approprié pour votre site.
