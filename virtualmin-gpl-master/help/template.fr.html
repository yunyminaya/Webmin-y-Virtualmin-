Template de serveur
Le modèle sélectionné détermine les configurations par défaut utilisés pour diverses fonctions, telles que la configuration du site Web du serveur, les Enregistrements de domaine DNS et les fichiers du répertoire de départ. Les modèles peuvent être créés et modifiés à l'aide de la icône Modèles du serveur sur la page principale du module (Administrateur).
