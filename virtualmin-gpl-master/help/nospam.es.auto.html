¿Revisar el correo electrónico para detectar spam y virus?
De manera predeterminada, cuando el filtrado de spam y virus está habilitado para un servidor virtual, se aplicará a todos los buzones de ese dominio. Sin embargo, es posible que desee deshabilitar esto para ciertas cuentas, como una que esté configurada para filtrar el correo no deseado en el cliente, o sea el destinatario de todo el correo no deseado del dominio. Para hacer esto, cambie esta opción a No.
