Afficher les bases de données dans la liste des utilisateurs ?
Lorsque la valeur Oui est définie, le nombre de bases de données auxquelles chaque utilisateur a accès sera affiché dans une colonne distincte sur la page Modifier les utilisateurs . Cette fonctionnalité fournit un aperçu rapide de l'accessibilité à la base de données pour chaque utilisateur, complétant les informations déjà disponibles dans la colonne « Accès à la connexion ». Opter pour Non masquera les « Bases de données ? » colonne de la liste des utilisateurs.
