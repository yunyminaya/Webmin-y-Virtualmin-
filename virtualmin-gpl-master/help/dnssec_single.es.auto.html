Número de claves DNSSEC
La práctica recomendada de DNSSEC es que cada zona tenga dos claves: una para firmar los registros (la clave de zona) y la otra para firmar solo la clave de zona. Sin embargo, puede usar esta opción para cambiar el número predeterminado de teclas para nuevas zonas a solo una, si eso tiene más sentido para su sistema. Esto evita la sobrecarga de tener que volver a generar periódicamente y volver a firmar la clave de zona.
