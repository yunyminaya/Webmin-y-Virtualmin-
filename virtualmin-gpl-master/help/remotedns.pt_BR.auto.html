Servidores DNS remotos
Esta página permite que você registre sistemas executando Webmin como servidores DNS, permitindo que zonas sejam hospedadas neles em vez do seu sistema Virtualmin primário. Os sistemas que você adicionar já devem ter o BIND em execução e o módulo Webmin DNS Server configurado corretamente.

Uma vez adicionadas, as zonas DNS podem ser movidas ou criadas nos sistemas remotos. Também é possível usar somente DNS remoto e desligar o servidor BIND neste sistema para economizar memória.
