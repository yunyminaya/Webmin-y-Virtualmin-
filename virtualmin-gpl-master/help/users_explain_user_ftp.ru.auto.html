Создать пользователя FTP
Создает нового пользователя Unix, предназначенного для доступа по FTP в каталоге веб-сайта, автоматически назначая идентификатор пользователя и группы владельца виртуального сервера. Этот специализированный пользователь создан с учетом доступа по FTP. Квоты для этого типа учетной записи доступны владельцу виртуального сервера. Примечательно, что возможности электронной почты недоступны для этого типа пользователей.
