Zusätzliche manuell konfigurierte Nameserver
Wenn Sie andere Nameserver manuell als sekundäre Nameserver für mit Virtualmin erstellte Domänen konfigurieren, sollten Sie deren Hostnamen zu diesem Textfeld hinzufügen. Dadurch wird sichergestellt, dass die erforderlichen NS- Einträge zur DNS-Zone hinzugefügt und die Sekundärdateien verwendet werden.

Wenn Sie Webmin so konfiguriert haben, dass auf sekundären Nameservern automatisch Slave-Domänen erstellt werden, müssen deren Hostnamen hier nicht eingegeben werden. Stattdessen fügt Virtualmin sie automatisch zur DNS-Zone hinzu.
