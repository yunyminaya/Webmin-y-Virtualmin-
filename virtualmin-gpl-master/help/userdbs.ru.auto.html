Разрешить доступ к базам данных
Это поле позволяет выбрать, в какие базы данных, принадлежащие этому виртуальному серверу, пользователь также может войти. Чтобы добавить базу данных, выберите ее из списка слева и щелкните стрелку ->.

Логин и пароль пользователя будут такими же, как те, которые он использовал бы для FTP или IMAP. По умолчанию ему будет предоставлен полный доступ ко всем таблицам в выбранных базах данных.

MySQL поддерживает имена пользователей длиной до 16 символов, поэтому для пользователя с длинным именем учетной записи при входе в базу данных потребуются только первые 16 символов.
