Оболочка для пользователей почтовых ящиков
Этот параметр указывает оболочку, настроенную для пользователей почтовых ящиков. Пользователь почтового ящика является наименее привилегированным типом пользователя и обычно должен иметь очень ограниченный доступ. Однако, если ваш сервер Virtualmin используется только доверенными пользователями или пользователями, которые должны иметь доступ к оболочке, вы можете изменить это значение по умолчанию / dev / null на один из реальных системных оболочек. Обратите внимание, что / bin / false - это специальная оболочка, которая закрывает удаленные входы по SSH, но все же разрешает вход по FTP.
