Crear servidor virtual
Este formulario le permite crear un nuevo servidor virtual de nivel superior, que tendrá un nuevo inicio de sesión de administración asociado. Al crear el servidor, debe ingresar al menos el nombre de dominio, la descripción y la contraseña de administración. Si se definen varias plantillas de servidor en el sistema, puede seleccionar una del menú: las plantillas determinan varios valores predeterminados para el nuevo servidor, como la configuración del sitio web, los registros DNS, etc.

En la sección Opciones avanzadas , puede seleccionar un nombre de usuario de administración diferente, un nombre de grupo y un nombre de base de datos para el servidor, en lugar de los valores predeterminados que se toman de la primera parte del nombre de dominio. También se puede ingresar una dirección de correo electrónico de contacto para el servidor; idealmente, esta debería ser la dirección existente de la persona que administrará el servidor, en un dominio diferente.

Para seleccionar qué funciones están habilitadas para el servidor, use las casillas de verificación en la segunda parte de la sección Opciones avanzadas . Algunas funciones dependen de otras y no se pueden seleccionar por sí mismas; por ejemplo, los informes de Webalizer sobre las estadísticas del sitio web dependen de tener un sitio web.

En la parte inferior de la sección Opciones avanzadas , puede seleccionar que se reenvíe todo el tráfico web a este nuevo sitio a otra URL y que se reenvíe todo el correo electrónico. También puede seleccionar que el nuevo servidor virtual use sus propias direcciones IP privadas, que se requieren para el alojamiento SSL y FTP anónimo.
