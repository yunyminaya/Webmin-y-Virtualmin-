Creare il login MySQL?
Se selezionato, verrà creato un account MySQL per questo dominio, in genere con lo stesso nome dell'account di accesso amministrativo del dominio. Se si utilizzano le impostazioni del modello predefinito, verrà creato un database MySQL anche per il server virtuale. Il nome del database sarà la prima parte del nome di dominio, quindi se si crea il dominio foo.com , il database si chiamerebbe foo .
