Apache-Benutzer zur Unix-Gruppe für neue Server hinzufügen?
Diese Option bestimmt, ob der Benutzer, unter dem Apache ausgeführt wird, der neuen virtuellen Hostgruppe hinzugefügt wird. Wenn diese Option aktiviert ist, können die Berechtigungen für Benutzer-Home-Verzeichnisse erheblich eingeschränkt werden. Wenn strengere Berechtigungen (z. B. 750 ) mit suEXEC kombiniert werden, ist es für Domänenbesitzer unmöglich, die Verzeichnisse und Dateien anderer Domänenbesitzer zu sehen. Diese Option wird auf jeder Plattform empfohlen, die sekundäre Gruppen unterstützt.
