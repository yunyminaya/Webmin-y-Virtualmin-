Kann Backups migrieren?
Standardmäßig dürfen Wiederverkäufer keine virtuellen Server erstellen, indem sie Sicherungen von anderen Kontrollfeldern (wie cPanel und Plesk) migrieren, da eine schädliche Sicherungsdatei verwendet werden kann, um Root- Zugriff auf das System zu erhalten. Wenn Sie jedoch einem Reseller vertrauen, können Sie die Berechtigung zum Migrieren von Backups erteilen, indem Sie hier Ja auswählen.
