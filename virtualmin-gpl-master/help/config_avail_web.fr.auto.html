Serveur Web Apache (pour hôte virtuel)
Activez cette option si le propriétaire du domaine doit avoir la possibilité de modifier la configuration Apache de son domaine. Bien que ce niveau de contrôle puisse être écrasant pour les utilisateurs inexpérimentés, il offre une grande flexibilité aux utilisateurs expérimentés. Il est également possible, en modifiant la configuration du contrôle d'accès par défaut, d'autoriser l'accès direct à la section des domaines virtuels du fichier de configuration d'Apache. Bien que des protections soient en place pour empêcher les actions malveillantes, elles peuvent ne pas être parfaites. Par défaut, les privilèges accordés sont généralement sûrs pour une utilisation dans un environnement d'hébergement partagé.
