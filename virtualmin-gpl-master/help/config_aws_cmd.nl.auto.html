Volledig pad naar aws-opdracht voor S3-bewerkingen
Als de aws opdracht op uw systeem is geïnstalleerd en dit veld is ingesteld op het volledige pad ernaar (zoals /usr/bin/aws ), zal Virtualmin het aanroepen om S3-bewerkingen uit te voeren voor back-ups, zoals geüploade en gedownloade bestanden. Omdat deze opdracht is ontwikkeld door Amazon, is deze betrouwbaarder dan de S3 API die is ingebouwd in Virtualmin.
