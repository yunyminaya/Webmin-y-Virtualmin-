Пакетное создание псевдонимов
Эта страница позволяет вам создать несколько псевдонимов электронной почты одновременно из текстового файла. Этот файл должен содержать один псевдоним на строку, и каждая строка должна иметь следующий формат:

name:[comment]:destination1:[destination2:...]

Значения полей:

имя
Имя почтового ящика псевдонима с продажами . Чтобы создать псевдоним, который соответствует всей почте домена, введите * здесь.
комментарий
Это необязательное поле можно использовать для ввода описательного комментария для псевдонима, такого как Корпоративные продажи .
пункт назначения
Можно указать одно или несколько мест назначения, обычно в форме адресов электронной почты для пересылки почты, например, joe@yourdomain.com . Также поддерживаются следующие специальные направления:
  • bounce
    Создает псевдоним, который возвращает письмо отправителю.
  • autoreply Your autoreply message
    Отправляет автоматический ответ отправителю, содержащий текст после слова autoreply .
  • local username
    Доставляет электронную почту прямо в почтовый ящик указанного пользователя.
