Durée maximale d'exécution du script PHP
Lorsque les scripts PHP sont exécutés en mode FCGI, ce champ définit la durée maximale d'exécution de chaque appel de script. Vous devrez peut-être l'augmenter lors de l'hébergement de scripts dont les temps d'exécution sont particulièrement élevés et qui sont supprimés par votre serveur Web avant leur achèvement.
