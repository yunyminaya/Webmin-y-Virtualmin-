SSL-certificaten per IP kopiëren naar Dovecot?
Wanneer deze optie is ingeschakeld en er een virtuele server wordt gemaakt met SSL ingeschakeld en een eigen privé-IP, wordt Dovecot geconfigureerd om het SSL-certificaat van de server te gebruiken voor verbindingen met dat IP-adres. Dit kan handig zijn als e-mailclients in SSL-modus verbinding maken met de domeinnaam en een overeenkomend certificaat verwachten.
