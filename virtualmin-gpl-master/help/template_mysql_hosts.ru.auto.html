Разрешенные клиентские хосты MySQL
Если дополнительные узлы должны быть разрешены для запроса базы данных MySQL, вы можете указать их здесь. Это может быть полезно, если эта система используется в качестве сервера базы данных, а другие машины будут использовать базу данных. Необходимо соблюдать меры безопасности, чтобы не допустить непреднамеренного незаконного доступа.
