Настроить задание Webalizer Cron для каждого виртуального сервера?
Если для этого параметра установлено значение Да (как по умолчанию), Virtualmin будет создавать отдельное задание Cron для каждого виртуального сервера, на котором включен Webalizer, для ежедневного создания отчета из его журналов.

Однако на сервере с большим количеством доменов вы можете вместо этого изменить эту опцию на Нет и вместо этого вручную запланировать команду /usr/libexec/webmin/virtual-server/run-all-webalizer.pl используя Cron. Это гарантирует, что Webalizer будет запускаться для каждого домена по очереди, а не обрабатывать несколько доменов одновременно, и дает вам больший контроль над точно, когда обработка будет завершена.
