Permitir o uso de inclusões do lado do servidor?
As inclusões do servidor são um recurso de servidor da web que permite que diretivas macro simples sejam usadas em arquivos HTML. Eles podem ser usados para incluir outros arquivos HTML, definir e exibir o conteúdo de variáveis e controlar qual conteúdo aparece na página com instruções if . Eles oferecem uma maneira mais simples de criar páginas dinâmicas do que uma linguagem de script completa como PHP ou Perl.

Para obter mais informações, consulte a documentação do Apache em http://httpd.apache.org/docs/current/howto/ssi.html
