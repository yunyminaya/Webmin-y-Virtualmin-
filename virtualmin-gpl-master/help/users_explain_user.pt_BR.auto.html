Criar usuário
Cria uma nova conta de usuário Unix, com o mesmo grupo do proprietário do servidor virtual. Esta configuração apresenta amplas opções de personalização, como configuração de senha, ajuste de permissões de login e grupos secundários e controle de cotas. Também abrange a configuração do diretório inicial e oferece flexibilidade para ativar ou desativar uma variedade de serviços como e-mail, bancos de dados e acesso à web. Além disso, inclui a capacidade de controlar outras configurações de acesso oferecidas por plugins adicionais.
