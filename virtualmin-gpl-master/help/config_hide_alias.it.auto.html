Includere alias server virtuali nel menu?
Per impostazione predefinita, il menu a sinistra in Virtualmin includerà tutti i server virtuali, inclusi gli alias. Tuttavia, se questa opzione è impostata su No , i server alias verranno esclusi dall'elenco. Questo può essere utile se il tuo sistema ospita un gran numero di alias, come per i domini parcheggiati. È ancora possibile trovare e gestire questi server alias utilizzando i collegamenti dalla pagina Modifica server virtuale per i loro genitori.
