Aggiorna server virtuali
Questa pagina consente di aggiornare contemporaneamente impostazioni e limiti in più server virtuali, come selezionato nella pagina precedente. Per impostazione predefinita, nulla verrà modificato sui server selezionati fino a quando non si sceglie qualche attributo o limite da modificare utilizzando i campi in questa pagina. Le impostazioni modificabili sono suddivise nelle seguenti categorie:

Modifiche di quote e larghezza di banda
Questi campi possono essere utilizzati per modificare le quote del disco per i server selezionati o per modificare i limiti della larghezza di banda. La modifica delle quote ha effetto solo sui server di livello superiore, in quanto i server secondari ereditano le loro quote dal dominio padre.

Modifiche alle funzioni abilitate
In questa sezione, è possibile selezionare le funzionalità da attivare o disattivare per tutti i server selezionati. Queste sono le stesse funzionalità che possono essere attivate per i singoli server nella pagina Modifica server virtuale .

Modifiche ai limiti del proprietario
Questi campi possono essere utilizzati per modificare i vari limiti che si applicano ai proprietari di server virtuali di livello superiore, come il numero di cassette postali, alias e domini che possono creare. Le modifiche apportate qui non hanno alcun effetto sui server secondari, che rientrano nei limiti imposti ai loro genitori.

Modifiche alle funzionalità del proprietario
I campi in questa sezione controllano le funzionalità di Virtualmin disponibili per gli amministratori dei server virtuali selezionati. Ancora una volta, sono rilevanti solo per server di livello superiore.

Modifiche all'opzione PHP
Se disponibile, questa sezione consente di modificare sia la versione PHP predefinita sia il metodo di esecuzione degli script PHP su tutti i server selezionati. È utile solo su server che dispongono di siti Web.

Modifiche alla configurazione
Questa sezione consente di apportare varie modifiche alla configurazione dei server selezionati, ad esempio il metodo utilizzato per la cancellazione dello spam o la shell di accesso dell'amministratore del dominio.

È anche possibile disabilitare temporaneamente i server selezionati utilizzando il pulsante Disabilita server nella parte inferiore della pagina. Quando lo usi, puoi anche inserire un motivo per disabilitare (come il mancato pagamento ) nella casella di testo adiacente.

Per riattivare più server che sono stati disabilitati su questa pagina o altrove, utilizzare il pulsante Abilita server .
