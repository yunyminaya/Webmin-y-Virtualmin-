Permitir acesso a bancos de dados
Este campo permite que você selecione em quais bancos de dados de propriedade deste servidor virtual o usuário também pode efetuar login. Para adicionar um banco de dados, selecione-o na lista à esquerda e clique na seta ->.

O login e a senha do usuário serão os mesmos que eles usariam para FTP ou IMAP. Por padrão, eles terão acesso total a todas as tabelas nos bancos de dados selecionados.

O MySQL só suporta nomes de usuário com até 16 caracteres, então, para um usuário com um nome de conta longo, apenas os primeiros 16 caracteres são necessários ao efetuar login no banco de dados.
