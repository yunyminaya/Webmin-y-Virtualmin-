Destinazioni di inoltro e-mail
Questa tabella di campi determina cosa succede alle email inviate a questo utente. Normalmente, verrebbero recapitate direttamente alla casella di posta dell'utente, ma se sono definite delle destinazioni qui, verranno utilizzate al loro posto. Questo può essere utilizzato per impostare un risponditore automatico se l'utente è assente, o per inoltrare la sua email a un indirizzo su un altro server.

Ogni destinazione ha una tipologia (scelta dal menù), che può essere una delle seguenti:

  • Indirizzo e-mail
    L'email verrà inoltrata all'indirizzo inserito nel campo adiacente.

  • Indirizzi in archivio
    L'email verrà inoltrata a tutti gli indirizzi elencati nel file di testo il cui percorso è immesso nel campo adiacente.

  • Scrivi sul file
    L'email all'alias verrà aggiunta al file specificato. Saranno inclusi tutti gli header e il corpo grezzo non decodificato.

  • Feed al programma
    Il programma immesso nel campo adiacente verrà eseguito e inviato via email all'alias passatogli come input.

  • Risposta automatica dal file
    Quando l'utente riceve un'email, un messaggio di risposta automatica verrà rispedito al mittente. Il testo di questa risposta automatica verrà preso dal file immesso nel campo di testo adiacente.

  • Applica file filtro
    Le e-mail inviate all'utente verranno elaborate in base alle regole del file di filtro specificato, il che potrebbe comportare l'inoltro ad altri indirizzi o file a seconda del contenuto effettivo del messaggio.

  • Casella di posta dell'utente
    L'email verrà consegnata direttamente alla casella di posta dell'utente immesso nel campo di testo adiacente. In genere, dovresti immettere il nome dell'utente stesso, in modo che riceva una copia dell'email oltre a essere inoltrata a qualsiasi altra destinazione. Tuttavia, devi immettere il nome utente di accesso POP3 completo, come joe.foo invece di solo joe .

Se si scelgono i tipi di destinazione Indirizzi nel file , Risposta automatica dal file o Applica filtro al file , quando si modificano nuovamente gli utenti , accanto a tali destinazioni verranno visualizzati i collegamenti etichettati Modifica per modificare il contenuto del file specificato.
