Atualização dinâmica de IP
Esta página permite configurar o Virtualmin para detectar alterações na interface de rede principal do sistema e atualizar uma entrada dinâmica de DNS quando ela for alterada. Isso é útil se você estiver executando o Virtualmin em um sistema com um endereço que ocasionalmente muda, como um que esteja sendo executado em um ambiente de máquina virtual como o EC2 da Amazon. Também é útil para servidores conectados à Internet por meio de uma conexão sem um endereço IP fixo, como ADSL ou Cable.

Para usar esta página, você deve selecionar um serviço DNS dinâmico, inserir o nome do host usado pelo seu sistema nesse serviço e também os detalhes de login do serviço. Por padrão, o Virtualmin examinará o endereço IP atribuído à interface de rede principal (como eth0 ) para verificar se há alterações, mas se o seu sistema estiver atrás de um roteador NAT, você deverá selecionar a opção Endereço visível na Internet para descobrir o IP. que o resto da Internet realmente vê.

Quando uma alteração é detectada, todos os servidores virtuais com endereços IP compartilhados usando o endereço antigo serão atualizados para o novo. Essa alteração será feita na configuração do Apache e nos registros DNS.

Se o serviço de script externo estiver selecionado, você deverá inserir o caminho completo para um script no campo adjacente que será chamado quando o endereço IP do sistema for alterado. O novo IP e nome do host serão passados para o script como parâmetros de linha de comando.
