Modules Webmin supplémentaires pour accorder l'accès à
Cette option vous permet de fournir aux utilisateurs des fonctionnalités Webmin supplémentaires au-delà des modules de domaine normaux (Apache, BIND, Read Mail, etc., abordés ci-dessous dans les modules de fonctionnalité disponibles pour les propriétaires de domaine et les modules supplémentaires disponibles pour les propriétaires de domaine ). Soyez très prudent avec cette option, car la plupart des modules Webmin fournissent par défaut un contrôle complètement dangereux sur les services qu'ils gèrent. Il y a cependant des occasions où cela peut être très utile. Par exemple, si vous avez un module personnalisé qui fournit un service spécifique à votre environnement, cette option peut être très utile. De plus, de nombreux modules peuvent être configurés avec un accès très strict par défaut, ou vous pouvez cloner un module et le configurer avec un ensemble de fonctions très spécifique pour le rendre sûr pour une utilisation par tous les propriétaires de domaine. Vous pouvez accorder sélectivement des privilèges supplémentaires à certains utilisateurs de domaine à l'aide du module Utilisateurs Webmin .
