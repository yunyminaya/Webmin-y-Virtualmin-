Максимальное количество подключений для пользователей почтовых ящиков
Это поле контролирует количество одновременных подключений к MySQL, которые может установить почтовый ящик с доступом к базе данных. По умолчанию разрешено неограниченное количество подключений, но поскольку каждое из них потребляет память в процессе сервера MySQL, вы можете ограничить общее количество подключений в загруженной системе.

Ограничение, установленное здесь, будет применяться только при создании нового почтового ящика или при включении входа в MySQL для существующего почтового ящика.
