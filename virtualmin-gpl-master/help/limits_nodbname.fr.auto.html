Peut-on choisir les noms de bases de données ?
Si cette restriction est définie sur Non , le propriétaire du serveur ne sera pas autorisé à sélectionner le nom de base de données par défaut lors de la création d'un sous-serveur sur lequel une fonction de base de données est activée. Au lieu de cela, la première partie du nom de domaine sera toujours utilisée par défaut.

De plus, l'utilisateur ne pourra pas supprimer la base de données par défaut d'un domaine existant, car cela lui permettrait de renommer efficacement la base de données par défaut une fois le domaine créé.
