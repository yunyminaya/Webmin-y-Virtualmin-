Dirección de correo electrónico
Este campo sirve para configurar el nombre de usuario de la dirección de correo electrónico del usuario, que debe ser único dentro del dominio. El nombre de inicio de sesión del usuario para IMAP/POP3 y FTP se basará en lo que se ingrese en este campo más el nombre de dominio. Por ejemplo, si el nombre de usuario es user1 y el dominio example.com , el inicio de sesión sería user1@example.com .
