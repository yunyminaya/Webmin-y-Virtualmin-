Ścieżka do utworzenia polecenia pomocnika API
Domyślnie Virtualmin utworzy skrypt pomocniczy API wiersza polecenia virtualmin w katalogu /usr/sbin lub /usr/local/sbin , w zależności od tego, co można zapisać w systemie. Tego pola konfiguracji można jednak użyć do wybrania innej lokalizacji, takiej jak /opt/bin/virtualmin która jest odpowiednia dla Twojej witryny.
