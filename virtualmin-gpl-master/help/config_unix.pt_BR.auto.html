Criação de usuário de administração ativada?
Esta opção determina se a criação de um usuário Unix para um novo servidor virtual de nível superior é sempre feita (e a opção oculta) ou se é opcional ao criar um servidor. Geralmente, é uma boa ideia definir essa opção como Sempre , pois é necessário um usuário Unix, a menos que seja criado um domínio somente DNS ou somente proxy da Web.
