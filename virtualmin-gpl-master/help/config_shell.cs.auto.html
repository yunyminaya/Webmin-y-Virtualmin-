Shell pro uživatele poštovní schránky
Tato volba určuje prostředí nakonfigurované pro uživatele poštovní schránky. Uživatel poštovní schránky je nejméně privilegovaným typem uživatele a obvykle by měl mít velmi omezený přístup. Pokud však váš server Virtualmin používají pouze důvěryhodní uživatelé nebo uživatelé, kteří musí mít přístup k shellu, můžete to změnit z výchozí / dev / null na jednu ze skutečných systémových skořápek. Všimněte si, že / bin / false je speciální prostředí, které uzavře vzdálené přihlášení SSH, ale přesto povolí přihlášení FTP.
