Número da porta externa para hosts virtuais SSL
Essa configuração pode ser usada para alterar o número da porta que o Virtualmin usa em URLs que fazem referência a domínios criados a partir deste modelo. Pode ser útil se o seu sistema tiver um proxy ou cache na frente, escutando na porta 443, e o Virtualmin foi configurado para criar hosts virtuais do Apache em uma porta diferente.
