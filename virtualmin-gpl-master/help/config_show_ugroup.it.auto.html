Consentire la selezione di un gruppo diverso per gli amministratori del server?
Quando questa opzione è impostata su Sì, il creatore di un nuovo server virtuale può selezionare un diverso gruppo Unix per il quale il suo proprietario deve essere membro, anziché il nuovo gruppo creato per il server virtuale. Ciò può essere utile se è necessario che tutti gli account dei proprietari del server sul proprio sistema siano membri di un determinato gruppo, ma devono essere utilizzati con cautela in quanto possono consentire ai proprietari dei server di eludere le quote di gruppo.
