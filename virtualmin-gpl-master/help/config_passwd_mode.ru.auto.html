Тип поля пароля
Этот параметр управляет отображением поля для ввода пароля для нового виртуального сервера. Доступные варианты:
Случайно сгенерированный пароль
Отображается одно видимое поле, содержащее случайно сгенерированный пароль, выбранный Virtualmin. Вы можете ввести другой пароль, хотя.

Введите пароль один раз
Отображается одно поле для ввода собственного пароля, а действительный пароль скрыт.

Введите пароль два
Отображаются два поля для ввода собственного пароля, опять же, со скрытым паролем. Оба пароля должны быть одинаковыми.
