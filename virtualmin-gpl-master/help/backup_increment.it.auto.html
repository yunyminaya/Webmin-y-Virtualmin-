Livello di backup
Per impostazione predefinita, i backup di Virtualmin includeranno tutti i file nelle directory home dei domini selezionati. Tuttavia, se si esegue regolarmente il backup di server virtuali di grandi dimensioni, potrebbe essere necessario molto tempo per copiare file che non sono stati modificati di recente.

Se qui viene selezionata l'opzione Differenziale , verranno inclusi solo i file che sono stati modificati o che sono stati creati dall'ultimo backup completo. Ciò è particolarmente utile quando si hanno due backup pianificati, ad esempio un backup completo settimanale e uno differenziale giornaliero. Supponendo che la maggior parte dei file in un dominio non vengano modificati quotidianamente, ciò ridurrà sostanzialmente le dimensioni e il tempo dei backup giornalieri.

Tieni presente che durante il ripristino dovrai ripristinare prima il backup completo e poi quello differenziale per ripristinare tutti i file. Inoltre, altri dati come configurazioni e database Apache non vengono sottoposti a backup differenziale e quindi verranno sempre inclusi indipendentemente da questa opzione.

Anche l'opzione Nessuna esegue un backup completo, ma non modifica lo stato del backup. Ciò può essere utile quando si esegue un backup ad hoc di domini per cui sono pianificati anche backup completi e differenziali.
