Comandos y directorios adicionales para que Jailkit los copie
Comandos y directorios adicionales para copiar usando el comando jk_cp .

jk_cp es un comando proporcionado por el paquete Jailkit para copiar archivos y directorios al entorno chroot . El comando jk_cp también manejará y copiará automáticamente las dependencias de biblioteca necesarias para archivos binarios, asegurando que los ejecutables funcionen correctamente dentro del chroot .

En lugar de copiar manualmente comandos adicionales y todas las dependencias en el directorio chroot del dominio, use este campo para agregar comandos adicionales (separados por espacios) para llevarlos al chroot , asegurándose de que todo funcione como se esperaba. Por ejemplo, para copiar los comandos rsync y zip , coloque rsync zip en este campo.
