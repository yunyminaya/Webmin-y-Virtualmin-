Comando para enumerar las cuotas de usuario
Cuando se utilizan comandos de cuotas externas, se llamará al programa configurado en esta opción cuando Virtualmin necesite encontrar las cuotas de usuarios existentes. No se llama con ningún parámetro, pero debe generar una línea de texto por usuario, y cada línea contiene campos separados por espacios. Los campos en cada fila deben ser:
  1. El nombre de usuario de Unix.
  2. Cuota actual utilizada en bloques de 1 kB.
  3. La cuota flexible en bloques de 1 kB, o 0 para ilimitado.
  4. La cuota fija en bloques de 1 kB, o 0 para ilimitado.
