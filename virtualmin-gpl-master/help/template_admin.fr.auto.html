Rediriger le domaine admin vers Usermin?
Lorsque ce champ est défini sur Oui (comme c'est le cas par défaut), tous les nouveaux serveurs virtuels auront une entrée d' admin ajoutée à leurs domaines DNS, avec la même adresse IP que le site Web du serveur. Lorsque http://admin.thedomain.com/ est accessible, le navigateur sera redirigé vers Virtualmin sur le port 10000 du même domaine, pour accéder à l'interface Web d'administration.

Si vous souhaitez utiliser une URL différente pour l'accès à Virtualmin (telle que celle pour laquelle vous avez un certificat SSL), l'URL de redirection peut être configurée dans l' URL suivante pour le champ de redirection administrateur .
