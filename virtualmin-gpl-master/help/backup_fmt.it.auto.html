Formato di backup
Questa opzione controlla il formato file che Virtualmin utilizzerà per questo backup. Le opzioni disponibili sono:
File di archivio singolo
Tutti i server virtuali vengono archiviati in un singolo file, nella posizione specificata dalla destinazione Backup .
Un file per server
Viene eseguito il backup di ciascun server virtuale in un proprio file separato, nella directory specificata dalla destinazione Backup .
Un file per server, nel nuovo formato
Viene eseguito il backup di ciascun server virtuale in un proprio file separato, ma in un formato diverso dall'opzione precedente. Questo nuovo formato è superiore in quanto non richiede prima di creare un file di grandi dimensioni nella /tmp , risparmiando spazio su disco.
Si consiglia l'opzione finale, poiché ha permesso di ripristinare più facilmente i singoli server virtuali e di risparmiare spazio temporaneo sul disco durante il backup.
