Максимум процессов сервера Mongrel для Rails
При установке сценариев Ruby on Rails администратор сервера может выбрать, как мой сервер Mongrel будет обрабатывать запросы к пути сценария, если mod_proxy_balancer модуль Apache mod_proxy_balancer . Чтобы предотвратить чрезмерное количество процессов сервера Mongrel от перегрузки системы, этот параметр можно использовать для ограничения количества, которое администраторы домена могут настраивать из Virtualmin.
