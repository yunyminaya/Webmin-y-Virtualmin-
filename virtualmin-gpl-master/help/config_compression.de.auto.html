Backup-Komprimierungsformat
Standardmäßig verwendet Virtualmin die gzip Komprimierung für seine Sicherungsdateien. Diese Option kann jedoch verwendet werden, um stattdessen das platzsparende (aber CPU-intensive) bzip2 Format auszuwählen. Sie können auch nicht komprimierte Sicherungen im tar Format verwenden. Dies kann hilfreich sein, wenn Sie eine schnelle Sicherung wünschen oder wenn Sie auf Band schreiben. Oder das zip Format, das auf anderen Betriebssystemen einfacher zu extrahieren ist.

Diese Option wird nur verwendet, wenn virtuelle Server in einem Verzeichnis mit einer Datei pro Domäne gesichert werden. Beim Sichern in einer einzelnen Datei bestimmt der Dateiname das Komprimierungsformat (wie domains.tar.gz oder domains.tar ).
