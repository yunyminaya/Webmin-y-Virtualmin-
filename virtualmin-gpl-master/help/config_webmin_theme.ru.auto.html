Тема для новых владельцев доменов Webmin
Вы можете выбрать тему по умолчанию для входа в систему владельца домена. По умолчанию используется глобальное значение по умолчанию для Webmin, которое является темой Virtualmin-Nuvola в установках Virtualmin Professional. Возможно, вам будет полезно различать цветовые схемы для различных уровней привилегий, например, выбрать красную тему для административных учетных записей, поскольку эти учетные записи наиболее опасны при небрежном использовании, когда используется стандартная мягкая серо-голубая тема для вашего домена. владельцы и посредники. Чисто дело вкуса.
