Ruta para crear el comando auxiliar API
De manera predeterminada, Virtualmin creará el script auxiliar de API de línea de comandos virtualmin en el directorio /usr/sbin o /usr/local/sbin , dependiendo de cuál sea editable en su sistema. Sin embargo, este campo de configuración se puede utilizar para seleccionar una ubicación diferente, como /opt/bin/virtualmin que sea apropiada para su sitio.
