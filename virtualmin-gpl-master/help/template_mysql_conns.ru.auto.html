Максимальное количество подключений для владельцев доменов
Это поле контролирует количество одновременных подключений к MySQL, которые может выполнить учетная запись владельца виртуального сервера. Сюда входят соединения из веб-приложений, установленных Virtualmin, таких как Wordpress или Drupal. По умолчанию разрешено неограниченное количество подключений, но поскольку каждое из них потребляет память внутри процесса сервера MySQL, вы можете ограничить общее количество подключений в загруженной системе.

Установленное здесь ограничение будет применяться только при создании нового виртуального сервера или при включении входа в MySQL для существующего сервера.
