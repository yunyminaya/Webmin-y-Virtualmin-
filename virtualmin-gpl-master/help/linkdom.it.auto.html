Nome di dominio da utilizzare nei collegamenti
Per impostazione predefinita, quando Virtualmin visualizza un collegamento al sito Web di questo server virtuale (ad esempio dopo l'installazione di uno script), utilizzerà il nome di dominio del server. Tuttavia, puoi invece utilizzare questo campo per selezionare un dominio alias da utilizzare nei collegamenti. Ciò può essere utile se il server è accessibile solo esternamente tramite il nome di dominio alias.
