Répertoire d'accueil
Par défaut, le répertoire de base d'un utilisateur FTP de site Web est le même que le répertoire Web de base du domaine. Cependant, vous pouvez le remplacer par le répertoire de base du domaine (en utilisant une valeur vide) ou par un sous-répertoire (en utilisant un chemin relatif sous le répertoire Web).

Cela suppose que le serveur FTP est configuré pour restreindre les clients à leurs répertoires personnels, ce qui devrait être la valeur par défaut dans les installations Virtualmin standard.
