¿Copiar certificados SSL por IP a Postfix?
Cuando esta opción está habilitada y se crea un servidor virtual con SSL habilitado y su propia IP privada, Dovecot se configurará para usar el certificado SSL del servidor para las conexiones a esa dirección IP. Esto puede ser útil si los clientes de correo se conectan al nombre de dominio en modo SSL y esperan un certificado coincidente.
