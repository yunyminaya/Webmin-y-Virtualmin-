Page HTML par défaut du site Web
Cette option vous permet de contrôler la page par défaut qui sera créée et affichée pour le nouveau serveur virtuel.

Vous pouvez choisir « Désactivé » pour ignorer la création d'une page ou opter pour une page par défaut du site Web Virtualmin en sélectionnant l'option « Oui, avec le modèle de page par défaut du site Web ».
