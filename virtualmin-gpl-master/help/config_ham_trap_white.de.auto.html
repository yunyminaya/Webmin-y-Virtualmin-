Absender von Nicht-Spam, die an hamtrap gesendet wurden, zur Whitelist hinzufügen?
Wenn der Hamtrap- Alias für einen virtuellen Server aktiviert ist, wird standardmäßig jede an ihn gesendete E-Mail zur SpamAssassin-Lerndatenbank für den Benutzer hinzugefügt, der sie weitergeleitet hat. Mit dieser Option können jedoch auch die ursprünglichen Absender der Nachrichten zu den Whitelists auf System- oder Domänenebene hinzugefügt werden, sodass sie in Zukunft niemals als Spammer klassifiziert werden.

Dies kann nützlich sein, wenn Sie legitime Absender haben, die fälschlicherweise als Spammer eingestuft werden.
