Aggiungere l'utente Apache al gruppo Unix per i nuovi server?
Questa opzione determina se l'utente con cui viene eseguito Apache verrà aggiunto al nuovo gruppo host virtuale. Se abilitato, i permessi possono essere ridotti in modo significativo sulle directory home degli utenti. Quando permessi più ristretti (ad esempio 750 ) vengono combinati con suEXEC, diventa impossibile per i proprietari di dominio vedere le directory e i file di altri proprietari di dominio. Questa opzione è consigliata su qualsiasi piattaforma che supporti i gruppi secondari.
