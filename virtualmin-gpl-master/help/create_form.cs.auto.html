Vytvořte virtuální server
Tento formulář umožňuje vytvořit nový virtuální server nejvyšší úrovně, který bude mít nové přidružené přihlašovací jméno pro správu. Při vytváření serveru musíte zadat alespoň název domény, popis a heslo pro správu. Pokud je v systému definováno více šablon serverů, můžete si vybrat jednu z nabídky - šablony určují různá výchozí nastavení pro nový server, například konfiguraci webových stránek, záznamy DNS atd.

V části Pokročilé možnosti můžete zvolit jiné uživatelské jméno pro správu, název skupiny a název databáze pro server, nikoli výchozí hodnoty převzaté z první části názvu domény. Můžete také zadat kontaktní e-mailovou adresu pro server - ideálně by měla být stávající adresa osoby, která bude server spravovat, v jiné doméně.

Chcete-li vybrat, které funkce jsou pro server povoleny, použijte zaškrtávací políčka v druhé části části Pokročilé možnosti . Některé funkce závisí na jiných a nemohou být vybrány samy o sobě - například hlášení webového statistik o statistikách webů závisí na tom, zda web skutečně existuje.

V dolní části části Pokročilé možnosti si můžete vybrat, zda chcete veškerý webový provoz na tomto novém webu přeposlat na jinou adresu URL a nechat přeposlat veškerý e-mail. Můžete také zvolit, aby nový virtuální server používal vlastní soukromé adresy IP, což je vyžadováno pro SSL a anonymní FTP hosting.
