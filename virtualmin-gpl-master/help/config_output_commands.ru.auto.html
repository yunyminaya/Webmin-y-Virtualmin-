Всегда показывать вывод команд pre и post?
По умолчанию Virtualmin будет отображать выходные данные команд до или после изменения виртуального сервера (заданных выше) только в случае их сбоя. Однако, если это значение будет изменено на Да, оно всегда будет отображать этот вывод, который может содержать сообщения для пользователя.
