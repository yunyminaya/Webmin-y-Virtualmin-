È possibile creare più server da file batch?
La pagina Crea server batch di Virtualmin consente agli utenti con autorizzazioni di aggiungere server per crearne diversi contemporaneamente da un file di testo. Poiché ciò potrebbe comportare la creazione rapida di un gran numero di domini, è possibile impedire agli amministratori dei server di utilizzare questa funzione. Per fare ciò, modifica questa impostazione su No : ciò limiterà l'utilizzo da parte dell'amministratore principale e dei rivenditori.
